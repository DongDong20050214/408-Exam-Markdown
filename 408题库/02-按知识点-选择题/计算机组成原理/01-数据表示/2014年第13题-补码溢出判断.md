## 2014年第13题

**题干**

若 x = 103, y = -25，则下列表达式采用8位定点补码运算实现时，会发生溢出的是（）。A. x + y  B. -x + y  C. x - y  D. -x - y

**答案**：C

**考点分析**
- C：103-(-25)=128，超出8位补码范围[-128,127]

---

## 解题思路（逐项计算法）

### 第一步：明确8位补码范围
```
8位有符号数范围：-128 ~ +127
```

### 第二步：逐项计算

**A. x + y = 103 + (-25) = 78 ✅**
```
78 ∈ [-128, 127]，未溢出
```

**B. -x + y = -103 + (-25) = -128 ✅**
```
-128刚好等于下界，未溢出（边界值）
```

**C. x - y = 103 - (-25) = 103 + 25 = 128 ❌**
```
128 > 127，溢出！
```

**D. -x - y = -103 - (-25) = -103 + 25 = -78 ✅**
```
-78 ∈ [-128, 127]，未溢出
```

### 第三步：确认答案
只有C溢出 → 答案：**C**

## 易错点分析

### 易错点1：减负数时忘记变加法
❌ **错误计算**：x - y = 103 - 25 = 78
✅ **正确计算**：x - y = x - (-25) = x + 25 = 128

### 易错点2：误认为-128溢出
❌ **错误理解**：B选项结果-128超出范围
✅ **正确理解**：8位补码范围是[-128, 127]
- -128是最小值，在范围内
- 只有<-128或>127才溢出

### 易错点3：符号判断错误
补码减法溢出规律：
- 正数 - 负数：结果可能溢出（变更大的正数）
- 负数 - 正数：结果可能溢出（变更小的负数）
- 同号相减：一般不溢出

## 知识点对比

### 补码加减法溢出规律

| 运算 | 操作数符号 | 可能溢出 | 原因 |
|------|----------|---------|------|
| x + y | 同号 | 是 | 正+正→更大，负+负→更小 |
| x + y | 异号 | 否 | 结果绝对值减小 |
| x - y | 异号 | 是 | 正-负=正+正，负-正=负+负 |
| x - y | 同号 | 否 | 结果绝对值减小 |

**记忆口诀**：
> **同号相加异号减，这两种情况可能溢**  
> **异号相加同号减，绝对值小不会溢**
