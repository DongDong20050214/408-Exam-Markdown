**题干**

五段流水线无转发，下列指令序列存在数据冒险的指令对是（）。

**答案**：B (I2和I3)

**考点分析**
- I2写R5，I3读R5
- RAW数据相关

---

## 解题思路

### 第一步：理解题目核心
五段流水线(取指IF、译码ID、执行EX、访存MEM、写回WB)无转发机制下的数据冒险检测

### 第二步：逐项分析
假设指令序列为：
```
I1: ADD R1, R2, R3
I2: SUB R5, R1, R4
I3: MUL R6, R5, R7
```

**I1和I2关系** ✅可能存在RAW  
I1写R1(WB阶段)，I2读R1(ID阶段)，若无转发需暂停

**I2和I3关系** ✅存在RAW冒险  
I2写R5(WB阶段)，I3读R5(ID阶段)，无转发时I3需等待I2的WB完成

**判断标准**：前一条指令写的寄存器是否被后一条指令读取

### 第三步：确认答案
✅ **B. I2和I3**存在数据冒险

## 易错点分析

### 易错点1：认为有足够的间隔就无冒险
❌ 错误理解：I2和I3之间没有其他指令，但五段流水线中，I2的WB在I3的ID之后，不会冲突  
✅ 正确理解：五段无转发流水线中，I3在I2之后2拍进入，当I3到达ID(读寄存器)时，I2才在EX阶段，还未写回R5，产生RAW冒险。需暂停I3直到I2的WB完成

### 易错点2：数据冒险时间线分析
| 周期 | 1 | 2 | 3 | 4 | 5 | 6 |
|------|---|---|---|---|---|---|
| I2 | IF | ID | EX | MEM | WB |  |
| I3 |  | IF | ID暂停 | ID暂停 | ID | EX |
| 冒险 |  |  | I3需R5但I2未写回 |  |  |  |
