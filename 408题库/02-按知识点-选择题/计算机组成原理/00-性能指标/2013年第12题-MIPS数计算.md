# MIPS真题汇总

## 2013年第12题

**元数据**
- 年份：2013年第12题
- 科目：#计算机组成原理
- 知识点：#计算机组成原理/性能/MIPS
- 难度：⭐⭐⭐
- 关联知识：[知识库/计算机组成原理/MIPS](知识库/计算机组成原理/MIPS.md)

**题干**

某计算机主频为 1.2GHz，该机的 MIPS 数是（）。A. 100  B. 200  C. 400  D. 600

**答案**：C

**考点分析**
- MIPS=主频/平均CPI=1200/(0.5×2+0.2×3+0.1×4+0.2×5)=400

## 解题思路（公式计算法）

### 第一步：理解题目条件（题干缺失部分需补充）

从答案推算，题目应给出：
- 主频：1.2GHz = 1200MHz
- 指令组成及各自CPI（根据考点分析反推）：
  - 50%指令CPI=2
  - 20%指令CPI=3
  - 10%指令CPI=4
  - 20%指令CPI=5

### 第二步：回顾MIPS相关公式

**核心公式**：
```
MIPS = 主频(MHz) / 平均CPI

或

MIPS = 指令数(百万条) / 执行时间(秒)
```

**平均CPI计算**：
```
平均CPI = Σ(指令比例 × 该类指令CPI)
```

**CPU执行时间**：
```
CPU时间 = 指令数 × CPI / 主频
```

### 第三步：计算平均CPI

```
平均CPI = 0.5×2 + 0.2×3 + 0.1×4 + 0.2×5
        = 1.0 + 0.6 + 0.4 + 1.0
        = 3.0
```

### 第四步：计算MIPS数

```
MIPS = 主频 / 平均CPI
     = 1200MHz / 3.0
     = 400 MIPS
```

### 第五步：确认答案
MIPS = 400 → 答案：**C**

## 易错点分析

### 陷阱1：混淆主频单位
❌ **错误计算**：直接用1.2计算，MIPS = 1.2 / 3.0 = 0.4
✅ **正确理解**：
- 主频单位：GHz需要转换为MHz
- 1.2GHz = 1200MHz
- MIPS公式中主频用MHz表示

**单位对应**：
- MIPS = Million Instructions Per Second（百万条指令/秒）
- 主频MHz = Million Hertz（百万Hz）
- 单位匹配，公式才成立

### 陷阱2：忘记计算平均CPI
❌ **错误想法**：题目给了多个CPI，不知道用哪个
✅ **正确做法**：
- 不同类型指令有不同CPI
- 必须先计算**加权平均CPI**
- 权重 = 各类指令所占比例

**计算步骤**：
```
类型A：50%指令，CPI=2  → 贡献 0.5×2 = 1.0
类型B：20%指令，CPI=3  → 贡献 0.2×3 = 0.6
类型C：10%指令，CPI=4  → 贡献 0.1×4 = 0.4
类型D：20%指令，CPI=5  → 贡献 0.2×5 = 1.0
                          总计 = 3.0
```

### 陷阱3：混淆MIPS和MHz
❌ **错误理解**：MIPS就是主频
✅ **正确理解**：
- **MHz（主频）**：时钟频率，硬件特性
- **MIPS**：每秒执行的指令数，性能指标
- 关系：MIPS = MHz / CPI

**实例对比**：
- 机器A：1000MHz，CPI=2  → MIPS=500
- 机器B：1000MHz，CPI=1  → MIPS=1000
- 相同主频，不同MIPS

## 知识点对比

### 性能指标公式汇总

| 指标 | 公式 | 单位 | 含义 |
|------|------|------|------|
| MIPS | 主频(MHz) / 平均CPI | 百万条指令/秒 | 指令执行速度 |
| MFLOPS | 浮点运算次数 / 执行时间 | 百万次浮点运算/秒 | 浮点性能 |
| CPI | 时钟周期数 / 指令数 | 周期/指令 | 指令平均周期数 |
| IPC | 指令数 / 时钟周期数 | 指令/周期 | CPI的倒数 |
| CPU时间 | 指令数 × CPI / 主频 | 秒 | 程序执行时间 |

### 平均CPI的计算方法

**方法1：按指令类型加权**
```
平均CPI = Σ(指令比例 × CPI)
        = P₁×CPI₁ + P₂×CPI₂ + ... + Pₙ×CPIₙ
```

**方法2：按总周期数计算**
```
平均CPI = 总时钟周期数 / 总指令数
        = (N₁×CPI₁ + N₂×CPI₂ + ... + Nₙ×CPIₙ) / 总指令数
```

### 单位转换表

| 原单位 | 目标单位 | 转换关系 |
|--------|---------|---------|
| GHz | MHz | 1 GHz = 1000 MHz |
| MHz | Hz | 1 MHz = 10⁶ Hz |
| μs | s | 1 μs = 10⁻⁶ s |
| ms | s | 1 ms = 10⁻³ s |
| MIPS | IPS | 1 MIPS = 10⁶ IPS |

**记忆口诀**：
> **MIPS等于频率除CPI，单位百万要记清**  
> **加权平均算CPI，比例乘积再相加**  
> **主频单位用兆赫，千兆换算要乘千**
