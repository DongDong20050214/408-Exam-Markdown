# CPI真题汇总

## 2017年第12题

**元数据**
- 年份：2017年第12题
- 科目：#计算机组成原理
- 知识点：#计算机组成原理/性能/CPI
- 难度：⭐⭐⭐
- 关联知识：[知识库/计算机组成原理/CPI](知识库/计算机组成原理/CPI.md)

**题干**

假定计算机 M1 和 M2 具有相同的指令集体系结构（ISA），主频分别为 1.5GHz 和 1.2GHz。在 M1 和 M2 上运行某基准程序P，若平均CPI分别为2和1，则程序P在 M1 和 M2 上运行时间的比值是（）。A. 0.4  B. 0.625  C. 1.6  D. 2.5

**答案**：C

**考点分析**
- 时间比=(CPI1/频率1)/(CPI2/频率2)=(2/1.5)/(1/1.2)=1.6

---

## 解题思路（公式计算法）

### 第一步：回顾执行时间公式
```
CPU执行时间 = 指令数 × CPI × 时钟周期
           = 指令数 × CPI / 频率

同一程序指令数相同，则：
时间比 = (CPI₁/频率₁) / (CPI₂/频率₂)
```

### 第二步：代入数据
```
M1: CPI₁ = 2, 频率₁ = 1.5 GHz
M2: CPI₂ = 1, 频率₂ = 1.2 GHz

T₁/T₂ = (2/1.5) / (1/1.2)
     = (2/1.5) × (1.2/1)
     = 2.4/1.5
     = 1.6
```

### 第三步：确认答案
时间比值1.6 → 答案：**C**

## 易错点分析

### 易错点1：直接用CPI或频率比值
❌ **错误计算**：
```
CPI比 = 2/1 = 2（错误）
频率比 = 1.5/1.2 = 1.25（错误）
```

✅ **正确理解**：时间同时受CPI和频率影响
```
时间 ∝ CPI（CPI越大时间越长）
时间 ∝ 1/频率（频率越大时间越短）
```

### 易错点2：比值方向搞反
❌ **错误**：T₂/T₁ = (1/1.2)/(2/1.5) = 0.625
✅ **正确**：T₁/T₂ = (2/1.5)/(1/1.2) = 1.6

题目问"M1和M2上运行时间的比值"= T₁/T₂

### 易错点3：计算顺序混乱
**标准步骤**：
```
1) 写出公式：T = I × CPI / f
2) 求比值：T₁/T₂ = (I×CPI₁/f₁)/(I×CPI₂/f₂)
3) 化简：= (CPI₁/f₁) / (CPI₂/f₂)
4) 代入：= (2/1.5) / (1/1.2) = 1.6
```

## 知识点对比

### 性能比较的三个维度

| 维度 | M1 | M2 | 影响 |
|------|----|----|------|
| CPI | 2 | 1 | M2更优（指令效率高） |
| 频率 | 1.5GHz | 1.2GHz | M1更优（时钟快） |
| 时间 | 1.6T | T | M2更快（综合更优） |

**结论**：M2虽然频率低，但CPI优势更大，整体更快

**记忆口诀**：
> **时间正比CPI，反比看频率**  
> **求比先列式，化简再代数**
