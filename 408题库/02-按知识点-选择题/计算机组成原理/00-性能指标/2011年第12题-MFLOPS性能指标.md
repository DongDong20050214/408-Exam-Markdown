**元数据**

- 知识点：#计算机组成原理/性能/MFLOPS

**题干**

下列选项中，描述浮点数操作速度指标的是（）。A. MIPS  B. CPI  C. IPC  D. MFLOPS

**答案**：D

**考点分析**
- MFLOPS：每秒百万次浮点运算

---

## 解题思路（概念辨析法）

### 第一步：理解关键词
题目问：描述**浮点数操作速度**的指标

关键词：
- 浮点数操作
- 速度指标（单位时间完成的操作数）

### 第二步：逐项分析

**A. MIPS ❌**
```
全称：Million Instructions Per Second
含义：每秒执行百万条指令
单位：百万条指令/秒

衡量对象：所有指令（包括整数、分支、访存等）
不是专门针对浮点运算

结论：错误 ✗
```

**B. CPI ❌**
```
全称：Cycles Per Instruction
含义：每条指令的平均时钟周期数
单位：周期/指令

衡量对象：指令执行效率，不是速度
CPI越小越快，但不是速度的直接度量

结论：错误 ✗
```

**C. IPC ❌**
```
全称：Instructions Per Cycle
含义：每个时钟周期执行的指令数
单位：指令/周期

衡量对象：指令并行度，不是浮点专用指标
IPC = 1/CPI

结论：错误 ✗
```

**D. MFLOPS ✅**
```
全称：Million FLoating-point Operations Per Second
含义：每秒百万次浮点运算
单位：百万次浮点操作/秒

衡量对象：专门针对浮点运算速度
完全符合题意

结论：正确 ✓
```

### 第三步：确认答案
只有MFLOPS是浮点运算速度指标 → 答案：**D**

## 易错点分析

### 易错点1：混淆MIPS和MFLOPS
❌ **错误理解**：MIPS也是速度指标，应该也对
✅ **正确理解**：
- **MIPS**：衡量**所有指令**的执行速度（通用指标）
- **MFLOPS**：专门衡量**浮点运算**速度（专用指标）
- 题目明确问"浮点数操作"，必须用MFLOPS

**类比**：
- MIPS = 汽车总速度（包括各种路况）
- MFLOPS = 高速公路速度（专门场景）

### 易错点2：认为CPI也能反映速度
❌ **错误想法**：CPI小说明执行快，也是速度指标
✅ **正确理解**：
- CPI是**效率指标**，不是**速度指标**
- CPI低 + 频率高 = 速度快
- 单独CPI无法确定速度（还需要时钟频率）

**对比**：
```
MIPS = 时钟频率 / CPI  （速度 = 频率/效率）
      ↑                ↑
   速度指标        效率指标
```

### 易错点3：混淆IPC和CPI的关系
❌ **错误理解**：IPC是每周期指令数，也是速度指标
✅ **正确理解**：
- IPC = 1/CPI（互为倒数）
- IPC也是**并行度指标**，不是速度指标
- IPC高 + 频率高 = 速度快

**示例**：
- CPU1：IPC=2，频率1GHz → 速度=2GIPS
- CPU2：IPC=1，频率3GHz → 速度=3GIPS
- IPC高不一定速度快！

## 知识点对比

### 常见性能指标分类

| 指标 | 全称 | 含义 | 类型 | 适用场景 |
|------|------|------|------|----------|
| MIPS | Million Instructions/s | 每秒百万条指令 | 速度 | 通用指令 |
| MFLOPS | Million FLOPs/s | 每秒百万次浮点运算 | 速度 | 浮点运算 |
| CPI | Cycles/Instruction | 每指令周期数 | 效率 | 指令级分析 |
| IPC | Instructions/Cycle | 每周期指令数 | 并行度 | 流水线分析 |

### 浮点性能指标体系

| 指标 | 单位 | 数量级 |
|------|------|--------|
| FLOPS | 次/秒 | 基本单位 |
| KFLOPS | 千次/秒 | 10³ |
| MFLOPS | 百万次/秒 | 10⁶ |
| GFLOPS | 十亿次/秒 | 10⁹ |
| TFLOPS | 万亿次/秒 | 10¹² |
| PFLOPS | 千万亿次/秒 | 10¹⁵ |

**记忆口诀**：
> **MIPS衡量通用速，MFLOPS专测浮点数**  
> **CPI效率IPC并行，速度指标看OPS数**
