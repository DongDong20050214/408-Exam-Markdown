# CPI和CPU时间真题汇总

## 2022年第12题

**元数据**
- 年份：2022年第12题
- 科目：#计算机组成原理
- 知识点：#计算机组成原理/性能/CPI
- 难度：⭐⭐
- 关联知识：[知识库/计算机组成原理/性能指标](知识库/计算机组成原理/性能指标.md)

**题干**

某计算机主频为 1 GHz, 程序 P 运行过程中, 共执行了 10000 条指令, 其中, 80% 的指令执行平均需 1 个时钟周期, 20% 的指令执行平均需 10 个时钟周期。程序 P 的平均 CPI 和 CPU 执行时间分别是 ( )。

A. 2.8, 28 μs  B. 28, 28 μs  C. 2.8, 28 ms  D. 28, 28 ms

**答案**：A

**考点分析**
- CPI=0.8×1+0.2×10=2.8
- 时间=10000×2.8/1G=28μs

## 解题思路（公式计算法）

### 第一步：明确题目条件

给定信息：
- 主频：1 GHz = 1000 MHz = 10⁹ Hz
- 总指令数：10000条
- 指令分布：
  - 80%指令：CPI = 1
  - 20%指令：CPI = 10

要求：平均CPI 和 CPU执行时间

### 第二步：回顾相关公式

**平均CPI公式**：
```
平均CPI = Σ(指令比例 × 该类指令CPI)
```

**CPU执行时间公式**：
```
CPU时间 = 指令数 × CPI / 主频
```

或
```
CPU时间 = 时钟周期数 × 时钟周期时间
       = (指令数 × CPI) × (1/主频)
```

### 第三步：计算平均CPI

```
平均CPI = 80% × 1 + 20% × 10
        = 0.8 × 1 + 0.2 × 10
        = 0.8 + 2.0
        = 2.8
```

### 第四步：计算CPU执行时间

**方法1：直接公式**
```
CPU时间 = 指令数 × CPI / 主频
        = 10000 × 2.8 / (10⁹ Hz)
        = 28000 / 10⁹ 秒
        = 28 × 10⁻⁶ 秒
        = 28 μs
```

**方法2：分步计算**
```
总时钟周期数 = 指令数 × 平均CPI
            = 10000 × 2.8
            = 28000 周期

时钟周期时间 = 1 / 主频
            = 1 / 10⁹ 秒
            = 1 ns

CPU时间 = 28000 × 1ns
        = 28000 ns
        = 28 μs
```

### 第五步：确认答案
平均CPI = 2.8，CPU时间 = 28 μs → 答案：**A**

## 易错点分析

### 陷阱1：时间单位搞错
❌ **错误计算**：28000 / 10⁹ = 0.000028，以为是0.000028秒 = 28ms
✅ **正确理解**：
- 28000 / 10⁹ = 28 × 10⁻⁶ 秒
- 10⁻⁶ 秒 = 1 微秒（μs）
- 所以是 28 μs，不是 28 ms

**单位换算**：
- 1 s = 10³ ms（毫秒）
- 1 s = 10⁶ μs（微秒）
- 1 s = 10⁹ ns（纳秒）

**检验方法**：
```
1 GHz 主频 → 时钟周期 = 1ns
10000条指令，平均CPI=2.8 → 需要28000个周期
28000 × 1ns = 28000ns = 28μs ✓
```

### 陷阱2：忘记用加权平均
❌ **错误计算**：CPI = (1 + 10) / 2 = 5.5
✅ **正确计算**：
- 不是简单平均，是**加权平均**
- 权重 = 各类指令的比例
- CPI = 0.8×1 + 0.2×10 = 2.8

**为什么要加权**：
- 80%指令占大多数，对平均值影响大
- 20%指令占少数，影响小
- 简单平均会高估慢指令的影响

### 陷阱3：主频单位换算错误
❌ **错误**：1 GHz = 10⁶ Hz
✅ **正确**：1 GHz = 10⁹ Hz

**记忆方法**：
- k (kilo) = 10³ (千)
- M (Mega) = 10⁶ (百万)
- G (Giga) = 10⁹ (十亿)
- T (Tera) = 10¹² (万亿)

## 知识点对比

### 时间单位对比表

| 单位 | 符号 | 与秒的关系 | 数量级 | 典型应用 |
|------|------|----------|--------|---------|
| 秒 | s | 1 s | 10⁰ | 程序总运行时间 |
| 毫秒 | ms | 10⁻³ s | 千分之一秒 | 磁盘访问时间 |
| 微秒 | μs | 10⁻⁶ s | 百万分之一秒 | 内存访问时间 |
| 纳秒 | ns | 10⁻⁹ s | 十亿分之一秒 | CPU时钟周期 |
| 皮秒 | ps | 10⁻¹² s | 万亿分之一秒 | 光在芯片内传播 |

### CPU执行时间的三种等价公式

**公式1：基于指令数**
```
CPU时间 = 指令数 × CPI / 主频
```

**公式2：基于时钟周期**
```
CPU时间 = 时钟周期数 / 主频
        = (指令数 × CPI) / 主频
```

**公式3：基于周期时间**
```
CPU时间 = 指令数 × CPI × 时钟周期时间
        = 指令数 × CPI × (1/主频)
```

### 计算步骤对比

| 步骤 | 本题示例 | 通用公式 |
|------|---------|---------|
| 1. 计算平均CPI | 0.8×1 + 0.2×10 = 2.8 | Σ(P_i × CPI_i) |
| 2. 计算总周期数 | 10000 × 2.8 = 28000 | N × CPI |
| 3. 计算时钟周期时间 | 1/10⁹ = 1ns | 1 / f |
| 4. 计算CPU时间 | 28000 × 1ns = 28μs | 周期数 × 周期时间 |

**记忆口诀**：
> **加权平均算CPI，比例相乘再相加**  
> **时间等于数乘CPI，除以主频要记牢**  
> **单位换算莫出错，微秒毫秒差千倍**  
> **千是ms百万μ，十亿纳秒记心间**
