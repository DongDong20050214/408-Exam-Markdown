# 编译程序真题汇总

## 2016年第12题

**元数据**
- 年份：[[../../01-真题原卷/2016年408真题#12|2016年第12题]]
- 科目：#计算机组成原理
- 知识点：#计算机组成原理/概述/编译
- 难度：⭐
- 关联知识：[[知识库/计算机组成原理/编译]]

**题干**

将高级语言源程序转换为机器级目标代码文件的程序是（）。A. 汇编程序  B. 链接程序  C. 编译程序  D. 解释程序

**答案**：C

**考点分析**
- C：编译程序完成转换

---

## 解题思路（概念辨析法）

### 第一步：明确关键词
- 输入：高级语言源程序（.c/.cpp等）
- 输出：机器级目标代码文件（.obj/.o等）
- 关键："转换为...文件"

### 第二步：逐项分析

**A. 汇编程序 ❌**
```
功能：汇编语言 → 机器码
输入：.asm（汇编语言）
输出：.obj（机器码）

错误原因：输入是汇编语言，不是高级语言
结论：错误 ✗
```

**B. 链接程序 ❌**
```
功能：目标文件组装 → 可执行文件
输入：.obj（已经是机器码）
输出：.exe/.out

错误原因：输入已经是机器码，不涉及语言翻译
结论：错误 ✗
```

**C. 编译程序 ✅**
```
功能：高级语言 → 机器码
输入：.c/.cpp（高级语言）
输出：.obj/.o（机器级目标代码）

完全符合题意
结论：正确 ✓
```

**D. 解释程序 ❌**
```
功能：高级语言 → 边译边执行
输出：执行结果（无独立文件）

错误原因：不生成目标代码文件
结论：错误 ✗
```

### 第三步：确认答案
只有编译程序符合 → 答案：**C**

## 易错点分析

### 易错点1：混淆"汇编程序"和"编译程序"
❌ **错误理解**：汇编程序也生成机器码，应该也对
✅ **正确理解**：
- 题目明确说"高级语言源程序"
- 汇编程序处理的是汇编语言（低级语言）
- 汇编语言 ≠ 高级语言

**语言层次**：
```
高级语言(C/Java) → [编译器] → 汇编语言 → [汇编器] → 机器码
                    ↑题目问的            ↑不是题目问的
```

### 易错点2：混淆"目标代码"和"可执行文件"
❌ **错误想法**：最终要运行程序，应该选链接程序
✅ **正确理解**：
- **目标代码文件**(.obj)：编译器输出，不能直接运行
- **可执行文件**(.exe)：链接器输出，可以直接运行
- 题目问的是生成"目标代码文件"，不是"可执行文件"

**完整流程**：
```
hello.c → [编译] → hello.obj → [链接] → hello.exe
          ↑题目问的这步        ↑不是这步
```

### 易错点3：认为"解释程序"也生成机器码
❌ **错误理解**：解释程序也要翻译成机器码才能执行
✅ **正确理解**：
- 解释程序：读一句→翻译→执行→读下一句
- 中间产物在内存，**不保存为文件**
- 题目要求"转换为...文件"，解释程序不符合

**对比**：
| 方式 | 编译 | 解释 |
|------|------|------|
| 过程 | 整体翻译 | 逐句翻译 |
| 生成文件 | ✅ 是(.obj) | ❌ 否 |
| 执行速度 | 快 | 慢 |

## 知识点对比

### 四种程序处理工具对比

| 工具 | 输入 | 输出 | 是否生成文件 | 题目匹配 |
|------|------|------|------------|---------|
| 编译程序 | 高级语言(.c) | 机器码(.obj) | ✅ 是 | ✅ 完全匹配 |
| 汇编程序 | 汇编语言(.asm) | 机器码(.obj) | ✅ 是 | ❌ 输入不符 |
| 链接程序 | 机器码(.obj) | 可执行文件(.exe) | ✅ 是 | ❌ 输入输出都不符 |
| 解释程序 | 高级语言(.py) | 执行结果 | ❌ 否 | ❌ 不生成文件 |

**记忆口诀**：
> **高级转机器，编译是正解**  
> **汇编处理汇编语，链接不翻译**  
> **解释边跑边消失，不留文件在磁盘**
