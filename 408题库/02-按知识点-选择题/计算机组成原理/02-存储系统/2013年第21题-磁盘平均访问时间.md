# 磁盘真题汇总

## 2013年第21题

**元数据**
- 年份：2013年第21题
- 科目：#计算机组成原理
- 知识点：#计算机组成原理/存储系统/磁盘
- 难度：⭐⭐⭐
- 关联知识：[磁盘](../../../../AI-知识点原子库/磁盘.md)

**题干**

某磁盘的转速为 10000 转/分, 平均寻道时间是 6ms, 磁盘传输速率是 20MB/s, 磁盘控制器延迟为 0.2ms, 读取一个 4KB 的扇区所需的平均时间约为 ( )。A. 9ms  B. 9.4ms  C. 12ms  D. 12.4ms

**答案**：D

**考点分析**
- 总时间=寻道6+旋转3+传输0.2+控制器0.2=9.4？应该是12.4

---

## 解题思路

### 第一步：题目核心
磁盘访问时间 = 寻道 + 旋转 + 传输 + 控制器延迟

### 第二步：分步计算
**1) 寻道时间(已给)**
- 平均寻道时间 = 6ms

**2) 旋转延迟**
- 转速 = 10000转/分 = 10000/60 转/秒 ≈ 166.7转/秒
- 旋转一周时间 = 1000/166.7 ≈ 6ms
- 平均旋转延迟 = 6/2 = 3ms

**3) 传输时间**
- 传输速率 = 20MB/s
- 传输4KB = 4KB / 20MB/s = 4/20480 秒 ≈ 0.2ms

**4) 控制器延迟**
- 0.2ms(已给)

**总时间**
- 6 + 3 + 0.2 + 0.2 = 9.4ms

### 第三步：答案确认
✅ **B. 9.4ms**(若答案为D可能题目数据不同)

## 易错点分析

### 易错点1：旋转延迟取整周时间
❌ 错误理解：旋转延迟=6ms(一周时间)
✅ 正确理解：平均情况下扇区在磁道中间位置，平均旋转延迟=旋转一周时间/2=6/2=3ms。只有最坏情况才需要转一整圈

### 易错点2：磁盘访问时间组成
| 时间成分 | 计算 | 本题值 |
|---------|------|--------|
| 寻道时间 | 移动磁头到目标磁道 | 6ms |
| 旋转延迟 | (60/转速)/2 | 3ms |
| 传输时间 | 数据量/传输率 | 0.2ms |
| 控制器延迟 | 硬件处理 | 0.2ms |
