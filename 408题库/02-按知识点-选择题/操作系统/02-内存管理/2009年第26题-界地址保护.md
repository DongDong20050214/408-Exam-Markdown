**题干**

分区分配内存管理方式的主要保护措施是（）。

A. 界地址保护  
B. 程序代码保护  
C. 数据保护  
D. 栈保护

**答案**：A

**考点分析**
- 分区分配：每个进程占用连续的内存分区
- 界地址寄存器：存储上界和下界地址
- 访问时检查是否越界，防止访问其他进程空间

---


## 解题思路

### 第一步：理解分区分配
分区分配：每个进程占用连续内存区域

### 第二步：逐项分析
**A. 界地址保护 ✅**
```
界地址寄存器：存储分区上界和下界
访问内存时检查地址是否越界
防止访问其他进程的分区
分区分配的核心保护机制
结论：正确
```

**B. 程序代码保护 ❌**
```
代码保护：防止代码被修改
与内存分配方式无关
不是分区分配特有的
结论：错误
```

**C. 数据保护 ❌**
```
数据保护：防止数据被非法访问
不是分区分配主要保护措施
结论：错误
```

**D. 栈保护 ❌**
```
栈保护：防止栈溢出
与分区分配无关
结论：错误
```

### 第三步：确认答案
界地址保护是主要措施 → 答案：**A**

## 易错点分析

### 易错点1：混淆保护类型
❌ 错误：所有保护都是分区分配的主要措施
✅ 正确：界地址保护是分区分配特有的

### 易错点2：不理解界地址保护
| 保护类型 | 目的 | 适用范围 |
|----------|------|----------|
| 界地址保护 | 防止越界访问 | 分区分配 |
| 存储保护键 | 进程隔离 | 多道系统 |
| 页表保护位 | 页面保护 | 分页系统 |
