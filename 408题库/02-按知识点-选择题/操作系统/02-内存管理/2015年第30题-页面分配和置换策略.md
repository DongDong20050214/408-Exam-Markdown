- 关联知识：[知识库/操作系统/页面置换](知识库/操作系统/页面置换.md)

**题干**

在请求分页系统中，页面分配策略与页面置换策略不能组合使用的是（）。A. 可变分配，全局置换  B. 可变分配，局部置换  C. 固定分配，全局置换  D. 固定分配，局部置换

**答案**：C

**考点分析**
- C错误：固定分配不能用全局置换

---


## 解题思路

### 第一步：理解分配和置换策略
分配策略：固定分配 vs 可变分配
置换策略：局部置换 vs 全局置换

### 第二步：逐项分析
**A. 可变分配+全局置换 ✅可组合**
```
可变分配：页框数可动态调整
全局置换：可从其他进程置换页面
配合良好：缺页时从全局置换，调整分配
结论：可组合
```

**B. 可变分配+局部置换 ✅可组合**
```
可变分配：页框数可调整
局部置换：只在本进程内置换
配合：根据缺页率调整分配
结论：可组合
```

**C. 固定分配+全局置换 ❌矛盾**
```
固定分配：页框数不变
全局置换：可能从其他进程抢页框
矛盾：抢了页框就不是固定分配了
结论：不能组合
```

**D. 固定分配+局部置换 ✅可组合**
```
固定分配：页框数固定
局部置换：只在本进程内置换
配合：页框数不变，内部置换
结论：可组合
```

### 第三步：确认答案
固定分配+全局置换矛盾 → 答案：**C**

## 易错点分析

### 易错点1：不理解策略冲突
❌ 错误：所有策略都可以组合
✅ 正确：固定分配与全局置换矛盾

### 易错点2：混淆策略含义
| 分配策略 | 页框数 | 置换策略 | 置换范围 |
|---------|--------|---------|----------|
| 固定分配 | 不变 | 局部置换 | 本进程内 ✅ |
| 固定分配 | 不变 | 全局置换 | 所有进程 ❌矛盾 |
| 可变分配 | 可变 | 局部置换 | 本进程内 ✅ |
| 可变分配 | 可变 | 全局置换 | 所有进程 ✅ |

**关键点**：
- 固定分配：进程页框数不变
- 全局置换：可能增加本进程页框
- 两者矛盾
