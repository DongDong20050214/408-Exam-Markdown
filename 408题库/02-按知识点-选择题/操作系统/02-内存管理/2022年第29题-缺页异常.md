**题干**

某进程访问的页 b 不在内存中，导致产生缺页异常，该缺页异常处理过程中不一定包含的操作是（）。

A. 淘汰内存中的页
B. 建立页号与页框号的对应关系
C. 将页 b 从外存读入内存
D. 修改页表中页 b 对应的存在位

**答案**：A

**考点分析**
- A不一定：若有空闲页框，无需淘汰
- BCD必须执行

---


## 解题思路

### 第一步：理解缺页处理流程
缺页处理：调入缺失的页面到内存

### 第二步：逐项分析
**A. 淘汰内存中的页 ❌不一定**
```
若有空闲页框：直接分配，无需淘汰
若无空闲页框：需要淘汰页面
结论：不一定执行
```

**B. 建立页号与页框号对应关系 ✅一定**
```
页面调入后必须建立映射
更新页表中页号到页框号的映射
结论：一定执行
```

**C. 将页b从外存读入内存 ✅一定**
```
缺页的根本原因是页不在内存
必须从外存读入
结论：一定执行
```

**D. 修改页表中页b的存在位 ✅一定**
```
页面调入后存在位置1
表示页面已在内存
结论：一定执行
```

### 第三步：确认答案
只有淘汰操作不一定 → 答案：**A**

## 易错点分析

### 易错点1：认为缺页必须淘汰
❌ 错误：缺页一定要淘汰页面
✅ 正确：有空闲页框时不需要淘汰

### 易错点2：不理解缺页处理步骤
| 步骤 | 是否必须 | 条件 |
|------|---------|------|
| 检查地址合法性 | ✅ | 无 |
| 查找页面位置 | ✅ | 无 |
| 选择页框 | ✅ | 无 |
| 淘汰页面 | ❌ | 无空闲页框时 |
| 读入页面 | ✅ | 无 |
| 更新页表 | ✅ | 无 |

**缺页处理流程**：
1. 保存现场
2. 查找页面在外存位置
3. 选择页框（有空闲则分配，无空闲则淘汰）
4. 从外存读入页面
5. 更新页表（存在位、页框号）
6. 恢复现场
