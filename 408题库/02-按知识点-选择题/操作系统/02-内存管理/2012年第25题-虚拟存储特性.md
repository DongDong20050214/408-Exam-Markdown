**题干**

下列关于虚拟存储器的叙述中，正确的是（）。

A. 虚拟存储只能基于连续分配技术  
B. 虚拟存储只能基于非连续分配技术  
C. 虚拟存储容量只受外存容量的限制  
D. 虚拟存储容量只受内存容量的限制

**答案**：B

**考点分析**
- A错误：不能基于连续分配
- B正确：必须基于分页或分段等非连续分配
- C错误：还受地址位数限制
- D错误：受外存和地址位数限制，不受内存限制

---


## 解题思路

### 第一步：理解虚拟存储原理
虚拟存储：程序部分装入内存，按需调入调出

### 第二步：逐项分析
**A. 只能基于连续分配 ❌**
```
连续分配：整个程序连续存放
无法实现部分装入
虚拟存储需要非连续分配
结论：错误
```

**B. 只能基于非连续分配 ✅**
```
虚拟存储需要：
- 分页：页面可分散存放
- 分段：段可分散存放
- 段页式：结合两者
必须支持非连续分配才能按需调入调出
结论：正确
```

**C. 容量只受外存限制 ❌**
```
虚拟存储容量受限于：
1. 地址位数（最大可寻址空间）
2. 外存大小
取最小值
32位系统：最大4GB，即使外存更大
结论：错误
```

**D. 容量只受内存限制 ❌**
```
虚拟存储容量远大于内存
目的就是突破内存限制
不受内存容量限制
结论：错误
```

### 第三步：确认答案
只能基于非连续分配 → 答案：**B**

## 易错点分析

### 易错点1：认为虚拟存储可用连续分配
❌ 错误：连续分配也能实现虚拟存储
✅ 正确：连续分配无法部分装入，必须非连续

### 易错点2：混淆容量限制因素
| 因素 | 是否限制 | 说明 |
|------|----------|------|
| 内存大小 | ❌ | 虚拟存储突破内存限制 |
| 外存大小 | ✅ | 程序存储在外存 |
| 地址位数 | ✅ | 32位最大4GB |
| CPU性能 | ❌ | 不影响容量 |

**容量计算**：
min(2^地址位数, 外存大小)
