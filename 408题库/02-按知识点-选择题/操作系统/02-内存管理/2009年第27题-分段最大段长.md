# 分段真题汇总

## 2009年第27题

**元数据**
- 年份：2009年第27题
- 科目：#操作系统
- 知识点：#操作系统/内存管理/分段
- 难度：⭐⭐

**题干**

一个分段存储管理系统中，地址长度为 32 位，其中段号占 8 位，则最大段长是（）。A. 2^8 字节  B. 2^16 字节  C. 2^24 字节  D. 2^32 字节

**答案**：C

**考点分析**
- 段内偏移=32-8=24位

---


## 解题思路

### 第一步：分析地址结构
```
地址长度：32位
段号：8位
段内偏移：32-8=24位
```

### 第二步：计算最大段长
```
段内偏移24位
可表示范围：0 ~ 2^24-1
最大段长：2^24字节
```

### 第三步：逐项分析
**A. 2^8字节 ❌**
```
2^8=256字节
是段号位数，不是段长
结论：错误
```

**B. 2^16字节 ❌**
```
2^16=64KB
计算错误
结论：错误
```

**C. 2^24字节 ✅**
```
段内偏移24位
可寻址2^24个字节
最大段长2^24字节=16MB
结论：正确
```

**D. 2^32字节 ❌**
```
2^32是整个地址空间
不是单个段的长度
结论：错误
```

### 第四步：确认答案
最大段长2^24字节 → 答案：**C**

## 易错点分析

### 易错点1：用段号位数计算段长
❌ 错误：段号8位，段长2^8
✅ 正确：段长由段内偏移位数决定

### 易错点2：混淆地址组成
| 概念 | 位数 | 作用 |
|------|------|------|
| 段号 | 8位 | 区分2^8=256个段 |
| 段内偏移 | 24位 | 每段最大2^24字节 |
| 地址总长 | 32位 | 总地址空间2^32字节 |

**分段地址计算**：
```
逻辑地址=段号+段内偏移
段号：确定是哪个段
段内偏移：段内位置
最大段长=2^(段内偏移位数)
```
