**题干**

若 P 访问虚拟地址为 02A01H 的存储单元，则经地址变换后得到的物理地址是（）。

A. 00A01H  B. 20A01H  C. 60A01H  D. 80A01H

**答案**：B

**考点分析**
- 虚拟地址02A01H，页大小4KB=1000H
- 页号=02H，页内偏移=A01H
- 页号2存在位为0，发生缺页，需要置换
- 置换后页号2映射到页框20H
- 物理地址=20×1000+A01=20A01H

---


## 解题思路

### 第一步：分析虚拟地址
```
虚拟地址：02A01H
页大小：4KB = 1000H
页号 = 02A01H / 1000H = 02H
页内偏移 = 02A01H % 1000H = A01H
```

### 第二步：查页表（需要题目给出）
```
根据考点分析：
页号2的页框号 = 20H
（可能经过缺页和置换）
```

### 第三步：计算物理地址
```
物理地址 = 页框号 × 页大小 + 页内偏移
        = 20H × 1000H + A01H
        = 20000H + A01H
        = 20A01H
```

### 第四步：确认答案
物理地址20A01H → 答案：**B**

## 易错点分析

### 易错点1：页号页内偏移划分错误
❌ 错误：随意划分地址
✅ 正确：根据页大小准确划分

### 易错点2：地址计算公式错误
| 步骤 | 公式 |
|------|------|
| 虚拟→页号 | 页号 = 虚拟地址 / 页大小 |
| 虚拟→偏移 | 偏移 = 虚拟地址 % 页大小 |
| 物理地址 | 页框号×页大小+偏移 |

**关键点**：
- 页号和偏移从虚拟地址提取
- 页框号从页表查找
- 物理地址 = 页框基址 + 偏移
