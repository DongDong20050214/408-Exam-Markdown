**题干**

在分段存储管理系统中，若进程 P1 和 P2 共享段 S，下列叙述中，错误的是（）。A. 在物理内存中仅保存一份段 S 的内容  B. 段 S 在 P1 和 P2 中应该具有相同的段号  C. P1 和 P2 共享段 S 在共享段表中的段表项  D. P1 和 P2 都不再使用段 S 时才回收段 S 所占的内存空间

**答案**：B

**考点分析**
- B错误：段号可以不同

---


## 解题思路

### 第一步：理解段共享机制
共享段：多个进程共享同一段物理内存

### 第二步：逐项分析
**A. 物理内存仅保存一份 ✅**
```
共享的目的：节省内存
多个进程映射到同一物理段
结论：正确
```

**B. 段号必须相同 ❌**
```
段号是逻辑地址的一部分
不同进程可以用不同段号
通过段表映射到同一物理地址
结论：错误
```

**C. 共享段表项 ✅**
```
共享段表记录共享段信息
多个进程引用同一段表项
结论：正确
```

**D. 都不用时才回收 ✅**
```
引用计数机制
只要有进程使用就不能回收
所有进程都不用才回收
结论：正确
```

### 第三步：确认答案
段号可以不同 → 答案：**B**

## 易错点分析

### 易错点1：认为共享段号必须相同
❌ 错误：共享段在各进程中段号相同
✅ 正确：段号可以不同，映射到同一物理地址

### 易错点2：不理解段表映射
| 进程 | 段号 | 段表项 | 物理地址 |
|------|------|--------|---------|
| P1 | 2 | →共享段表 | 1000 |
| P2 | 5 | →共享段表 | 1000 |

**共享段机制**：
- 逻辑：各进程段号可不同
- 物理：映射到同一内存区域
- 管理：引用计数，都不用才回收
