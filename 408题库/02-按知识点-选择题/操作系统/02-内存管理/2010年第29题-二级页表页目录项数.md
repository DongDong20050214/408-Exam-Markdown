# 二级页表页目录项数真题汇总

## 2010年第29题

**元数据**
- 年份：2010年第29题
- 科目：#操作系统
- 知识点：#操作系统/内存管理/二级页表
- 难度：⭐⭐⭐

**题干**

某计算机采用二级页表的分页存储管理方式，按字节编址，页大小为2^10字节，页表项大小为2字节，逻辑地址空间大小为2^16页，则表示整个逻辑地址空间的页目录表中包含表项的个数至少是（）。

A. 64  
B. 128  
C. 256  
D. 512

**答案**：B

**考点分析**
- 总页数：2^16页
- 每页可容纳页表项：2^10/2=512项
- 页目录项数：2^16/512=2^16/2^9=2^7=128项

---


## 解题思路

### 第一步：理解二级页表结构
页目录→页表→物理页

### 第二步：计算步骤
```
已知：
- 页大小：2^10字节
- 页表项大小：2字节
- 逻辑地址空间：2^16页

计算：
1. 每页可容纳页表项数
   = 页大小/页表项大小
   = 2^10/2 = 2^9 = 512项

2. 需要页表数
   = 总页数/每页容纳页表项数
   = 2^16/2^9 = 2^7 = 128个页表

3. 页目录项数
   = 页表数
   = 128项
```

### 第三步：确认答案
页目录项数=128 → 答案：**B**

## 易错点分析

### 易错点1：混淆页数和页表项数
❌ 错误：2^16页就是2^16个页目录项
✅ 正确：需要除以每页能容纳的页表项数

### 易错点2：计算错误
| 步骤 | 计算 | 结果 |
|------|------|------|
| 每页容纳页表项 | 2^10÷2 | 512项 |
| 需要页表数 | 2^16÷512 | 128个 |
| 页目录项数 | 128 | **B选项** |
