# 共享内存真题汇总

## 2023年第30题

**元数据**
- 年份：[[../../01-真题原卷/2023年408真题#30|2023年第30题]]
- 科目：#操作系统
- 知识点：#操作系统/内存管理/共享内存 #页表
- 难度：⭐⭐
- 关联知识：[[2023年第30题-共享内存]]

**题干**

进程 R 和 S 共享数据 data，若 data 在 R 和 S 中所在页的页号分别为 p1 和 p2，两个页所对应的页框号分别为 f1 和 f2，则下列叙述中，正确的是（）。

A. p1 和 p2 一定相等，f1 和 f2 一定相等
B. p1 和 p2 一定相等，f1 和 f2 不一定相等
C. p1 和 p2 不一定相等，f1 和 f2 一定相等
D. p1 和 p2 不一定相等，f1 和 f2 不一定相等

**答案**：C

**考点分析**
- 虚拟页号（p1,p2）可以不同（各进程独立地址空间）
- 物理页框号（f1,f2）必须相同（共享同一物理页）

---


## 解题思路

### 第一步：理解共享内存机制
共享内存：多个进程映射到同一物理页

### 第二步：分析页号和页框号
```
页号(p)：虚拟地址的一部分，各进程独立
页框号(f)：物理地址，共享时必须相同
```

### 第三步：逐项分析
**A. p1=p2, f1=f2 ❌**
```
页号是虚拟地址，各进程独立
不要求相同
结论：错误
```

**B. p1=p2, f1≠f2 ❌**
```
共享内存要求映射到同一物理页
页框号必须相同
结论：错误
```

**C. p1≠p2, f1=f2 ✅**
```
页号：各进程虚拟地址空间独立，可以不同
页框号：共享同一物理页，必须相同
结论：正确
```

**D. p1≠p2, f1≠f2 ❌**
```
页框号不同说明不是同一物理页
无法实现共享
结论：错误
```

### 第四步：确认答案
页号可不同，页框号必须相同 → 答案：**C**

## 易错点分析

### 易错点1：认为共享内存页号必须相同
❌ 错误：共享内存在各进程中页号相同
✅ 正确：页号是虚拟的，可以不同

### 易错点2：混淆虚拟和物理地址
| 概念 | 范围 | 共享时 |
|------|------|--------|
| 页号 | 虚拟地址 | 可以不同 |
| 页框号 | 物理地址 | 必须相同 |

**共享内存映射**：
```
进程R：页p1 → 页框f
进程S：页p2 → 页框f（同一页框）
```

**关键点**：
- 虚拟：各进程独立，页号可不同
- 物理：共享同一内存，页框号必须相同
