**题干**

某系统采用页式存储管理，用位图管理空闲页框。若页大小为 4KB，物理内存大小为 16GB，则位图所占空间的大小是（）。

A. 128B  B. 128KB  C. 512KB  D. 4MB

**答案**：C

**考点分析**
- 页框数 = 16GB / 4KB = 2^24 = 4M个
- 位图大小 = 4M位 / 8 = 512KB

---


## 解题思路

### 第一步：计算页框数量
```
物理内存：16GB = 2^34 字节
页大小：4KB = 2^12 字节
页框数 = 2^34 / 2^12 = 2^22 = 4M个
```

### 第二步：计算位图大小
```
位图：每个页框对应1位
总位数 = 4M位 = 2^22 位
字节数 = 2^22 / 8 = 2^22 / 2^3 = 2^19 字节
      = 512KB
```

### 第三步：逐项分析
**A. 128B ❌** 太小
**B. 128KB ❌** 偏小
**C. 512KB ✅** 正确
**D. 4MB ❌** 太大

### 第四步：确认答案
位图512KB → 答案：**C**

## 易错点分析

### 易错点1：忘记除以8
❌ 错误：4M位=4MB
✅ 正确：4M位=4M/8字节=512KB

### 易错点2：计算错误
| 步骤 | 计算 | 结果 |
|------|------|------|
| 页框数 | 16GB/4KB | 4M |
| 位数 | 4M位 | 2^22位 |
| 字节数 | 2^22/8 | 512KB |

**位图计算公式**：
```
位图大小(字节) = 页框数 / 8
              = (内存大小/页大小) / 8
```
