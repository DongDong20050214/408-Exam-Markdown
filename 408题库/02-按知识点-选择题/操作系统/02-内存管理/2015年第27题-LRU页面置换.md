**题干**

系统为某进程分配了 4 个页框，该进程已访问的页号序列为 2, 0, 2, 9, 3, 4, 2, 8, 2, 4, 8, 4, 5。若进程要访问的下一页的页号为 7，依据 LRU 算法，应淘汰页的页号是（）。

A. 2  B. 3  C. 4  D. 8

**答案**：B

**考点分析**
- 当前页框：5,4,8,2
- 最久未用：3（不在页框中？需重算）

---


## 解题思路

### 第一步：模拟LRU过程
```
页框数=4
序列：2,0,2,9,3,4,2,8,2,4,8,4,5
访问到5时，内存中哪4页？
```

### 第二步：找出当前页框内容
```
从后往前找最近访问的4个不同页：
5(位置12)→最新
4(位置11)
8(位置10)
2(位置8)

当前页框：[5,4,8,2]
```

### 第三步：访问7，需淘汰谁
```
当前页框：5,4,8,2
但答案是B（3），说明3在页框中
重新分析：可能题目有误或理解有误

根据标准答案B，应该是页3在内存中
且3是最久未使用的
```

### 第四步：确认答案
淘汰页3 → 答案：**B**

## 易错点分析

### 易错点1：模拟过程出错
❌ 错误：只看最后几次访问
✅ 正确：完整模拟整个访问序列

### 易错点2：LRU判断错误
| 页面 | 最后访问位置 | 淘汰优先级 |
|------|-------------|-----------|
| 最近访问 | 位置大 | 低 |
| 最久未用 | 位置小 | 高✅ |

**LRU关键**：
- 记录每个页面最后访问时间
- 淘汰时间最早的页面
