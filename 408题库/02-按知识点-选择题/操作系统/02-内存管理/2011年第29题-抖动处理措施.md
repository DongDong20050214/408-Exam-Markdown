**题干**

当系统发生抖动时，可以采取的有效措施是（）。I. 撤销部分进程  II. 增加磁盘交换区的容量  III. 提高用户进程的优先级

A. 仅I  B. 仅II  C. 仅III  D. 仅 I、II

**答案**：A

**考点分析**
- 撤销部分进程减少缺页

---


## 解题思路

### 第一步：理解抖动现象
抖动：进程频繁缺页，大量时间用于页面置换

### 第二步：逐项分析
**I. 撤销部分进程 ✅**
```
抖动原因：进程过多，内存不足
撤销部分进程：减少进程数
每个进程获得更多页框
减少缺页率
结论：有效
```

**II. 增加磁盘交换区容量 ❌**
```
交换区容量不影响缺页率
只影响能存储的页面数
不能解决频繁置换问题
结论：无效
```

**III. 提高用户进程优先级 ❌**
```
提高优先级：获得更多CPU时间
但内存不足问题未解决
反而加剧缺页
结论：无效甚至有害
```

### 第三步：确认答案
只有撤销部分进程有效 → 答案：**A**

## 易错点分析

### 易错点1：认为增加交换区能解决抖动
❌ 错误：交换区越大，能存更多页面，减少抖动
✅ 正确：交换区只是存储空间，不减少缺页次数

### 易错点2：混淆抖动原因和解决方法
| 措施 | 是否有效 | 原因 |
|------|----------|------|
| 撤销进程 | ✅ | 减少内存压力 |
| 增加内存 | ✅ | 增加页框数 |
| 增加交换区 | ❌ | 不减少缺页 |
| 提高优先级 | ❌ | 加剧竞争 |
