**题干**

下列选项中，属于多级页表优点的是（）。A. 加快地址变换速度  B. 减少缺页中断次数  C. 减少页表项所占字节数  D. 减少页表所占的连续内存空间

**答案**：D

**考点分析**
- 多级页表减少连续内存需求

---


## 解题思路

### 第一步：理解多级页表目的
单级页表：需要大块连续内存
多级页表：分散存储，按需分配

### 第二步：逐项分析
**A. 加快地址变换速度 ❌**
```
多级页表需要多次访问内存
单级页表只需一次访问
反而变慢，不是优点
结论：错误
```

**B. 减少缺页中断次数 ❌**
```
缺页次数由页面置换算法决定
与页表级数无关
结论：错误
```

**C. 减少页表项所占字节数 ❌**
```
页表项大小不变
多级页表项数总和可能更多
不减少字节数
结论：错误
```

**D. 减少页表所占连续内存空间 ✅**
```
单级页表：整个页表必须连续
多级页表：可以分散存储
每级页表单独分配
不需要大块连续内存
结论：正确
```

### 第三步：确认答案
减少连续内存需求 → 答案：**D**

## 易错点分析

### 易错点1：认为多级页表加快地址转换
❌ 错误：层次越多查找越快
✅ 正确：多级需多次内存访问，反而变慢

### 易错点2：混淆多级页表优缺点
| 特性 | 单级页表 | 多级页表 |
|------|----------|----------|
| 连续内存需求 | 大 | 小✅优点 |
| 地址转换速度 | 快 | 慢❌缺点 |
| 内存利用率 | 低 | 高✅优点 |
| 实现复杂度 | 简单 | 复杂❌缺点 |
