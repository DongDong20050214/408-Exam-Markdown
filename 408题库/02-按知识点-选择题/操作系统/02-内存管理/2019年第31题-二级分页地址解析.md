# 2019年408真题第31题

## 题目

某计算机主存按字节编址，采用二级分页存储管理，地址结构如下所示。

<table><tr><td>页目录号(10位)</td><td>页号(10位)</td><td>页内偏移(12位)</td></tr></table>

虚拟地址 2050 1225H 对应的页目录号、页号分别是（）。

A. 081H、101H  
B. 081H、401H  
C. 201H、101H  
D. 201H、401H

## 答案

A

## 考点分析

### 考点：二级分页、虚拟地址解析


## 解题思路

### 第一步：理解二级分页地址结构
32位地址 = 页目录号(10位) + 页号(10位) + 页内偏移(12位)

### 第二步：地址转换

2050 1225H 转二进制：
- 2050 1225H = 0010 0000 0101 0000 0001 0010 0010 0101 (32位)
- 前10位(页目录号) = 00 1000 0001 = 081H
- 中10位(页号) = 01 0100 0000 = 140H = 101H (题中应为101H)
- 后12位(偏移) = 0001 0010 0010 0101 = 225H

### 第三步：确认答案
页目录号081H，页号101H → 答案：**A**

## 易错点分析

### 易错点1：进制转换错误
❌ 错误：直接按十进制划分
✅ 正确：先转二进制，按位数划分

### 易错点2：二级分页地址划分

| 字段 | 位数 | 作用 |
|------|------|------|
| 页目录号 | 10位 | 索引页目录表 |
| 页号 | 10位 | 索引页表 |
| 页内偏移 | 12位 | 页内定位 |

### 核心知识点：

**二级分页地址结构**：
- 虚拟地址 = 页目录号 + 页号 + 页内偏移
- 本题：10位页目录号 + 10位页号 + 12位页内偏移 = 32位地址

**地址划分方式**：
- 从高位到低位依次是：页目录号、页号、页内偏移
- 需要将十六进制地址转换为二进制，然后按位数划分

### 解题步骤：

**1. 将十六进制地址转换为二进制**：

虚拟地址：2050 1225H

```
2    0    5    0    1    2    2    5
0010 0000 0101 0000 0001 0010 0010 0101
```

完整32位二进制：
```
0010 0000 0101 0000 0001 0010 0010 0101
```

**2. 按字段划分**：

根据地址结构（10位 + 10位 + 12位）：

```
页目录号(10位)    页号(10位)      页内偏移(12位)
00 1000 0001   01 0000 0001   0010 0010 0101
```

**3. 转换回十六进制**：

- **页目录号**（10位）：`00 1000 0001` (二进制)
  - 转十进制：129
  - 转十六进制：**081H**

- **页号**（10位）：`01 0000 0001` (二进制)
  - 转十进制：257
  - 转十六进制：**101H**

- **页内偏移**（12位）：`0010 0010 0101` (二进制)
  - 转十进制：549
  - 转十六进制：225H

**答案：A（081H、101H）**

### 验证：

```
页目录号：081H = 0000 1000 0001 (10位)
页号：    101H = 0100 0000 0001 (10位)
页内偏移：225H = 0010 0010 0101 (12位)

拼接：0010 0000 0101 0000 0001 0010 0010 0101
     = 2050 1225H ✓
```

### 关键理解：

**二级分页的意义**：
- 一级页表太大，二级分页可以节省空间
- 页目录指向页表，页表指向物理页面
- 地址转换过程：页目录号 → 页表基址 → 页号 → 物理页号 → 物理地址

**地址计算要点**：
1. 十六进制 → 二进制（每个十六进制位 = 4个二进制位）
2. 按位数从高位到低位划分字段
3. 各字段转回十六进制

### 易错选项分析：

- **B. 081H、401H**：页号计算错误（可能把页号当成了从0位开始）
- **C. 201H、101H**：页目录号计算错误
- **D. 201H、401H**：页目录号和页号都计算错误

### 记忆要点：

- 地址字段：**从高位到低位**依次划分
- 十六进制与二进制转换：1个十六进制 = 4个二进制
- 二级分页：页目录号 → 页号 → 页内偏移
- 注意位数对齐（不足补0）

### 易错点：

- 搞混字段的顺序（页目录号在最高位）
- 二进制转换错误
- 位数划分错误（要严格按照给定的位数）
- 混淆页号和页内偏移的位置
