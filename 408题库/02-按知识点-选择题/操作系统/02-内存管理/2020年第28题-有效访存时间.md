# 有效访存时间真题汇总

## 2020年第28题

**元数据**
- 年份：[[../../01-真题原卷/2020年408真题#28|2020年第28题]]
- 科目：#操作系统
- 知识点：#操作系统/内存管理/访存时间
- 难度：⭐⭐
- 关联知识：[[知识库/操作系统/分页]]

**题干**

下列因素中，影响请求分页系统有效（平均）访存时间的是（）。

I. 缺页率  II. 磁盘读写时间  III. 内存访问时间  IV. 执行缺页处理程序的 CPU 时间

A. 仅II、III  B. 仅 I、IV  C. 仅 I、III、IV  D. I、II、III 和 IV

**答案**：D

**考点分析**
- 所有因素都影响有效访存时间

---


## 解题思路

### 第一步：理解有效访存时间
有效访存时间 = 命中时间×(1-缺页率) + 缺页处理时间×缺页率

### 第二步：逐项分析
**I. 缺页率 ✅**
```
缺页率直接影响公式
缺页率高→有效访存时间长
结论：影响
```

**II. 磁盘读写时间 ✅**
```
缺页时需要从磁盘读取页面
磁盘时间是缺页处理的主要部分
结论：影响
```

**III. 内存访问时间 ✅**
```
命中时需要访问内存
访问时间直接影响命中时的耗时
结论：影响
```

**IV. 执行缺页处理程序CPU时间 ✅**
```
缺页处理包括：
- 中断处理
- 页表更新
- 进程切换等
CPU时间也是缺页处理的一部分
结论：影响
```

### 第三步：确认答案
I、II、III、IV都影响 → 答案：**D**

## 易错点分析

### 易错点1：忽略CPU处理时间
❌ 错误：缺页只涉及磁盘和内存时间
✅ 正确：还包括CPU的缺页处理时间

### 易错点2：不理解有效访存时间公式
| 因素 | 影响部分 | 权重 |
|------|---------|------|
| 缺页率 | 整体 | 关键参数 |
| 内存时间 | 命中部分 | 1-p |
| 磁盘时间 | 缺页部分 | p |
| CPU时间 | 缺页部分 | p |

**公式**：
```
EAT = (1-p)×内存时间 + p×(CPU处理+磁盘+内存)
其中p=缺页率
```
