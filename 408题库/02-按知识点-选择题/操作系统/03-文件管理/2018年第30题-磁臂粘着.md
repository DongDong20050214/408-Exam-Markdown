# 磁臂粘着真题汇总

## 2018年第30题

- 关联知识：[2021年第26题-磁盘调度](2021年第26题-磁盘调度.md)

**题干**

系统总是访问磁盘的某个磁道而不响应对其他磁道的访问请求，这种现象称为磁臂粘着。下列磁盘调度算法中，不会导致磁臂粘着的是（）。

A. 先来先服务（FCFS）  B. 最短寻道时间优先（SSTF）  C. 扫描算法（SCAN）  D. 循环扫描算法（CSCAN）

**答案**：A

**考点分析**
- FCFS不会产生磁臂粘着
- SSTF可能导致饥饿

---


## 解题思路

### 第一步：理解磁臂粘着现象
磁臂粘着（饥饿）：某些磁道的请求长时间得不到服务，磁头总是服务其他磁道。

### 第二步：逐项分析

**A. 先来先服务（FCFS）✅**
```
详细分析：按请求到达顺序服务，公平性最好
判断依据：
- 调度策略：严格按照请求到达的时间顺序
- 公平性：每个请求都会被服务，不存在饥饿
- 不会粘着：无论磁道位置如何，都会依次服务
- 缺点：平均寻道时间可能较长，但不会饥饿
结论：正确，FCFS不会导致磁臂粘着
```

**B. 最短寻道时间优先（SSTF）❌**
```
详细分析：贪心策略可能导致远端请求饥饿
判断依据：
- 调度策略：每次选择距离当前磁头最近的请求
- 磁臂粘着场景：
  - 磁头在中间磁道（如磁道100）
  - 不断有中间磁道的新请求到达
  - 磁头持续在中间区域移动
  - 外围磁道（如磁道0和199）的请求长期得不到服务
- 示例：磁头在100，有98、102的请求不断到达
         磁道0的请求永远排在后面
结论：错误，SSTF可能导致磁臂粘着
```

**C. 扫描算法（SCAN）❌**
```
详细分析：电梯算法仍可能导致局部饥饿
判断依据：
- 调度策略：磁头单向移动到边界，再反向
- 可能粘着：刚扫描过的磁道新请求需等待完整往返
- 示例：
  - 磁头向外扫描刚经过磁道50
  - 此时磁道50来新请求
  - 需等待磁头到达外侧边界→反向→到达内侧边界→再次外扫
  - 等待时间可能很长
结论：错误，SCAN也可能导致某种形式的饥饿
```

**D. 循环扫描算法（CSCAN）❌**
```
详细分析：改进的SCAN仍无法完全避免饥饿
判断依据：
- 调度策略：单向扫描到边界，快速返回起点，再扫描
- 优于SCAN：但刚扫描过的位置的新请求仍需等待
- 等待时间：比SCAN短，但仍可能较长
- 不如FCFS：无法保证每个请求的等待时间上限
结论：错误，CSCAN虽有改进但仍可能饥饿
```

### 第三步：确认答案
只有FCFS不会导致磁臂粘着 → 答案：**A**

## 易错点分析

### 易错点1：认为SCAN/CSCAN不会饥饿
❌ 错误理解：SCAN和CSCAN是改进算法，不会有饥饿问题
✅ 正确理解：SCAN/CSCAN虽比SSTF公平，但刚扫描过的磁道若有新请求到达，需等待磁头完成一次完整扫描（SCAN需往返，CSCAN需跳转+单程），可能等待数百毫秒。严格意义上这也是饥饿的一种，只是程度轻于SSTF。只有FCFS按时间顺序服务，绝对不会出现任何请求被推迟的情况

### 易错点2：不理解磁臂粘着的本质

| 算法 | 是否公平 | 可能饥饿 | 原因 |
|------|---------|---------|------|
| FCFS | 完全公平 | ❌不会 | 严格按时间顺序 |
| SSTF | 不公平 | ✅会 | 远端请求长期得不到服务 |
| SCAN | 相对公平 | ✅会（轻微） | 刚扫描过的位置新请求等待久 |
| CSCAN | 相对公平 | ✅会（轻微） | 同SCAN但改进 |

**SSTF饥饿示例**：
```
时间线：
t0: 磁头在100，请求队列：[50, 150]
    选择：100→150（近）
    
t1: 磁头在150，新请求到达105，队列：[50, 105]
    选择：150→105（近）
    
t2: 磁头在105，新请求到达110，队列：[50, 110]
    选择：105→110（近）
    
...中间位置不断有新请求...

t100: 磁道50的请求仍未被服务（磁臂粘着在中间区域）
```

**SCAN饥饿示例**：
```
扫描方向：→（向右）

t0: 磁头在50向右扫，经过磁道51
t1: 磁道51新请求到达
    → 磁头继续向右到199
    → 反向向左到0
    → 再次向右才能服务磁道51
    等待时间：几乎完整两次扫描
```

**关键点**：
- 磁臂粘着 = 请求饥饿
- FCFS：牺牲效率保证公平
- SSTF：追求效率可能不公平
- SCAN/CSCAN：效率和公平的折中，但仍有小幅饥饿可能
