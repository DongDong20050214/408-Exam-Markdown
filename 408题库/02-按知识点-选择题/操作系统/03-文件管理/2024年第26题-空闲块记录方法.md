**题干**

文件系统需占用部分外存空间记录空闲块位置。下列方法中，占用外存空间的大小与当前空闲块数量无关的是（）。

A. 位图法  B. 空闲表法  C. 成组链接法  D. 空闲链表法

**答案**：A

**考点分析**
- 位图法：空间固定，与空闲块数无关

---


## 解题思路

### 第一步：理解空闲块管理方法
不同方法占用空间不同

### 第二步：逐项分析
**A. 位图法 ✅**
```
每个块对应1位
总位数=磁盘块总数
空间固定，与空闲块数无关
结论：无关
```

**B. 空闲表法 ❌**
```
记录空闲块的起始和长度
空闲块多→表项多→空间大
结论：有关
```

**C. 成组链接法 ❌**
```
记录空闲块号
空闲块多→链接信息多
结论：有关
```

**D. 空闲链表法 ❌**
```
空闲块组成链表
空闲块多→链表长→空间大
结论：有关
```

### 第三步：确认答案
位图法空间固定 → 答案：**A**

## 易错点分析

### 易错点1：认为位图法空间可变
❌ 错误：空闲块少时位图也少
✅ 正确：位图大小取决于总块数

### 易错点2：混淆管理方法
| 方法 | 空间大小 |
|------|---------|
| 位图法 | 固定✅ |
| 空闲表法 | 可变 |
| 空闲链表 | 可变 |
| 成组链接 | 可变 |
