**题干**

下列文件物理结构中，适合随机访问且易于文件扩展的是（）。A. 连续结构  B. 索引结构  C. 链式结构且磁盘块定长  D. 链式结构且磁盘块变长

**答案**：B

**考点分析**
- 索引结构支持随机访问和扩展

---


## 解题思路

### 第一步：理解文件物理结构
连续、链式、索引三种结构

### 第二步：逐项分析
**A. 连续结构 ❌**
```
随机访问：✅支持
文件扩展：❌困难（需连续空间）
结论：不满足
```

**B. 索引结构 ✅**
```
随机访问：✅通过索引表直接定位
文件扩展：✅增加索引项即可
结论：都满足
```

**C. 链式结构定长 ❌**
```
随机访问：❌需顺序遍历链表
文件扩展：✅容易
结论：不满足
```

**D. 链式结构变长 ❌**
```
随机访问：❌需顺序遍历
文件扩展：✅容易
结论：不满足
```

### 第三步：确认答案
索引结构满足两个条件 → 答案：**B**

## 易错点分析

### 易错点1：认为连续结构易扩展
❌ 错误：连续结构随机访问快也易扩展
✅ 正确：扩展需要连续空间，困难

### 易错点2：混淆文件结构特点
| 结构 | 随机访问 | 易扩展 |
|------|---------|--------|
| 连续 | ✅ | ❌ |
| 链式 | ❌ | ✅ |
| 索引 | ✅ | ✅ |
