**题干**

文件系统中，文件访问控制信息存储的合理位置是（）。

A. 文件控制块  
B. 文件分配表  
C. 用户口令表  
D. 系统注册表

**答案**：A

**考点分析**
- FCB(文件控制块)包含：文件名、物理位置、访问权限等
- 访问控制信息：读、写、执行权限
- 文件分配表(FAT)：存储文件物理块分配信息

---


## 解题思路

### 第一步：理解文件访问控制信息
访问控制：读、写、执行等权限

### 第二步：逐项分析
**A. 文件控制块FCB ✅**
```
FCB包含：
- 文件名
- 物理位置
- 访问权限（rwx）
- 所有者等
结论：正确
```

**B. 文件分配表FAT ❌**
```
FAT记录：
- 文件占用的磁盘块
- 块链接关系
不包含权限信息
结论：错误
```

**C. 用户口令表 ❌**
```
存储用户登录口令
与文件访问权限无关
结论：错误
```

**D. 系统注册表 ❌**
```
Windows特有
存储系统配置
不是文件权限位置
结论：错误
```

### 第三步：确认答案
FCB包含访问控制信息 → 答案：**A**

## 易错点分析

### 易错点1：混淆FCB和FAT
❌ 错误：FAT也存权限信息
✅ 正确：FAT只存物理块分配

### 易错点2：不理解FCB内容
| 数据结构 | 存储内容 |
|---------|---------|
| FCB | 文件属性+权限 |
| FAT | 磁盘块分配 |
| 索引节点 | 类似FCB |
