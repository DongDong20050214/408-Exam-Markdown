# 进程切换真题汇总

## 2024年第25题

**元数据**
- 年份：2024年第25题
- 科目：#操作系统
- 知识点：#操作系统/进程管理/进程切换
- 难度：⭐⭐

**题干**

在支持页式存储管理的系统中，进程切换时操作系统需要执行的操作是（）。

I. 更新程序计数器的值  II. 更新栈基址寄存器的值  III. 更新页表基地址寄存器的值

A. 仅III  B. 仅I、II  C. 仅I、III  D. I、II、III

**答案**：D

**考点分析**
- 所有操作都需要执行

---


## 解题思路

### 第一步：理解进程切换操作
进程切换：保存旧进程上下文，恢复新进程上下文

### 第二步：逐项分析
**I. 更新程序计数器PC ✅**
```
PC指向下一条指令
切换后要执行新进程的指令
必须更新PC
结论：需要执行
```

**II. 更新栈基址寄存器 ✅**
```
每个进程有独立的栈
切换后需要指向新进程的栈
必须更新栈基址寄存器
结论：需要执行
```

**III. 更新页表基址寄存器 ✅**
```
页式存储：每个进程有独立页表
切换后需要指向新进程的页表
必须更新页表基址寄存器（PTBR）
结论：需要执行
```

### 第三步：确认答案
I、II、III都需要 → 答案：**D**

## 易错点分析

### 易错点1：认为不需要更新页表基址
❌ 错误：页表是共享的，不需要更新
✅ 正确：每个进程独立页表，必须更新PTBR

### 易错点2：忽视栈基址更新
| 寄存器 | 是否更新 | 原因 |
|--------|---------|------|
| PC | ✅ | 指令位置不同 |
| 栈基址 | ✅ | 栈空间不同 |
| 页表基址 | ✅ | 地址空间不同 |
| 通用寄存器 | ✅ | 上下文不同 |

**进程切换步骤**：
1. 保存旧进程上下文
   - PC、寄存器、栈指针等
2. 更新调度信息
3. 恢复新进程上下文
   - 更新PC、栈基址、页表基址等
4. 返回新进程执行

**关键点**：
- 所有进程相关寄存器都要更新
- 页表基址切换实现地址空间隔离
