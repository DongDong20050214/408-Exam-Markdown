**题干**

在支持多线程的系统中，进程P创建的若干个线程不能共享的是（ ）。A. 进程 P 的代码段  B. 进程 P 中打开的文件  C. 进程 P 的全局变量  D. 进程 P 中某线程的栈指针

**答案**：D

**考点分析**
- D：栈指针是线程私有的

---


## 解题思路

### 第一步：理解线程共享和私有
线程共享：进程资源（代码、数据、文件）
线程私有：执行状态（栈、寄存器、PC）

### 第二步：逐项分析
**A. 代码段 ✅可共享**
```
同一进程的所有线程共享代码段
代码是只读的，可以安全共享
所有线程执行相同的程序代码
结论：可以共享
```

**B. 打开的文件 ✅可共享**
```
文件描述符表属于进程
所有线程可访问进程打开的文件
通过文件描述符共享文件资源
结论：可以共享
```

**C. 全局变量 ✅可共享**
```
全局变量在进程的数据段
所有线程可以访问和修改
需要同步机制避免冲突
结论：可以共享
```

**D. 某线程的栈指针 ❌不可共享**
```
每个线程有独立的栈
栈指针(SP)指向线程私有栈
不同线程栈指针值不同
结论：不能共享
```

### 第三步：确认答案
只有栈指针不能共享 → 答案：**D**

## 易错点分析

### 易错点1：认为全局变量不共享
❌ 错误：每个线程有独立的变量空间
✅ 正确：全局变量在进程空间，所有线程共享

### 易错点2：混淆线程共享和私有
| 资源 | 是否共享 | 说明 |
|------|----------|------|
| 代码段 | ✅ | 只读，共享 |
| 全局变量 | ✅ | 需同步 |
| 堆 | ✅ | 动态分配共享 |
| 栈 | ❌ | 每线程独立 |
| 寄存器 | ❌ | 线程私有 |
| PC | ❌ | 线程私有 |
