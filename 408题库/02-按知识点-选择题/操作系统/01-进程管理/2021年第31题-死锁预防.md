# 死锁预防真题汇总

## 2021年第31题

**元数据**
- 年份：2021年第31题
- 科目：#操作系统
- 知识点：#操作系统/进程管理/死锁 #资源分配
- 难度：⭐⭐⭐

**题干**

若系统中有 n (n ≥ 2) 个进程, 每个进程均需要使用某类临界资源 2 个, 则系统不会发生死锁所需的该类资源总数至少是 ( )。

A. 2  B. n  C. n + 1  D. 2n

**答案**：C

**考点分析**
- 最坏情况：n个进程各持有1个资源，都在等待第2个
- 需要n+1个资源，至少1个进程能获得2个资源完成

---


## 解题思路

### 第一步：分析死锁条件
每个进程需要2个资源，n个进程
最坏情况：每个进程持有1个，都等待第2个

### 第二步：推导公式
```
最坏情况分析：
n个进程各持有1个资源 → 用了n个资源
每个进程都差1个 → 死锁

避免死锁：
至少1个进程能获得2个资源完成
需要资源数：n + 1

证明：
n个资源：最坏每个进程1个，死锁
n+1个资源：至少1个进程获得2个，完成后释放，其他进程依次完成
```

### 第三步：逐项分析
**A. 2个 ❌**
```
n≥2个进程，2个资源不够
最多1个进程完成
结论：会死锁
```

**B. n个 ❌**
```
n个进程各持有1个
每个都差1个，形成死锁
结论：会死锁
```

**C. n+1个 ✅**
```
最坏：n个进程各持有1个
剩余1个资源分配给某进程
该进程获得2个完成，释放2个
其他进程依次完成
结论：不会死锁
```

**D. 2n个 ❌**
```
2n个资源充足但不是最少
n+1就够了
结论：不是最少
```

### 第四步：确认答案
至少n+1个 → 答案：**C**

## 易错点分析

### 易错点1：认为需要2n个资源
❌ 错误：n个进程×2个资源=2n个
✅ 正确：n+1个就够，不需要同时满足所有进程

### 易错点2：不理解最坏情况分析
| 资源数 | 最坏分配 | 是否死锁 |
|--------|---------|---------|
| n-1 | 部分进程各1个 | 死锁 |
| n | n个进程各1个 | 死锁 |
| n+1 | n个各1个+1个 | 不死锁 |
| 2n | 充足 | 不死锁 |

**死锁预防公式**：
```
进程数：n
每进程需要：m个资源
避免死锁最少资源：n×(m-1) + 1

本题：n×(2-1) + 1 = n+1
```

**原理**：
- 最坏：每个进程持有(m-1)个
- 需要：n×(m-1) + 1
- 至少1个进程能获得m个完成
