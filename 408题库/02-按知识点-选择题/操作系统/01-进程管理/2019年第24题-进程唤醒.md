# 进程唤醒真题汇总

## 2019年第24题

**元数据**
- 年份：[[../../01-真题原卷/2019年408真题#24|2019年第24题]]
- 科目：#操作系统
- 知识点：#操作系统/进程管理/进程唤醒
- 难度：⭐⭐
- 关联知识：[[知识库/操作系统/进程状态]]

**题干**

下列选项中，可能会将进程唤醒的事件是（）。

I. I/O 结束  II. 某进程退出临界区  III. 当前进程的时间片用完

A. 仅I  B. 仅III  C. 仅 I、II  D. I、II、III

**答案**：C

**考点分析**
- I、II可能唤醒阻塞进程
- III导致进程就绪，不是唤醒

---


## 解题思路

### 第一步：理解唤醒含义
唤醒：阻塞态→就绪态

### 第二步：逐项分析
**I. I/O结束 ✅**
```
进程等待I/O时处于阻塞态
I/O完成后唤醒进程
阻塞态→就绪态
结论：会唤醒
```

**II. 某进程退出临界区 ✅**
```
其他进程等待临界区时阻塞
退出临界区释放资源
唤醒等待进程
结论：可能唤醒
```

**III. 当前进程时间片用完 ❌**
```
时间片用完：执行态→就绪态
不涉及阻塞态进程
不是唤醒操作
结论：不会唤醒
```

### 第三步：确认答案
I、II会唤醒 → 答案：**C**

## 易错点分析

### 易错点1：混淆唤醒和调度
❌ 错误：时间片用完也是唤醒
✅ 正确：
- 唤醒：阻塞→就绪
- 调度：就绪→执行

### 易错点2：不理解唤醒条件
| 事件 | 是否唤醒 | 原因 |
|------|----------|------|
| I/O完成 | ✅ | 等待事件结束 |
| 释放资源 | ✅ | 资源可用 |
| 时间片用完 | ❌ | 不涉及阻塞 |
| V操作 | ✅ | 释放信号量 |
