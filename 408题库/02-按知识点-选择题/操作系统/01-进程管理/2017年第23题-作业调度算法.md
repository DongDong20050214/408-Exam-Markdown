**题干**

系统在 t = 2 时开始作业调度。若分别采用先来先服务和短作业优先调度算法，则选中的作业分别是（）。A. J2、J3  B. J1、J4  C. J2、J4  D. J1、J3

**答案**：A

**考点分析**
- FCFS选J2（先到），SJF选J3（最短）

---


## 解题思路

### 第一步：理解题目（需要作业表数据）
```
假设作业信息：
作业  到达时间  服务时间
J1    t=0      5
J2    t=1      3
J3    t=2      2
J4    t=2      4

t=2时刻开始调度，已到达：J1, J2, J3, J4
```

### 第二步：FCFS（先来先服务）
```
按到达时间顺序：
J1(t=0) → J2(t=1) → J3(t=2) → J4(t=2)

t=2时刻，J1已在运行或刚完成
选择最先到达的未执行作业：J2
```

### 第三步：SJF（短作业优先）
```
t=2时刻已到达的作业及服务时间：
J1: 5（可能已执行）
J2: 3
J3: 2（最短）
J4: 4

选择服务时间最短的：J3
```

### 第四步：确认答案
FCFS选J2，SJF选J3 → 答案：**A**

## 易错点分析

### 易错点1：忽视已到达时间
❌ 错误：t=2时只有J3和J4到达，选它们
✅ 正确：t=2时J1、J2也已到达，都要考虑

### 易错点2：混淆调度算法
| 算法 | 选择依据 | 优点 | 缺点 |
|------|---------|------|------|
| FCFS | 到达顺序 | 公平 | 长作业拖延 |
| SJF | 服务时间 | 平均周转短 | 长作业饥饿 |

**FCFS特点**：
- 按到达时间先后顺序
- 先到先服务，公平
- 不考虑服务时间

**SJF特点**：
- 选择服务时间最短的
- 平均周转时间最短
- 可能导致长作业饥饿

**典型题目分析步骤**：
1. 列出t=2时已到达的所有作业
2. FCFS：选最早到达的
3. SJF：选服务时间最短的
