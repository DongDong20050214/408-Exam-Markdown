# 二级反馈队列真题汇总

## 2019年第27题

**元数据**
- 年份：2019年第27题
- 科目：#操作系统
- 知识点：#操作系统/进程管理/反馈队列
- 难度：⭐⭐⭐

**题干**

系统采用二级反馈队列调度算法进行进程调度，P1、P2需要的CPU时间分别为30ms和20ms，则进程P1、P2在系统中的平均等待时间为( )。

A. 25 ms  B. 20 ms  C. 15 ms  D. 10 ms

**答案**：C

**考点分析**
- P1: 0-10(Q1), 10-20等待, 20-30(Q2), 30-50(Q2)
- P2: 10-20(Q1), 20-30(Q2)
- 需详细计算等待时间

---

## 解题思路

### 第一步：理解二级反馈队列
Q1时间片10ms(FCFS)，Q2时间片20ms。Q1优先级高于Q2。

### 第二步：模拟执行过程

**时间线：**
- 0-10ms: P1在Q1执行10ms，用完进入Q2
- 10-20ms: P2在Q1执行10ms，用完进入Q2
- 20-40ms: P1在Q2执行20ms，完成（总30ms）
- 40-50ms: P2在Q2执行10ms，完成（总20ms）

**等待时间计算：**
- P1: 在10-20ms等待P2 = 10ms等待
- P2: 在20-40ms等待P1 = 20ms等待
- 平均等待 = (10+20)/2 = 15ms

### 第三步：确认答案
平均等待时间15ms → 答案：**C**

## 易错点分析

### 易错点1：混淆等待时间和周转时间
❌ 错误：等待时间包括执行时间
✅ 正确：等待时间是进程在就绪队列中等待的时间

### 易错点2：P1/P2执行过程

| 时间段 | Q1 | Q2 | P1等待 | P2等待 |
|--------|----|----|--------|--------|
| 0-10 | P1 | - | 0 | 0 |
| 10-20 | P2 | - | 10 | 0 |
| 20-40 | - | P1 | 0 | 20 |
| 40-50 | - | P2 | - | 0 |
