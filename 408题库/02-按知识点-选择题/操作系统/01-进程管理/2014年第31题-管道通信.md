# 管道通信真题汇总

## 2014年第31题

**元数据**
- 年份：[2014年第31题](../../../01-真题原卷/2014年408真题.md#31)
- 科目：#操作系统
- 知识点：#操作系统/进程管理/管道
- 难度：⭐⭐
- 关联知识：[进程通信](../../../../AI-知识点原子库/进程通信.md)

**题干**

下列关于管道（Pipe）通信的叙述中，正确的是（）。A. 一个管道可实现双向数据传输  B. 管道的容量仅受磁盘容量大小限制  C. 进程对管道进行读操作和写操作都可能被阻塞  D. 一个管道只能有一个读进程或一个写进程对其操作

**答案**：C

**考点分析**
- 管道满时写阻塞，管道空时读阻塞

---


## 解题思路

### 第一步：理解管道特性
管道：半双工通信，单向数据流，有限缓冲区

### 第二步：逐项分析
**A. 双向数据传输 ❌**
```
管道是半双工通信
数据只能单向流动
读端和写端固定
需双向通信需要两个管道
结论：错误
```

**B. 容量受磁盘限制 ❌**
```
管道缓冲区在内存中
不在磁盘上
容量受内存限制，通常4KB-64KB
结论：错误
```

**C. 读写都可能阻塞 ✅**
```
管道空时：读操作阻塞（等待数据）
管道满时：写操作阻塞（等待空间）
这是管道同步机制
结论：正确
```

**D. 只能一个读进程或一个写进程 ❌**
```
管道可以有多个读进程和写进程
但通常一个写多个读，或多个写一个读
不限制进程数量
结论：错误
```

### 第三步：确认答案
读写都可能阻塞 → 答案：**C**

## 易错点分析

### 易错点1：认为管道是全双工
❌ 错误：管道可以双向读写
✅ 正确：管道半双工，单向通信

### 易错点2：混淆管道类型
| 管道类型 | 方向 | 位置 | 持久性 |
|----------|------|------|--------|
| 匿名管道 | 单向 | 内存 | 临时 |
| 命名管道 | 单向 | 文件系统 | 持久 |
| Socket | 双向 | 网络 | 持久 |
