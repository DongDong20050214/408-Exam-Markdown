# 并发执行时间计算真题汇总

## 2016年第24题

**元数据**
- 年份：[[../../01-真题原卷/2016年408真题#24|2016年第24题]]
- 科目：#操作系统
- 知识点：#操作系统/进程管理/CPU调度 #流水并发
- 难度：⭐⭐⭐
- 关联知识：[[知识库/操作系统/进程调度]]

**题干**

某单CPU系统中有输入和输出设备各1台，现有3个并发执行的作业，每个作业的输入、计算和输出时间均分别为2ms、3ms和4ms，且都按输入、计算和输出的顺序执行，则执行完3个作业需要的时间最少是（）。

A. 15ms  
B. 17ms  
C. 22ms  
D. 27ms

**答案**：B

**考点分析**
- 流水线并发：输入设备、CPU、输出设备可并行工作
- 第1个作业：0-2输入，2-5计算，5-9输出，总耗时9ms
- 第2个作业：2-4输入，5-8计算，9-13输出
- 第3个作业：4-6输入，8-11计算，13-17输出
- 最后一个作业输出结束时刻为17ms

**时间线分析**
```
作业1: [输入2][计算3][输出4]
作业2:   [输入2][计算3][输出4]
作业3:     [输入2][计算3][输出4]
时刻: 0--2--4--6--8--11--13--17
```

---


## 解题思路

### 第一步：理解流水线并发
输入设备、CPU、输出设备可同时工作
关键：后一个作业可在前一个作业进入下一阶段后开始

### 第二步：绘制时间轴
```
每个作业：输入2ms → 计算3ms → 输出4ms

时间轴：
0-2:   作业1输入 | CPU空闲  | 输出空闲
2-4:   作业2输入 | 作业1计算 | 输出空闲
4-5:   作业3输入 | 作业1计算 | 输出空闲
5-6:   作业3输入 | 作业2计算 | 作业1输出
6-8:   输入空闲 | 作业2计算 | 作业1输出
8-9:   输入空闲 | 作业3计算 | 作业1输出
9-11:  输入空闲 | 作业3计算 | 作业2输出
11-13: 输入空闲 | CPU空闲  | 作业2输出
13-17: 输入空闲 | CPU空闲  | 作业3输出
```

### 第三步：计算完成时间
```
作业1：0-2输入，2-5计算，5-9输出
作业2：2-4输入，5-8计算，9-13输出
作业3：4-6输入，8-11计算，13-17输出

最后完成时刻：17ms
```

### 第四步：确认答案
最少时间17ms → 答案：**B**

## 易错点分析

### 易错点1：忽视设备并行
❌ 错误：3个作业顺序执行，3×(2+3+4)=27ms
✅ 正确：流水线并发，设备可同时工作

### 易错点2：计算流水线时间公式错误
❌ 错误：简单相加所有时间段
✅ 正确：使用流水线公式

**流水线时间计算**：
| 公式 | 说明 |
|------|------|
| 第1个作业完成 | t1 + t2 + t3 = 2+3+4 = 9ms |
| n个作业完成 | (t1+t2+t3) + (n-1)×max(t1,t2,t3) |
| 本题 | 9 + 2×4 = 17ms |

**关键点**：
- 第1个作业走完全程：9ms
- 后续作业每个增加最长阶段时间：4ms
- 总时间：9 + 2×4 = 17ms
