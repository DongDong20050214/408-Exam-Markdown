# 进程调度时机真题汇总

## 2021年第27题

**元数据**
- 年份：2021年第27题
- 科目：#操作系统
- 知识点：#操作系统/进程管理/调度时机
- 难度：⭐⭐

**题干**

下列事件中，可能引起进程调度程序执行的是（）。

I. 中断处理结束  II. 进程阻塞  III. 进程执行结束  IV. 进程的时间片用完

A. 仅 I、III  B. 仅 II、IV  C. 仅 III、IV  D. I、II、III 和 IV

**答案**：D

**考点分析**
- 所有选项都可能引起进程调度
- 进程状态改变、时间片用完、中断返回等都可能触发调度

---


## 解题思路

### 第一步：理解调度时机
调度发生：进程状态改变、时间片用完、中断返回

### 第二步：逐项分析
**I. 中断处理结束 ✅**
```
中断返回前检查是否需要调度
可能有更高优先级进程就绪
时钟中断后触发调度
结论：可能引起调度
```

**II. 进程阻塞 ✅**
```
进程阻塞释放CPU
必须选择新进程执行
执行态→阻塞态，需要调度
结论：引起调度
```

**III. 进程执行结束 ✅**
```
进程结束释放CPU
必须选择新进程
需要立即调度
结论：引起调度
```

**IV. 时间片用完 ✅**
```
时间片轮转调度
时间片用完进程降级
选择新进程执行
结论：引起调度
```

### 第三步：确认答案
I、II、III、IV都引起调度 → 答案：**D**

## 易错点分析

### 易错点1：认为中断不引起调度
❌ 错误：中断处理在内核态，不能调度
✅ 正确：中断返回前可以调度

### 易错点2：混淆调度时机
| 事件 | 是否调度 | 原因 |
|------|---------|------|
| 中断返回 | ✅ | 检查就绪队列 |
| 进程阻塞 | ✅ | 释放CPU |
| 进程结束 | ✅ | 释放CPU |
| 时间片用完 | ✅ | 轮转调度 |
| 临界区内 | ✅可以 | 可抢占 |

**调度时机分类**：
1. **主动放弃CPU**：
   - 进程结束
   - 进程阻塞
   
2. **被动失去CPU**：
   - 时间片用完
   - 被高优先级抢占
   
3. **中断后**：
   - 时钟中断
   - I/O中断完成

**关键点**：
- 非抢占式：只在主动放弃时调度
- 抢占式：以上所有情况都可调度
