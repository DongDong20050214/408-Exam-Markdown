# 线程共享真题汇总

## 2024年第28题

**元数据**
- 年份：2024年第28题
- 科目：#操作系统
- 知识点：#操作系统/进程管理/线程 #资源共享
- 难度：⭐⭐

**题干**

若进程P中的线程T先打开文件, 得到文件描述符fd, 再创建两个线程Ta和Tb, 则下列资源中, Ta与Tb可共享的是 ( )。

I. 进程P的地址空间  II. 线程T的栈  III. 文件描述符fd

A. 仅I  B. 仅I、III  C. 仅II、III  D. I、II、III

**答案**：B

**考点分析**
- 地址空间和文件描述符共享
- 栈不共享

---


## 解题思路

### 第一步：理解线程资源共享
同一进程的线程：共享进程资源，独立栈空间

### 第二步：逐项分析
**I. 进程P的地址空间 ✅共享**
```
所有线程共享进程地址空间
包括代码段、数据段、堆
Ta和Tb都可以访问
结论：可共享
```

**II. 线程T的栈 ❌不共享**
```
每个线程有独立的栈
T的栈只属于T
Ta和Tb有各自的栈，不能访问T的栈
结论：不可共享
```

**III. 文件描述符fd ✅共享**
```
文件描述符属于进程资源
同一进程的所有线程共享
Ta和Tb可以用fd操作文件
结论：可共享
```

### 第三步：确认答案
I、III可共享 → 答案：**B**

## 易错点分析

### 易错点1：认为线程栈也共享
❌ 错误：所有资源都共享
✅ 正确：栈是线程私有的

### 易错点2：混淆线程共享资源
| 资源类型 | 是否共享 | 说明 |
|---------|---------|------|
| 地址空间 | ✅ | 代码、数据、堆 |
| 文件描述符 | ✅ | 进程资源 |
| 栈 | ❌ | 线程私有 |
| 寄存器 | ❌ | 线程私有 |
| 线程ID | ❌ | 线程私有 |

**线程资源分类**：
- **共享资源**：
  - 进程地址空间
  - 全局变量
  - 堆
  - 文件描述符
  - 信号处理

- **私有资源**：
  - 线程ID
  - 栈
  - 寄存器
  - 程序计数器
