# 时间片轮转真题汇总

## 2024年第30题

**元数据**
- 年份：2024年第30题
- 科目：#操作系统
- 知识点：#操作系统/进程管理/时间片轮转
- 难度：⭐⭐⭐

**题干**

假设某系统使用时间片轮转调度算法进行CPU调度，时间片大小为5ms，系统共有10个进程，初始时均处于就绪队列，执行结束前仅处于执行态或就绪态。若队尾的进程P所需CPU时间最短，时间为25ms，在不考虑系统开销的情况下，则进程P的周转时间为（）。

A. 200 ms  B. 205 ms  C. 250 ms  D. 295 ms

**答案**：C

**考点分析**
- P执行5次，每次间隔45ms
- 周转时间：5+45+5+45+5+45+5+45+5+45+5=250ms

---


## 解题思路

### 第一步：分析时间片轮转调度
```
10个进程，时间片5ms
P在队尾，需要25ms（5个时间片）
```

### 第二步：模拟执行过程
```
P需要执行5次，每次5ms
其他9个进程轮流执行

轮次1：其他9个进程各执行5ms = 45ms
       P执行5ms（第1个时间片）
轮次2：其他9个进程各执行5ms = 45ms
       P执行5ms（第2个时间片）
轮次3：其他9个进程各执行5ms = 45ms
       P执行5ms（第3个时间片）
轮次4：其他9个进程各执行5ms = 45ms
       P执行5ms（第4个时间片）
轮次5：其他9个进程各执行5ms = 45ms
       P执行5ms完成（第5个时间片）

总时间 = 45×5 + 5×5 = 225 + 25 = 250ms
```

### 第三步：计算周转时间
```
周转时间 = 完成时刻 - 到达时刻
P到达时刻 = 0
P完成时刻 = 250ms
周转时间 = 250 - 0 = 250ms
```

### 第四步：确认答案
周转时间250ms → 答案：**C**

## 易错点分析

### 易错点1：忽略其他进程执行时间
❌ 错误：P需要25ms，周转时间=25ms
✅ 正确：P每次执行前其他进程先执行

### 易错点2：计算轮次错误
| 轮次 | 其他进程 | P执行 | 累计时间 |
|------|---------|-------|---------|
| 1 | 45ms | 5ms | 50ms |
| 2 | 45ms | 5ms | 100ms |
| 3 | 45ms | 5ms | 150ms |
| 4 | 45ms | 5ms | 200ms |
| 5 | 45ms | 5ms | 250ms |

**时间片轮转计算**：
```
进程数：n
时间片：q
某进程需要时间：T
执行轮数：⌈T/q⌉

周转时间 = (n-1)×q×轮数 + T
         = 9×5×5 + 25
         = 225 + 25
         = 250ms
```

**关键点**：
- P在队尾，每轮最后执行
- 每轮前其他9个进程先执行
- 总共5轮
