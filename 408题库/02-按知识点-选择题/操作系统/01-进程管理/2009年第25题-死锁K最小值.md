**题干**

某计算机系统中有 8 台打印机, 由 K 个进程竞争使用, 每个进程最多需要 3 台打印机。该系统可能会发生死锁的 K 的最小值是 ( )。A. 2  B. 3  C. 4  D. 5

**答案**：C

**考点分析**
- K≥8/3+1=4时可能死锁

---


## 解题思路

### 第一步：理解死锁条件
死锁最坏情况：每个进程都分到(max-1)个资源，都差1个

### 第二步：推导公式
```
总资源：R=8
每进程最多需要：max=3
进程数：K

最坏情况：K个进程各持有2台
K×2 = 8（所有资源用完）
K = 4

此时每个进程都差1台，陷入死锁
```

### 第三步：逐项分析
**A. K=2 ❌**
```
2个进程各持有2台，还剩4台
有空闲资源，不会死锁
结论：不会死锁
```

**B. K=3 ❌**
```
3个进程各持有2台，还剩2台
有空闲资源，不会死锁
结论：不会死锁
```

**C. K=4 ✅**
```
4个进程各持有2台，刚好用完8台
每个进程都差1台打印机
无空闲资源，发生死锁
结论：可能死锁
```

**D. K=5 ✅但不是最小**
```
5个进程，资源不够每人2台
更容易死锁，但不是最小值
结论：会死锁但不是最小
```

### 第四步：确认答案
K的最小值是4 → 答案：**C**

## 易错点分析

### 易错点1：用错公式
❌ 错误：总资源/每进程需求 = 8/3 ≈ 3
✅ 正确：K×(max-1) ≥ R，即 K ≥ R/(max-1)+1 = 8/2+1 = 5

**注意**：题目问"可能死锁"，不是"一定死锁"
最坏情况：K×(max-1) = R
K = R/(max-1) = 8/2 = 4

### 易错点2：不理解最坏情况
| 进程数K | 最坏分配 | 剩余 | 是否死锁 |
|---------|---------|------|----------|
| 2 | 2×2=4 | 4 | ❌ 有空闲 |
| 3 | 3×2=6 | 2 | ❌ 有空闲 |
| 4 | 4×2=8 | 0 | ✅ 死锁 |
| 5 | 5×2=10>8 | - | ✅ 更易死锁 |

**死锁公式**：
- 不死锁：K ≤ R/(max-1)
- 可能死锁：K ≥ R/(max-1)+1
- 本题：K ≥ 8/2+1 = 5，但4就可能死锁了
