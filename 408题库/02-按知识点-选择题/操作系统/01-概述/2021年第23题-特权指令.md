**题干**

下列指令中，只能在内核态执行的是（）。

A. trap 指令  B. I/O 指令  C. 数据传送指令  D. 设置断点指令

**答案**：B

**考点分析**
- B正确：I/O指令是特权指令，只能在内核态执行
- trap、数据传送、设置断点都可在用户态执行

---


## 解题思路

### 第一步：区分特权指令和非特权指令
特权指令：只能内核态执行（I/O、关中断等）
非特权指令：用户态可执行（算术、trap等）

### 第二步：逐项分析
**A. trap指令 ❌**
```
trap是陷入指令
用户态执行，触发软中断
用于请求系统调用
结论：用户态可执行
```

**B. I/O指令 ✅**
```
I/O指令直接操作硬件
必须在内核态执行
用户态执行会触发异常
结论：只能内核态执行
```

**C. 数据传送指令 ❌**
```
mov等数据传送指令
访问用户空间内存
是基本指令
结论：用户态可执行
```

**D. 设置断点指令 ❌**
```
调试器设置断点
用户态可执行
用于程序调试
结论：用户态可执行
```

### 第三步：确认答案
只有I/O指令必须内核态 → 答案：**B**

## 易错点分析

### 易错点1：认为trap是特权指令
❌ 错误：trap陷入内核，必须是特权指令
✅ 正确：trap在用户态执行，用于请求OS服务

### 易错点2：不理解特权指令
| 指令类型 | 用户态 | 示例 |
|----------|--------|------|
| 特权指令 | ❌ | I/O、关中断、修改PSW |
| 陷入指令 | ✅ | trap、int、syscall |
| 普通指令 | ✅ | 算术、逻辑、传送 |
