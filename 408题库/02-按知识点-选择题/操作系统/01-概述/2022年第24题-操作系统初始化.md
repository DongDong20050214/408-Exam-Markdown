**题干**

下列选项中, 需要在操作系统进行初始化过程中创建的是 ( )。

A. 中断向量表  B. 文件系统的根目录  C. 硬盘分区表  D. 文件系统的索引节点表

**答案**：A

**考点分析**
- A正确：中断向量表由OS初始化时创建
- C：硬盘分区表在硬盘格式化时创建
- BD：文件系统结构在文件系统挂载时创建

---


## 解题思路

### 第一步：理解OS初始化时机
OS启动时创建基本数据结构

### 第二步：逐项分析
**A. 中断向量表 ✅**
```
存储中断服务程序入口地址
OS初始化时建立
填入各中断的处理程序
结论：OS初始化创建
```

**B. 文件系统根目录 ❌**
```
根目录在磁盘上
文件系统挂载时读取
不是OS初始化创建
结论：挂载时创建
```

**C. 硬盘分区表 ❌**
```
分区表在磁盘上
硬盘格式化时创建
OS启动前就存在
结论：格式化时创建
```

**D. 索引节点表 ❌**
```
i-node表在磁盘上
文件系统创建时生成
挂载时读入内存
结论：文件系统创建时
```

### 第三步：确认答案
只有中断向量表在OS初始化时创建 → 答案：**A**

## 易错点分析

### 易错点1：混淆创建时机
❌ 错误：OS启动要访问文件，必须先创建文件系统
✅ 正确：文件系统在磁盘上，OS启动时挂载

### 易错点2：不理解各结构位置
| 结构 | 位置 | 创建时机 |
|------|------|----------|
| 中断向量表 | 内存 | OS初始化 |
| 分区表 | 磁盘 | 格式化磁盘 |
| 根目录 | 磁盘 | 创建文件系统 |
| i-node表 | 磁盘 | 创建文件系统 |
