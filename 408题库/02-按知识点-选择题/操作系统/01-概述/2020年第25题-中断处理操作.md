# 中断处理真题汇总

## 2020年第25题

**元数据**
- 年份：[2020年第25题](../../../01-真题原卷/2020年408真题.md#25)
- 科目：#操作系统
- 知识点：#操作系统/概述/中断处理
- 难度：⭐⭐
- 关联知识：[中断处理](../../../../AI-知识点原子库/中断处理.md)

**题干**

下列与中断相关的操作中，由操作系统完成的是（）。

I. 保存被中断程序的中断点  II. 提供中断服务  III. 初始化中断向量表  IV. 保存中断屏蔽字

A. 仅 I、II  B. 仅 I、II、IV  C. 仅 III、IV  D. 仅II、III、IV

**答案**：D

**考点分析**
- 操作系统完成：提供中断服务、初始化中断向量表、保存中断屏蔽字
- 保存中断点由硬件完成

---


## 解题思路

### 第一步：区分硬件和OS的中断任务
硬件：保存PC/PSW（自动）
OS：中断服务、初始化、保存寄存器

### 第二步：逐项分析
**I. 保存中断点 ❌**
```
中断点（PC）由硬件自动保存
中断发生时硬件立即将PC压栈
不需要OS参与
结论：硬件完成
```

**II. 提供中断服务 ✅**
```
中断服务程序由OS编写
处理具体的中断请求
如I/O完成、时钟中断等
结论：OS完成
```

**III. 初始化中断向量表 ✅**
```
中断向量表存储中断服务程序地址
由OS在启动时初始化
填入各中断的处理程序入口
结论：OS完成
```

**IV. 保存中断屏蔽字 ✅**
```
中断屏蔽字在PSW中
硬件保存PSW，但OS需要进一步保存
为中断嵌套做准备
结论：OS完成
```

### 第三步：确认答案
II、III、IV由OS完成 → 答案：**D**

## 易错点分析

### 易错点1：认为保存PC是OS的工作
❌ 错误：中断保存现场都是OS完成的
✅ 正确：PC和PSW由硬件自动保存，OS保存其他寄存器

### 易错点2：混淆硬件和OS的分工
| 操作 | 完成者 | 时机 |
|------|--------|------|
| 保存PC | 硬件 | 中断发生瞬间 |
| 保存PSW | 硬件 | 中断发生瞬间 |
| 保存通用寄存器 | OS | 进入中断服务后 |
| 中断服务 | OS | 中断处理过程 |
| 初始化向量表 | OS | 系统启动时 |
