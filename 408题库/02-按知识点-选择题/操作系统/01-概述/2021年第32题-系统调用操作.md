**题干**

下列选项中，通过系统调用完成的操作是（）。

A. 页置换  B. 进程调度  C. 创建新进程  D. 生成随机整数

**答案**：C

**考点分析**
- C正确：创建进程需要fork等系统调用
- AB是内核内部操作，D是库函数

---


## 解题思路

### 第一步：理解系统调用
应用程序通过系统调用请求OS服务

### 第二步：逐项分析
**A. 页置换 ❌**
```
页置换由OS内核自动完成
发生缺页时内核处理
不需要应用程序调用
结论：内核内部操作
```

**B. 进程调度 ❌**
```
进程调度由调度程序完成
时间片用完或阻塞时触发
不通过系统调用
结论：内核内部操作
```

**C. 创建新进程 ✅**
```
fork()系统调用创建进程
应用程序主动请求
需要OS分配资源
结论：通过系统调用完成
```

**D. 生成随机整数 ❌**
```
rand()是C库函数
在用户态执行算法
不需要OS服务
结论：库函数
```

### 第三步：确认答案
只有创建进程通过系统调用 → 答案：**C**

## 易错点分析

### 易错点1：认为所有OS操作都是系统调用
❌ 错误：页置换、调度由OS完成，是系统调用
✅ 正确：这些是内核内部操作，不对外提供接口

### 易错点2：混淆系统调用和库函数
| 类型 | 执行态 | 示例 |
|------|--------|------|
| 系统调用 | 内核态 | fork、read、write |
| 库函数 | 用户态 | rand、printf、sin |
| 内核操作 | 内核态 | 调度、页置换 |
