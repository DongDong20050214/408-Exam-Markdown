**题干**

下列关于多道程序系统的叙述中，不正确的是（）。

A. 支持进程的并发执行
B. 不必支持虚拟存储管理
C. 需要实现对共享资源的管理
D. 进程数越多 CPU 利用率越高

**答案**：D

**考点分析**
- D错误：进程数过多导致频繁切换，利用率反而下降
- 多道程序可以不用虚拟内存

---


## 解题思路

### 第一步：理解多道程序特点
多道：多个程序并发，提高资源利用率

### 第二步：逐项分析
**A. 支持进程并发执行 ✅**
```
多道程序的核心特征
多个进程交替执行
宏观并行，微观串行
结论：正确
```

**B. 不必支持虚拟存储 ✅**
```
多道程序可以用固定分区
不一定需要虚拟内存
虚拟存储是可选技术
结论：正确
```

**C. 需要管理共享资源 ✅**
```
多进程访问共享资源
需要同步和互斥机制
避免冲突和竞争
结论：正确
```

**D. 进程数越多利用率越高 ❌**
```
进程数适中时利用率最高
进程过多导致频繁切换
切换开销反而降低利用率
存在最优进程数
结论：错误
```

### 第三步：确认答案
D不正确 → 答案：**D**

## 易错点分析

### 易错点1：认为进程越多越好
❌ 错误：多道程序就是进程多，越多越好
✅ 正确：过多进程导致颠簸，利用率下降

### 易错点2：认为必须有虚拟内存
| 技术 | 是否必须 | 作用 |
|------|----------|------|
| 虚拟存储 | ❌ | 扩展内存空间 |
| 中断机制 | ✅ | 实现并发 |
| 进程调度 | ✅ | 切换进程 |
