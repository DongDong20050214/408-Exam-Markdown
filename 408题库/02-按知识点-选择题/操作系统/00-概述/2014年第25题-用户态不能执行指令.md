**题干**

下列指令中，不能在用户态执行的是（）。A. trap 指令  B. 跳转指令  C. 压栈指令  D. 关中断指令

**答案**：D

**考点分析**
- D：关中断是特权指令

---


## 解题思路

### 第一步：区分特权指令和非特权指令
特权指令：只能在内核态执行（关中断、修改PSW等）
非特权指令：用户态可执行（算术、跳转等）

### 第二步：逐项分析
**A. trap指令 ✅可以**
```
trap是陷入指令（软中断）
用户程序通过trap请求OS服务
在用户态执行trap，然后切换到内核态
结论：用户态可执行
```

**B. 跳转指令 ✅可以**
```
跳转指令是基本指令
改变PC值，控制程序流程
不涉及特权操作
结论：用户态可执行
```

**C. 压栈指令 ✅可以**
```
压栈操作访问用户栈
修改栈指针SP
是普通内存操作
结论：用户态可执行
```

**D. 关中断指令 ❌不可以**
```
关中断是特权指令
禁止CPU响应中断
只能在内核态执行
用户态执行会引发异常
结论：用户态不可执行
```

### 第三步：确认答案
只有关中断指令不能在用户态执行 → 答案：**D**

## 易错点分析

### 易错点1：认为trap是特权指令
❌ 错误：trap陷入内核，肯定是特权指令
✅ 正确：trap在用户态执行，用于请求OS服务

### 易错点2：混淆特权指令
| 指令类型 | 用户态 | 示例 |
|----------|--------|------|
| 特权指令 | ❌ | 关中断、修改PSW、I/O |
| 陷入指令 | ✅ | trap、int、syscall |
| 普通指令 | ✅ | 算术、跳转、访存 |
