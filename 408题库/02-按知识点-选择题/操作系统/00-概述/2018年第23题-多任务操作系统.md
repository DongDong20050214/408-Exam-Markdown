**题干**

下列关于多任务操作系统的叙述，正确的是（）。I. 具有并发和并行的特点  II. 需要实现对共享资源的保护  III. 需要运行在多 CPU 的硬件平台上

A. 仅I  B. 仅II  C. 仅I、II  D. I、II、III

**答案**：C

**考点分析**
- I、II正确，III错误（单CPU也可多任务）

---


## 解题思路

### 第一步：理解多任务系统
多任务：支持多个任务同时运行

### 第二步：逐项分析
**I. 具有并发和并行特点 ✅**
```
并发：宏观同时（单CPU时间片轮转）
并行：真正同时（多CPU同时执行）
多任务系统至少支持并发
结论：正确
```

**II. 需要保护共享资源 ✅**
```
多任务访问共享资源需保护
避免冲突和不一致
使用锁、信号量等机制
结论：正确
```

**III. 需要多CPU平台 ❌**
```
单CPU也可实现多任务
通过时间片轮转实现并发
多CPU只是提供并行能力
结论：错误
```

### 第三步：确认答案
I、II正确，III错误 → 答案：**C**

## 易错点分析

### 易错点1：认为多任务必须多CPU
❌ 错误：多任务需要多个CPU同时执行
✅ 正确：单CPU通过时间片轮转也能多任务

### 易错点2：混淆并发和并行
| 概念 | 含义 | CPU要求 |
|------|------|---------|
| 并发 | 交替执行 | 单CPU即可 |
| 并行 | 同时执行 | 需要多CPU |
| 多任务 | 并发或并行 | 单CPU或多CPU |
