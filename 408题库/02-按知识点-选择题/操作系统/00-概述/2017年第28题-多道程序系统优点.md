**题干**

与单道程序系统相比，多道程序系统的优点是（）。I. CPU 利用率高  II. 系统开销小  III. 系统吞吐量大  IV. I/O 设备利用率高

A. 仅I、III  B. 仅 I、IV  C. 仅Ⅱ、Ⅲ  D. 仅 I、III、IV

**答案**：D

**考点分析**
- I、III、IV正确，II错误（系统开销更大）

---


## 解题思路

### 第一步：理解多道程序特点
多道：内存中多个程序，并发执行

### 第二步：逐项分析
**I. CPU利用率高 ✅**
```
单道：CPU等待I/O时空闲
多道：CPU可切换执行其他程序
CPU利用率显著提高
结论：正确
```

**II. 系统开销小 ❌**
```
多道需要：进程调度、内存管理、保护机制
系统开销比单道更大
代价换取效率
结论：错误
```

**III. 系统吞吐量大 ✅**
```
单位时间完成更多作业
CPU和I/O并行工作
总体效率提高
结论：正确
```

**IV. I/O设备利用率高 ✅**
```
单道：I/O结束后设备空闲
多道：其他程序可使用I/O设备
I/O设备利用率提高
结论：正确
```

### 第三步：确认答案
I、III、IV正确，II错误 → 答案：**D**

## 易错点分析

### 易错点1：认为多道系统开销小
❌ 错误：多道减少了空闲，开销应该更小
✅ 正确：多道需要调度、保护等机制，开销更大

### 易错点2：混淆效率和开销
| 对比项 | 单道 | 多道 |
|--------|------|------|
| CPU利用率 | 低 | 高✅ |
| 系统开销 | 小 | 大❌ |
| 吞吐量 | 低 | 高✅ |
| I/O利用率 | 低 | 高✅ |
