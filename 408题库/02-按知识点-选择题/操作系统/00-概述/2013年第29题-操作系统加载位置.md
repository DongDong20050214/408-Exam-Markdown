**题干**

计算机开机后，操作系统最终被加载到（）。A. BIOS  B. ROM  C. EPROM  D. RAM

**答案**：D

**考点分析**
- 操作系统加载到RAM运行

---


## 解题思路

### 第一步：理解计算机启动过程
BIOS(ROM) → 引导程序 → OS加载到RAM → OS运行

### 第二步：逐项分析
**A. BIOS ❌**
```
BIOS是基本输入输出系统
存储在ROM中
负责引导，不是OS加载位置
结论：错误
```

**B. ROM ❌**
```
ROM是只读存储器
断电不丢失，但不可写
OS需要频繁读写，不能在ROM运行
结论：错误
```

**C. EPROM ❌**
```
EPROM是可擦除只读存储器
写入速度慢，容量小
不适合运行OS
结论：错误
```

**D. RAM ✅**
```
RAM是随机存取存储器（内存）
读写速度快
OS加载到RAM后才能运行
结论：正确
```

### 第三步：确认答案
OS必须加载到RAM运行 → 答案：**D**

## 易错点分析

### 易错点1：混淆存储和运行位置
❌ 错误：BIOS在ROM中，OS也应该在ROM
✅ 正确：
- ROM：存储BIOS等固定程序
- RAM：运行OS和应用程序
- OS从磁盘加载到RAM运行

### 易错点2：不理解各种存储器特点
| 存储器 | 可写 | 速度 | 断电 | 用途 |
|--------|------|------|------|------|
| RAM | ✅ | 快 | 丢失 | 运行程序 |
| ROM | ❌ | 慢 | 保留 | 固定程序 |
| EPROM | 紫外线擦除 | 慢 | 保留 | 固件 |
