**题干**

单处理机系统中，可并行的是（）。I进程与进程  II处理机与设备  III处理机与通道  IV 设备与设备

A. I、II和III  B. I、II 和 IV

**答案**：需查看完整选项

**考点分析**
- I、II、III、IV都可能并行

---


## 解题思路

### 第一步：理解并行概念
并行：多个活动同时进行。单处理机系统中，CPU同一时刻只能执行一个进程，但可以与其他部件并行工作。

### 第二步：逐项分析
**I. 进程与进程 ❌**
```
单处理机：CPU同一时刻只能执行一个进程
结论：进程间是并发（交替执行），不是并行
```

**II. 处理机与设备 ✅**
```
CPU执行进程时，I/O设备可同时工作
例：CPU计算的同时，磁盘在读写数据
结论：可以并行
```

**III. 处理机与通道 ✅**
```
通道是独立于CPU的I/O处理器
CPU执行指令时，通道可独立传输数据
结论：可以并行
```

**IV. 设备与设备 ✅**
```
多个I/O设备可同时工作
例：打印机打印同时磁盘读取
结论：可以并行
```

### 第三步：确认答案
II、III、IV可并行，I不能并行 → 答案：**B**

## 易错点分析

### 易错点1：混淆并发和并行
❌ 错误理解：单处理机系统中进程可以并行
✅ 正确理解：
- 并发：多个进程交替执行（宏观同时，微观交替）
- 并行：真正同时执行（需要多个处理单元）
- 单处理机中进程只能并发，不能并行

### 易错点2：忽视I/O设备的独立性
| 概念 | 是否独立 | 能否与CPU并行 |
|------|----------|--------------|
| 进程 | 依赖CPU | ❌ 不能 |
| I/O设备 | 独立硬件 | ✅ 可以 |
| 通道 | 独立处理器 | ✅ 可以 |
