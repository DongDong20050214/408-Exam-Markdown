**题干**

中断处理和子程序调用都需要压栈保护现场，中断处理一定会保存而子程序调用不需要保存其内容的是（）。

A. 程序计数器  
B. 程序状态字寄存器  
C. 通用数据寄存器  
D. 通用地址寄存器

**答案**：B

**考点分析**
- A都需要：保存返回地址
- B中断需要：PSW包含中断屏蔽位等状态，子程序调用不涉及
- C都可能：根据需要保存
- D都可能：根据需要保存

---


## 解题思路

### 第一步：理解题目关键
找出"中断一定保存，子程序不需要"的内容

### 第二步：逐项分析
**A. 程序计数器PC ❌**
```
中断：需保存断点地址（返回地址）
子程序调用：需保存返回地址
结论：都需要保存
```

**B. 程序状态字PSW ✅**
```
PSW包含：中断屏蔽位、条件码、CPU状态等
中断：必须保存PSW（保护CPU状态、中断屏蔽位）
子程序：不需保存PSW（不涉及CPU状态切换）
结论：中断需要，子程序不需要
```

**C. 通用数据寄存器 ❌**
```
中断：根据需要保存
子程序：根据需要保存（遵循调用约定）
结论：都是根据需要，不是一定
```

**D. 通用地址寄存器 ❌**
```
中断：根据需要保存
子程序：根据需要保存
结论：都是根据需要，不是一定
```

### 第三步：确认答案
只有PSW是中断一定保存，子程序不需要 → 答案：**B**

## 易错点分析

### 易错点1：认为PC只有中断才保存
❌ 错误：子程序调用不需要保存返回地址
✅ 正确：子程序调用也要保存返回地址（在栈中）

### 易错点2：不理解PSW的作用
| 内容 | 中断是否保存 | 子程序是否保存 | 原因 |
|------|------------|--------------|------|
| PC | ✅ | ✅ | 都需返回地址 |
| PSW | ✅ | ❌ | 中断需保护CPU状态 |
| 通用寄存器 | 按需 | 按需 | 根据使用情况 |

**PSW为什么中断必须保存**：
- 中断屏蔽位：防止中断嵌套错误
- 条件码：恢复程序执行状态
- CPU模式位：用户态/内核态切换
