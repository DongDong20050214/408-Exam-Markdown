# 内核态用户态切换真题汇总

## 2015年第24题

**元数据**
- 年份：[[../../01-真题原卷/2015年408真题#24|2015年第24题]]
- 科目：#操作系统
- 知识点：#操作系统/概述/内核态用户态
- 难度：⭐⭐
- 关联知识：[[2023年第26题-态转换]]

**题干**

假定下列指令已装入指令寄存器，则执行时不可能导致CPU从用户态变为内核态的是（）。

A. DIV R0, R1  
B. INT n  
C. NOT R0  
D. （缺失选项）

**答案**：C

**考点分析**
- A可能：除法可能触发除0异常，进入内核态
- B一定：INT指令产生软中断，切换到内核态
- C不可能：NOT是普通算术运算，不触发异常或中断
- 用户态→内核态：中断、异常、系统调用

---


## 解题思路

### 第一步：判断能否切换态
能切换：中断、异常、系统调用
不能切换：普通算术/逻辑指令

### 第二步：逐项分析
**A. DIV R0,R1 ✅可能**
```
除法指令
可能触发除0异常
异常处理需切换到内核态
结论：可能切换
```

**B. INT n ✅一定**
```
软中断指令
主动产生中断
立即切换到内核态执行中断处理
结论：一定切换
```

**C. NOT R0 ❌不可能**
```
按位取反指令
普通逻辑运算
不触发任何异常或中断
始终在用户态执行
结论：不可能切换
```

### 第三步：确认答案
只有NOT不可能切换态 → 答案：**C**

## 易错点分析

### 易错点1：认为除法指令一定切换
❌ 错误：除法运算不需要OS，不切换态
✅ 正确：除法可能除0异常，异常时才切换

### 易错点2：混淆指令类型
| 指令 | 能否切换 | 条件 |
|------|----------|------|
| 算术指令 | 可能 | 触发异常时（除0、溢出） |
| 中断指令 | 一定 | 主动产生中断 |
| 逻辑指令 | 不能 | 无异常条件 |
| 访存指令 | 可能 | 缺页异常 |
