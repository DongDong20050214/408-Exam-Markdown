# 2019年408真题第10题

## 题目

排序过程中，对尚未确定最终位置的所有元素进行一遍处理称为一"趟"。下列序列中，不可能是快速排序第二趟结果的是（）。

A. 5, 2, 16, 12, 28, 60, 32, 72  
B. 2, 16, 5, 28, 12, 60, 32, 72  
C. 2, 12, 16, 5, 28, 32, 72, 60  
D. 5, 2, 12, 28, 16, 32, 72, 60

## 答案

C

## 考点分析

### 考点：快速排序的过程特征

本题考查快速排序的分区特征和排序过程的性质。

### 核心知识点：

**快速排序的特点**：
1. 每一趟选择一个基准元素（pivot）
2. 将序列分为两部分：小于基准的在左，大于基准的在右
3. 基准元素放在最终位置（左右两边的元素可以无序）
4. 递归处理左右两个子序列

**关键性质**：
- **第一趟**：至少有1个元素（基准）确定最终位置
- **第二趟**：至少有2个元素确定最终位置（第一趟的基准 + 第二趟的基准）
- 确定位置的元素：左边所有元素 < 它 < 右边所有元素

### 逐项分析：

**原始序列推测**（根据选项反推，可能是）：5, 2, 16, 12, 28, 60, 32, 72

**A. 5, 2, 16, 12, 28, 60, 32, 72（✓ 可能）**

分析：
- 第一趟：选5作为基准，分为 [2] 5 [16, 12, 28, 60, 32, 72]
- 第二趟：对右边选16作为基准，分为 [2] 5 [12] 16 [28, 60, 32, 72]
- 或者：[5, 2] [16, 12, 28, 60, 32, 72] 都没有完全处理
- 可能性：✓

**B. 2, 16, 5, 28, 12, 60, 32, 72（✓ 可能）**

分析：
- 第一趟：选28作为基准，分为 [2, 16, 5, 12] 28 [60, 32, 72]
- 第二趟：对左边处理但未完成分区
- 可能性：✓

**C. 2, 12, 16, 5, 28, 32, 72, 60（✗ 不可能）**

分析：
- 观察28：左边是 [2, 12, 16, 5]，右边是 [32, 72, 60]
- 如果28在最终位置，左边应该都 < 28，右边都 > 28（✓ 满足）
- 观察16：左边是 [2, 12]，右边是 [5]
- **问题**：5 < 16，但5在16的右边！
- 这违反了快速排序的分区性质
- **结论**：不可能

**为什么C不可能？**
- 如果16已经确定位置，那么16左边的所有元素应该 < 16，右边的所有元素应该 > 16
- 但5 < 16，却出现在16的右边
- 这与快速排序的分区原则矛盾

**D. 5, 2, 12, 28, 16, 32, 72, 60（✓ 可能）**

分析：
- 第一趟：选28作为基准，分为 [5, 2, 12, 16] 28 [32, 72, 60]
- 第二趟：左边或右边某个继续分区但未完成
- 可能性：✓

### 快速排序的判定规则：

对于第k趟排序后的序列，检查是否满足：
1. 至少有k个元素在最终位置
2. 对于在最终位置的元素x：
   - x左边的所有元素 < x
   - x右边的所有元素 > x
3. 不在最终位置的元素可以无序，但应该在正确的"大致区域"

### 记忆要点：

- 快速排序的核心：**分区**
- 基准左边 < 基准 < 基准右边
- 第k趟至少k个元素就位
- 逆向检查：看是否存在"小元素在大元素右边"的违规情况

### 易错点：

- 认为快速排序每趟都完全有序（实际只有基准就位）
- 忽略分区的严格性（左小右大）
- 混淆"部分有序"和"分区正确"

## 解题思路

### 判定方法：检查分区一致性

快速排序第二趟后，至少有2个元素在最终位置，满足"左边所有元素<它<右边所有元素"。

**检查C选项中的矛盾**：
序列：2, 12, 16, 5, 28, 32, 72, 60

观察16的位置（下标2）：
- 左边：2, 12（都<16）✓
- 右边：5, 28, 32, 72, 60
- **问题**：5<16，但5在16右边！

这违反了快速排序的核心性质：如果16已经在最终位置，它右边的所有元素都应该≥16。

**其他选项验证**：
- A选项：5可能是枢轴，左边2<5，右边都>5 ✓
- B选项：28可能是枢轴，左右分区合理 ✓
- D选项：28可能是枢轴，分区合理 ✓

## 易错点分析

### 易错点1：认为快速排序每趟后子序列内部有序
❌ 错误理解：快速排序每做完一趟，划分出的子序列应该内部有序或接近有序。看到某个序列左边部分没有完全排好，就认为不可能是快速排序的中间结果。

✅ 正确理解：快速排序每趟只保证枢轴就位，左右子序列内部仍无序。例如第二趟后，有2个枢轴到达最终位置，其余元素仍可能乱序。判断标准是已就位的枢轴满足"左边所有元素≤它，右边所有元素≥它"，而非整体有序度。快速排序的核心是分治+划分，不是逐步排序。只有当所有元素都成为过枢轴（递归到底）才完全有序。中间阶段可能出现[3,9,5,2,6,8,11,15]这样局部无序但符合划分性质（6是枢轴，左边都≤6，右边都≥6）。记忆：快排看枢轴就位，不看局部有序。

### 易错点2：混淆"已就位元素"和"部分有序"
❌ 错误理解：看序列中某段连续递增就认为这些元素已就位。例如看到[2,12,16]连续递增，认为这三个元素都已确定最终位置。

✅ 正确理解：元素就位的唯一标准是"左边所有元素≤它≤右边所有元素"，而非局部有序。连续递增的元素不一定都已就位，可能只有其中某个是枢轴。C选项中16左边是[2,12]都≤16，但右边的5<16却出现在右侧，违反了已就位的定义。判断方法：对每个候选枢轴，检查其左侧最大值和右侧最小值是否满足max(左侧)≤枢轴≤min(右侧)。本题C选项16的问题：16右侧最小值是5，5<16，说明16不在最终位置（若在最终位置则右侧应全≥16）。快排判定法则：已就位元素必须满足全局分割性质，局部有序不等于就位。
