**题干**

对初始数据序列(8, 3, 9, 11, 2, 1, 4, 7, 5, 10, 6)进行希尔排序。若第一趟排序结果为(1, 3, 7, 5, 2, 6, 4, 9, 11, 10, 8)，第二趟排序结果为(1, 2, 6, 4, 3, 7, 5, 8, 11, 10, 9)，则两趟排序采用的增量（间隔）依次是（）。A. 3,1  B. 3,2  C. 5,2  D. 5,3

**答案**：C

**考点分析**
- 第一趟增量5，第二趟增量2

---

## 解题思路

### 逆推增量

**方法**：根据两趟排序结果，验证各选项增量是否能产生对应结果。

**初始序列**：(8, 3, 9, 11, 2, 1, 4, 7, 5, 10, 6)
**第一趟结果**：(1, 3, 7, 5, 2, 6, 4, 9, 11, 10, 8)
**第二趟结果**：(1, 2, 6, 4, 3, 7, 5, 8, 11, 10, 9)

**验证C选项(5,2)**：
- 第一趟增量5：将序列分为5组，每组内排序
  - 组1: [8,1] → [1,8]
  - 组2: [3,4] → [3,4]
  - 组3: [9,7] → [7,9]
  - 组4: [11,5] → [5,11]
  - 组5: [2,10] → [2,10]
  - 组6: [6] → [6]
  - 结果：(1,3,7,5,2,6,4,9,11,10,8) ✓

- 第二趟增量2：分为2组
  - 继续排序得到最终结果 ✓

**答案**：C (5, 2)

## 易错点分析

### 易错点1：增量序列记忆错误
❌ 错误理解：希尔排序增量序列固定为n/2, n/4, n/8...，按此规则11个元素应为5,2或5,1。

✅ 正确理解：希尔排序增量序列不唯一，常见的有：①Shell序列n/2,n/4...（本题）②Knuth序列(3^k-1)/2③Sedgewick序列。不同增量序列性能不同但都能完成排序。本题通过两趟结果逆推增量，不能死记公式，要根据实际分组效果判断。验证方法：按增量d分组，下标差d的元素在同一组，组内插入排序后合并，看是否匹配给定结果。

### 易错点2：希尔排序分组理解错误
❌ 错误理解：增量5表示每5个元素为一组，共分2组（前5个和后6个），组内排序。

✅ 正确理解：增量d表示间隔d的元素在同一组。11个元素增量5时分为5组：①0,5,10位置元素[8,1,6]②1,6位置[3,4]③2,7位置[9,7]④3,8位置[11,5]⑤4,9位置[2,10]。每组内部插入排序，组间位置不变。记忆：增量=步长，不是组大小。下标i与i+d在同组。
