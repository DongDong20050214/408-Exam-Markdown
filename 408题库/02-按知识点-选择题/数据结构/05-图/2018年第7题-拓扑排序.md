# 2018年第7题

**元数据**
- 年份：2018年第7题
- 难度：⭐⭐

**题干**：下列选项中, 不是题 07 图中有向图的拓扑序列的是 ( )。

A. 1, 5, 2, 3, 6, 4 
B. 5, 1, 2, 6, 3, 4 
C. 5, 1, 2, 3, 6, 4 
D. 5,2,1,6,3,4 

**答案**：D

**考点分析**：本题考查拓扑排序。拓扑序列必须满足：若有向边<u,v>，则u必须在v之前出现。根据题图的边关系，需要检查每个选项是否违反了某条有向边的约束。选项D中某个结点的排序位置违反了其前驱结点的约束条件，因此不是合法的拓扑序列。

## 解题思路

### 第一步：理解题目核心
拓扑序列：若有边<u,v>，则u必须在v前

### 第二步：逐项分析
（本题需查看题图的边关系，无法从文本直接判断）

**拓扑排序验证方法**：
- 检查每条有向边<u,v>，u是否在v之前出现
- 违反任意一条边的约束即不是拓扑序列

根据标准答案D，选项D违反了某条边的约束。

**A. 1,5,2,3,6,4** ✅ 可能
**B. 5,1,2,6,3,4** ✅ 可能
**C. 5,1,2,3,6,4** ✅ 可能
**D. 5,2,1,6,3,4** ❌ 违反约束

### 第三步：确认答案
D

## 易错点分析

### 易错点1：拓扑序列的验证
❌ 错误理解：只看数字顺序是否合理，忽略必须检查所有有向边的约束。
✅ 正确理解：拓扑排序的核心是有向无环图的线性化。验证方法：对于图中每条有向边<u,v>，在序列中u必须出现在v之前。若存在边<1,2>，则1必须在2前面；若序列中2在1前面，则违反约束，不是合法拓扑序列。需要逐一检查图中所有边。

### 易错点2：拓扑排序特点
| 特点 | 说明 |
|-------|------|
| 唯一性 | 拓扑序列可能不唯一 |
| 前提 | 必须是有向无环图(DAG) |
| 验证 | 检查所有边的顺序约束 |
