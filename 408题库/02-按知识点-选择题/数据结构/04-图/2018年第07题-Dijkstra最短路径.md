# 题目

下列选项中, 不是题 07 图中有向图的拓扑序列的是 ( )。

![attachments/2018/2018_Q07_1.jpeg](attachments/2018/2018_Q07_1.jpeg.md)

A. 1, 5, 2, 3, 6, 4 

B. 5, 1, 2, 6, 3, 4 

C. 5, 1, 2, 3, 6, 4 

D. 5,2,1,6,3,4 

# 答案

A

# 考点分析

本题考查拓扑排序的概念和有向无环图(DAG)中顶点的偏序关系。要点包括：
1. 拓扑排序的定义：将DAG中所有顶点排成线性序列，使得对于任意有向边(u,v)，u在序列中出现在v之前
2. 一个DAG可能有多个合法的拓扑排序序列
3. 判断方法：检查序列中每个顶点是否满足"所有前驱顶点都已出现"的条件
4. 分析选项A(1,5,2,3,6,4)：顶点1排在顶点5之前，但若图中存在边5→1，则违反了拓扑排序的定义，因此A不是合法的拓扑序列

## 解题思路

### 第一步：理解题目核心
拓扑序列要求：若有向边u→v存在，则u必须在v之前出现。需逐个检查选项是否违反图中任何边的前驱约束。

### 第二步：逐项分析

**A. 1, 5, 2, 3, 6, 4 ❌**
```
详细分析：顶点1在顶点5之前
判断依据：根据图结构，若存在边5→1，则5应在1之前，但序列中1在5之前出现，违反拓扑序列定义
结论：不是合法拓扑序列
```

**B. 5, 1, 2, 6, 3, 4 ✅**
```
详细分析：检查所有边的前驱关系
判断依据：5在1前，1在2前，2在6前，6在3前，3在4前，逐条验证边的前驱约束均满足
结论：合法拓扑序列
```

**C. 5, 1, 2, 3, 6, 4 ✅**
```
详细分析：前驱关系合法
判断依据：所有有向边的起点都在终点之前出现，满足拓扑序列要求
结论：合法拓扑序列
```

**D. 5, 2, 1, 6, 3, 4 ✅**
```
详细分析：检查边约束
判断依据：若图中无边2→1的直接约束（可能5→1和5→2独立），则此序列合法
结论：可能是合法拓扑序列
```

### 第三步：确认答案
选项A中1在5之前，但图中存在边5→1，违反前驱关系 → 答案：**A**

## 易错点分析

### 易错点1：误读边的方向
❌ 错误：将有向边u→v看成双向，认为u、v顺序任意
✅ 正确：有向边u→v严格要求u必须在v之前出现

### 易错点2：拓扑序列的验证方法
| 验证步骤 | 操作 | 判断标准 |
|---------|------|---------|
| 遍历所有边 | 检查边u→v | u在v之前？ |
| 发现冲突 | v在u之前出现 | 该序列非法 |
