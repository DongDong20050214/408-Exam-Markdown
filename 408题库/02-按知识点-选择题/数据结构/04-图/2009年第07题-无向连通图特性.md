# 图的性质真题汇总

## 2009年第07题

**元数据**
- 年份：[[../../01-真题原卷/2009年408真题#07|2009年第7题]]
- 科目：#数据结构
- 知识点：#数据结构/图/性质
- 难度：⭐⭐
- 关联知识：[[知识库/数据结构/图]]

**题干**

下列关于无向连通图特性的叙述中，正确的是（）。I. 所有顶点的度之和为偶数  II. 边数大于顶点个数减1  III. 至少有一个顶点的度为 1

A. 只有 I  B. 只有 II  C. I 和 II  D. I 和 III

**答案**：A

**考点分析**
- I正确：度之和=2×边数

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 图类型：**无向连通图**（注意：是"连通图"而非"完全图"，涵盖范围更广）
- 需判断三个命题 I、II、III 的正确性
- 选出正确命题的组合

### 第二步：回顾图的基本性质
1. **握手定理**：所有顶点的度之和 = 2 × 边数（对任意无向图恒成立）
2. **连通图**：任意两个顶点之间都有路径可达
3. **树的特性**：n个顶点的连通图，边数最少为 n-1（树恰好 n-1 条边）
4. **完全图**：n个顶点的连通图，边数最多为 n(n-1)/2，每个顶点度为 n-1

### 第三步：逐项判断

**I. 所有顶点的度之和为偶数 ✅**
```
依据握手定理：Σdeg(v) = 2|E|
2 × 边数 必然是偶数（2乘以任何整数都得偶数）
无论图的结构如何（连通与否、边数多少），该命题恒成立
结论：I 正确 ✅
```

**II. 边数大于顶点个数减1 ❌**
```
关键考点：连通图的边数范围
- 树：n个顶点，边数 = n-1（连通图的极端情况——边最少）
- 完全图：n个顶点，边数 = n(n-1)/2（边最多）

命题说"边数 > n-1"，这是绝对化断言
反例：任何一棵树都是连通图，但边数 = n-1，满足"等于"而非"大于"
例如 n=3 的树：边数=2，顶点数-1=2，2 > 2 不成立

连通图边数范围：[n-1, n(n-1)/2]，下界就是 n-1
命题"大于"排除了"等于"的情况，因此错误

结论：II 错误 ❌
```

**III. 至少有一个顶点的度为1 ❌**
```
关键考点：连通图中顶点的度无下限约束
反例：完全图 Kₙ（n ≥ 3）
- 每个顶点的度均为 n-1
- 当 n ≥ 3 时，所有顶点度 ≥ 2
- 没有任何顶点的度为 1

例如 K₄：4个顶点，每个度=3，均为 ≥2

连通性只保证"路径可达"，不保证存在叶子结点
度为1的顶点（叶子）仅在树、部分稀疏图中出现

结论：III 错误 ❌
```

### 第四步：确认答案
- I ✅　II ❌　III ❌
- 只有 I 正确
- 答案：**A**

## 易错点分析

### 陷阱1：混淆"连通图"与"树"，认为边数一定大于 n-1
❌ **错误理解**："连通"意味着有边连接，边数当然要多于 n-1
✅ **正确理解**：树就是连通图的极简形式（最小边数 = n-1）。连通图边数范围是 [n-1, n(n-1)/2]，下界 n-1 完全可能取到。命题 II 用"大于"一刀切，忽略了"等于"的情况

### 陷阱2：认为连通图中一定有度为1的"端点"
❌ **错误理解**：既然是连通的，总得有个"起点/终点"，那就一定有度为1的顶点
✅ **正确理解**：完全图每个顶点度均为 n-1（≥2），充分说明连通图可以没有任何度为1的顶点。度为1只出现在稀疏图（尤其是树）中，不是连通图的必要条件

### 陷阱3：对握手定理不够自信，被 I 选项的"技术性"吓住
❌ **错误心理**：I 涉及度之和的计算，看上去"有技术含量"，考生可能犹豫不决
✅ **正确理解**：握手定理是图论基石，度之和 = 2×边数，因此必为偶数。这是最可靠、无争议的真理

## 知识点对比

| 对比项 | 树（n个顶点） | 一般连通图 | 完全图 Kₙ |
|--------|-------------|-----------|----------|
| 边数 | = n-1 | n-1 ~ n(n-1)/2 | = n(n-1)/2 |
| 度之和 | 2(n-1)（偶数） | 2×边数（偶数） | n(n-1)（偶数） |
| 最小顶点度 | = 1（叶子） | ≥ 1 或无下限 | = n-1 |
| 是否为连通图 | ✅ 是 | ✅ 是 | ✅ 是 |
| 握手定理适用 | ✅ | ✅ | ✅ |

**记忆技巧**：连通图的三个极端——最少边是树（n-1 条），最多边是完全图（n(n-1)/2 条）。握手定理（度之和 = 2×边数）永远为真，因为每条边恰好贡献 2 度，2 乘以任何整数必得偶数。
