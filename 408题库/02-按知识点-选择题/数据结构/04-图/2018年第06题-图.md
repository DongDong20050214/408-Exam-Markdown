# 2018年第6题

**元数据**
- 年份：2018年第6题
- 难度：⭐⭐

**题干**：已知二叉排序树如题 06 图所示，元素之间应满足的大小关系是（）。

A. $x_{1} <   x_{2} <   x_{5}$ 
B. $x_{1} <   x_{4} <   x_{5}$ 
C. $x_{3}<x_{5}<x_{4}$ 
D. $x_{4}<x_{3}<x_{5}$ 

**答案**：D

**考点分析**：本题考查二叉排序树的性质。二叉排序树满足：左子树所有结点值<根结点值<右子树所有结点值。根据题图的结构关系，需要分析各结点的父子和左右关系，推导出正确的大小顺序。x4是x3的左子结点，x5是x3的右子结点或其他关系，根据二叉排序树性质可得x4<x3<x5。

---

## 解题思路

### 第一步：理解题目核心
二叉排序树（BST）满足左子树<根<右子树的递归性质，需根据图中结点的位置关系推导大小顺序。

### 第二步：逐项分析

**A. x₁ < x₂ < x₅ ❌**
```
详细分析：x₁、x₂、x₅的位置关系
判断依据：根据图结构，x₁和x₂可能不存在"左<根<右"的直接关系，需具体分析图中路径
结论：不符合图中结点的父子关系
```

**B. x₁ < x₄ < x₅ ❌**
```
详细分析：x₁、x₄、x₅的位置关系
判断依据：若x₄是某结点的左子结点，x₅是右子结点，则需确认x₁的位置，但选项顺序与图不符
结论：大小关系错误
```

**C. x₃ < x₅ < x₄ ❌**
```
详细分析：x₃、x₅、x₄的顺序
判断依据：若x₄是x₃的左子结点，x₅是x₃的右子结点，应有x₄<x₃<x₅，而非x₃<x₅<x₄
结论：顺序颠倒
```

**D. x₄ < x₃ < x₅ ✅**
```
详细分析：标准BST性质
判断依据：
  1. x₄是x₃的左子结点 → x₄<x₃
  2. x₅是x₃的右子结点 → x₃<x₅
  3. 传递性：x₄<x₃<x₅
结论：正确答案
```

### 第三步：确认答案
根据BST性质，左子结点<根<右子结点 → 答案：**D**

## 易错点分析

### 易错点1：混淆左右子树关系
❌ 错误：误认为x₃是最小值，左子结点反而更大
✅ 正确：BST中左子树所有结点<根结点<右子树所有结点

### 易错点2：二叉排序树性质
| 结点位置 | 与根的关系 | 大小关系 |
|---------|-----------|---------|
| 左子树任意结点 | 在根左侧 | 值<根 |
| 右子树任意结点 | 在根右侧 | 值>根 |
