# 2021年408真题第07题

## 题目

给定如下有向图，该图的拓扑有序序列的个数是（）。

![2021_Q08_1.jpeg](2021_Q08_1.jpeg.md)

A. 1

B. 2

C. 3

D. 4

## 答案

C

## 考点分析

### 考点：有向无环图（DAG）的拓扑排序

本题考查对给定有向无环图进行拓扑排序，计算所有可能的拓扑有序序列个数。

### 解题思路：

1. **拓扑排序的定义**：
   - 对有向无环图的顶点进行线性排序
   - 若存在边 <u, v>，则 u 在序列中必须出现在 v 之前
   - 一个 DAG 可能有多个拓扑有序序列

2. **拓扑排序的方法**：
   - 每次选择一个入度为 0 的顶点输出
   - 删除该顶点及其出边
   - 重复上述过程直到所有顶点输出完毕

3. **计算拓扑序列个数**：
   - 根据图的结构（需查看图片），分析顶点之间的依赖关系
   - 在每一步中，可能有多个入度为 0 的顶点可选
   - 每种选择导致不同的拓扑序列
   - 通过枚举或分析得出所有可能的序列

4. **具体分析**：
   - 答案为 C (3个)，说明该图结构允许 3 种不同的拓扑排序方式
   - 通常是因为某些步骤中有 2-3 个入度为 0 的顶点可供选择

### 关键点：

- 拓扑排序只适用于有向无环图（DAG）
- 拓扑序列的个数取决于图的结构和顶点间的依赖关系
- 每一步选择入度为 0 的顶点时，若有多个选择，会产生不同序列
- 需要系统地枚举所有可能的选择路径

### 易错点：

- 遗漏某些合法的拓扑序列
- 计算入度时出错，导致选择顺序错误
- 没有考虑到某些步骤中可能有多个入度为 0 的顶点
- 混淆拓扑排序与其他图遍历算法（如 DFS、BFS）
