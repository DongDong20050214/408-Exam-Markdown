# 2020年第8题

**元数据**
- 年份：[[../../01-真题原卷/2020年408真题#08|2020年第8题]]
- 科目：#数据结构
- 知识点：#数据结构/图/关键路径
- 难度：⭐⭐⭐

**题干**

若使用 AOE 网估算工程进度，则下列叙述中正确的是（）。

A. 关键路径是从原点到汇点边数最多的一条路径

B. 关键路径是从原点到汇点路径长度最长的路径

C. 增加任一关键活动的时间不会延长工程的工期

D. 缩短任一关键活动的时间将会缩短工程的工期

**答案**：B

**考点分析**

本题考查AOE网（Activity On Edge Network）和关键路径的概念。

**AOE网基本概念**：
- 顶点表示事件（event）
- 边表示活动（activity），边上的权值表示活动持续时间
- 原点：入度为0的顶点（工程开始）
- 汇点：出度为0的顶点（工程结束）

**关键路径**：
- 从原点到汇点的所有路径中，路径长度（各边权值之和）最长的路径
- 关键路径上的活动称为关键活动
- 关键路径的长度决定了整个工程的最短完成时间（工期）

**选项分析**：
- A错误：不是边数最多，而是路径长度（权值和）最长
- B正确：关键路径定义即路径长度最长的路径
- C错误：增加关键活动的时间必然延长工期
- D错误：缩短关键活动时间不一定缩短工期，因为可能存在多条关键路径，缩短一条路径上的活动后，该路径可能不再是关键路径，工期由其他关键路径决定

**重要性质**：
- AOE网可能有多条关键路径
- 只有缩短所有关键路径上的共有活动，才能确保缩短工期

## 解题思路

### 第一步：理解题目核心
AOE网关键路径是从源点到汇点路径长度（边权值和）最长的路径，它决定工程最短完成时间，关键活动时间余量为0。

### 第二步：逐项分析

**A. 关键路径是从原点到汇点边数最多的一条路径 ❌**
```
详细分析：边数与路径长度的区别
判断依据：关键路径看的是路径长度（所有边权值之和），不是边的数量。例如：路径1有5条边权为1，总长5；路径2有2条边权为10，总长20，路径2是关键路径
结论：混淆了边数和路径长度
```

**B. 关键路径是从原点到汇点路径长度最长的路径 ✅**
```
详细分析：关键路径的定义
判断依据：
  1. 路径长度=该路径上所有活动持续时间之和
  2. 最长路径决定工程最短完成时间
  3. 这条最长路径即为关键路径
结论：正确答案
```

**C. 增加任一关键活动的时间不会延长工程的工期 ❌**
```
详细分析：关键活动的特性
判断依据：关键活动在关键路径上，时间余量为0，增加其时间直接增加关键路径长度，必然延长工期
结论：逻辑错误，与关键活动定义矛盾
```

**D. 缩短任一关键活动的时间将会缩短工程的工期 ❌**
```
详细分析：多条关键路径的情况
判断依据：若存在多条关键路径，缩短其中一条上的活动，其他关键路径仍为瓶颈，工期不变。只有缩短所有关键路径的共有活动才能缩短工期
结论："任一"过于绝对
```

### 第三步：确认答案
关键路径=路径长度（权值和）最长的路径 → 答案：**B**

## 易错点分析

### 易错点1：混淆边数与路径长度
❌ 错误：认为经过活动多的路径就是关键路径
✅ 正确：路径长度指权值总和（持续时间总和），与边数无关

### 易错点2：缩短关键活动与工期的关系
| 情况 | 缩短关键活动 | 工期变化 |
|------|------------|---------|
| 单条关键路径 | 任一关键活动 | 工期缩短 |
| 多条关键路径 | 仅某一条上的活动 | 工期可能不变 |
| 多条关键路径 | 所有路径的共有活动 | 工期缩短 |
