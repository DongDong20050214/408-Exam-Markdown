# 2019年408真题第06题

## 题目

用有向无环图描述表达式 $(x + y)((x + y) / x)$，需要的顶点个数至少是（）。

A. 5  
B. 6  
C. 8  
D. 9

## 答案

A

## 考点分析

### 考点：有向无环图（DAG）表示表达式

本题考查用DAG表示表达式时的**公共子表达式复用**。

### 核心思想：

**DAG的优势**：
- 可以共享公共子表达式
- 每个相同的子表达式只需要一个节点
- 比表达式树更节省空间

### 表达式分析：

**原表达式**：$(x + y)((x + y) / x)$

**展开**：$(x + y) \times ((x + y) / x)$

**识别操作和操作数**：
1. 叶节点（变量）：$x$, $y$
2. 中间结果：$x + y$（公共子表达式，出现2次）
3. 中间结果：$(x + y) / x$
4. 最终结果：$(x + y) \times ((x + y) / x)$

### 构造DAG：

**节点列表**（共5个）：
1. 节点 $x$（变量）
2. 节点 $y$（变量）
3. 节点 $x+y$（**复用**，被引用2次）
4. 节点 $(x+y)/x$（除法运算）
5. 节点 $(x+y) \times ((x+y)/x)$（乘法运算，根节点）

**DAG结构**：
```
        × (节点5，根)
       / \
      /   \
   x+y    / (节点4)
 (节点3) / \
    /\  x+y  x
   /  \(复用节点3)
  x    y
(节点1)(节点2)
```

### 关键点：

- **公共子表达式 $(x+y)$ 只需1个节点**
- 该节点被2个父节点引用（作为乘法左操作数，作为除法左操作数）
- 如果用表达式树表示（不复用），需要6个节点

### 对比：

**表达式树（不复用）**：
- $x$, $y$, $x+y$ (第1次)
- $x$ (第2次), $y$ (第2次), $x+y$ (第2次)
- $(x+y)/x$, 最终乘法
- 至少需要 **8个节点**

**DAG（复用）**：
- $x$, $y$（各1个，可以复用）
- $x+y$（1个，复用）
- $(x+y)/x$（1个）
- 最终乘法（1个）
- 只需 **5个节点**

### 记忆要点：

- DAG = 有向无环图 = 允许多个父节点（共享）
- 表达式树 = 每个节点只有1个父节点（不共享）
- DAG通过复用公共子表达式减少节点数
- 实际编译器优化中常用DAG表示

### 易错点：

- 忘记复用公共子表达式，按表达式树计算
- 忘记变量 $x$ 可以复用（出现2次）
- 混淆节点数和边数

## 解题思路

### 第一步：理解题目核心
DAG表示表达式可复用公共子表达式，表达式(x+y)((x+y)/x)中x+y出现2次但在DAG中仅需1个节点，需识别所有独立子表达式。

### 第二步：逐项分析

**A. 5 ✅**
```
详细分析：最少节点数=独立变量+独立运算
判断依据：
  节点清单：
  1. x（变量）
  2. y（变量）
  3. x+y（运算，复用）
  4. (x+y)/x（运算）
  5. 最终乘法（运算）
  总计：5个节点
结论：正确答案
```

**B. 6 ❌**
```
详细分析：多算1个节点
判断依据：可能误将x+y计数2次，或重复计数某个变量
结论：未完全复用公共子表达式
```

**C. 8 ❌**
```
详细分析：用表达式树部分复用
判断依据：可能复用了变量x、y，但x+y计算了2次，导致节点数为2+2+2+1+1=8
结论：未复用中间结果
```

**D. 9 ❌**
```
详细分析：完全不复用，用纯树结构
判断依据：x出现3次算3个节点，y出现2次算2个节点，x+y出现2次算2个节点，加上/和*共9个
结论：忽略DAG的核心特性
```

### 第三步：确认答案
DAG复用公共子表达式，5个独立节点：x、y、x+y、(x+y)/x、乘法 → 答案：**A**

## 易错点分析

### 易错点1：忘记复用公共子表达式
❌ 错误：认为x+y出现2次就需要2个节点
✅ 正确：DAG中相同子表达式只需1个节点，可被多个父节点引用

### 易错点2：DAG与树的节点复用对比
| 结构 | 公共子表达式处理 | 本题节点数 |
|------|----------------|-----------|
| 表达式树 | 不复用，重复存储 | 8-9个 |
| DAG | 复用，共享节点 | 5个 |
