# 2022年第6题

**元数据**
- 年份：[[../../01-真题原卷/2022年408真题#06|2022年第6题]]
- 科目：#数据结构
- 知识点：#数据结构/图/连通性判定
- 难度：⭐⭐⭐

**题干**

对于无向图 $G = (V, E)$，下列选项中，正确的是（）。

A. 当 $|V|>|E|$ 时，G一定是连通的

B. 当 $|V| < |E|$ 时，$G$ 一定是连通的

C. 当 $|V|=|E|-1$ 时，G一定是不连通的

D. 当 $|V| > |E| + 1$ 时，G一定是不连通的

**答案**：D

**考点分析**

本题考查无向图连通性与顶点数、边数的关系。

**核心知识点**：
- 含 $n$ 个顶点的连通无向图至少有 $n-1$ 条边（树）
- 含 $n$ 个顶点的无向图最多有 $\frac{n(n-1)}{2}$ 条边（完全图）

**选项分析**：

**A. $|V| > |E|$ 时，G一定连通**：错误。反例：$|V| = 4, |E| = 2$，满足 $|V| > |E|$，但可能是两个独立的边，图不连通。

**B. $|V| < |E|$ 时，G一定连通**：错误。边数多不保证连通性。反例：$|V| = 4, |E| = 5$，若4个顶点分为两组，一组3个顶点有3条边（形成三角形），另一组1个孤立顶点，再加2条自环（若允许）或重边，仍不连通。

**C. $|V| = |E| - 1$ 时，G一定不连通**：错误。这恰好是连通图（树）的特征。例如 $|V| = 4, |E| = 3$，可以构成一棵树，是连通的。

**D. $|V| > |E| + 1$ 时，G一定不连通**：正确。连通图至少需要 $|V| - 1$ 条边。若 $|V| > |E| + 1$，即 $|E| < |V| - 1$，边数不足以连通所有顶点，因此图一定不连通。

**结论**：图的连通性需要最少边数支撑，当边数少于 $|V| - 1$ 时，图必然不连通。

---

## 解题思路

### 第一步：回顾连通图的基本性质
**核心定理**：
- n个顶点的**连通图**最少需要 **n-1** 条边（树的特征）
- 少于n-1条边，图**必然不连通**（无法形成连接所有顶点的路径）
- 等于或多于n-1条边，图**可能连通也可能不连通**（取决于边的具体分布）

**关键推论**：
```
边数 < n-1  →  一定不连通（充分条件）
边数 ≥ n-1 →  可能连通，也可能不连通
```

### 第二步：逐项分析

**A. 当|V|>|E|时，G一定是连通的 ❌**
分析：|V|>|E| 等价于 |E|<|V|
- 例如：|V|=5, |E|=3，满足|V|>|E|
- 连通需要至少4条边（5-1=4）
- 但只有3条边，不足以连通
反例构造：
```
5个顶点：v1, v2, v3, v4, v5
3条边：v1-v2, v3-v4, v4-v5
结果：v1,v2一组；v3,v4,v5一组，两组不连通
```
结论：边数可能不足，不保证连通 ❌

**B. 当|V|<|E|时，G一定是连通的 ❌**
分析：|V|<|E| 意味着边数多于顶点数
- 边多不保证连通性（关键是边的分布）
反例构造：
```
4个顶点：v1, v2, v3, v4
5条边，但分布不均：
  - v1,v2,v3三个顶点形成完全图K₃（3条边）
  - v1-v2, v2-v3, v3-v1
  - v4孤立
  - 剩余2条边：v1-v2重边（或自环）
结果：v4孤立，图不连通
```
**注意**：虽然边数多，但若集中在部分顶点，仍可能不连通
结论：边数多不保证分布均匀 ❌

**C. 当|V|=|E|-1时，G一定是不连通的 ❌**
分析：|V|=|E|-1 → |E|=|V|+1
- 例如：|V|=4, |E|=5
- 连通需要至少3条边（4-1=3）
- 现有5条边，**超过最小需求**
反例构造：
```
4个顶点的树（连通）：v1-v2-v3-v4（3条边）
再加2条边：v1-v3, v2-v4（形成环）
总共5条边，图连通
```
**错误点**：题目说"一定不连通"，但边数充足时完全可以连通
结论：边数超过n-1，可以构成连通图 ❌

**D. 当|V|>|E|+1时，G一定是不连通的 ✅**
分析：|V|>|E|+1 → |E|<|V|-1
- 连通图最少需要 |V|-1 条边
- 现有边数 < |V|-1，**严格不足**
证明：
```
设图G有n个顶点，e条边，且e < n-1
连通图需要至少n-1条边才能将n个顶点连成一棵树
当e < n-1时，边数不足以连接所有顶点
必然存在至少一个顶点无法与其他顶点连通
因此G一定不连通
```
数值例子：
```
|V|=5, |E|=3，满足5>3+1
连通需要至少4条边，但只有3条
不可能连通所有5个顶点
```
结论：边数严格不足以支撑连通性 ✅

### 第三步：确认答案
当|V|>|E|+1即|E|<|V|-1时，边数严格少于连通所需的最小边数，图必然不连通 → 答案：**D**

## 易错点分析

### 易错点1：混淆"可能连通"与"一定连通"的逻辑
❌ 错误理解：边数多（如|V|<|E|）就一定连通
✅ 正确理解：连通性取决于两个因素：
1. **边数是否足够**（必要条件）：至少n-1条
2. **边的分布是否合理**（充分条件）：不能集中在部分顶点

| 边数范围 | 连通性判断 | 原因 |
|---------|-----------|------|
| e < n-1 | **一定不连通** | 边数绝对不足 ✅确定性结论 |
| e = n-1 | **可能连通** | 可形成树（连通），也可能不连通 |
| e > n-1 | **可能连通** | 边多但分布可能不均 |
| e ≥ C(n,2) | **一定连通** | 完全图，所有顶点都相连 |

### 易错点2：对选项C的理解偏差，误认为|E|=|V|+1一定不连通
❌ 错误推理：选项D说|V|>|E|+1不连通，那C选项|V|=|E|-1（即|E|=|V|+1）边更少，更不连通
✅ 正确分析：
```
选项C：|E|=|V|+1，例如|V|=4, |E|=5
  - 连通需要至少3条边
  - 现有5条边 > 3，边数充足
  - 可以构成连通图（如树+2条额外边）

选项D：|E|<|V|-1，例如|V|=5, |E|=3
  - 连通需要至少4条边
  - 现有3条边 < 4，边数不足
  - 必然不连通
```

**关键区别**：
| 选项 | 不等式 | 与n-1比较 | 结论 |
|------|-------|----------|------|
| C | |E|=|V|+1 | |E| > |V|-1（边数超过阈值）| 可能连通 |
| D | |E|<|V|-1 | |E| < |V|-1（边数低于阈值）| 一定不连通 |

### 易错点3：忽略"一定"与"可能"的逻辑严谨性
题目要求判断"一定"的结论，需要充分条件（100%保证）

| 命题类型 | 逻辑要求 | 反例检验 |
|---------|---------|---------|
| "一定连通" | 任何边的配置都连通 | 找到一种配置不连通即推翻 |
| "一定不连通" | 任何边的配置都不连通 | 找到一种配置连通即推翻 ✅严格 |
| "可能连通" | 存在某种配置连通 | 举一个连通例子即证明 |

**选项D的严密性**：
```
|E| < |V|-1 → 边数绝对不足 → 无论如何配置都无法连通所有顶点
这是数学上的充分条件，无反例
```
