**题干**

若使用 AOE 网估算工程进度，则下列叙述中正确的是（）。

A. 关键路径是从原点到汇点边数最多的一条路径  B. 关键路径是从原点到汇点路径长度最长的路径  C. 增加任一关键活动的时间不会延长工程的工期  D. 缩短任一关键活动的时间将会缩短工程的工期

**答案**：B

**考点分析**
- 关键路径：路径长度最长

---

## 解题思路

### 第一步：理解题目核心
AOE网关键路径是从源点到汇点路径长度（权值和）最长的路径，决定工程最短完成时间，关键活动时间余量为0。

### 第二步：逐项分析

**A. 关键路径是从原点到汇点边数最多的一条路径 ❌**
```
详细分析：边数多不等于路径长度长
判断依据：关键路径看的是路径权值和（持续时间总和），不是边的数量。例如：路径1有3条边权值1+1+1=3，路径2有2条边权值5+5=10，路径2才是关键路径
结论：混淆边数与路径长度
```

**B. 关键路径是从原点到汇点路径长度最长的路径 ✅**
```
详细分析：路径长度=所有边权值之和
判断依据：
  1. AOE网中边权表示活动持续时间
  2. 路径长度=该路径所有活动时间之和
  3. 最长路径决定工程最短完成时间
  4. 这条最长路径就是关键路径
结论：正确答案
```

**C. 增加任一关键活动的时间不会延长工程的工期 ❌**
```
详细分析：关键活动时间余量为0
判断依据：关键活动在关键路径上，增加其时间会使关键路径更长，直接延长工程工期
结论：逻辑错误，增加关键活动时间必然延长工期
```

**D. 缩短任一关键活动的时间将会缩短工程的工期 ❌**
```
详细分析：可能存在多条关键路径
判断依据：若有多条关键路径，缩短其中一条上的关键活动，其他关键路径仍是瓶颈，工期不变。只有缩短所有关键路径上的共同活动才能缩短工期
结论："任一"过于绝对
```

### 第三步：确认答案
关键路径=路径长度（权值和）最长的路径 → 答案：**B**

## 易错点分析

### 易错点1：混淆边数与路径长度
❌ 错误：认为边数多的路径就是关键路径
✅ 正确：路径长度指权值和（持续时间总和），不是边的数量

### 易错点2：关键路径与工期的关系
| 操作 | 对工期影响 | 前提条件 |
|------|-----------|---------|
| 增加关键活动时间 | 必然延长工期 | 无条件 |
| 缩短关键活动时间 | 可能缩短工期 | 仅当该活动在所有关键路径上 |
