# 图的连通性真题汇总

## 2010年第07题

**元数据**
- 年份：[2010年第7题](../../../01-真题原卷/2010年408真题.md#07)
- 科目：#数据结构
- 知识点：#数据结构/图/连通性
- 难度：⭐⭐⭐
- 关联知识：[图](../../../../AI-知识点原子库/图.md)

**题干**

若无向图 G = (V, E) 中含有 7 个顶点，要保证图 G 在任何情况下都是连通的，则需要的边数最少是（）。A. 6  B. 15  C. 16  D. 21

**答案**：C

**考点分析**
- 最坏情况：6个顶点全连通+1个顶点，需C(6,2)+1=16条边

---

## 解题思路

### 第一步：理解题意关键词"任何情况下"
题目问"要保证图G在**任何情况下**都是连通的"，这是最坏情况分析问题。需要找到一种边的配置，使得无论如何删边（在不破坏连通性前提下），图始终保持连通。

关键：不是问"连通图最少需要几条边"（答案是6），而是问"如何配置边才能对抗最坏情况"。

### 第二步：分析最坏情况的构造
**最坏情况：6个顶点形成完全图K₆，剩余1个孤立顶点**
```
7个顶点分组：{6个顶点构成K₆} + {1个孤立顶点}
此时：
  - K₆内部有C(6,2) = 15条边
  - 孤立顶点与K₆之间需至少1条边连接
  - 总边数：15 + 1 = 16条
```

**为什么这样配置？**
```
假设边数<16：
  - 若只有15条边，可能出现7个顶点分成{6个全连通}+{1个孤立}
  - 此时1个孤立顶点无边连接，图不连通
  - 16条边强制至少有1条边连出K₆，保证连通

假设边数=16：
  - 即使最坏情况下6个顶点耗尽15条边形成K₆
  - 剩余1条边必然连接孤立顶点与K₆，保证连通
```

### 第三步：逐项分析

**A. 6 ❌**
```
6条边可以形成树（最小连通图）
但无法保证"任何情况下"都连通
反例：6条边全部用于连接其中6个顶点成树，第7个顶点孤立
结论：不满足"任何情况下" ❌
```

**B. 15 ❌**
```
15条边恰好等于C(6,2)，可形成6个顶点的完全图K₆
反例配置：6个顶点用完15条边构成K₆，第7个顶点无边连接
此时图不连通
结论：边数不足 ❌
```

**C. 16 ✅**
```
16 = C(6,2) + 1 = 15 + 1
分析：
  - 最坏情况：6个顶点耗尽15条边形成K₆
  - 剩余1条边必然连接第7个顶点与K₆中某顶点
  - 保证7个顶点全部连通
任何边的配置下，16条边足以保证连通性
结论：正确 ✅
```

**D. 21 ❌**
```
21 = C(7,2)，7个顶点的完全图K₇的边数
虽然绝对保证连通，但超过最少需求
题目问"最少"，16条已足够，21条冗余
结论：过量但非最少 ❌
```

### 第四步：确认答案
最坏情况需C(6,2)+1=16条边，保证任何配置下都连通 → 答案：**C**

## 易错点分析

### 易错点1：误选A（6条边），忽略"任何情况下"的约束
❌ 错误理解：连通图最少需要n-1=6条边（树的性质），所以选A
✅ 正确理解：6条边在**理想配置**下（构成树）能连通，但题目问"任何情况下"——最坏配置可能让6条边无法覆盖所有顶点。例如6条边全部用在6个顶点间形成树，第7个顶点孤立

| 条件 | 最少边数 | 原因 |
|------|----------|------|
| 理想配置的连通图 | n-1=6 | 构成树 |
| 任何情况下都连通 | C(n-1,2)+1=16 | 对抗最坏分组 |

### 易错点2：不理解"最坏情况"的构造逻辑
❌ 错误理解：随便配置边，只要边数够多就能连通
✅ 正确理解：最坏情况是"对手"恶意配置边，尽量让图不连通。需要的边数要多到"即使对手恶意配置，图仍然连通"

**最坏配置模型**：
```
将n个顶点分成两组：k个顶点 + (n-k)个顶点
组内边数最多：C(k,2) + C(n-k,2)
组间边数最少：0条（最坏情况）
要保证连通，总边数必须 > max{C(k,2) + C(n-k,2)}
```

对于n=7，最坏分组是6+1：C(6,2)+C(1,2)=15+0=15
因此需要15+1=16条边，强制至少1条边跨组

### 易错点3：混淆"保证连通"与"可能连通"
| 概念 | 边数要求 | 例子 |
|------|----------|------|
| 可能连通 | ≥n-1 | 6条边可以构成树（连通），也可能不连通 |
| 保证连通（任何配置） | >C(n-1,2) | 16条边在任何配置下都连通 |
| 必然连通（完全图） | C(n,2) | 21条边形成K₇，绝对连通但冗余 |

题目问"保证...任何情况下"属于第二种，需要C(6,2)+1=16条边
