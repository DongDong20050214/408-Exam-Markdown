# 2011年第1题

**元数据**
- 年份：[[../../01-真题原卷/2011年408真题#1|2011年第1题]]
- 科目：#408真题
- 难度：⭐⭐

**题干**

设 $n$ 是描述问题规模的非负整数，下面程序片段的时间复杂度是（ ）。

$$
\begin{array}{r l} & \mathbf {x} = 2; \\ & \text { while } (\mathbf {x} <   n / 2) \\ & \quad \mathbf {x} = 2 ^ {*} \mathbf {x}; \end{array}
$$

A. $O(\log_2 n)$ 

B. $O(n)$ 

C. $O(n\log_2n)$ 

D. $O(n^{2})$ 

**答案**：A

**考点分析**

本题考查算法时间复杂度分析。循环中 x 每次乘以 2（即 $x = 2x$），因此 x 的变化序列为 2, 4, 8, 16, ..., 直到 $x \geq n/2$ 时停止。假设循环执行 k 次，则有 $2^{k+1} \geq n/2$，即 $k+1 \geq \log_2(n/2) = \log_2 n - 1$，所以 $k = O(\log_2 n)$。这是典型的指数增长导致的对数时间复杂度。

---

## 解题思路

### 第一步：分析循环变量变化规律

**代码分析**：
```c
x = 2;
while (x < n/2)
    x = 2 * x;
```

**x的值变化序列**：
- 初始：x = 2 = 2¹
- 第1次循环后：x = 4 = 2²
- 第2次循环后：x = 8 = 2³
- 第3次循环后：x = 16 = 2⁴
- ...
- 第k次循环后：x = 2^(k+1)

**循环条件**：x < n/2

### 第二步：确定循环次数

**终止条件**：当 x ≥ n/2 时停止

假设循环执行k次，则循环后：
$$x = 2^{k+1} \geq \frac{n}{2}$$

两边取对数：
$$k+1 \geq \log_2(\frac{n}{2})$$
$$k+1 \geq \log_2 n - \log_2 2$$
$$k+1 \geq \log_2 n - 1$$
$$k \geq \log_2 n - 2$$

因此：**循环次数 k = O(log₂n)**

### 第三步：逐项分析

**A. O(log₂n) ✅**
- 循环次数k ≈ log₂n
- x每次翻倍，典型对数复杂度
- 正确答案

**B. O(n) ❌**
- 线性复杂度：循环n次
- 但x指数增长，不是线性增长
- 错误

**C. O(nlog₂n) ❌**
- 归并排序、快速排序的平均复杂度
- 需要嵌套循环或递归
- 本题只有单层循环
- 错误

**D. O(n²) ❌**
- 需要两层嵌套循环
- 本题只有一层循环
- 错误

### 第四步：验证

**示例：n = 16**
- n/2 = 8
- x变化：2 → 4 → 8（停止）
- 循环2次
- log₂16 = 4，k ≈ 2 符合O(log n)

**示例：n = 32**
- n/2 = 16
- x变化：2 → 4 → 8 → 16（停止）
- 循环3次
- log₂32 = 5，k ≈ 3 符合O(log n)

## 易错点分析

### 易错点1：误认为循环n次
❌ 错误理解：看到循环条件"x<n/2"中有n，直觉认为要循环n/2次，所以时间复杂度是O(n)
✅ 正确理解：时间复杂度取决于循环次数，循环次数取决于循环变量的增长速度。虽然条件是"x<n/2"，但x每次乘2(x=2*x)是指数增长而非线性增长(线性是x=x+1)。指数增长意味着x快速逼近n/2，循环很快结束。初始x=2，第k次循环后x=2^(k+1)，当2^(k+1)≥n/2时停止，即k≥log₂n-2，循环次数k=O(log₂n)。指数增长对应对数复杂度

### 易错点2：混淆增长方式与复杂度的关系
❌ 错误理解：x增长到n/2需要O(n)时间
✅ 正确理解：
- 线性增长：x = x + c → 循环次数O(n)
- 指数增长：x = x * c → 循环次数O(log n)
- 增长越快，循环次数越少

### 易错点3：不会用对数求循环次数
❌ 错误做法：凭感觉猜测复杂度，没有严格数学推导
✅ 正确方法：分析循环复杂度必须严格按数学推导。标准四步：①找通项公式(本题x每次乘2，初始x=2，第k次后x=2^(k+1))；②列终止条件不等式(x<n/2停止，即2^(k+1)≥n/2)；③两边取对数求解k(log₂(2^(k+1))≥log₂(n/2)，得k≥log₂n-2)；④根据大O定义忽略低阶项，得O(log₂n)。关键：指数式两边取对数将指数降为系数。口诀：乘除法循环看对数

## 知识点对比

| 循环变量变化方式 | 示例代码 | 循环次数 | 时间复杂度 |
|---------------|---------|---------|-----------|
| **加法（线性）** | x = x + 1 | n次 | O(n) |
| **减法（线性）** | x = x - 1 | n次 | O(n) |
| **乘法（指数）** | x = x * 2 | log n次 | **O(log n)** |
| **除法（指数）** | x = x / 2 | log n次 | **O(log n)** |
| **平方（快速）** | x = x * x | log log n次 | O(log log n) |

**本题规律**：x每次乘2，属于指数增长，对应对数复杂度。

| 变化类型 | 增长速度 | 达到n需要的步数 | 典型算法 |
|---------|---------|---------------|---------|
| x = x + c | 慢（线性） | n/c步 | 顺序查找 |
| x = 2*x | 快（指数） | log₂n步 | 二分查找、本题 |
| x = x*x | 极快 | log log n步 | - |

**记忆要点**：
- 加减法 → O(n)
- 乘除法（常数倍）→ O(log n)
- 看到x倍增，首选O(log n)
