# 循环队列真题汇总

## 2014年第03题

**元数据**
- 年份：[[../../01-真题原卷/2014年408真题#03|2014年第3题]]
- 科目：#数据结构
- 知识点：#数据结构/栈队列串/循环队列
- 难度：⭐⭐⭐
- 关联知识：[[知识库/数据结构/循环队列]]

**题干**

循环队列放在一维数组 A[0...M-1]中，end1 指向队头元素，end2 指向队尾元素的后一个位置。假设队列两端均可进行入队和出队操作，队列中最多能容纳 M-1 个元素。初始时为空。下列判断队空和队满的条件中，正确的是（）。

A. 队空: end1 == end2; 队满：end1 == (end2 + 1) mod M
B. 队空: end1 == end2; 队满：end2==(end1+1) mod(M-1)
C. 队空: end2 == (end1 + 1) mod M; 队满：end1 == (end2 + 1) mod M
D. 队空: end1 == (end2 + 1) mod M; 队满：end2 == (end1 + 1) mod (M - 1)

**答案**：A

**考点分析**
- 循环队列判空判满条件

---

## 解题思路

### 第一步：理解题目定义
- 数组：A[0...M-1]（M个位置）
- end1：指向队头元素
- end2：指向队尾元素的**后一个位置**
- 最多容纳：M-1个元素（牺牲一个空位区分空满）

### 第二步：分析队空和队满条件

**队空**：
- 无元素时，end1和end2指向同一位置
- 条件：**end1 == end2**

**队满**：
- M-1个元素时，end2的下一个位置就是end1
- end2指向队尾后一位，再前进一步会碰到end1
- 条件：**end1 == (end2 + 1) % M**

### 第三步：逐项分析

#### ✅ A. 队空: end1 == end2; 队满: end1 == (end2 + 1) mod M（正确答案）
- 队空：两指针重合 ✓
- 队满：end2再进一位就是end1，留一个空位 ✓
**正确**

#### ❌ B. 队空: end1 == end2; 队满: end2 == (end1+1) mod(M-1)
- 队空正确
- 队满错误：模数应为M不是M-1，且方向相反
**错误**

#### ❌ C. 队空: end2 == (end1 + 1) mod M; 队满: end1 == (end2 + 1) mod M
- 队空错误：end2在end1后一位表示有1个元素，不是空
- 队满正确
**错误**

#### ❌ D. 队空: end1 == (end2 + 1) mod M; 队满: end2 == (end1 + 1) mod (M - 1)
- 队空错误：end1在end2后一位表示满，不是空
- 队满模数错误
**错误**

### 对比表格

| 状态 | 元素个数 | end1与end2关系 | 条件 |
|------|---------|---------------|------|
| 队空 | 0 | 重合 | end1 == end2 |
| 有1个元素 | 1 | end2比end1多1 | end2 == (end1+1)%M |
| 队满 | M-1 | end2再进1就是end1 | end1 == (end2+1)%M |

### 图示说明

```
队空：end1,end2 → [ ]
       ↑

有1个元素：end1 → [x] ← end2
           ↑       ↑

队满（M=5）：
end1 → [x][x][x][x][ ] ← end2
       ↑              ↑
       队头           队尾后一位
       
end2再进一位就回到end1位置（空位被占用）
```

## 易错点分析

### 易错点1：混淆end2的定义
❌ 误认为end2指向队尾元素（而非后一位）
✅ 正确理解：end2指向队尾元素的**后一个位置**，这影响判满条件

### 易错点2：模数M和M-1混淆
❌ 误用mod(M-1)
✅ 循环队列的循环范围是0到M-1，模数应为M

### 易错点3：队空队满条件记反
❌ 把队满条件end1==(end2+1)%M误记为队空条件
✅ 记忆技巧：队空最简单（两指针重合），队满是end2"追上"end1前一位
