# 循环队列真题汇总

## 2011年第03题

**元数据**
- 年份：[[../../01-真题原卷/2011年408真题#03|2011年第3题]]
- 科目：#数据结构
- 知识点：#数据结构/栈队列串/循环队列
- 难度：⭐⭐
- 关联知识：[[知识库/数据结构/循环队列]]

**题干**

已知循环队列存储在一维数组 A[0...n-1] 中，且队列非空时 front 和 rear 分别指向队头元素和队尾元素。若初始时队列为空，且要求第 1 个进入队列的元素存储在 A[0] 处，则初始时 front 和 rear 的值分别是（）。

A. 0,0  B. 0, n - 1  C. n-1,0  D. n - 1, n - 1

**答案**：B

**考点分析**
- 第一个元素在A[0]
- front=0, rear=n-1

---

## 解题思路

### 第一步：理解题目关键条件
- 循环队列数组：A[0...n-1]
- **非空时**：front指向队头元素，rear指向队尾元素（注意：指向元素本身，非下一位置）
- **初始状态**：队列为空
- **第一个元素**：必须存储在A[0]处

### 第二步：分析第一个元素入队操作
假设初始front=f，rear=r，第一个元素x入队：
```
入队操作：rear = (rear + 1) % n; A[rear] = x;
```
要让x存储在A[0]，需要：
```
(r + 1) % n = 0
```
解得：**r = n - 1**

### 第三步：分析front初始值
第一个元素入队后：
- rear指向A[0]（队尾元素）
- front应指向A[0]（队头元素）
- 因此第一个元素入队后：front = 0, rear = 0

回推初始状态：
- 入队前rear = n-1
- 入队不改变front（队列空→非空时front才开始有效）
- **初始front = 0**

### 第四步：逐项分析

#### ❌ A. 0, 0
初始front=0, rear=0
```
入队：rear = (0+1) % n = 1
第一个元素存在A[1]，不符合题意
```
**错误**

#### ✅ B. 0, n-1（正确答案）
初始front=0, rear=n-1
```
入队：rear = (n-1+1) % n = 0
第一个元素存在A[0]，符合题意 ✓
此时队列：front=0指向队头A[0]，rear=0指向队尾A[0]
```
**正确**

#### ❌ C. n-1, 0
初始front=n-1, rear=0
```
入队：rear = (0+1) % n = 1
第一个元素存在A[1]，不符合题意
```
**错误**

#### ❌ D. n-1, n-1
初始front=n-1, rear=n-1
```
入队：rear = (n-1+1) % n = 0
第一个元素存在A[0]，看似可行

但第一个元素入队后：front=n-1, rear=0
此时front=n-1不指向队头元素A[0]，违反题意
```
**错误**（front值不符合非空时的约定）

### 对比表格：不同循环队列定义方式

| 定义方式 | front指向 | rear指向 | 空队列条件 | 本题类型 |
|---------|----------|----------|-----------|---------|
| 类型1 | 队头元素 | 队尾元素 | front=0, rear=n-1 | ✓ |
| 类型2 | 队头元素 | 队尾下一位置 | front=rear | 常见 |
| 类型3 | 队头前一位置 | 队尾元素 | front=rear | 少见 |

**本题特点**：非空时front和rear都指向实际元素（类型1），空队列时需特殊设置rear=n-1。

### 完整模拟入队过程

| 操作 | front | rear | 队列内容 | 说明 |
|------|-------|------|---------|------|
| 初始 | 0 | n-1 | 空 | 空队列 |
| 入x | 0 | 0 | A[0]=x | front和rear都指向A[0] |
| 入y | 0 | 1 | A[0]=x, A[1]=y | front指向A[0]，rear指向A[1] |
| 出队 | 1 | 1 | A[1]=y | front和rear都指向A[1] |
| 出队 | ? | ? | 空 | 回到空队列状态？ |

**注意**：题目未说明出队后如何回到空状态，只关注初始到第一个元素的过程。

## 易错点分析

### 易错点1：混淆不同的循环队列定义
❌ 错误理解：使用常见的"rear指向队尾下一位置"定义，认为空队列front=rear=0，入队后rear=1
✅ 正确理解：题目明确"非空时front和rear分别指向队头元素和队尾元素"，不是指向下一位置，需用类型1定义

### 易错点2：忽略"第一个元素在A[0]"的约束
❌ 错误做法：只考虑front和rear的关系，未验证第一个元素的实际存储位置
✅ 正确方法：通过入队公式rear=(rear+1)%n反推，要让结果为0，则初始rear必须为n-1

### 易错点3：误认为空队列时front=rear
❌ 错误类比：套用常见定义（类型2）的空队列条件front=rear，选A或D
✅ 正确理解：本题定义下，空队列时front≠rear，具体为front=0且rear=n-1（rear在front"前面"一个循环位置）
