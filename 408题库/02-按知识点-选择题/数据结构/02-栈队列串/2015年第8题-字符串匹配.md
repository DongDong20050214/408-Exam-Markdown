# 2015年第8题

**元数据**
- 年份：2015年第8题
- 难度：⭐⭐

**题干**：已知字符串 s 为 "abaabaabacacaabaabcc"，模式串 t 为 "abaabc"。采用 KMP 算法进行匹配，第一次出现"失配"（s[i] ≠ t[j]）时，i = j = 5，下次开始匹配时，i 和 j 的值分别是（）。

A. i = 1, j = 0 

B. i = 5, j = 0 

C. i = 5, j = 2 

D. i = 6, j = 2

**答案**：C

**考点分析**：
- **考点**：数据结构 - KMP字符串匹配算法
- **知识点**：
  - KMP算法通过next数组避免主串指针i回溯
  - 失配时，模式串指针j移动到next[j]位置，主串指针i不变
  - next[j]表示模式串前j个字符中最长相同前后缀的长度
- **解析**：
  - 模式串 t = "abaabc"，下标从0开始
  - 计算next数组：
    - next[0] = -1（特殊值）
    - next[1] = 0（a，无相同前后缀）
    - next[2] = 0（ab，无相同前后缀）
    - next[3] = 1（aba，相同前后缀"a"，长度1）
    - next[4] = 2（abaa，相同前后缀"ab"，长度2）
    - next[5] = 2（abaab，相同前后缀"ab"，长度2）
  - 失配时i=5, j=5，即s[5]≠t[5]
  - 下次匹配：i保持5不变，j移动到next[5]=2
  - 因此i=5, j=2

---

## 解题思路

### next数组

t="abaabc": next=[−1,0,0,1,1,2]

失配时i=5,j=5 → 下次i=5,j=next[5]=2

### 逐项分析

#### ❌ A. i=1, j=0
i回退，违反KMP
**错误**

#### ❌ B. i=5, j=0
j应为next[5]=2
**错误**

#### ✅ C. i=5, j=2
i不变，j=next[5]=2
**正确**

#### ❌ D. i=6, j=2
i不应前进
**错误**

### 对比表格

| 算法 | i | j | 特点 |
|------|---|---|------|
| 暴力 | 回退 | 0 | 低效 |
| KMP | 不变 | next[j] | 高效 |

## 易错点分析

### 易错点1：i回退
❌ 选A
✅ KMP中i不回退

### 易错点2：next计算错误
❌ next[5]=0
✅ next[5]=2（"abaab"前后缀"ab"）
