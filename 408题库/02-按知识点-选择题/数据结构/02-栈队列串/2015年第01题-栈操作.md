# 2015年第1题

**元数据**
- 年份：2015年第1题
- 难度：⭐⭐

**题干**：已知程序如下:

```c
int S(int n)
{ return (n<=0)?0:S(n-1)+n; }
void main()
{ cout<< S(1); }
```

程序运行时使用栈来保存调用过程的信息，自栈底到栈顶保存的信息依次对应的是（）。

A. main()→S(1)→S(0) 

B. S(0)→S(1)→main() 

C. main()→S(0)→S(1) 

D. S(1)→S(0)→main()

**答案**：A

**考点分析**：
- **考点**：数据结构 - 栈的应用、函数调用栈
- **知识点**：
  - 函数调用时，系统使用栈来保存函数的返回地址、参数、局部变量等信息
  - 函数调用遵循"后进先出"(LIFO)原则
  - 栈底存放最先调用的函数，栈顶存放最后调用的函数
- **解析**：
  - 程序从main()开始执行，main()先入栈（栈底）
  - main()调用S(1)，S(1)入栈
  - S(1)调用S(0)，S(0)入栈（栈顶）
  - 因此从栈底到栈顶依次是：main()→S(1)→S(0)

---

## 解题思路

### 第一步：追踪函数调用流程
```c
main() { cout << S(1); }
  ↓ 调用S(1)
S(1): return S(0) + 1
  ↓ 调用S(0)
S(0): return 0
```

栈状态演变：
1. [main]
2. [main, S(1)]
3. [main, S(1), S(0)] ← 题目考查此时刻

### 逐项分析

#### ✅ A. main()→S(1)→S(0)
栈底到栈顶：最早调用到最后调用
**正确**

#### ❌ B. S(0)→S(1)→main()
完全颠倒
**错误**

#### ❌ C. main()→S(0)→S(1)
S调用顺序反了
**错误**

#### ❌ D. S(1)→S(0)→main()
main应在栈底
**错误**

### 对比表格

| 函数 | 调用顺序 | 栈位置 | 返回值 |
|------|---------|--------|--------|
| main() | 1 | 栈底 | S(1)的结果 |
| S(1) | 2 | 栈中 | S(0)+1=1 |
| S(0) | 3 | 栈顶 | 0 |

## 易错点分析

### 易错点1：栈方向混淆
❌ 栈顶在左
✅ 栈底（左）→栈顶（右）

### 易错点2：递归顺序错误
❌ S(0)先调用
✅ S(1)→S(0)
