**题干**

一个栈的入栈序列为 1, 2, 3, ..., n，其出栈序列是 p1, p2, p3, ..., pn。若 p2 = 3，则 p3 可能取值的个数是（）。A. n - 3  B. n - 2  C. n - 1  D. 无法确定

---

**答案**：C

**考点分析**
- p2=3说明第二个出栈的是3
- p1可能是1或2（其中之一）
- p3可以是{1,2,4,5,...,n}中除3外的任何值，共n-1个

---

## 解题思路

### 第一步：分析p2=3的含义

**p2=3表示**：
- 3是第二个出栈的元素
- 1和2必须在3之前入栈（入栈顺序固定）
- p1（第一个出栈）只能是1或2

### 第二步：枚举p1的可能性

**情况1：p1=1**
```
操作：1入→1出→2入→3入→3出
此时栈：[2]，待入：4,5,...,n
p3可以是：2（直接出）或 4~n（依次入栈后出栈顶）
可能值：{2, 4, 5, ..., n}
```

**情况2：p1=2**
```
操作：1入→2入→2出→3入→3出
此时栈：[1]，待入：4,5,...,n
p3可以是：1（直接出）或 4~n（依次入栈后出栈顶）
可能值：{1, 4, 5, ..., n}
```

### 第三步：计算p3可能值的总数

两种情况的并集：
```
{2, 4, 5, ..., n} ∪ {1, 4, 5, ..., n} = {1, 2, 4, 5, ..., n}
```

**p3不能是3**（3已在p2位置出栈）

**可能值个数**：n-1

### 逐项分析

#### ❌ A. n-3
误认为1,2,3都不能是p3
**错误**

#### ❌ B. n-2
误认为1,2都不能是p3（实际上二者之一可以是p3）
**错误**

#### ✅ C. n-1（正确答案）
p3可以是除3外的任何数：{1,2,4,5,...,n}
共n-1个
**正确**

#### ❌ D. 无法确定
p3的可能值完全由p2=3确定
**错误**

### 对比表格

| p1 | 栈状态 | p3可能值 | 个数 |
|----|--------|---------|------|
| 1 | [2] | 2,4,5,...,n | n-1 |
| 2 | [1] | 1,4,5,...,n | n-1 |
| 并集 | - | 1,2,4,5,...,n | n-1 |

## 易错点分析

### 易错点1：误认为1和2都不能是p3
❌ 错误理解：1,2在3之前入栈，都被"压"在栈底，无法在p3出栈
✅ 正确理解：p1会出栈1或2中的一个，另一个留在栈中可在p3出栈

### 易错点2：忽略4~n的灵活性
❌ 错误推理：只考虑栈中剩余的1或2，得出p3只有1种可能
✅ 正确分析：4~n可在p3前依次入栈，任何一个都可作为栈顶在p3出栈
