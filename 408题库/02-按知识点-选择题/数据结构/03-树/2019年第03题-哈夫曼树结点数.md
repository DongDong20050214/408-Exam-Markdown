# 2019年408真题第03题

## 题目

对 $n$ 个互不相同的符号进行哈夫曼编码。若生成的哈夫曼树共有 115 个结点，则 $n$ 的值是（）。

A. 56  
B. 57  
C. 58  
D. 60

## 答案

C

## 考点分析

### 考点：哈夫曼树的性质

本题考查哈夫曼树（最优二叉树）的结点数关系。

### 核心性质：

**哈夫曼树的结点数公式**：
- 设有 $n$ 个叶子节点（待编码的符号）
- 哈夫曼树是**严格二叉树**（每个非叶节点都有2个孩子）
- 总结点数 = 叶子节点数 + 分支节点数 = $n + (n-1) = 2n - 1$

### 解题步骤：

1. **已知条件**：
   - 总结点数 = 115
   - 叶子节点数 = $n$（待求）

2. **应用公式**：
   - $2n - 1 = 115$
   - $2n = 116$
   - $n = 58$

### 推导说明：

**为什么分支节点数是 $n-1$？**

在哈夫曼树的构造过程中：
- 初始：$n$ 个独立节点（都是叶子）
- 每次合并操作：选2个节点合并成1个新节点
- 减少节点数：每次 $-2 + 1 = -1$
- 最终形成1棵树：需要 $n-1$ 次合并
- 因此产生 $n-1$ 个分支节点

### 验证：

- 叶子节点：58
- 分支节点：57
- 总结点数：58 + 57 = 115 ✓

### 记忆要点：

- **哈夫曼树总结点数 = $2n - 1$**（$n$ 为叶子数）
- 哈夫曼树是严格二叉树（度为0或2）
- 度为1的节点数 = 0
- 度为2的节点数 = 度为0的节点数 - 1

## 易错点分析

### 易错点1：混淆总结点数和叶结点数
❌ 错误理解：认为115是叶结点数，计算2×115-1
✅ 正确理解：115是总结点数，需反推叶结点数n，公式：n = (总数+1)/2

### 易错点2：忘记哈夫曼树是严格二叉树
❌ 错误理解：认为哈夫曼树可能有度为1的结点
✅ 正确理解：哈夫曼树构造过程决定所有非叶结点度为2，无度为1结点

## 知识点对比

| 树的类型 | 度为0(叶结点) | 度为1 | 度为2 | 总结点数公式 |
|---------|--------------|-------|-------|------------|
| 哈夫曼树 | n | 0 | n-1 | 2n-1 |
| 满二叉树 | n | 0 | n-1 | 2n-1 |
| 一般二叉树 | n₀ | n₁ | n₂ | n₀+n₁+n₂ |

| 公式 | 适用范围 | 说明 |
|------|---------|------|
| 总数 = 2n-1 | 哈夫曼树、满二叉树 | n为叶结点数 |
| n₀ = n₂+1 | 所有二叉树 | 叶结点比度为2的结点多1 |
| 反推公式：n=(总数+1)/2 | 哈夫曼树 | 已知总数求叶数 |
