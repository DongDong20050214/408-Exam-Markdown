# 前序后序推导真题汇总

## 2012年第03题

**元数据**
- 年份：[[../../01-真题原卷/2012年408真题#03|2012年第3题]]
- 科目：#数据结构
- 知识点：#数据结构/树/前序后序遍历
- 难度：⭐⭐⭐
- 关联知识：[[知识库/数据结构/二叉树]]

**题干**

若一棵二叉树的前序遍历序列为a,e,b,d,c，后序遍历序列为b,c,d,e,a，则根结点的孩子结点（）。

A. 只有e  
B. 有e,b  
C. 有e,c  
D. 无法确定

**答案**：A

**考点分析**
- 前序：根→左→右，a是根，e是左子树或右子树的根
- 后序：左→右→根，a是根
- 前序第二个e，后序倒数第二个e
- 说明e是a的唯一子树（只有一个孩子）

---

## 解题思路

### 第一步：理解题目核心

**题目条件**：
- 前序遍历：a, e, b, d, c
- 后序遍历：b, c, d, e, a
- 求：根结点的孩子结点

**遍历规则**：
- 前序：根 → 左 → 右（第一个是根）
- 后序：左 → 右 → 根（最后一个是根）

**关键推导**：
- 前序第一个 = 根 → a是根
- 后序最后一个 = 根 → a是根（验证一致）
- 前序第二个 = 左子树的根或右子树的根 → e
- 后序倒数第二个 → e

### 第二步：分析树结构

**确定根结点**：
- 前序第一个a = 根
- 后序最后一个a = 根

**分析e的位置**：
- 前序：a之后是e → e是a的某个子树的根
- 后序：a之前是e → e是a的某个子树的根

**找e在后序中的位置**：
后序：b, c, d, e, a
- e是倒数第二个
- e之前的b,c,d都是e的子树

**在前序中找e的范围**：
前序：a, e, b, d, c
- e之后是b,d,c
- 这些都是e子树的结点

**对比两个序列中e的子树**：
- 前序中e的子树：b, d, c
- 后序中e的子树：b, c, d

验证：
- 前序e的子树：b,d,c（e之后到下一个子树前）
- 后序e的子树：b,c,d（e之前，从某个位置开始）

**判断a的孩子数量**：
- 前序：a, e, b, d, c
- 如果a有两个孩子：左子树是e(b,d,c)，右子树是？
- 后序：b, c, d, e, a
- 如果a有两个孩子：左子树(...)e，右子树(...)

分析发现：整个序列除了a，其余都在e的子树中
- 前序：a后面全是e的子树
- 后序：a前面全是e的子树

**结论**：a只有一个孩子结点e

### 第三步：逐项分析

**A. 只有e ✅**
```
详细分析：
前序：a, e, b, d, c
后序：b, c, d, e, a

a是根，除a外所有结点(e,b,c,d)都在e的子树中
说明a只有一个孩子e，e的子树包含b,c,d

判断依据：前序除a外首元素是e，后序除a外尾元素是e，中间部分都是e的子树
结论：✅ a只有一个孩子e
```

**B. 有e,b ❌**
```
详细分析：
如果a有两个孩子e和b，则：
- 前序应为：a, e, ..., b, ...（e子树和b子树分开）
- 后序应为：..., e, ..., b, a（e子树和b子树分开）

但实际后序中b在e之前，说明b在e的子树中，不是a的孩子

判断依据：b在后序中位于e之前，是e的子树结点
结论：❌ b不是a的孩子
```

**C. 有e,c ❌**
```
详细分析：
同理，c在后序中位于e之前，是e的子树结点

判断依据：c在e的子树中
结论：❌ c不是a的孩子
```

**D. 无法确定 ❌**
```
详细分析：
通过前序和后序可以明确确定：
- a是根
- e是a的唯一子树的根
- b,c,d都在e的子树中

判断依据：树结构可以确定
结论：❌ 可以确定
```

### 第四步：确认答案

根结点a只有一个孩子结点e → 答案：**A**

## 易错点分析

### 易错点1：误以为前序第二个一定是左孩子
❌ 错误理解：前序第二个e是a的左孩子，所以a有左孩子
✅ 正确理解：前序第二个e是a的某个孩子，但不能确定是左还是右，也不能确定有几个孩子。需要结合后序判断。

### 易错点2：误以为b也是a的孩子
❌ 错误推理：前序中b紧跟在e后面，认为b是a的另一个孩子
✅ 正确推理：需要看b在后序中的位置。b在e之前，说明b是e的子树结点，不是a的直接孩子。

### 易错点3：不会用前序后序划分子树
❌ 错误方法：只看前序或只看后序，无法判断
✅ 正确方法：
1. 确定根（前序第一个=后序最后一个）
2. 找根的孩子在后序中的位置（紧邻根之前）
3. 对比前序后序，确定子树范围

## 知识点对比

| 遍历方式 | 顺序 | 根的位置 | 本题序列 |
|---------|------|---------|---------|
| 前序 | 根→左→右 | 第一个 | a,e,b,d,c |
| 后序 | 左→右→根 | 最后一个 | b,c,d,e,a |
| 中序 | 左→根→右 | 中间 | 无法仅用前序后序推出 |

| 关键判断 | 方法 | 本题应用 |
|---------|------|---------|
| 确定根 | 前序第一个=后序最后一个 | a是根 ✓ |
| 确定根的孩子 | 看其他结点在后序中的分布 | e独占后序倒数第二，其余都在e之前 |
| 判断孩子数 | 看根之后的结点是否分属不同子树 | 所有结点都在e的子树中，所以a只有1个孩子e |

| 树结构 | 前序特征 | 后序特征 | 本题是否符合 |
|-------|---------|---------|-------------|
| 根有2个孩子 | a,左子树,右子树 | 左子树,右子树,a | ❌ |
| 根有1个孩子 | a,子树 | 子树,a | ✅ |
