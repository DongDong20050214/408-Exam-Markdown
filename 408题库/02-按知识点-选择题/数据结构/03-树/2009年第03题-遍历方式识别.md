## 解题思路

### 第一步：理解题目核心
根据给定序列 3, 1, 7, 5, 6, 2, 4 和二叉树结构，判断是哪种遍历方式。N=根，L=左子树，R=右子树。

### 第二步：回顾遍历方式特征
| 遍历 | 顺序 | 根位置 | 特点 |
|------|------|--------|------|
| LRN | 左→右→根 | 末尾 | 后序遍历 |
| NRL | 根→右→左 | 开头 | 前序的镜像 |
| RLN | 右→左→根 | 末尾 | 后序的镜像 |
| RNL | 右→根→左 | 中间 | 中序的镜像 |

**判断依据**：先找根在序列中的位置，再验证左右子树的遍历顺序。

### 第三步：逐项分析

**A. LRN ❌**
```
规则：先遍历左子树，再遍历右子树，最后访问根
特征：根在序列最后
序列：3, 1, 7, 5, 6, 2, 4
分析：若为LRN，则4应为根，但原图中根不是4
结论：❌ 不符合
```

**B. NRL ❌**
```
规则：先访问根，再遍历右子树，最后遍历左子树
特征：根在序列开头
序列：3, 1, 7, 5, 6, 2, 4
分析：若为NRL，则3应为根，但原图中根不是3
结论：❌ 不符合
```

**C. RLN ❌**
```
规则：先遍历右子树，再遍历左子树，最后访问根
特征：根在序列最后
序列：3, 1, 7, 5, 6, 2, 4
分析：若为RLN，则4应为根，且序列应为[右子树][左子树][根]
结论：❌ 不符合
```

**D. RNL ✅**
```
规则：先遍历右子树，再访问根，最后遍历左子树
特征：根在序列中间
序列：3, 1, 7, 5, 6, 2, 4
分析：序列结构为 [右子树部分] [根] [左子树部分]
- 右子树：3, 1, 7（3个元素）
- 根：5（第4个元素）
- 左子树：6, 2, 4（3个元素）
验证：右子树内部也按RNL遍历，左子树内部也按RNL遍历
结论：✅ 完全符合
```

### 第四步：确认答案
RNL遍历方式产生序列 3,1,7,5,6,2,4，符合题意。

答案：**D**

## 易错点分析

### 易错点1：只看根的位置不验证子树
❌ **错误做法**：看到根在中间就选RNL，不验证左右子树结构
✅ **正确方法**：三步验证法
1. 根据根的位置缩小范围（RNL或LNR）
2. 分割序列为左右子树部分
3. 验证每个子树内部也符合该遍历规则

**验证示例**：
```
序列：3,1,7 | 5 | 6,2,4
右子树：3,1,7 应按RNL排列（右→根→左）
- 若右子树根为1，则3在1的右侧，7在1的左侧
- 递归验证直到叶子
```

### 易错点2：混淆镜像遍历与标准遍历
❌ **错误理解**：认为只有NLR/LNR/LRN三种标准遍历
✅ **正确理解**：408考纲包含镜像遍历（L↔R互换）

| 标准遍历 | 镜像遍历 | 关系 |
|---------|---------|------|
| NLR（前序） | NRL | 左右交换 |
| LNR（中序） | RNL | 左右交换 |
| LRN（后序） | RLN | 左右交换 |

### 易错点3：忘记子树的递归性质
❌ **片面验证**：只验证根和第一层子树，不递归验证深层
✅ **完整验证**：每个子树都是一棵二叉树，必须递归应用相同遍历规则
```
若整棵树按RNL，则：
- 根的右子树按RNL遍历
- 根的左子树按RNL遍历
- 右子树的右子树仍按RNL遍历
- …以此类推
```

## 知识点对比

| 遍历方式 | 根访问时机 | 序列分段特征 | 常见用途 |
|---------|-----------|-------------|---------|
| LRN | 最后 | [左全部][右全部][根] | 后序表达式、删除操作 |
| NRL | 最先 | [根][右全部][左全部] | 镜像前序 |
| RLN | 最后 | [右全部][左全部][根] | 镜像后序 |
| RNL | 中间 | [右全部][根][左全部] | 降序遍历排序树 |

**记忆口诀**：先定位根（N的位置），再看左右顺序（L在R前还是R在L前），根+左右顺序=遍历方式。
