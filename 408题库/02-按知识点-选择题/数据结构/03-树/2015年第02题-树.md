# 2015年第2题

**元数据**
- 年份：2015年第2题
- 难度：⭐⭐

**题干**：先序序列为 a, b, c, d 的不同二叉树的个数是（）。

A. 13 

B. 14 

C. 15 

D. 16

**答案**：B

**考点分析**：
- **考点**：数据结构 - 二叉树的计数、卡特兰数
- **知识点**：
  - n个节点的不同二叉树个数为第n个卡特兰数：Cn = C(2n,n)/(n+1)
  - 对于给定先序序列，第一个节点必为根节点
  - 剩余节点可分配到左右子树，形成不同的结构
- **解析**：
  - 先序序列的第一个元素a必为根节点
  - 剩余b,c,d三个节点可以有多种分配方式：
    - 左子树0个，右子树3个：C3 = 5种
    - 左子树1个，右子树2个：C1×C2 = 1×2 = 2种（3种选择：b为左，c为左，d为左，共3×2=6种）
    - 左子树2个，右子树1个：C2×C1 = 2×1 = 2种（3种选择，共3×2=6种）
    - 左子树3个，右子树0个：C3 = 5种
  - 总计：5 + 3×2 + 3×2 + 5 = 5 + 6 + 6 + 5 = 22种
  - 实际应为C4 = C(8,4)/5 = 70/5 = 14种

---

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 先序序列为 a, b, c, d（4个结点）
- 问：有多少种不同的二叉树
- 选项：A.13  B.14  C.15  D.16

### 第二步：回顾卡特兰数与二叉树计数
**n个结点的不同二叉树个数 = 第n个卡特兰数**：
Cn = C(2n, n) / (n+1)

对于n=4：
C4 = C(8, 4) / 5 = (8×7×6×5)/(4×3×2×1) / 5 = 1680/24 / 5 = 70/5 = 14

**关键点**：先序序列固定后，第一个元素必为根，但剩余元素在左右子树的分配仍有多种可能。

### 第三步：逐项判断

**A. 13 ❌**
```
13不是卡特兰数
C3 = 5, C4 = 14, C5 = 42
13不在卡特兰数列中
结论：❌ 错误
```

**B. 14 ✅**
```
C4 = C(8,4) / 5 = 70/5 = 14

验证：4个结点的不同二叉树有14种
先序序列a,b,c,d固定了第一个结点为a（根）
剩余b,c,d三个结点可分配到左右子树

分配方式：
- 左0右3：1×C3 = 1×5 = 5种（右子树先序为b,c,d的5种形态）
- 左1右2：3×C1×C2 = 3×1×2 = 6种（b/c/d任一为左子树，剩余2个为右子树）
  但这里要考虑先序的约束：左子树先序必须紧跟根a
  实际：左1右2时，左子树只能是b/c/d之一，共C1×C2 = 1×2 = 2种
  但有3种选择（选谁为左），实际不对...
  
正确理解：先序固定，意味着遍历顺序固定
a必为根，b必在a后首先访问
如果b在左子树，则b是左子树根；如果b在右子树，则左子树为空，b是右子树根

重新分析：
先序a,b,c,d，a为根后：
- 若左子树为空，则右子树先序b,c,d → C3=5种
- 若左子树为b，右子树为c,d → 1×C2=2种
- 若左子树为b,c，右子树为d → C2×1=2种
- 若左子树为b,c,d，右子树为空 → C3=5种
总计：5+2+2+5 = 14 ✓

结论：✅ 正确
```

**C. 15 ❌**
```
15不是C4的值
可能是误算（如14+1）
结论：❌ 错误
```

**D. 16 ❌**
```
16 = 2^4，可能误认为每个结点2种选择
但二叉树计数不是2^n
结论：❌ 错误
```

### 第四步：确认答案
4个结点的二叉树有C4 = 14种不同形态。

答案：**B**

## 易错点分析

### 易错点1：误用排列组合公式
❌ **错误计算**：4个结点有4! = 24种排列，或2^4 = 16种。
✅ **正确理解**：二叉树的形态由结构决定，不是简单的排列。n个结点的二叉树个数是卡特兰数Cn，不是n!或2^n。

### 易错点2：忽略先序序列的约束
❌ **错误想法**：先序序列固定后，树的形态就唯一了。
✅ **正确理解**：先序序列只确定了访问顺序，不能唯一确定树的结构。例如先序a,b，可以是"a左孩子b"或"a右孩子b"两种。给定先序序列后，仍有C(n) = C(2n,n)/(n+1)种不同形态。

## 知识点对比

| n | Cn（n个结点二叉树数） | 计算公式 | 常见值 |
|---|---------------------|---------|--------|
| 1 | 1 | C(2,1)/2 | 1 |
| 2 | 2 | C(4,2)/3 = 6/3 | 2 |
| 3 | 5 | C(6,3)/4 = 20/4 | 5 |
| 4 | 14 | C(8,4)/5 = 70/5 | 14 ✅ |
| 5 | 42 | C(10,5)/6 = 252/6 | 42 |

**卡特兰数递推公式**：Cn = Σ(Ci × Cn-1-i)，其中i从0到n-1。
