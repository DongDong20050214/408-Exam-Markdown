# 题目

设一棵非空完全二叉树 $T$ 的所有叶结点均位于同一层，且每个非叶结点都有 2 个子结点。若 $T$ 有 $k$ 个叶结点，则 $T$ 的结点总数是（）。

A. $2k - 1$ 

B. 2k 

C. $k^{2}$ 

D. $2^{k} - 1$ 

# 答案

A

# 考点分析

本题考查满二叉树（完美二叉树）的性质和结点数计算。关键点：
1. 题目描述的是满二叉树：所有叶结点在同一层，每个非叶结点都有2个子结点
2. 满二叉树中，叶结点数与非叶结点数的关系：叶结点数 = 非叶结点数 + 1，即k = n₀，n₂ = k-1（其中n₀为叶结点数，n₂为度为2的结点数）
3. 由于没有度为1的结点，总结点数 n = n₀ + n₂ = k + (k-1) = 2k - 1
4. 也可从二叉树性质推导：n₀ = n₂ + 1，因此n = n₀ + n₁ + n₂ = n₀ + 0 + (n₀ - 1) = 2n₀ - 1 = 2k - 1

## 解题思路

### 第一步：理解题目核心

**题目条件**：
- 完全二叉树T
- 所有叶结点在同一层
- 每个非叶结点都有2个子结点
- k个叶结点

**关键识别**：这是**满二叉树**（Perfect Binary Tree）

### 第二步：利用二叉树性质

**性质1：度数关系**
```
二叉树中：n₀ = n₂ + 1
其中：n₀为叶结点数，n₂为度为2的结点数
```

**性质2：满二叉树无度为1的结点**
```
满二叉树：n₁ = 0
```

**推导总结点数**：
```
总结点数 n = n₀ + n₁ + n₂
           = n₀ + 0 + n₂
           = n₀ + (n₀ - 1)    [因为n₀ = n₂ + 1]
           = 2n₀ - 1
           = 2k - 1
```

### 第三步：逐项分析

**A. 2k - 1 ✅**
```
详细分析：
已知：k个叶结点，即n₀ = k
满二叉树性质：n₂ = n₀ - 1 = k - 1
总结点数：n = n₀ + n₂ = k + (k-1) = 2k - 1

结论：✅ 正确
```

**B. 2k ❌**
```
分析：
若总结点数为2k，则非叶结点数为k
这与满二叉树的性质矛盾（非叶结点应为k-1）

结论：❌ 错误
```

**C. k² ❌**
```
分析：
k²与k无线性关系
举例：k=3时，2k-1=5，但k²=9，不符

结论：❌ 错误
```

**D. 2^k - 1 ❌**
```
分析：
2^k - 1是高度为k的满二叉树的总结点数
但题目k是叶结点数，不是高度
举例：k=4时，2k-1=7，但2^4-1=15，不符

结论：❌ 错误
```

### 第四步：确认答案

答案：**A**

## 易错点分析

### 易错点1：混淆k的含义
❌ 错误理解：k是树的高度，总结点数为2^k - 1
✅ 正确理解：k是叶结点数，总结点数为2k - 1

### 易错点2：忘记满二叉树的性质
❌ 错误理解：随意计算，未利用n₀ = n₂ + 1
✅ 正确理解：满二叉树中叶结点数比非叶结点数多1

## 知识点对比

| 树的类型 | 定义 | n₁(度为1) | 结点数公式 |
|---------|------|----------|-----------|
| 满二叉树 | 所有非叶结点度为2，叶在同一层 | 0 | n = 2n₀ - 1 |
| 完全二叉树 | 除最后一层外都满，最后一层靠左 | 0或1 | 无固定公式 |
| 任意二叉树 | 无特殊要求 | ≥0 | n = n₀ + n₁ + n₂ |

| 公式 | 适用范围 | 说明 |
|------|---------|------|
| n₀ = n₂ + 1 | 所有二叉树 | 叶结点数=度为2的结点数+1 |
| n = 2n₀ - 1 | 满二叉树 | 由n₀ = n₂ + 1和n₁ = 0推导 |
| n = 2^h - 1 | 高度为h的满二叉树 | h是高度，不是叶结点数 |
