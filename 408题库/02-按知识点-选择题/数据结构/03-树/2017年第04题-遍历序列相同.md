**题干**

要使一棵非空二叉树的先序序列与中序序列相同，所有非叶结点须满足的条件是（）。

A. 只有左子树  B. 只有右子树  C. 结点的度均为 1  D. 结点的度均为 2

**答案**：B

**考点分析**
- 先序=中序：无左子树

---

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 非空二叉树
- 先序序列与中序序列相同
- 问：所有非叶结点须满足什么条件
- 选项：A.只有左子树  B.只有右子树  C.度均为1  D.度均为2

### 第二步：回顾遍历定义
**先序遍历**：根→左→右
**中序遍历**：左→根→右

**先序=中序的条件**：
- 先序：根→左→右
- 中序：左→根→右
- 要使两者相同，必须"根"在两种遍历中位置相同
- 先序根在最前，中序根在中间（左子树后）
- 只有当**左子树为空**时，根才能在两种遍历中都排在最前

### 第三步：逐项判断

**A. 只有左子树 ❌**
```
若所有非叶结点只有左子树：
例：根A有左孩子B，B有左孩子C
     A
    /
   B
  /
 C

先序：A→B→C（根→左→左）
中序：C→B→A（左→左→根）
结论：❌ 序列不同
```

**B. 只有右子树 ✅**
```
若所有非叶结点只有右子树：
例：根A有右孩子B，B有右孩子C
 A
  \
   B
    \
     C

先序：A→B→C（根→右→右）
中序：A→B→C（根→右→右）
因为左子树为空，中序变为：空→根→右 = 根→右

关键：无左子树时，中序的"左"部分为空
中序：空→根→右 = 根→右
先序：根→空→右 = 根→右
结论：✅ 序列相同
```

**C. 结点的度均为1 ❌**
```
度为1包括"只有左孩子"和"只有右孩子"两种情况
如果有结点只有左孩子，则先序≠中序（见选项A）
结论：❌ 不够精确（包含了A的情况）
```

**D. 结点的度均为2 ❌**
```
若所有非叶结点度为2：
例：完全二叉树
     A
    / \
   B   C

先序：A→B→C（根→左→右）
中序：B→A→C（左→根→右）
结论：❌ 序列不同
```

### 第四步：确认答案
所有非叶结点只有右子树，即无左子树。

答案：**B**

## 易错点分析

### 易错点1：混淆"只有右子树"和"度为1"
❌ **错误理解**：度为1就是只有一个孩子，所以C也对
✅ **正确理解**：
- 度为1包括两种：只有左孩子 或 只有右孩子
- 只有"只有右孩子"才能保证先序=中序
- C选项范围太大，包含了错误情况（只有左孩子）

### 易错点2：不理解遍历顺序的本质
❌ **错误想法**：只要结构特殊，先序和中序就可能相同
✅ **正确理解**：
- 先序：根最先访问
- 中序：根在左子树之后访问
- 只有左子树为空时，根才能在两种遍历中都排第一

## 知识点对比

| 树结构 | 先序 | 中序 | 是否相同 |
|--------|------|------|---------|
| 只有左子树 | 根→左 | 左→根 | ❌ 不同 |
| 只有右子树 | 根→右 | 根→右 | ✅ 相同 |
| 完全二叉树 | 根→左→右 | 左→根→右 | ❌ 不同 |
| 单结点 | 根 | 根 | ✅ 相同 |

**通用规律**：先序=中序 ⟺ 所有结点无左子树
