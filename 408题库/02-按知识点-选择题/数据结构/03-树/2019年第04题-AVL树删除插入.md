# 2019年408真题第04题

## 题目

在任意一棵非空平衡二叉树（AVL 树） $T_{1}$ 中，删除某结点 $\nu$ 之后形成平衡二叉树 $T_{2}$，再将 $\nu$ 插入 $T_{2}$ 形成平衡二叉树 $T_{3}$。下列关于 $T_{1}$ 与 $T_{3}$ 的叙述中，正确的是（）。

I. 若 $\nu$ 是 $T_{1}$ 的叶结点，则 $T_{1}$ 与 $T_{3}$ 可能不相同

II. 若 $\nu$ 不是 $T_{1}$ 的叶结点，则 $T_{1}$ 与 $T_{3}$ 一定不同

III. 若 $\nu$ 不是 $T_{1}$ 的叶结点，则 $T_{1}$ 与 $T_{3}$ 一定相同

A. 仅I  
B. 仅II  
C. 仅I、II  
D. 仅I、III

## 答案

A

## 考点分析

### 考点：AVL树的删除和插入操作

本题考查AVL树在删除和插入操作后的结构变化，需要理解旋转调整的影响。

### 核心知识点：

**AVL树的性质**：
- 任意节点的左右子树高度差不超过1
- 插入或删除可能破坏平衡，需要通过旋转调整

**关键理解**：
- 删除节点后可能触发旋转
- 重新插入同一节点时，插入位置可能改变
- 旋转会改变树的结构

### 逐项分析：

**I. 若 $\nu$ 是 $T_{1}$ 的叶结点，则 $T_{1}$ 与 $T_{3}$ 可能不相同（✓ 正确）**

**原因**：
- 删除叶节点可能导致父节点失衡，触发旋转
- 旋转改变了树的结构
- 重新插入 $\nu$ 时，插入位置由新的树结构决定
- $\nu$ 可能被插入到不同的位置

**反例**：
```
原始 T1:      删除叶节点3后 T2:    重新插入3后 T3:
    2              2                    4
   / \            / \                  / \
  1   4          1   4                2   5
     / \              \              / \
    3   5              5            1   3
```
删除3触发右旋后，结构改变，再插入3的位置不同。

**II. 若 $\nu$ 不是 $T_{1}$ 的叶结点，则 $T_{1}$ 与 $T_{3}$ 一定不同（✗ 错误）**

**反例**：
- 删除非叶节点时，用前驱或后继替换
- 如果删除过程没有触发旋转调整
- 重新插入可能恢复原结构

**III. 若 $\nu$ 不是 $T_{1}$ 的叶结点，则 $T_{1}$ 与 $T_{3}$ 一定相同（✗ 错误）**

**反例**：
- 删除非叶节点可能触发多次旋转
- 树结构改变后，重新插入无法恢复原结构

### 总结：

- **叶节点删除**：更容易触发旋转，导致结构不可恢复
- **非叶节点删除**：情况复杂，可能相同也可能不同
- **关键因素**：是否发生旋转调整

### 记忆要点：

- AVL树的删除和插入不满足**可逆性**
- 旋转是单向的结构变化
- 同一组关键字可能构造出不同的AVL树（取决于插入顺序）

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 平衡二叉树T₁删除结点v → T₂
- T₂插入v → T₃
- 问：T₁与T₃的关系

### 第二步：理解AVL树操作特点
- 删除可能触发旋转调整
- 旋转改变树结构
- 重新插入时位置由当前结构决定
- **关键**：操作不满足可逆性

### 第三步：逐项判断

**I. 若v是T₁的叶结点，则T₁与T₃可能不相同 ✅**
```
示例：
T₁:    5           删除3→旋转后T₂:    6
      / \                              / \
     3   6                            5   7
        / \                                
       7   8                               

重新插入3→T₃:    6
                / \
               5   7
              /     
             3       

T₁ ≠ T₃（结构改变）
结论：✅ 正确（可能不相同）
```

**II. 若v不是T₁的叶结点，则T₁与T₃一定不同 ❌**
```
反例：删除非叶结点，无旋转发生
T₁:    5           删除5→用后继6替换
      / \          
     3   7         T₂:    6
        / \              / \
       6   8            3   7
                             \
                              8

重新插入5:  6
           / \
          5   7（可能恢复或不恢复）
           \   \
            3   8

不一定不同，所以"一定不同"错误
结论：❌ 错误
```

**III. 若v不是T₁的叶结点，则T₁与T₃一定相同 ❌**
```
反例：删除非叶结点触发旋转
结构改变后无法恢复
结论：❌ 错误
```

### 第四步：确认答案
仅I正确

答案：**A**

## 易错点分析（补充）

### 易错点1：认为删除和插入互逆
❌ **错误理解**：删除v再插入v，树结构必然恢复
✅ **正确理解**：AVL树的删除和插入不满足可逆性，因为旋转会改变结构，影响后续插入位置

### 易错点2：认为删除非叶结点一定改变结构
❌ **错误想法**：II正确，非叶结点删除必然导致T₁≠T₃
✅ **正确理解**：删除非叶结点的结果不确定，可能相同也可能不同，"一定不同"是错的
