# 森林遍历真题汇总

## 2020年第04题

**元数据**
- 年份：[2020年第4题](../../../01-真题原卷/2020年408真题.md#04)
- 科目：#数据结构
- 知识点：#数据结构/树/森林遍历
- 难度：⭐⭐⭐
- 关联知识：[知识库/数据结构/森林](知识库/数据结构/森林.md)

**题干**

已知森林 F 及与之对应的二叉树 T, 若 F 的先根遍历序列是 a, b, c, d, e, f, 后根遍历序列是 b, a, d, f, e, c, 则 T 的后序遍历序列是（）。

A. b, a, d, f, e, c  B. b, d, f, e, c, a  C. b, f, e, d, c, a  D. f, e, d, c, b, a

**答案**：C

**解题思路**

**关键对应关系**：
- 森林的先根遍历 = 对应二叉树的**先序遍历**
- 森林的后根遍历 = 对应二叉树的**中序遍历**

**已知**：
- 森林F的先根遍历：a, b, c, d, e, f
- 森林F的后根遍历：b, a, d, f, e, c

**转换为二叉树T**：
- T的先序遍历：a, b, c, d, e, f
- T的中序遍历：b, a, d, f, e, c

**步骤1：根据先序和中序构造二叉树T**

**先序**：a, b, c, d, e, f
**中序**：b, a, d, f, e, c

1. 先序第一个a是根
2. 中序找a的位置，左边：b，右边：d, f, e, c
3. 左子树（先序：b，中序：b）→ 只有根b
4. 右子树（先序：c, d, e, f，中序：d, f, e, c）

**右子树继续分解**：
- 先序：c, d, e, f
- 中序：d, f, e, c
- 先序第一个c是右子树的根
- 中序找c，左边：d, f, e，右边：空
- c的左子树（先序：d, e, f，中序：d, f, e）

**c的左子树继续分解**：
- 先序：d, e, f
- 中序：d, f, e
- 先序第一个d是根
- 中序找d，左边：空，右边：f, e
- d的右子树（先序：e, f，中序：f, e）

**d的右子树继续分解**：
- 先序：e, f
- 中序：f, e
- 先序第一个e是根
- 中序找e，左边：f，右边：空
- e的左子树：f

**二叉树T的结构**：
```
      a
     / \
    b   c
       /
      d
       \
        e
       /
      f
```

**步骤2：求二叉树T的后序遍历**

后序遍历：左-右-根

- 从a开始：左子树b → 右子树c → 根a
- b是叶子：b
- c的左子树：d → 根c
  - d的右子树：e → 根d
    - e的左子树：f → 根e
    - f是叶子：f
    - e的后序：f, e
  - d的后序：f, e, d
- c的后序：f, e, d, c
- a的后序：b, f, e, d, c, a

**后序遍历结果**：b, f, e, d, c, a

等等，答案是C(b, f, e, d, c, a)，但选项C是b, f, e, d, c, a...让我检查选项。

哦，我看错了，让我重新看选项：
- C. b, f, e, d, c, a

这正是我计算的结果！但题目选项显示的是"b, f, e, d, c, a"？

让我重新读题目中的选项：
A. b, a, d, f, e, c
B. b, d, f, e, c, a
C. b, f, e, d, c, a
D. f, e, d, c, b, a

我的结果b, f, e, d, c, a对应选项C ✓

**选项分析**：
- ❌ A. b, a, d, f, e, c —— 这是森林的后根遍历（二叉树中序），不是后序
- ❌ B. b, d, f, e, c, a —— 顺序错误
- ✅ C. b, f, e, d, c, a —— 正确的后序遍历
- ❌ D. f, e, d, c, b, a —— 完全反过来了

**易错点分析**

❌ **错误1：混淆森林遍历与二叉树遍历的对应关系**
- 误认为森林的后根遍历 = 二叉树的后序遍历
- ✅ 正确：
  - 森林先根 = 二叉树先序
  - 森林中根（不存在） 
  - 森林后根 = 二叉树中序

❌ **错误2：直接选A选项**
- 误以为森林的后根遍历就是答案
- ✅ 正确：题目问的是二叉树的后序遍历，需要重新计算

❌ **错误3：构造二叉树时出错**
- 先序中序构造时，左右子树划分错误
- ✅ 正确：中序中根节点左边是左子树，右边是右子树

**知识点对比**

| 遍历方式 | 森林F | 对应二叉树T |
|---------|-------|-----------|
| **先根遍历** | 先访问根，再依次遍历子树 | **先序遍历**（根-左-右） |
| **后根遍历** | 先依次遍历子树，再访问根 | **中序遍历**（左-根-右） |
| **中序遍历** | 二叉树特有，森林无此概念 | 左-根-右 |
| **后序遍历** | 不直接对应 | 左-右-根 |

**森林与二叉树转换关键**

| 转换规则 | 说明 |
|---------|------|
| **左孩子右兄弟** | 树的第一个孩子→左孩子，兄弟节点→右孩子 |
| **遍历对应** | 森林先根=二叉树先序，森林后根=二叉树中序 |
| **还原方法** | 已知先序+中序可唯一确定二叉树 |

**本题解题步骤总结**

| 步骤 | 内容 | 本题应用 |
|------|------|---------|
| 1 | 识别对应关系 | 森林先根→二叉树先序，森林后根→二叉树中序 |
| 2 | 构造二叉树 | 先序a,b,c,d,e,f + 中序b,a,d,f,e,c → 构造树 |
| 3 | 求目标遍历 | 对构造的二叉树进行后序遍历 |

**构造过程可视化**

```
先序：a, b, c, d, e, f
中序：b, a, d, f, e, c

根：a
├─左（先序b，中序b）：b
└─右（先序c,d,e,f，中序d,f,e,c）
   根：c
   ├─左（先序d,e,f，中序d,f,e）
   │  根：d
   │  └─右（先序e,f，中序f,e）
   │     根：e
   │     └─左：f
   └─右：无
```

**后序遍历路径**：
1. 最左下f → 2. e → 3. d → 4. c → 5. b → 6. a
结果：b, f, e, d, c, a

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 森林F的先根遍历：a,b,c,d,e,f
- 森林F的后根遍历：b,a,d,f,e,c
- 问：对应二叉树T的后序遍历

### 第二步：识别对应关系
**关键对应**：
- 森林先根遍历 = 二叉树先序遍历
- 森林后根遍历 = 二叉树中序遍历

转换：
- T的先序：a,b,c,d,e,f
- T的中序：b,a,d,f,e,c

### 第三步：构造二叉树T
先序+中序可唯一确定二叉树：
```
先序a→根是a
中序中a的左边：b（左子树）
中序中a的右边：d,f,e,c（右子树）

构造结果：
      a
     / \
    b   c
       /
      d
       \
        e
       /
      f
```

### 第四步：求后序遍历
后序：左→右→根
- a：左(b)→右(c的后序)→a
- c：左(d的后序)→c
- d：右(e的后序)→d
- e：左(f)→e

后序：b, f, e, d, c, a

### 第五步：逐项判断

**A. b,a,d,f,e,c ❌**
```
这是森林的后根遍历（二叉树中序）
不是后序遍历
结论：❌
```

**B. b,d,f,e,c,a ❌**
```
顺序不对
结论：❌
```

**C. b,f,e,d,c,a ✅**
```
完全符合后序遍历结果
结论：✅ 正确
```

**D. f,e,d,c,b,a ❌**
```
完全反向
结论：❌
```

### 第六步：确认答案
答案：**C**

## 易错点分析（补充）

### 易错点1：对应关系记错
❌ **错误理解**：森林后根遍历 = 二叉树后序遍历
✅ **正确理解**：
- 森林先根 = 二叉树先序
- 森林后根 = 二叉树中序
- 二叉树后序需要重新计算

### 易错点2：直接选A
❌ **错误做法**：看到A选项就是森林的后根遍历，直接选A
✅ **正确做法**：题目问的是二叉树的后序遍历，不是森林的后根遍历
