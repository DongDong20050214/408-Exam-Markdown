# 2011年第5题

**元数据**
- 年份：2011年第5题
- 科目：#数据结构
- 难度：⭐⭐

**题干**

若一棵二叉树的前序遍历序列和后序遍历序列分别为 1, 2, 3, 4 和 4, 3, 2, 1，则该二叉树的中序遍历序列不会是（）。

A. 1,2,3,4 

B. $2, 3, 4, 1$ 

C. $3,2,4,1$ 

D. 4, 3, 2, 1 

**答案**：C

**考点分析**

本题考查二叉树遍历序列的性质。前序遍历是"根-左-右"，后序遍历是"左-右-根"。从前序序列可知根为1，从后序序列可知1在最后，确认根为1。由于前序中1后面是2，说明2是1的左子树或右子树的根；后序中2在1前面倒数第二个，说明2也是靠近根的子树。通过分析，可以构造出多种满足条件的二叉树，对应的中序序列包括A、B、D选项。但选项C (3,2,4,1) 无法由任何满足给定前序和后序的二叉树产生，因为它违反了遍历的基本结构关系。

---

## 解题思路

### 第一步：确定根节点

**前序**：1, 2, 3, 4（根-左-右）
**后序**：4, 3, 2, 1（左-右-根）

根节点：**1**

### 第二步：分析可能的树结构

前序中1后是2，后序中2在1前，说明2是1的子树根。

**关键观察**：前序和后序无法唯一确定树，可能有多种结构。

**可能结构1**：全部右偏
```
    1
     \
      2
       \
        3
         \
          4
```
中序：1,2,3,4 ✓（对应选项A）

**可能结构2**：全部左偏
```
        1
       /
      2
     /
    3
   /
  4
```
中序：4,3,2,1 ✓（对应选项D）

**可能结构3**：混合结构
```
    1
     \
      2
     / \
    3   (无)
   /
  4
```
前序：1,2,3,4 ✓
后序：4,3,2,1 ✓
中序：1,4,3,2（不在选项中）

再尝试：
```
      1
     /
    2
   / \
  (无) 3
       \
        4
```
前序：1,2,3,4 ✓
后序：4,3,2,1 ✓
中序：2,3,4,1（不在选项中）

**尝试构造选项B**：2,3,4,1
```
    1
   /
  2
   \
    3
     \
      4
```
前序：1,2,3,4 ✓
后序：4,3,2,1 ✓
中序：2,3,4,1 ✓（对应选项B）

### 第三步：验证选项C不可行

**选项C**：3,2,4,1

尝试构造使中序为3,2,4,1的树：
- 中序：3在最前→3是最左节点
- 1在最后→1是根
- 但前序1,2,3,4中2在3前，说明2在3的上方
- 中序中2在3后、4前，说明2在3和4之间

矛盾分析：
- 前序中2排第2，说明2是靠近根的节点
- 若中序为3,2,4,1，2应该在3之后，说明2在3的右侧
- 但这会导致前序中3应该在2之前（左子树先访问）
- 与前序1,2,3,4矛盾

**无法构造出满足条件的树使中序为3,2,4,1**

### 第四步：逐项分析

**A. 1,2,3,4 ✅** - 可行（右偏树）
**B. 2,3,4,1 ✅** - 可行（混合结构）
**C. 3,2,4,1 ❌** - 不可行（矛盾）
**D. 4,3,2,1 ✅** - 可行（左偏树）

## 易错点分析

### 易错点1：认为前序+后序可唯一确定树
❌ 错误理解：两个遍历序列一定能唯一确定树结构
✅ 正确理解：前序+后序可能对应多种树结构，只有前序/后序+中序才能唯一确定

### 易错点2：没有验证所有选项
❌ 错误理解：找到一个可行的树结构就认为其他都不可行
✅ 正确理解：需要尝试构造每个选项对应的树，验证是否满足前序和后序

### 易错点3：构造树时违背遍历规则
❌ 错误理解：随意安排节点位置而不考虑遍历顺序约束
✅ 正确理解：前序（根左右）、中序（左根右）、后序（左右根）的顺序必须严格满足

## 知识点对比

| 遍历组合 | 能否唯一确定 | 原因 |
|---------|------------|------|
| **前序+中序** | ✓ 唯一 | 前序定根，中序分左右 |
| **后序+中序** | ✓ 唯一 | 后序定根，中序分左右 |
| **前序+后序** | ✗ 不唯一 | 无法区分左右子树边界 |

**本题关键**：前序+后序不唯一，但可以排除不可能的中序序列。
