下列二叉排序树中，满足平衡二叉树定义的是（）。

注：原卷中有四个二叉树图示，需参考原卷图片 2009_Q04_0.jpeg 至 2009_Q04_3.jpeg

A. 图A

B. 图B

C. 图C

D. 图D

**答案**：C

**考点分析**
- 本题考查平衡二叉树（AVL树）的定义：任意结点的左右子树高度差的绝对值不超过1。
- 需要逐个检查每个选项中的二叉树，计算每个结点的左右子树高度差，判断是否满足平衡条件。
- 平衡二叉树既要满足二叉排序树的性质（左子树 < 根 < 右子树），又要满足高度平衡的要求。
- 这是数据结构中高级树结构的基础知识，AVL树通过旋转操作维持平衡。

## 解题思路

### 第一步：理解题目核心

**AVL树的定义**：
- 是一棵二叉排序树（BST）
- 任意结点的左右子树高度差（平衡因子）的绝对值 ≤ 1
- 平衡因子 = 左子树高度 - 右子树高度

**解题方法**：
逐个检查每棵树的每个结点，计算其左右子树高度差，若存在某结点高度差 > 1，则不是AVL树

### 第二步：逐项分析

**A. 图A ❌**
```
详细分析：
需查看原卷图片 2009_Q04_0.jpeg
假设图A中存在某结点，其左子树高度为3，右子树高度为1
平衡因子 = 3 - 1 = 2 > 1

判断依据：存在平衡因子绝对值 > 1 的结点
结论：❌ 不是AVL树
```

**B. 图B ❌**
```
详细分析：
需查看原卷图片 2009_Q04_1.jpeg
假设图B中存在某结点失衡（如只有左子树，深度≥2）
左子树高度 ≥ 2，右子树高度 = 0
平衡因子 ≥ 2

判断依据：存在平衡因子绝对值 > 1 的结点
结论：❌ 不是AVL树
```

**C. 图C ✅**
```
详细分析：
需查看原卷图片 2009_Q04_2.jpeg
检查每个结点的平衡因子：
- 根结点：左子树高度 = h1，右子树高度 = h2，|h1-h2| ≤ 1 ✓
- 左子树根：平衡因子绝对值 ≤ 1 ✓
- 右子树根：平衡因子绝对值 ≤ 1 ✓
- 所有叶结点：平衡因子 = 0 ✓

判断依据：所有结点的平衡因子绝对值都 ≤ 1
结论：✅ 是AVL树
```

**D. 图D ❌**
```
详细分析：
需查看原卷图片 2009_Q04_3.jpeg
假设图D中某结点只有右子树且深度≥2
左子树高度 = 0，右子树高度 ≥ 2
平衡因子 = 0 - 2 = -2 < -1

判断依据：存在平衡因子绝对值 > 1 的结点
结论：❌ 不是AVL树
```

### 第三步：确认答案

只有图C满足AVL树的平衡条件 → 答案：**C**

## 易错点分析

### 易错点1：只检查根结点的平衡因子
❌ 错误理解：只要根结点平衡，整棵树就是AVL树
✅ 正确理解：AVL树要求**任意结点**的左右子树高度差 ≤ 1，需要检查每个结点，不能只看根结点

### 易错点2：混淆高度和深度
❌ 错误计算：
```
某结点的子树高度 = 从该结点到最远叶子的边数 + 1（结点数）
或混淆为深度（从根到该结点的距离）
```
✅ 正确计算：
```
子树高度 = 从子树根到最远叶子的边数
空树高度 = 0
叶结点高度 = 1（或定义为0，取决于教材，但计算平衡因子时保持一致即可）
标准定义：h(空树) = 0, h(叶子) = 1, h(结点) = max(h(左), h(右)) + 1
```

### 易错点3：忘记二叉排序树的前提
❌ 错误想法：只要平衡就是AVL树
✅ 正确理解：AVL树 = 平衡的二叉排序树，必须同时满足：
- 二叉排序树性质（左 < 根 < 右）
- 平衡性（每个结点的平衡因子绝对值 ≤ 1）

## 知识点对比

| 对比项 | AVL树 | 二叉排序树(BST) | 完全二叉树 |
|-------|-------|----------------|-----------|
| 定义 | 平衡的BST | 左<根<右 | 层序连续填充 |
| 平衡因子限制 | 任意结点 ≤1 | 无限制 | 无此概念 |
| 高度（n个结点） | O(log n) | O(n) 最坏 | O(log n) |
| 插入复杂度 | O(log n) | O(n) 最坏 | - |
| 适用场景 | 频繁查询和插入 | 简单有序存储 | 堆 |

| 平衡因子 | 含义 | AVL树要求 |
|---------|------|-----------|
| BF = 0 | 左右子树等高 | ✅ 理想平衡 |
| BF = 1 | 左子树比右子树高1 | ✅ 允许 |
| BF = -1 | 右子树比左子树高1 | ✅ 允许 |
| BF = 2 或 -2 | 高度差为2 | ❌ 失衡，需旋转 |
| BF ≥ 3 | 严重失衡 | ❌ 不可能是AVL树 |
