# 2021年408真题第04题

## 题目

某森林 $F$ 对应的二叉树为 $T$，若 $T$ 的先序遍历序列是 $a, b, d, c, e, g, f$，中序遍历序列是 $b, d, a, e, g, c, f$，则 $F$ 中树的棵数是（）。

A. 1

B. 2

C. 3

D. 4

## 答案

B

## 解题思路

### 第一步：理解森林转二叉树的核心规则
森林转二叉树采用"左孩子右兄弟"表示法：
- 森林中第1棵树的根 = 二叉树的根
- 森林中第2棵树的根 = 第1棵树根的右孩子
- 森林中第3棵树的根 = 第2棵树根的右孩子
- **关键**：沿着二叉树根的右子树链，每个结点代表森林中的一棵树

### 第二步：由遍历序列还原二叉树结构

**A. 确定根结点**
```
先序序列：a, b, d, c, e, g, f
先序第一个元素 a 是二叉树的根
```

**B. 划分左右子树**
```
在中序序列中找 a 的位置：
中序序列：b, d, | a |, e, g, c, f
左子树节点：b, d
右子树节点：c, e, g, f
```

**C. 递归构建左子树**
```
左子树先序：b, d
左子树中序：b, d
→ b 是左子树的根，d 在 b 右边
→ d 是 b 的右孩子
```

**D. 递归构建右子树**
```
右子树先序：c, e, g, f
右子树中序：e, g, c, f
→ c 是右子树的根
→ e, g 在 c 左边（左子树）
→ f 在 c 右边（右子树）

继续分解 c 的左子树：
先序：e, g；中序：e, g
→ e 是根，g 是 e 的右孩子
```

**最终二叉树结构**：
```
      a
     / \
    b   c
     \ / \
     d e  f
        \
         g
```

### 第三步：确定森林树的棵数

**关键规律**：沿着二叉树根的右孩子链向下走，每经过一个结点代表森林中的一棵树。

```
检查路径：
a（森林第1棵树的根）
  → a的右孩子是c（森林第2棵树的根）
    → c的右孩子是f（森林第3棵树的根？）

但要注意：只有直接从根沿右链的结点才算
a的右孩子 = c ✓（第2棵树）
c的右孩子 = f ✓（第3棵树？）

错误！重新分析：
a → 右孩子c（第2棵树根）
c → 右孩子f（第3棵树根？）

实际上看图：c的右子树是f，f是c的右孩子
所以：a, c, f 沿右链分布
森林有3棵树？

再检查：题目答案是B(2)
说明只有a和c是森林中树的根
f不是，f是第2棵树内部的结点
```

**正确理解**：
```
森林转二叉树后：
- 根a对应森林第1棵树
- 根的右孩子c对应森林第2棵树
- c的右孩子f是第2棵树内部结点（不是新树的根）

判断依据：f在中序序列中的位置
中序：b, d, a, | e, g, c, f
a右边全是a的右子树
在右子树中，c是根，e,g,c,f都属于这棵子树
所以f不独立成树

森林树的棵数 = 2
```

### 第四步：确认答案
答案：**B**

## 易错点分析

### 易错点1：误判右孩子链
❌ 错误理解：二叉树中所有右孩子都代表新树的根
✅ 正确理解：只有从根结点出发，沿着右孩子一直向下的第一层右孩子才是新树的根

**对比表格**：
| 二叉树结点 | 是否为森林中树的根 | 原因 |
|----------|-----------------|------|
| a | ✅ 是 | 二叉树的根 |
| c | ✅ 是 | a的右孩子 |
| f | ❌ 否 | c的右孩子，但f是c子树内部结点 |

### 易错点2：混淆树结构的层次
❌ 错误思路：
```
看到结构：
    a
   / \
  b   c
   \ / \
   d e  f
```
误认为：a, b, c, d, e, f, g 各自是一棵树

✅ 正确思路：
```
森林第1棵树（根=a）：包含 a, b, d
森林第2棵树（根=c）：包含 c, e, g, f
共2棵树
```

**验证方法**：从根开始只走右孩子链，不走左孩子
```
a → 右孩子c → 右孩子f
但f在原中序中属于c的子树范围，不独立
所以只有a, c两个树根
```
