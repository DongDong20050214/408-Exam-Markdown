# 2010年第3题

**元数据**
- 年份：2010年第3题
- 科目：#数据结构
- 难度：⭐⭐

**题干**

下列线索二叉树中（用虚线表示线索），符合后序线索树定义的是（）。

![[attachments/2010/2010_Q03_2.jpeg]]

A. 选项A图  
B. 选项B图  
C. 选项C图  
D. 选项D图

（注：原题含图片，需参考原卷图片）

**答案**：C

**考点分析**

本题考查后序线索二叉树的定义和性质。后序遍历顺序是：左子树→右子树→根节点。

在后序线索二叉树中：
- 若节点的左子树为空，左指针指向该节点在后序序列中的前驱
- 若节点的右子树为空，右指针指向该节点在后序序列中的后继
- 需要根据后序遍历序列来判断线索的正确性

考生需要掌握线索二叉树的基本概念，能够根据遍历顺序判断线索指向是否正确，这是对二叉树遍历和线索化理解的综合考查。

## 解题思路

### 第一步：理解题目核心

**后序线索二叉树定义**：
- 后序遍历顺序：左子树 → 右子树 → 根节点
- 线索化规则：
  - 若结点左子树为空，左指针指向**后序前驱**
  - 若结点右子树为空，右指针指向**后序后继**
- 图中虚线表示线索，实线表示正常的左右孩子指针

**解题关键**：
1. 先确定树的后序遍历序列
2. 确定每个结点的后序前驱和后继
3. 检查虚线（线索）指向是否符合定义

### 第二步：逐项分析

根据图片描述，树结构为：
```
    a
   / \
  b   c
 /
d
```

**后序遍历序列**：d → b → c → a

**前驱后继关系**：
- d的前驱：无，后继：b
- b的前驱：d，后继：c
- c的前驱：b，后继：a
- a的前驱：c，后继：无

**A. 选项A ❌**
```
详细分析：
需查看原卷图片，假设线索指向错误
例如：某结点的左线索应指向前驱，却指向了其他结点

判断依据：线索指向与后序遍历的前驱后继关系不符
结论：❌ 不符合后序线索树定义
```

**B. 选项B ❌**
```
详细分析：
需查看原卷图片，检查线索指向
可能错误：某结点的右线索应指向后继，却指错了

判断依据：存在线索指向错误的情况
结论：❌ 不符合定义
```

**C. 选项C ✅**
```
详细分析：
根据后序序列 d → b → c → a，检查线索：
- d无左孩子 → 左线索应指向前驱（无，指向空或头结点）
- d无右孩子 → 右线索应指向后继b ✓
- b的左孩子是d（实线），右孩子空 → 右线索指向后继c ✓
- c无左右孩子 → 左线索指向前驱b，右线索指向后继a ✓
- a的左孩子是b，右孩子是c（实线）→ 无需线索
或a的右线索指向后继（无，指向空或头结点）✓

判断依据：所有线索指向都符合后序遍历的前驱后继关系
结论：✅ 符合后序线索树定义
```

**D. 选项D ❌**
```
详细分析：
需查看原卷图片，检查是否有线索指向错误
可能问题：混淆了前序或中序的前驱后继关系

判断依据：线索指向不符合后序遍历顺序
结论：❌ 不符合定义
```

### 第三步：确认答案

只有选项C的线索指向完全符合后序遍历的前驱后继关系 → 答案：**C**

## 易错点分析

### 易错点1：混淆不同遍历方式的前驱后继
❌ 错误理解：用中序或前序的遍历顺序判断前驱后继
✅ 正确理解：后序线索树必须按**后序遍历**顺序（左→右→根）确定前驱后继

| 遍历方式 | 本题树的序列 | 根结点a的前驱 |
|---------|------------|--------------|
| 前序(NLR) | a,b,d,c | 无（第一个） |
| 中序(LNR) | d,b,a,c | b |
| 后序(LRN) | d,b,c,a ✅ | c ✅ |

### 易错点2：忘记线索化的条件
❌ 错误想法：所有结点都需要线索
✅ 正确理解：只有当**左子树为空**时才用左线索，**右子树为空**时才用右线索

```
结点有孩子 → 用实线指向孩子（不需要线索）
结点孩子为空 → 用虚线指向前驱/后继（线索）
```

### 易错点3：不理解"前驱"和"后继"的定义
❌ 错误理解：前驱=父结点，后继=子结点
✅ 正确理解：前驱/后继是指**在遍历序列中**紧邻的前一个/后一个结点

| 概念 | 定义 | 示例（后序序列：d,b,c,a） |
|------|------|-------------------------|
| 前驱 | 遍历序列中的前一个结点 | b的前驱是d |
| 后继 | 遍历序列中的后一个结点 | b的后继是c |
| 父结点 | 树结构中的上层结点 | d的父结点是b |

## 知识点对比

| 线索类型 | 遍历顺序 | 左线索指向 | 右线索指向 | 特点 |
|---------|---------|-----------|-----------|------|
| 前序线索树 | 根→左→右 | 前序前驱 | 前序后继 | 根最先访问 |
| 中序线索树 | 左→根→右 | 中序前驱 | 中序后继 | 最常用 |
| 后序线索树 ✅ | 左→右→根 | 后序前驱 | 后序后继 | 根最后访问 |

| 指针类型 | 表示方式 | 指向内容 | 条件 |
|---------|---------|---------|------|
| 实线（正常指针） | 实线箭头 | 左/右孩子结点 | 孩子存在 |
| 虚线（线索） | 虚线箭头 | 前驱/后继结点 | 孩子为空 |
