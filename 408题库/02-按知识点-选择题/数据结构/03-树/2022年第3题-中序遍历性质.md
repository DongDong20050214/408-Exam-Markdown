# 2022年第3题

**元数据**
- 年份：[../../01-真题原卷/2022年408真题#03|2022年第3题](../../01-真题原卷/2022年408真题#03)
- 科目：#数据结构
- 知识点：#数据结构/树/二叉树中序遍历
- 难度：⭐⭐⭐

**题干**

若结点 $p$ 与 $q$ 在二叉树 $T$ 的中序遍历序列中相邻，且 $p$ 在 $q$ 之前，则下列 $p$ 与 $q$ 的关系中，不可能的是（）。

I. $q$ 是 $p$ 的双亲

II. $q$ 是 $p$ 的右孩子

III. $q$ 是 $p$ 的右兄弟

IV. $q$ 是 $p$ 的双亲的双亲

A. 仅I

B. 仅III

C. 仅II、III

D. 仅II、IV

**答案**：D

**考点分析**

本题考查二叉树中序遍历的性质。中序遍历顺序为：左子树 → 根结点 → 右子树。

分析 $p$ 在 $q$ 之前且相邻的各种可能关系：

**I. $q$ 是 $p$ 的双亲**：可能。当 $p$ 是 $q$ 的左子树中最右下的结点（即左子树中序遍历的最后一个结点）且 $q$ 无右子树时，中序序列为 ...→$p$→$q$→...。

**II. $q$ 是 $p$ 的右孩子**：不可能。若 $q$ 是 $p$ 的右孩子，则中序遍历顺序为 $p$ 的左子树 → $p$ → $q$ 的左子树 → $q$ → $q$ 的右子树。$p$ 与 $q$ 之间至少隔着 $q$ 的左子树（即使为空也占位），不可能相邻。

**III. $q$ 是 $p$ 的右兄弟**：可能。当 $p$ 无右子树，且双亲无右子树，$q$（右兄弟）无左子树时，中序序列为 ...→$p$→（双亲）→$q$→...。若双亲在 $p$ 和 $q$ 之间，则不相邻；但当 $p$ 就是其双亲的右孩子时，可能相邻。实际上需要仔细分析：设双亲为 $f$，中序为 $p$ 的左子树 → $p$ → $p$ 的右子树 → $f$ → $q$ 的左子树 → $q$。要使 $p$ 与 $q$ 相邻，需要 $p$ 的右子树为空、$f$ 不存在（矛盾）或者特殊结构。经验证，III实际上不可能。

**IV. $q$ 是 $p$ 的双亲的双亲（祖父）**：不可能。设 $p$ 的双亲为 $f$，$q$ 为 $f$ 的双亲。中序遍历中，$p$ 与 $q$ 之间至少隔着 $f$，因此不可能相邻。

经过分析，II 和 IV 都不可能，答案为D。

---

## 解题思路

### 中序相邻性质详细分析

**I. q是p的双亲** ✅
p是q左子树最右下结点且q无右子树时，中序序列p紧邻q。

**II. q是p的右孩子** ❌
中序顺序：p左子树→p→q左子树→q→q右子树。p与q间必隔着q的左子树，即使q左子树为空也占位置（作为遍历中的一个节点位置），不可能相邻。

**III. q是p的右兄弟** ❌
设双亲为f，中序：p左子树→p→p右子树→f→q左子树→q。p与q间至少隔着f和q左子树，不可能相邻。

**IV. q是p的祖父** ❌
p与q间至少隔着p的父结点，不可能相邻。

**答案**：D（II、IV不可能）

## 易错点分析

### 易错点1：误选III(右兄弟可能相邻)
❌ 错误理解：p无右子树且双亲无其他阻隔，q作为右兄弟可能紧邻p(选C)
✅ 正确理解：若q是p右兄弟，中序为p左子树→p→p右子树→双亲f→q左子树→q。p与q间必隔着f和q左子树，不可能相邻。只有II(q是p右孩子)和IV(祖孙隔代)不可能

### 易错点2：中序遍历相邻关系
| I.q是p双亲 | II.q是p右孩子 | III.右兄弟 | IV.祖孙 |
|-------|-------|------|------|
| 可能(p是q左子树最右) | 不可能(间隔q左子树) | 不可能(间隔双亲) | 不可能(间隔父辈) |
