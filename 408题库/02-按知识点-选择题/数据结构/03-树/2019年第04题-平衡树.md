# 2019年第4题

**元数据**
- 年份：2019年第4题
- 难度：⭐⭐

**题干**：在任意一棵非空平衡二叉树（AVL 树） $T_{1}$ 中，删除某结点 $\nu$ 之后形成平衡二叉树 $T_{2}$ ，再将 $\nu$ 插入 $T_{2}$ 形成平衡二叉树 $T_{3}$ 。下列关于 $T_{1}$ 与 $T_{3}$ 的叙述中，正确的是（）。

I. 若 $\nu$ 是 $T_{1}$ 的叶结点，则 $T_{1}$ 与 $T_{3}$ 可能不相同

II. 若 $\nu$ 不是 $T_{1}$ 的叶结点，则 $T_{1}$ 与 $T_{3}$ 一定不同

III. 若 $\nu$ 不是 $T_{1}$ 的叶结点，则 $T_{1}$ 与 $T_{3}$ 一定相同

A. 仅I
B. 仅II
C. 仅I、II
D. 仅 I、III

**答案**：A

**考点分析**：本题考查AVL树的动态操作特性。当删除和插入叶结点时，可能触发不同的旋转操作，导致树的结构改变。I正确：删除叶结点可能触发旋转，再插入同一结点时可能在不同位置，导致结构不同。II和III都不对：非叶结点的删除和插入情况更复杂，不能简单判断T1和T3是否相同。

---

## 解题思路

### 第一步：理解题目核心

**操作流程**：
1. T₁：原始AVL树
2. 删除结点v → T₂（仍是AVL树）
3. 插入结点v → T₃（仍是AVL树）
4. 比较T₁与T₃

**关键问题**：删除和插入可能触发旋转，改变树结构

### 第二步：分析两种情况

**情况1：v是叶结点**
```
示例1：T₁与T₃相同的情况
T₁:      4
        / \
       2   6
      / \
     1   3
     
删除叶结点1 → T₂:
        4
       / \
      2   6
       \
        3

插入1 → T₃:
        4
       / \
      2   6
     / \
    1   3
    
结果：T₁ = T₃

示例2：T₁与T₃不同的情况
T₁:      4
        / \
       2   5
      / \
     1   3
     
删除叶结点5 → T₂失衡:
        4
       / 
      2   
     / \
    1   3
    
T₂变为:  2
        / \
       1   4
          /
         3

插入5 → T₃:
        2
       / \
      1   4
         / \
        3   5

结果：T₁ ≠ T₃（发生了旋转）
```

**情况2：v不是叶结点**
```
不能一概而论，需具体分析
```

### 第三步：逐项分析

**I. 若v是T₁的叶结点，则T₁与T₃可能不同 ✅**
```
详细分析：
删除叶结点可能导致某些子树失衡，触发旋转
旋转后树结构改变
再插入v时，按AVL规则找位置，可能不回到原位置
甚至可能在不同位置上

结论：✅ 正确（"可能"不同，不是"一定"）
```

**II. 若v不是T₁的叶结点，则T₁与T₃一定不同 ❌**
```
分析：
非叶结点情况复杂，不能断言"一定"不同
存在某些情况下T₁ = T₃

反例：简单情况下可能相同

结论：❌ 错误（"一定"太绝对）
```

**III. 若v不是T₁的叶结点，则T₁与T₃一定相同 ❌**
```
分析：
与II矛盾，且同样太绝对
非叶结点删除更复杂，更可能导致结构变化

结论：❌ 错误
```

### 第四步：确认答案

仅I正确 → 答案：**A**

## 易错点分析

### 易错点1：类比BST的结论
❌ 错误理解：参考BST(叶结点删除再插入会回原位)，认为AVL树也有相同性质
✅ 正确理解：AVL树有平衡约束(左右子树高度差≤1)，删除叶结点可能导致祖先失衡，触发旋转改变树结构。旋转后再插入相同值结点时，AVL插入算法按新树结构寻找位置，可能落在不同位置。因此T₁与T₃可能不同

### 易错点2：逻辑量词"可能"与"一定"的区别
| 命题形式 | 要求 | 判断方法 |
|-------|-------|------|
| "可能" | 存在一个例子满足 | 找一个情况即可 |
| "一定" | 所有情况都满足 | 需证明或找反例 |

## 知识点对比

| 树类型 | 叶结点删除再插入 | 非叶结点删除再插入 | 原因 |
|--------|----------------|-------------------|------|
| BST | 叶：相同；非叶：不同 | 位置被替代 | 无旋转 |
| AVL | 可能不同 | 不确定 | 可能旋转 |

| 命题形式 | 要求 | 判断方法 |
|---------|------|---------|
| "可能" | 存在一个反例即可 | 找到一个情况满足 |
| "一定" | 所有情况都满足 | 需证明或找反例否定 |

| 选项 | 命题 | 是否绝对 | 正误 |
|------|------|---------|------|
| I | v是叶→可能不同 | 否 | ✅ |
| II | v非叶→一定不同 | 是 | ❌ |
| III | v非叶→一定相同 | 是 | ❌ |
