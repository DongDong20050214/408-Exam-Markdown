# 2020年第4题

**元数据**
- 年份：[../../01-真题原卷/2020年408真题#04|2020年第4题](../../01-真题原卷/2020年408真题#04|2020年第4题.md)
- 科目：#数据结构
- 知识点：#数据结构/树/森林与二叉树转换
- 难度：⭐⭐⭐

**题干**

已知森林 $F$ 及与之对应的二叉树 $T$，若 $F$ 的先根遍历序列是 $a, b, c, d, e, f$，后根遍历序列是 $b, a, d, f, e, c$，则 $T$ 的后序遍历序列是（）。

A. $b, a, d, f, e, c$ 

B. b, d, f, e, c, a 

C. $b, f, e, d, c, a$ 

D. $f, e, d, c, b, a$ 

**答案**：C

**考点分析**

本题考查森林与二叉树的转换以及遍历序列的对应关系。

关键对应关系：
1. 森林的先根遍历 = 对应二叉树的先序遍历
2. 森林的后根遍历 = 对应二叉树的中序遍历

已知条件：
- 森林先根遍历：a, b, c, d, e, f（对应二叉树先序）
- 森林后根遍历：b, a, d, f, e, c（对应二叉树中序）

求解二叉树后序遍历：
根据先序和中序确定二叉树结构：
- 先序第一个a是根，在中序中找a，其左侧b是左子树，右侧d,f,e,c是右子树
- 左子树：先序b，中序b，只有一个节点b
- 右子树：先序c,d,e,f，中序d,f,e,c
  - c是右子树的根，在中序中找c，左侧d,f,e，右侧无
  - 左子树：先序d,e,f，中序d,f,e
    - d是根，左侧无，右侧f,e
    - 右子树：先序e,f，中序f,e
      - e是根，左侧f
      
构建出的二叉树后序遍历：左-右-根
- 访问顺序：b, f, e, d, c, a

答案为C。

---

## 解题思路

### 关键对应关系

**核心公式**：
- 森林先根遍历 = 对应二叉树先序遍历
- 森林后根遍历 = 对应二叉树中序遍历

**已知**：
- 二叉树先序：a, b, c, d, e, f
- 二叉树中序：b, a, d, f, e, c

**根据先序中序还原二叉树**：
先序第一个a是根，中序中a左边是b（左子树），右边是d,f,e,c（右子树）。递归还原完整结构。

**求二叉树后序**：
按照"左右根"顺序遍历还原的二叉树，得到后序序列。

**答案**：C (b, f, e, d, c, a)

## 易错点分析

### 易错点1：遗忘遍历对应关系
❌ 错误理解：森林先根=二叉树先序，森林中序=二叉树中序，直接用二叉树中序做答
✅ 正确理解：森林后根遍历=二叉树中序遍历。已知森林先根a,b,c,d,e,f(对应二叉树先序)和森林后根b,a,d,f,e,c(对应二叉树中序)，可还原二叉树结构，再求二叉树后序b,f,e,d,c,a

### 易错点2：遍历序列对应表
| 森林遍历 | 对应二叉树遍历 | 记忆 |
|-------|-------|------|
| 先根 | 先序 | 根-左-右均优先根 |
| 后根 | 中序 | 后根访问顺序=中序 |
| 中根 | 不存在对应 | 森林无中根概念 |
