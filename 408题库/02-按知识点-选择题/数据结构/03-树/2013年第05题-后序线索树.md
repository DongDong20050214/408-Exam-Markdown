# 后序线索树真题汇总

## 2013年第05题

**元数据**
- 年份：[[../../01-真题原卷/2013年408真题#05|2013年第5题]]
- 科目：#数据结构
- 知识点：#数据结构/树/线索二叉树 #后序
- 难度：⭐⭐⭐
- 关联知识：[[知识库/数据结构/线索二叉树]]

**题干**

若 X 是后序线索二叉树中的叶结点，且 X 存在左兄弟结点 Y，则 X 的右线索指向的是（ ）。

A. X 的父结点  B. 以 Y 为根的子树的最左下结点  C. X 的左兄弟结点 Y  D. 以 Y 为根的子树的最右下结点

**答案**：A

**考点分析**
- 后序遍历：左右根
- X后继是父节点

---

## 解题思路

### 第一步：理解题目核心

**题目条件**：
- 后序线索二叉树
- X是叶结点
- X有左兄弟Y
- 求：X的右线索指向谁

**后序遍历规则**：左子树 → 右子树 → 根

**线索化规则**：
- 右孩子为空 → 右线索指向后序后继

### 第二步：分析树结构和后序序列

**树结构**（X是叶子，Y是X的左兄弟）：
```
      父结点P
      /      \
     Y        X
    / \
   ...  ...
```

**后序遍历顺序**：
1. 遍历Y的左子树
2. 遍历Y的右子树
3. 访问Y
4. 遍历X的左子树（X是叶子，无左子树）
5. 遍历X的右子树（X是叶子，无右子树）
6. 访问X
7. 访问父结点P

**X的后序后继**：
- X是右子树的根（叶子）
- 后序：左 → 右 → 根
- X遍历完后，下一个是父结点P

### 第三步：逐项分析

**A. X的父结点 ✅**
```
详细分析：
后序遍历顺序：
- Y的子树全部
- Y本身
- X（叶子，无子树）
- 父结点P

X的后序后继就是父结点P

判断依据：后序遍历规则，右子树遍历完后访问根
结论：✅ 正确
```

**B. 以Y为根的子树的最左下结点 ❌**
```
详细分析：
Y子树的最左下结点在后序中很早就被访问
不可能是X的后继（X在Y之后访问）

判断依据：后序中Y子树的结点都在Y之前，而Y在X之前
结论：❌ 错误
```

**C. X的左兄弟结点Y ❌**
```
详细分析：
后序遍历：先左子树（Y及其子树），再右子树（X）
Y在X之前被访问，不是X的后继

判断依据：Y是X的前驱，不是后继
结论：❌ 错误
```

**D. 以Y为根的子树的最右下结点 ❌**
```
详细分析：
Y子树的最右下结点也在Y被访问之前访问
仍然在X之前，不是X的后继

判断依据：Y子树的所有结点都在X之前
结论：❌ 错误
```

### 第四步：确认答案

X的右线索指向父结点 → 答案：**A**

## 易错点分析

### 易错点1：混淆前驱和后继
❌ 错误理解：X的左兄弟Y在X附近，可能是X的后继
✅ 正确理解：后序遍历中，左子树先于右子树，Y（左子树）在X（右子树）之前，Y是X的前驱方向

### 易错点2：不理解后序遍历的顺序
❌ 错误想法：后序是"根→左→右"
✅ 正确理解：后序是"左→右→根"，子树遍历完才访问根

**后序遍历示例**：
```
树结构：
      P
     / \
    Y   X
   / \
  A   B

后序序列：A, B, Y, X, P
X的后继：P（父结点）
```

### 易错点3：混淆不同遍历的线索指向
❌ 错误类比：中序线索树的规则
✅ 正确理解：不同遍历方式，后继不同

| 遍历方式 | X的后继 | 本题 |
|---------|--------|------|
| 前序 | X的第一个孩子或某个祖先 | - |
| 中序 | X的父结点或右兄弟方向 | - |
| 后序 ✅ | X的父结点 | ✅ |

## 知识点对比

| 树结构 | 后序序列 | X的位置 | X的后继 |
|-------|---------|--------|---------|
| P(Y,X) Y是左孩子 | ...Y的子树, Y, X, P | 倒数第二 | P ✅ |

| 遍历方式 | 顺序 | 叶结点后继特点 |
|---------|------|---------------|
| 前序 | 根→左→右 | 父结点的右子树或祖先 |
| 中序 | 左→根→右 | 父结点或父结点的右子树 |
| 后序 ✅ | 左→右→根 | 父结点 ✅ |

| 结点类型 | 后序后继 | 本题X |
|---------|---------|------|
| 左子树的叶子 | 右兄弟或父结点 | - |
| 右子树的叶子 ✅ | 父结点 | ✅ X是右孩子 |
| 根结点 | 无后继 | - |
