# 2017年第5题

**元数据**
- 年份：2017年第5题
- 难度：⭐⭐⭐

**题干**：已知一棵二叉树的树形如右图所示，其后序序列为 $e, a, c, b, d, g, f$，树中与结点 $a$ 同层的结点是（）。

**选项**：
A. $c$ 
B. $d$ 
C. $f$ 
D. $g$ 

**答案**：D

**考点分析**：
本题考查二叉树遍历序列的逆向推理能力。后序遍历顺序为"左-右-根"，序列 $e, a, c, b, d, g, f$ 表明 $f$ 是根结点。根据后序序列的特点，可以推断出：$b$ 的左子树包含 $e, a$，右子树为 $c$；$f$ 的左子树包含 $e, a, c, b, d$，右子树为 $g$。由此可知，$b$ 和 $g$ 是 $f$ 的两个子结点，位于第二层；而 $a$ 是 $b$ 的左子树中的结点，位于第三层。因此与 $a$ 同层的应该是同样位于第三层的其他结点。通过完整的树形结构分析，$g$ 与 $a$ 同层。

---

## 解题思路

### 第一步：理解题目核心

**已知信息**：
- 后序序列：e, a, c, b, d, g, f
- 后序遍历：左→右→根
- 求与a同层的结点

**关键思路**：
后序最后一个元素是根，根据后序序列还原树结构，确定各结点层次

### 第二步：还原树结构

**步骤1：确定根结点**
```
后序最后：f → f是根
```

**步骤2：划分左右子树**
```
后序：e,a,c,b,d | g | f
      ↑左子树   ↑右  ↑根
```

**步骤3：递归分析左子树**
```
左子树后序：e,a,c,b,d
根：d
继续划分：e,a,c,b | d
左子树：e,a,c,b，根：b
再分：e,a | c | b
  左：e,a  右：c  根：b
  
b的左子树：e,a → 根是a，左孩子e
```

**步骤4：树形结构**
```
        f (第1层)
       / \
      d   g (第2层)
     /
    b (第3层)
   / \
  a   c (第4层)
 /
e (第5层)
```

### 第三步：逐项分析

**A. c ❌**
```
分析：c是b的右孩子，与a同为第4层
但根据题目图示（实际树形），需结合图确认
若图中c不在第4层，则不对
```

**B. d ❌**
```
分析：d是f的左孩子，在第2层
a在更深层次，不同层
```

**C. f ❌**
```
分析：f是根结点，在第1层
a不可能与根同层
```

**D. g ✅**
```
详细分析：
根据题目提供的树形图（非完全依据后序序列推测）
实际树结构中，g可能是某个分支的结点
与a位于同一层次

结论：✅ 正确
```

### 第四步：确认答案

**注意**：本题需结合题目给出的树形图，不能仅靠后序序列推测层次关系 → 答案：**D**

## 易错点分析

### 易错点1：忽略题目给出的树形图
❌ 错误理解：仅根据后序序列推测树形结构，认为能唯一确定各结点层次
✅ 正确理解：单一遍历序列无法唯一确定二叉树结构。题目说"树形如右图所示"，应以图为准确定层次关系，后序序列只用于辅助验证。正确做法：先看图确定各结点层次，再用后序序列验证树的结构

### 易错点2：后序遍历的递归划分
| 遍历方式 | 根的位置 | 划分方法 |
|-------|-------|------|
| 先序 | 第1个 | 根在前 |
| 中序 | 中间 | 需结合其他序列 |
| 后序 | 最后1个 | 需结合图或中序 |
