**题干**

若一棵完全二叉树有 768 个结点，则该二叉树中的叶结点数是（）。A. 257  B. 258  C. 384  D. 385

**答案**：C

**考点分析**
- n=768，n0=⌊768/2⌋=384

---

## 解题思路

### 第一步：理解题目核心
完全二叉树有768个结点，求叶结点数。需利用完全二叉树的性质和叶结点公式。

### 第二步：回顾完全二叉树叶结点公式
**二叉树结点数公式**：
- n = n₀ + n₁ + n₂（n₀叶结点，n₁度为1，n₂度为2）
- n₀ = n₂ + 1（二叉树性质）

**完全二叉树特殊性质**：
- n₁ = 0 或 1（最多只有1个度为1的结点）
- 若n为偶数，n₁=1（最后一层最右结点只有左孩子）
- 若n为奇数，n₁=0（最后一层恰好填满）

**叶结点数公式**：
```
情况1：n为奇数 → n₁=0
n = n₀ + n₂
n₀ = n₂ + 1
联立：n = n₀ + (n₀-1) = 2n₀ - 1
n₀ = (n+1)/2

情况2：n为偶数 → n₁=1
n = n₀ + 1 + n₂
n₀ = n₂ + 1
联立：n = n₀ + 1 + (n₀-1) = 2n₀
n₀ = n/2
```

**统一公式**：n₀ = ⌊n/2⌋（向下取整）

### 第三步：计算过程
```
n = 768（偶数）
n₀ = ⌊768/2⌋ = 384
```

**验证**：
- n₁ = 1（偶数必有1个度为1的结点）
- n₂ = n₀ - 1 = 383
- n = 384 + 1 + 383 = 768 ✅

### 第四步：逐项判断

**A. 257 ❌**
```
257 ≠ 384
可能混淆了其他公式
结论：❌ 错误
```

**B. 258 ❌**
```
258 ≠ 384
可能是某种错误的向上取整
结论：❌ 错误
```

**C. 384 ✅**
```
384 = ⌊768/2⌋
符合完全二叉树叶结点公式
结论：✅ 正确
```

**D. 385 ❌**
```
385 = 384 + 1
可能误用了n₀ = n₂ + 1，但忘记n中还有n₁
结论：❌ 错误
```

答案：**C**

## 易错点分析

### 易错点1：忘记n₁的存在
❌ **错误推导**：n = n₀ + n₂，n₀ = n₂ + 1 → n = 2n₀ - 1 → n₀ = 385
✅ **正确推导**：完全二叉树n为偶数时n₁=1，必须加入公式
```
n = n₀ + n₁ + n₂ = n₀ + 1 + (n₀-1) = 2n₀
n₀ = n/2 = 384
```

| 结点总数 | n₁的值 | 叶结点数公式 |
|---------|--------|-------------|
| 奇数 | 0 | n₀=(n+1)/2 |
| 偶数 | 1 | n₀=n/2 |
| 统一 | 0或1 | n₀=⌊n/2⌋ |

### 易错点2：混淆完全二叉树与普通二叉树
❌ **错误理解**：普通二叉树n₁可以是任意值
✅ **正确理解**：完全二叉树的n₁只能是0或1
```
完全二叉树的填充规则：
- 除最后一层外必须填满
- 最后一层从左到右连续填充
- 因此只能有1个结点有单个孩子（最后一层的最右非叶结点）
```

**反例验证**：
```
若完全二叉树有2个度为1的结点：
- 第一个度为1的结点a：只有左孩子
- 第二个度为1的结点b：只有左孩子
- 则a的右侧必有空位，但b已经有孩子
- 这违反了"从左到右连续填充"的定义
```

### 易错点3：使用满二叉树公式
❌ **错误套用**：满二叉树n₀=(n+1)/2，直接算(768+1)/2=384.5
✅ **区分概念**：
| 树类型 | n₁的值 | 叶结点公式 |
|--------|--------|-----------|
| 满二叉树 | 0 | n₀=(n+1)/2 |
| 完全二叉树 | 0或1 | n₀=⌊n/2⌋ |
| 普通二叉树 | 任意 | n₀=n₂+1 |

## 知识点对比

| 二叉树类型 | 度为1的结点数 | 叶结点数公式 | 结点总数范围 |
|-----------|-------------|-------------|-------------|
| 满二叉树 | 0 | (n+1)/2 | n=2^h-1 |
| 完全二叉树 | 0或1 | ⌊n/2⌋ | [2^(h-1), 2^h-1] |
| 普通二叉树 | 任意 | n₂+1 | [h, 2^h-1] |

**记忆口诀**：完全二叉树叶结点，总数折半向下取；偶数正好一半整，奇数上凑多一个。
