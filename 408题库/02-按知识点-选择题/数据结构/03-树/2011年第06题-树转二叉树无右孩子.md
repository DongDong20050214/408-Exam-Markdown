- 关联知识：[[知识库/数据结构/树]]

**题干**

已知一棵有 2011 个结点的树，其叶结点个数为 116，该树对应的二叉树中无右孩子的结点个数是（）。

A. 115  B. 116  C. 1895  D. 1896

**答案**：D

**考点分析**
- 树转二叉树：无右孩子=非最后孩子
- 非叶节点=2011-116=1895
- 无右孩子=1895+1=1896

---

## 解题思路

### 第一步：理解题目核心

**题目条件**：
- 树有2011个结点
- 叶结点有116个
- 求：对应二叉树中无右孩子的结点个数

**树转二叉树规则（左孩子右兄弟）**：
- 原树的第一个孩子 → 二叉树的左孩子
- 原树的兄弟结点 → 二叉树的右孩子

**关键推导**：
- 二叉树中有右孩子 ⟺ 原树中有右兄弟（不是父结点的最后一个孩子）
- 二叉树中无右孩子 ⟺ 原树中是父结点的最后一个孩子，或是根结点

### 第二步：分析无右孩子的结点

**原树中的结点分类**：
1. 根结点：1个（没有兄弟，转换后无右孩子）
2. 非根结点：2011 - 1 = 2010个

**非根结点中，哪些无右孩子？**
- 每个父结点的孩子们中，只有**最后一个孩子**在二叉树中无右孩子
- 每个非叶结点（父结点）贡献1个"最后孩子"

**非叶结点个数**：
- 非叶结点 = 总结点数 - 叶结点数
- 非叶结点 = 2011 - 116 = 1895个

**无右孩子的结点总数**：
- 根结点：1个
- 每个非叶结点的最后孩子：1895个
- 总计：1 + 1895 = 1896个

### 第三步：逐项分析

**A. 115 ❌**
```
详细分析：
115 = 116 - 1（叶结点数-1）
这是错误的推导，没有考虑非叶结点的最后孩子

判断依据：不符合树转二叉树的规则
结论：❌ 错误
```

**B. 116 ❌**
```
详细分析：
116 = 叶结点数
错误理解：认为只有叶结点在二叉树中无右孩子
但实际上，非叶结点的最后孩子也无右孩子

判断依据：忽略了非叶结点的最后孩子
结论：❌ 错误
```

**C. 1895 ❌**
```
详细分析：
1895 = 非叶结点数
这只统计了每个父结点的最后孩子，忘记加上根结点

判断依据：忘记根结点也无右孩子
结论：❌ 少算1个
```

**D. 1896 ✅**
```
详细分析：
计算过程：
1. 根结点无右孩子：1个
2. 非叶结点个数：2011 - 116 = 1895个
3. 每个非叶结点的最后孩子无右孩子：1895个
4. 总计：1 + 1895 = 1896个

判断依据：符合树转二叉树的完整规则
结论：✅ 正确答案
```

### 第四步：确认答案

无右孩子的结点数 = 根结点 + 非叶结点数 = 1 + 1895 = 1896 → 答案：**D**

## 易错点分析

### 易错点1：只统计叶结点
❌ 错误理解：只有叶结点在二叉树中无右孩子
✅ 正确理解：所有"是父结点的最后孩子"的结点，以及根结点，在二叉树中都无右孩子

### 易错点2：忘记根结点
❌ 错误计算：无右孩子数 = 非叶结点数 = 1895
✅ 正确计算：根结点也没有右兄弟，转换后无右孩子，需要加1

### 易错点3：不理解"右孩子"的含义
❌ 错误理解：右孩子 = 原树中的第二个孩子
✅ 正确理解：在"左孩子右兄弟"表示法中，右孩子 = 右兄弟（同一父结点的下一个孩子）

| 原树关系 | 二叉树中的表示 | 是否有右孩子 |
|---------|--------------|-------------|
| 第一个孩子 | 左孩子 | 看是否有兄弟 |
| 中间孩子 | 在兄弟链上 | ✅ 有（下一个兄弟） |
| 最后孩子 | 兄弟链末尾 | ❌ 无 |
| 根结点 | 根 | ❌ 无 |
