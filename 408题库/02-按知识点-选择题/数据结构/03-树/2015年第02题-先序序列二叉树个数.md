# 二叉树个数真题汇总

## 2015年第02题

**元数据**
- 年份：[[../../01-真题原卷/2015年408真题#02|2015年第2题]]
- 科目：#数据结构
- 知识点：#数据结构/树/二叉树 #卡特兰数
- 难度：⭐⭐⭐
- 关联知识：[[知识库/数据结构/二叉树]]

**题干**

先序序列为 a, b, c, d 的不同二叉树的个数是（ ）。

A. 13  B. 14  C. 15  D. 16

**答案**：B

**解题思路**

**关键**：先序序列确定，求有多少种不同二叉树形态

先序遍历：根→左→右
- a是根（固定）
- b,c,d分配到左右子树

**递归分析**：
- 左子树i个节点，右子树n-1-i个节点
- 左子树形态数×右子树形态数

这是**卡特兰数**：$C_n = \frac{1}{n+1}C_{2n}^{n}$

**计算C_4**：
$$C_4 = \frac{1}{5}C_8^4 = \frac{1}{5} \times \frac{8!}{4!4!} = \frac{1}{5} \times 70 = 14$$

或递推：
- $C_0=1, C_1=1, C_2=2, C_3=5$
- $C_4 = C_0C_3 + C_1C_2 + C_2C_1 + C_3C_0$
- $= 1×5 + 1×2 + 2×1 + 5×1 = 14$

**选项分析**：
- ❌ A. 13
- ✅ B. 14
- ❌ C. 15
- ❌ D. 16

**易错点分析**

❌ **错误1：认为是排列数4!=24**
- 混淆序列个数与树形态个数
- ✅ 正确：先序固定，求树形态数（卡特兰数）

❌ **错误2：计算错误**
- 卡特兰数公式应用错误
- ✅ 正确：$C_4=14$

**知识点对比**

| n | 卡特兰数$C_n$ | 含义 |
|---|-------------|------|
| 1 | 1 | 1个节点1种树 |
| 2 | 2 | 2个节点2种树 |
| 3 | 5 | 3个节点5种树 |
| 4 | 14 | 4个节点14种树 |
| 5 | 42 | 5个节点42种树 |

**卡特兰数公式**

| 公式类型 | 表达式 |
|---------|--------|
| **组合数** | $C_n = \frac{1}{n+1}C_{2n}^{n}$ |
| **递推式** | $C_n = \sum_{i=0}^{n-1}C_iC_{n-1-i}$ |
| **递推式2** | $C_n = \frac{2(2n-1)}{n+1}C_{n-1}$ |

**本题C_4示例**

递推计算：
- i=0: 左0右3 → $C_0×C_3 = 1×5 = 5$
- i=1: 左1右2 → $C_1×C_2 = 1×2 = 2$
- i=2: 左2右1 → $C_2×C_1 = 2×1 = 2$
- i=3: 左3右0 → $C_3×C_0 = 5×1 = 5$
- 总计：5+2+2+5=14

**关键理解**：
- 先序序列确定根和子序列
- 不同形态 = 左右子树形态组合
- n个节点二叉树形态数 = 卡特兰数$C_n$

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 先序序列：a, b, c, d（4个结点）
- 问：不同二叉树的个数
- 先序遍历顺序：根→左→右

### 第二步：分析卡特兰数应用
a必为根，b,c,d分配到左右子树：
- 左0右3：C0×C3 = 1×5 = 5
- 左1右2：C1×C2 = 1×2 = 2
- 左2右1：C2×C1 = 2×1 = 2
- 左3右0：C3×C0 = 5×1 = 5
- 总计：14种

### 第三步：逐项判断

**A. 13 ❌**
```
13不是卡特兰数
可能是误算（14-1）
结论：❌
```

**B. 14 ✅**
```
C4 = C(8,4)/5 = 70/5 = 14
递推：5+2+2+5 = 14
结论：✅ 正确
```

**C. 15 ❌**
```
15不是C4
可能误算为C3+C3+...
结论：❌
```

**D. 16 ❌**
```
16 = 2^4
误用指数公式
结论：❌
```

### 第四步：确认答案
答案：**B**

## 易错点分析（补充）

### 易错点1：混淆序列数和形态数
❌ **错误理解**：4个结点有4!=24种排列
✅ **正确理解**：先序序列固定，只求树的形态数（结构不同），是卡特兰数C4=14

### 易错点2：认为先序确定树唯一
❌ **错误想法**：给定先序后树的形态就确定了
✅ **正确理解**：需要先序+中序（或其他组合）才能唯一确定。仅先序时，相同先序可对应多种不同形态。
