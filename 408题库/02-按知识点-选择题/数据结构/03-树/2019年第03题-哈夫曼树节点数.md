# 哈夫曼树节点数真题汇总

## 2019年第03题

**元数据**
- 年份：[[../../01-真题原卷/2019年408真题#03|2019年第3题]]
- 科目：#数据结构
- 知识点：#数据结构/树/哈夫曼树 #节点数
- 难度：⭐⭐
- 关联知识：[[知识库/数据结构/哈夫曼树]]

**题干**

对 n 个互不相同的符号进行哈夫曼编码。若生成的哈夫曼树共有 115 个结点，则 n 的值是（ ）。

A. 56  B. 57  C. 58  D. 60

**答案**：C

**解题思路**

**哈夫曼树的性质**：
- n个叶子节点（符号）
- 每次合并2个节点生成1个新节点
- 共合并n-1次
- 总节点数 = n（叶子）+ (n-1)（非叶子）= **2n - 1**

**已知条件**：
- 总节点数 = 115
- 求叶子节点数n（符号个数）

**计算**：
$$2n - 1 = 115$$
$$2n = 116$$
$$n = 58$$

**选项分析**：
- ❌ A. 56 —— 2×56-1=111
- ❌ B. 57 —— 2×57-1=113
- ✅ C. 58 —— 2×58-1=115 ✓
- ❌ D. 60 —— 2×60-1=119

**易错点分析**

❌ **错误1：认为节点数等于n**
- 忽略了内部节点
- ✅ 正确：总节点数包括叶子和非叶子，公式是2n-1

❌ **错误2：公式记错为2n或2n+1**
- 混淆其他树的节点数公式
- ✅ 正确：哈夫曼树（二叉树）n个叶子对应2n-1个总节点

❌ **错误3：计算错误**
- (115+1)/2 = 58，但写成57或56
- ✅ 正确：仔细计算115+1=116，116/2=58

**知识点对比**

| 树类型 | 叶子节点数n | 总节点数公式 |
|--------|-----------|------------|
| **哈夫曼树** | n | 2n - 1 |
| **满二叉树（高度h）** | $2^{h-1}$ | $2^h - 1$ |
| **完全二叉树** | 不定 | 不定 |
| **度为m的树** | n | $(m-1)n + 1$ |

**哈夫曼树构造过程节点数变化**

初始n=4个符号为例：

| 步骤 | 操作 | 当前节点数 |
|------|------|-----------|
| 初始 | 4个叶子 | 4 |
| 第1次合并 | 选2个，生成1个 | 4-2+1=3 |
| 第2次合并 | 选2个，生成1个 | 3-2+1=2 |
| 第3次合并 | 选2个，生成1个 | 2-2+1=1（根） |
| 最终 | - | 4+3=7=2×4-1 ✓ |

**公式推导**

n个叶子节点：
- 每次合并减少2个节点，增加1个节点，净减少1个
- 合并n-1次后，剩余1个根节点
- 过程中新增节点数 = n-1（每次合并新增1个）
- 总节点数 = n（初始叶子）+ (n-1)（新增非叶子）= 2n-1

**本题验证**

n=58：
- 叶子节点：58个
- 非叶子节点：57个（每次合并生成1个）
- 总节点：58+57=115 ✓

**度的角度理解**

哈夫曼树是**度为2的树**（每个非叶子节点恰有2个孩子）。

对于度为2的树：
- 设度为0的节点（叶子）为$n_0$，度为2的节点为$n_2$
- 有公式：$n_0 = n_2 + 1$
- 总节点：$n_0 + n_2 = (n_2+1) + n_2 = 2n_2 + 1$
- 若$n_0 = n$，则$n_2 = n-1$，总节点 = $n + (n-1) = 2n-1$ ✓

**关键理解**：
- 哈夫曼树是严格二叉树（每个节点0或2个孩子）
- n个叶子需要n-1次两两合并
- 每次合并产生1个新节点
- 最终2n-1个节点

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- n个互不相同的符号
- 哈夫曼树共115个结点
- 问：n的值

### 第二步：应用哈夫曼树公式
**哈夫曼树结点数公式**：总结点数 = 2n - 1
- n为叶结点数（符号个数）
- 2n-1 = 115
- n = 58

### 第三步：逐项判断

**A. 56 ❌**
```
2×56-1 = 111 ≠ 115
结论：❌
```

**B. 57 ❌**
```
2×57-1 = 113 ≠ 115
结论：❌
```

**C. 58 ✅**
```
2×58-1 = 115 ✓
结论：✅ 正确
```

**D. 60 ❌**
```
2×60-1 = 119 ≠ 115
结论：❌
```

### 第四步：确认答案
答案：**C**

## 易错点分析（补充）

### 易错点1：公式记忆错误
❌ **错误公式**：总结点数 = 2n 或 n+1
✅ **正确公式**：总结点数 = 2n-1（n个叶子，n-1个内部结点）

### 易错点2：计算粗心
❌ **常见错误**：(115+1)/2 = 58，但写错成57
✅ **正确步骤**：2n-1=115 → 2n=116 → n=58
