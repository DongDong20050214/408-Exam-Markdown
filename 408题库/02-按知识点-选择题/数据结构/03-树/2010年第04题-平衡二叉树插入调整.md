# 平衡二叉树插入调整真题汇总

## 2010年第04题

**元数据**
- 年份：[[../../01-真题原卷/2010年408真题#04|2010年第4题]]
- 科目：#数据结构
- 知识点：#数据结构/树/平衡二叉树 #旋转调整
- 难度：⭐⭐⭐⭐
- 关联知识：[[知识库/数据结构/平衡二叉树]]

**题干**

在平衡二叉树中插入关键字48后，关键字37所在结点的左、右子结点中保存的关键字分别是（）。

A. 13,48  
B. 24,48  
C. 24,53  
D. 24,90

**答案**：B

**考点分析**
- 插入48后可能破坏平衡
- 需要进行旋转调整
- 根据具体树形判断旋转类型
- 调整后37的左子为24，右子为48

---

## 解题思路

### 第一步：理解题目核心
在已有的平衡二叉树中插入48后，需判断37结点的左右子结点变化（原题应有图示）。

### 第二步：AVL树插入的四种旋转类型
| 失衡类型 | 插入位置 | 旋转方式 | 示意 |
|---------|---------|---------|------|
| LL | 左子树的左子树 | 右旋 | 顺时针 |
| RR | 右子树的右子树 | 左旋 | 逆时针 |
| LR | 左子树的右子树 | 先左旋后右旋 | 双旋 |
| RL | 右子树的左子树 | 先右旋后左旋 | 双旋 |

**旋转判断依据**：
- 找最小失衡子树的根结点
- 确定失衡路径（插入点到根的路径）
- 根据路径的左右转向选择旋转

### 第三步：推导插入48的调整过程
**典型情况**（需结合原图）：
假设插入前的部分结构为：
```
      50
     /  \
   37    ...
  /  \
 24   53
```

**插入48**：
- 48 < 50，向左；48 > 37，向右；48 < 53，向左
- 插入到53的左孩子位置

**检查平衡因子**：
```
插入后：
      50 (BF=?)
     /  \
   37    ...
  /  \
 24   53
     /
    48
```
- 53的BF = 1-0 = 1（平衡）
- 37的BF = 0-2 = -2（失衡！）

**旋转调整**：
- 失衡类型：37的右子树的左子树插入 → RL型
- 需要先对53右旋，再对37左旋

**RL旋转过程**：
第一步：对53右旋
```
   37              37
  /  \            /  \
 24   53    →    24   48
     /                 \
    48                 53
```

第二步：对37左旋
```
   37                48
  /  \              /  \
 24   48    →     37   53
       \          /
       53        24
```

**调整后37的子结点**：
- 左孩子：24
- 右孩子：48（旋转后上升到37右侧）

### 第四步：逐项判断

**A. 13, 48 ❌**
```
左孩子13不在原树中（假设原树无13）
结论：❌ 错误
```

**B. 24, 48 ✅**
```
符合RL旋转后的结果：
- 24保持为37的左孩子
- 48旋转上升为37的右孩子
结论：✅ 正确
```

**C. 24, 53 ❌**
```
这是插入前的状态（未调整）
插入48后必定需要旋转调整
结论：❌ 错误
```

**D. 24, 90 ❌**
```
90不在调整路径上
结论：❌ 错误
```

答案：**B**

## 易错点分析

### 易错点1：插入后不旋转
❌ **错误理解**：认为插入48后直接成为53的左孩子，37的子结点不变
✅ **正确理解**：插入可能破坏平衡，需沿路径检查所有祖先的平衡因子
```
检查顺序：53 → 37 → 50 → ...
发现37失衡（BF=-2），必须旋转
```

### 易错点2：混淆旋转类型
❌ **错误判断**：看到右子树失衡就认为是RR型，直接左旋
✅ **正确判断**：RL型需要两步旋转
```
判断依据：失衡路径的转向
37 → 53（右）→ 48（左）→ RL型

RR型示例：37 → 53（右）→ 60（右）
RL型示例：37 → 53（右）→ 48（左）
```

| 失衡类型 | 路径转向 | 旋转次数 | 中间结点位置 |
|---------|---------|---------|-------------|
| LL | 左-左 | 1次 | 上升到根 |
| RR | 右-右 | 1次 | 上升到根 |
| LR | 左-右 | 2次 | 左孩子的右孩子上升 |
| RL | 右-左 | 2次 | 右孩子的左孩子上升 |

### 易错点3：忘记旋转后结点的重新连接
❌ **遗漏细节**：只旋转关键路径，忘记其他子树的连接
✅ **完整旋转**：
```
RL旋转后：
- 48成为新子树的根
- 37成为48的左孩子（24仍是37的左孩子）
- 53成为48的右孩子
- 若48有左子树，需接到37的右侧
```

## 知识点对比

| 旋转类型 | 单旋 | 双旋 |
|---------|------|------|
| 包含 | LL, RR | LR, RL |
| 路径特征 | 同向转折 | 反向转折 |
| 旋转次数 | 1次 | 2次 |
| 上升结点 | 失衡点的孩子 | 失衡点的孙子 |
| 难度 | 较简单 | 较复杂 |

**记忆口诀**：同向单旋转，反向要双旋；LL右旋RR左旋，LR先左再右旋，RL先右再左旋。
