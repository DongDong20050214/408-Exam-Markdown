# 二叉树层次关系真题汇总

## 2017年第05题

**元数据**
- 年份：[../../01-真题原卷/2017年408真题#05|2017年第5题](../../01-真题原卷/2017年408真题#05|2017年第5题.md)
- 科目：#数据结构

- 难度：⭐⭐⭐
- 关联知识：[知识库/数据结构/二叉树](知识库/数据结构/二叉树.md)

**题干**

已知一棵二叉树的后序序列为e,a,c,b,d,g,f，树中与结点a同层的结点是（）。

A. c  
B. d  
C. f  
D. g

**答案**：D

**考点分析**
- 后序遍历：左子树→右子树→根
- 根据后序序列和树形结构确定层次关系

---

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 后序序列：e, a, c, b, d, g, f
- 问：与结点a同层的结点是哪个
- 选项：A.c  B.d  C.f  D.g
- 注意：仅有后序序列无法唯一确定树结构，题目应该还有图

### 第二步：回顾后序遍历规则
**后序遍历**：左子树→右子树→根
- 序列最后一个元素f是根
- f的左右子树分别在f之前
- 递归分析每个子树

### 第三步：分析树结构
```
后序：e, a, c, b, d, g, f
最后f是根

需要找出f的左右子树划分：
假设左子树后序：e,a,c,b,d
右子树后序：g
则树结构：
      f (根)
     / \
   ...  g

继续分析左子树(根为d)：
左子树后序：e,a,c,b,d
d是这个子树的根
d的左右子树划分：
假设左子树：e,a,c,b
右子树：空
则：
      d
     /
   b(根)
  /  \
 a    c
/
e

或者其他划分...

由于无图，无法确定。但根据答案D(g)，
推测g与a在同一层。
```

**关键推理**（根据答案反推）：
```
若g与a同层，则：
- f是根（第1层）
- f的某个子树包含a，另一个子树是或包含g
- a和g在同一深度

可能的结构：
     f (1层)
    / \
   d   g (2层)
  /
 b (3层)
/ \
a  c (4层)
/
e (5层)

这样a和g不同层...

重新推测：
     f (1层)
    / \
   b   g (2层) ← a和g同层
  / \
 a   d (3层)
/   /
e  c (4层)

后序验证：e(a的左)→a→c(d的左)→d(b的右)→b→g(f的右)→f
= e,a,c,d,b,g,f ✗ 不匹配（题目是e,a,c,b,d,g,f）

正确结构：
      f (1层)
     / \
    d   g (2层)
   /
  b (3层)
 / \
a  c (4层)
/
e (5层)

后序：e→a→c→b→d→g→f ✓
此时a在4层，g在2层，不同层

应该题目有图，这里根据答案选D
```

### 第四步：逐项判断

**A. c ❌**
```
c在后序中紧跟a，可能是a的兄弟或其他关系
但根据答案不是同层
结论：❌
```

**B. d ❌**
```
d在后序靠后，可能是更高层的结点
结论：❌
```

**C. f ❌**
```
f是根，在第1层
a不可能与根同层
结论：❌
```

**D. g ✅**
```
根据答案，g与a同层
需要原题图才能验证
结论：✅ 正确
```

### 第五步：确认答案
答案：**D**（需结合原题图验证）

## 易错点分析

### 易错点1：认为后序序列可唯一确定树
❌ **错误理解**：仅凭后序序列就能画出唯一的二叉树
✅ **正确理解**：需要后序+中序（或前序+中序，或其他组合）才能唯一确定二叉树。仅后序序列无法确定树结构，本题应有配图。

### 易错点2：后序相邻就是同层
❌ **错误想法**：后序序列中相邻的结点在树中同层
✅ **正确理解**：后序相邻可能是：
- 兄弟关系（如a,c可能是兄弟，都是b的子）
- 父子关系（如d,g,f中，g和d可能是f的子）
- 其他复杂关系

## 知识点对比

| 遍历类型 | 顺序 | 根的位置 | 能否唯一确定树 |
|---------|------|---------|--------------|
| 前序 | 根→左→右 | 最前 | ❌ 不能 |
| 中序 | 左→根→右 | 中间 | ❌ 不能 |
| 后序 | 左→右→根 | 最后 | ❌ 不能 |
| 前序+中序 | — | — | ✅ 能 |
| 后序+中序 | — | — | ✅ 能 |
