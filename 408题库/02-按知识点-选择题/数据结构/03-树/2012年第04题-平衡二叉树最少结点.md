**题干**

若平衡二叉树的高度为6，且所有非叶结点的平衡因子均为1，则该平衡二叉树的结点总数为（）。

A. 12  
B. 20  
C. 32  
D. 33

**答案**：B

**考点分析**
- 平衡因子=1：左子树高度-右子树高度=1
- 所有非叶结点平衡因子为1，是最少结点情况
- N(h) = N(h-1) + N(h-2) + 1（斐波那契递推）
- N(0)=0, N(1)=1, N(2)=2, N(3)=4, N(4)=7, N(5)=12, N(6)=20

---

## 解题思路

### 第一步：理解题目核心

**题目条件**：
- 平衡二叉树高度为6
- 所有非叶结点的平衡因子均为1
- 求：结点总数

**平衡因子BF=1的含义**：
- BF = 左子树高度 - 右子树高度 = 1
- 左子树比右子树高1层
- 这是高度为h的AVL树**结点数最少**的情况

**递推公式**：
设N(h)为高度h的最少结点数AVL树的结点数
- N(h) = N(h-1) + N(h-2) + 1
- 其中：左子树高度h-1，右子树高度h-2，根结点1个

### 第二步：递推计算

**初始条件**：
- N(0) = 0（空树）
- N(1) = 1（单个结点）

**逐步递推**：
- N(2) = N(1) + N(0) + 1 = 1 + 0 + 1 = 2
- N(3) = N(2) + N(1) + 1 = 2 + 1 + 1 = 4
- N(4) = N(3) + N(2) + 1 = 4 + 2 + 1 = 7
- N(5) = N(4) + N(3) + 1 = 7 + 4 + 1 = 12
- N(6) = N(5) + N(4) + 1 = 12 + 7 + 1 = 20

### 第三步：逐项分析

**A. 12 ❌**
```
详细分析：
12 = N(5)（高度为5的最少结点数）
错误原因：把高度看成5而不是6

判断依据：高度计算错误
结论：❌ 错误
```

**B. 20 ✅**
```
详细分析：
20 = N(6)
递推计算：
N(6) = N(5) + N(4) + 1 = 12 + 7 + 1 = 20

判断依据：符合递推公式和初始条件
结论：✅ 正确答案
```

**C. 32 ❌**
```
详细分析：
32 = 2^5（可能误以为是满二叉树）
错误理解：认为是满二叉树或完全二叉树的结点数

判断依据：题目要求的是最少结点数，不是满二叉树
结论：❌ 错误
```

**D. 33 ❌**
```
详细分析：
33无法通过AVL树的递推公式推导出来
可能是某种错误计算的结果

判断依据：不符合N(h) = N(h-1) + N(h-2) + 1
结论：❌ 错误
```

### 第四步：确认答案

高度为6，所有非叶结点BF=1的AVL树结点数 = 20 → 答案：**B**

## 易错点分析

### 易错点1：误以为是满二叉树
❌ 错误理解：高度为h的二叉树结点数 = 2^h - 1
✅ 正确理解：题目要求"所有非叶结点BF=1"是**最少结点**的AVL树，不是满二叉树

| 树类型 | 高度6的结点数 | 特征 |
|-------|-------------|------|
| 满二叉树 | 2^6-1=63 | 所有层都满 |
| 完全二叉树 | [32,63] | 最后一层左对齐 |
| 最少AVL树 ✅ | 20 | BF都为1 |

### 易错点2：不理解"BF=1"的含义
❌ 错误理解：BF=1只是平衡条件之一，不影响结点数
✅ 正确理解：所有非叶结点BF=1表示左子树总是比右子树高1，这是**最少结点**的极端情况

### 易错点3：递推公式记错
❌ 错误公式：N(h) = N(h-1) + N(h-2)（忘记+1）
✅ 正确公式：N(h) = N(h-1) + N(h-2) + 1（根结点要加1）
