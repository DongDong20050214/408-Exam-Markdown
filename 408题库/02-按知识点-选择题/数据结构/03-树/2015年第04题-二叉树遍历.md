# 2015年第4题

**元数据**
- 年份：2015年第4题
- 难度：⭐⭐

**题干**：现有一棵无重复关键字的平衡二叉树（AVL 树），对其进行中序遍历可得到一个降序序列。下列关于该平衡二叉树的叙述中，正确的是（）。

A. 根结点的度一定为 2

B. 树中最小元素一定是叶结点

C. 最后插入的元素一定是叶结点

D. 树中最大元素一定是无左子树

**答案**：D

**考点分析**：
- **考点**：数据结构 - 平衡二叉树(AVL树)、二叉排序树性质
- **知识点**：
  - AVL树是平衡的二叉排序树，左子树所有节点 < 根节点 < 右子树所有节点
  - 中序遍历：左子树→根→右子树
  - 中序遍历得降序，说明这是一棵"右-根-左"的遍历顺序，或者说是反向的二叉排序树
- **解析**：
  - 中序遍历为降序，说明遍历顺序是：右子树→根→左子树
  - A错误：根节点可以只有一个子树，如单个根节点的树
  - B错误：最小元素在最右边，不一定是叶节点
  - C错误：AVL树插入后可能需要旋转调整，最后插入的元素不一定还是叶节点
  - D正确：中序遍历为降序，最大元素在最左侧，按照二叉排序树性质，最大元素只能在左侧路径，因此无左子树（否则左子树中还有更大元素）

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 平衡二叉树（AVL树）
- 无重复关键字
- 中序遍历得降序序列

### 第二步：分析BST性质
正常BST：中序遍历→升序
本题：中序遍历→降序
说明：右子树值 < 根 < 左子树值（反向BST）

### 第三步：逐项判断

**A. 根结点的度一定为2 ❌**
```
反例：单节点树
    [10]
中序：10（降序）
根度=0，不是2
结论：错误
```

**B. 树中最小元素一定是叶结点 ❌**
```
反向BST：最小元素在最右路径
反例：
      5
       \
        3
         \
          1
中序：5,3,1（降序）
最小=1，是叶子 ✓

但考虑：
      5
     / \
    7   3
         \
          1
中序：7,5,3,1（降序）
最小=1，是叶子

再考虑：
      5
       \
        2
       /
      1
中序：5,2,1（降序）
最小=1，有右子树也行？不，反向BST中1<2，1应该在右边

实际上反例：可以不是叶子
结论：不一定
```

**C. 最后插入的元素一定是叶结点 ❌**
```
AVL树插入后会旋转
例：树[5,3]，插入1
插入前：5
         \
          3
插入1：需旋转
旋转后：3
       / \
      5   1
最后插入的1成为叶子 ✓

但插入后旋转可能让之前的叶子变非叶子
结论：不一定
```

**D. 树中最大元素一定是无左子树 ✅**
```
反向BST：左子树 > 根 > 右子树
最大元素在最左路径
若最大元素有左子树→左子树更大→矛盾
结论：正确 ✅
```

### 第四步：确认答案
答案：**D**

## 易错点分析

### 陷阱1：忘记是"反向"BST
❌ 错误理解：中序降序就是普通BST从右往左遍历
✅ 正确理解：值的大小关系反了，左>根>右

### 陷阱2：混淆AVL旋转影响
❌ 错误理解：最后插入的一定是叶子
✅ 正确理解：旋转后可能变成非叶子

## 知识点对比

| 项 | 正向BST | 反向BST |
|---|---------|---------|
| 性质 | 左<根<右 | 左>根>右 |
| 中序 | 升序 | 降序 |
| 最大 | 最右 | 最左 |
| 最小 | 最左 | 最右 |
