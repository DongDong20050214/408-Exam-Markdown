# 度为4的树叶子结点数真题汇总

## 2010年第05题

**元数据**
- 年份：[2010年第5题](../../../01-真题原卷/2010年408真题.md#05)
- 科目：#数据结构
- 知识点：#数据结构/树/树的性质
- 难度：⭐⭐⭐
- 关联知识：[树](../../../../AI-知识点原子库/树.md)

**题干**

在一棵度为4的树T中，若有20个度为4的结点，10个度为3的结点，1个度为2的结点，10个度为1的结点，则树T的叶结点个数是（）。

A. 41  
B. 82  
C. 113  
D. 122

**答案**：B

**考点分析**
- 总结点数 n = n₀ + n₁ + n₂ + n₃ + n₄
- 边数 = n - 1
- 边数 = 1×n₁ + 2×n₂ + 3×n₃ + 4×n₄
- n₀ + n₁ + n₂ + n₃ + n₄ - 1 = n₁ + 2n₂ + 3n₃ + 4n₄
- n₀ = 1 + n₂ + 2n₃ + 3n₄ = 1 + 1×1 + 2×10 + 3×20 = 82

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 度为4的树T
- n₄ = 20（度为4的结点）
- n₃ = 10（度为3的结点）
- n₂ = 1（度为2的结点）
- n₁ = 10（度为1的结点）
- 求：叶结点数 n₀

### 第二步：回顾树的基本公式
**公式1**：总结点数 = 各度结点数之和
n = n₀ + n₁ + n₂ + n₃ + n₄

**公式2**：树中边数 = 结点数 - 1 = n - 1

**公式3**：树中边数 = 所有结点的度之和（每个度为k的结点贡献k条边）
边数 = 1×n₁ + 2×n₂ + 3×n₃ + 4×n₄

### 第三步：联立求解
```
n - 1 = 1×n₁ + 2×n₂ + 3×n₃ + 4×n₄
n₀ + n₁ + n₂ + n₃ + n₄ - 1 = n₁ + 2n₂ + 3n₃ + 4n₄
n₀ = 1 + n₂ + 2n₃ + 3n₄
```

代入数据：
```
n₀ = 1 + 1 + 2×10 + 3×20
   = 1 + 1 + 20 + 60
   = 82
```

### 第四步：逐项判断
- A. 41 ❌ — 可能是 n₀ = n₂ + 2n₃ + 3n₄（忘记+1）
- B. 82 ✅ — 正确
- C. 113 ❌ — 可能是 n₀ = 1 + n₁ + n₂ + n₃ + n₄（全加）
- D. 122 ❌ — 可能是 n₀ = 2 + 2n₂ + 4n₃ + 6n₄（系数翻倍）

答案：**B**

## 易错点分析

### 陷阱1：忘记公式中的"+1"
❌ **错误推导**：n₀ = n₂ + 2n₃ + 3n₄ = 1 + 20 + 60 = 81
✅ **正确公式**：n₀ = **1** + n₂ + 2n₃ + 3n₄ = 82
```
n₀ + Σn_k - 1 = Σ(k×n_k)
n₀ = 1 + Σ((k-1)×n_k)
```

### 陷阱2：混淆"度"和"出度"
❌ 误以为度为k的结点贡献k+1条边
✅ 在树中，除根外每个结点恰好有一条入边，度为k的结点贡献k条出边（即k个孩子）

## 知识点对比

| 对比项 | 度为m的树 | 二叉树 | 公式差异 |
|--------|----------|--------|---------|
| 结点度定义 | 0~m | 0~2 | m可以>2 |
| 叶结点公式 | n₀=1+Σ((k-1)n_k) | n₀=n₂+1 | 二叉树是特例(k≤2) |
| 边数公式 | n-1 | n-1 | 相同 |
| 满m叉树结点数 | (m^(h+1)-1)/(m-1) | 2^(h+1)-1 | m=2时退化为二叉树公式 |

**记忆口诀**：叶结点数 = 1 + 一度贡献0 + 二度贡献1 + 三度贡献2 + 四度贡献3（每个非叶结点贡献度-1个"额外"叶结点位置）
