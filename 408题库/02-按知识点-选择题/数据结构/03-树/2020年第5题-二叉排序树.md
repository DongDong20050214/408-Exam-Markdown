# 2020年第5题

**元数据**
- 年份：[[../../01-真题原卷/2020年408真题#05|2020年第5题]]
- 科目：#数据结构
- 知识点：#数据结构/树/二叉排序树构建
- 难度：⭐⭐

**题干**

下列给定的关键字输入序列中，不能生成右侧二叉排序树的是（）。

A. 4, 5, 2, 1, 3 

B. 4, 5, 1, 2, 3 

C. 4, 2, 5, 3, 1 

D. 4,2,1,3,5 

**答案**：C

**考点分析**

本题考查二叉排序树的构建过程。二叉排序树（BST）的插入规则：小于根节点的插入左子树，大于根节点的插入右子树。

根据题目描述的二叉排序树结构应该是：
```
    4
   / \
  2   5
 / \
1   3
```

逐一验证各选项：

**A. 4, 5, 2, 1, 3**
- 4作为根
- 5 > 4，插入右子树
- 2 < 4，插入左子树
- 1 < 4且1 < 2，插入2的左子树
- 3 < 4且3 > 2，插入2的右子树
- ✓ 可以生成

**B. 4, 5, 1, 2, 3**
- 4作为根
- 5 > 4，插入右子树
- 1 < 4，插入左子树
- 2 < 4且2 > 1，插入1的右子树
- 3 < 4且3 > 1且3 > 2，插入2的右子树
- ✗ 生成的树中1是2的父节点，与目标不符

**C. 4, 2, 5, 3, 1**
- 4作为根
- 2 < 4，插入左子树
- 5 > 4，插入右子树
- 3 < 4且3 > 2，插入2的右子树
- 1 < 4且1 < 2，插入2的左子树
- ✓ 可以生成

**D. 4, 2, 1, 3, 5**
- 4作为根
- 2 < 4，插入左子树
- 1 < 4且1 < 2，插入2的左子树
- 3 < 4且3 > 2，插入2的右子树
- 5 > 4，插入右子树
- ✓ 可以生成

实际上选项B无法生成目标树，但答案是C，需要重新审视目标树的结构。

## 解题思路

### 完整模拟验证

需要根据题目给出的目标二叉排序树结构，逐一验证各选项的插入顺序是否能生成该结构。

**关键**：二叉排序树的插入顺序不同会生成不同的树结构，第一个插入的元素必定是根节点。

**验证方法**：
1. 确认目标树的根节点
2. 模拟每个选项的插入过程
3. 对比生成的树结构是否与目标一致

由于答案是C，说明选项C的插入顺序无法生成目标树结构。具体原因需要结合原题图示进行完整模拟验证。

## 易错点分析

### 易错点1：模拟过程出错
❌ 错误理解：只检查前几步插入，未完整模拟到最后一个元素
✅ 正确理解：必须完整模拟每个序列的插入过程。C选项4,2,5,3,1：插入3后3在2的右子树，再插1时1在2的左子树，最终结构4(2(1,3),5)与目标4(2(1,3),5)相同，可能题目目标树不同或需核对原图

### 易错点2：BST插入规则
| 当前值vs根 | 操作 | 示例 |
|-------|-------|------|
| <根 | 插入左子树 | 2<4插左 |
| >根 | 插入右子树 | 5>4插右 |
| 递归直到空位 | 找到叶子插入 | 1<2插2的左 |
