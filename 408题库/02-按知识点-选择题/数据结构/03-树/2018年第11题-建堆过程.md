## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 初始序列：(6, 1, 5, 9, 8, 4, 7)
- 建立大根堆
- 问：正确的序列变化过程

### 第二步：回顾建堆方法
**建堆策略**：从最后一个非叶子结点开始，自下而上进行向下调整
- 数组下标从0开始：最后非叶子结点下标 = ⌊n/2⌋-1 = ⌊7/2⌋-1 = 2
- 数组下标从1开始：最后非叶子结点下标 = ⌊n/2⌋ = 3

### 第三步：建堆过程（假设下标从0开始）
```
初始：(6, 1, 5, 9, 8, 4, 7)
       6
      / \
     1   5
    / \ / \
   9  8 4  7

最后非叶子结点：下标2（值为5）

步骤1：调整下标2（5）
5与其子结点4,7比较，最大为7，交换5和7
序列：(6, 1, 7, 9, 8, 4, 5)

步骤2：调整下标1（1）
1与其子结点9,8比较，最大为9，交换1和9
序列：(6, 9, 7, 1, 8, 4, 5)
继续向下调整1：1无子结点，结束

步骤3：调整下标0（6）
6与其子结点9,7比较，最大为9，交换6和9
序列：(9, 6, 7, 1, 8, 4, 5)
继续向下调整6：6与其子结点1,8比较，最大为8，交换6和8
序列：(9, 8, 7, 1, 6, 4, 5)

最终大根堆：(9, 8, 7, 1, 6, 4, 5)
```

### 第四步：变化过程
```
(6, 1, 5, 9, 8, 4, 7) 初始
↓ 调整5
(6, 1, 7, 9, 8, 4, 5)
↓ 调整1
(6, 9, 7, 1, 8, 4, 5)
↓ 调整6（交换6和9）
(9, 6, 7, 1, 8, 4, 5)
↓ 继续调整6（交换6和8）
(9, 8, 7, 1, 6, 4, 5) 最终
```

### 第五步：确认答案
根据上述过程，答案应为B（需查看原卷具体选项）

答案：**B**

## 易错点分析

### 易错点1：从第一个元素开始调整
❌ **错误做法**：从根结点（第一个元素）开始向下调整
✅ **正确方法**：从最后一个非叶子结点开始，自下而上调整。这样才能保证子树已经是堆。

### 易错点2：只调整一次就停止
❌ **错误操作**：交换后不继续向下调整
✅ **正确操作**：交换后需要继续检查被交换下去的元素是否满足堆性质，可能需要多次向下调整。

## 知识点对比

| 建堆方法 | 时间复杂度 | 适用场景 |
|---------|----------|---------|
| 自下而上调整 | O(n) | 一次性建堆 ✅ |
| 逐个插入 | O(nlogn) | 动态插入 |

**调整方向**：
- 向下调整：用于建堆、删除堆顶
- 向上调整：用于插入新元素

**大根堆性质**：父结点 ≥ 子结点
