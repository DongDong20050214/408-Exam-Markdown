# 2011年第4题

**元数据**
- 年份：2011年第4题
- 科目：#数据结构
- 难度：⭐⭐

**题干**

若一棵完全二叉树有 768 个结点，则该二叉树中的叶结点数是

A. 257 

B. 258 

C. 384 

D. 385 

**答案**：C

**考点分析**

本题考查完全二叉树的性质。设叶结点数为 $n_0$，度为1的结点数为 $n_1$，度为2的结点数为 $n_2$。对于二叉树，有关系式 $n_0 = n_2 + 1$。总结点数 $n = n_0 + n_1 + n_2 = 768$。对于完全二叉树，$n_1$ 只能是 0 或 1。若 $n_1 = 0$，则 $n_0 + n_2 = 768$，结合 $n_0 = n_2 + 1$，得 $2n_0 - 1 = 768$，$n_0 = 384.5$（不是整数，舍去）。若 $n_1 = 1$，则 $n_0 + n_2 = 767$，结合 $n_0 = n_2 + 1$，得 $2n_0 - 1 = 767$，$n_0 = 384$。

---

## 解题思路

### 第一步：理解题目核心

**完全二叉树性质**：
- n₀ = 叶结点数（度为0）
- n₁ = 度为1的结点数
- n₂ = 度为2的结点数
- 总结点数：n = n₀ + n₁ + n₂

**二叉树基本性质**：
- n₀ = n₂ + 1（叶结点数 = 度为2的结点数 + 1）

**完全二叉树特殊性质**：
- n₁ 只能是 0 或 1（至多有一个度为1的结点）

### 第二步：分类讨论

**情况1：n₁ = 0（没有度为1的结点）**
```
总结点数：n₀ + n₂ = 768
基本性质：n₀ = n₂ + 1

代入消元：
(n₂ + 1) + n₂ = 768
2n₂ + 1 = 768
2n₂ = 767
n₂ = 383.5

判断依据：n₂必须是整数，383.5不是整数
结论：❌ 这种情况不可能
```

**情况2：n₁ = 1（有一个度为1的结点）**
```
总结点数：n₀ + 1 + n₂ = 768
即：n₀ + n₂ = 767
基本性质：n₀ = n₂ + 1

代入消元：
(n₂ + 1) + n₂ = 767
2n₂ + 1 = 767
2n₂ = 766
n₂ = 383

叶结点数：n₀ = n₂ + 1 = 383 + 1 = 384

判断依据：结果为整数，符合完全二叉树性质
结论：✅ 叶结点数 = 384
```

### 第三步：逐项分析

**A. 257 ❌**
```
详细分析：
257无法通过完全二叉树的性质推导出来
可能错误推导：某种错误的层数计算

判断依据：不符合n₀ = n₂ + 1的关系
结论：❌ 错误
```

**B. 258 ❌**
```
详细分析：
258也无法通过正确的性质推导
可能错误：误用了某些不适用的公式

判断依据：不满足完全二叉树的约束条件
结论：❌ 错误
```

**C. 384 ✅**
```
详细分析：
通过正确的分类讨论：
- 完全二叉树n₁ = 1
- n₀ + n₂ = 767
- n₀ = n₂ + 1
- 解得 n₀ = 384

验证：n₀ + n₁ + n₂ = 384 + 1 + 383 = 768 ✓

判断依据：符合所有完全二叉树性质
结论：✅ 正确答案
```

**D. 385 ❌**
```
详细分析：
若n₀ = 385，则n₂ = 384（由n₀ = n₂ + 1）
总数 = n₀ + n₁ + n₂ = 385 + n₁ + 384 = 769 + n₁

即使n₁ = 0，总数也是769，不是768

判断依据：不满足总结点数768的条件
结论：❌ 错误
```

### 第四步：确认答案

叶结点数 = 384 → 答案：**C**

## 易错点分析

### 易错点1：忘记完全二叉树的n₁约束
❌ 错误理解：直接用n₀ = n₂ + 1和n₀ + n₁ + n₂ = 768求解，忽略n₁的取值限制
✅ 正确理解：完全二叉树的n₁只能是0或1，必须分类讨论

### 易错点2：混淆n₀ = n₂ + 1的适用范围
❌ 错误想法：认为这个公式只适用于满二叉树
✅ 正确理解：n₀ = n₂ + 1适用于**所有二叉树**（不仅是完全二叉树）

**公式推导**：
```
边数 = 总结点数 - 1
边数 = n₁×1 + n₂×2（每个结点的度数之和）

n₀ + n₁ + n₂ - 1 = n₁ + 2n₂
n₀ + n₁ + n₂ = n₁ + 2n₂ + 1
n₀ = n₂ + 1
```

### 易错点3：不进行分类讨论直接求解
❌ 错误做法：
```
n₀ + n₁ + n₂ = 768
n₀ = n₂ + 1
解得 n₀ = (769 - n₁) / 2
```
没有利用完全二叉树n₁∈{0,1}的约束。

✅ 正确做法：先判断n₁=0是否可行，再判断n₁=1是否可行
