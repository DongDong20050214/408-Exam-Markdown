# 2021年408真题第03题

## 题目

已知二维数组 A 按行优先方式存储，每个元素占用 1 个存储单元。若元素 A[0][0] 的存储地址是 100，A[3][3] 的存储地址是 220，则元素 A[5][5] 的存储地址是（）。

A. 295

B. 300

C. 301

D. 306

## 答案

B

## 解题思路

### 第一步：理解二维数组按行优先存储规则
二维数组A[m][n]按行优先存储时：
- 存储顺序：A[0][0], A[0][1], ..., A[0][n-1], A[1][0], A[1][1], ...
- 地址公式：Loc(A[i][j]) = Loc(A[0][0]) + (i×n + j) × 每个元素大小
- 本题每个元素占1个存储单元

### 第二步：逐项分析

**建立方程求列数n**
```
已知条件：
- Loc(A[0][0]) = 100
- Loc(A[3][3]) = 220
- 每个元素占1个单元

代入公式：
Loc(A[3][3]) = Loc(A[0][0]) + (3×n + 3)×1
220 = 100 + 3n + 3
3n + 3 = 120
3n = 117
n = 39

结论：数组每行有39列
```

**计算A[5][5]的地址**
```
代入公式：
Loc(A[5][5]) = Loc(A[0][0]) + (5×39 + 5)×1
            = 100 + (195 + 5)
            = 100 + 200
            = 300

结论：A[5][5]的地址是300
```

**验证各选项**
- **A. 295 ❌**：偏移量错误，295-100=195，对应元素A[5][0]而非A[5][5]
- **B. 300 ✅**：正确，偏移200对应(5×39+5)
- **C. 301 ❌**：多算1个单元，可能误算为A[5][6]
- **D. 306 ❌**：计算错误，无对应规律

### 第三步：确认答案
通过列数n=39计算得A[5][5]地址为300 → 答案：**B**

## 易错点分析

### 易错点1：混淆行优先与列优先存储
❌ 错误理解：列优先存储时公式也是i×n+j
✅ 正确理解：
| 存储方式 | 地址公式 | 存储顺序示例 |
|----------|----------|--------------|
| 行优先 | i×n+j | A[0][0], A[0][1], A[1][0], A[1][1] |
| 列优先 | j×m+i | A[0][0], A[1][0], A[0][1], A[1][1] |

### 易错点2：偏移量计算漏掉j项
❌ 错误思路：从A[0][0]到A[3][3]偏移量是3×n（只算了3行）
✅ 正确理解：
```
完整偏移量 = i×n + j
从A[0][0]到A[3][3]：
- 走过3整行：3×n个元素
- 第4行走3列：+3个元素
- 总偏移：3n+3
```

**选项分析**：
- ❌ A. 295 —— 偏移量错误，可能误算为其他元素
- ✅ B. 300 —— 正确：建立方程220=100+3n+3得n=39，则Loc(A[5][5])=100+(5×39+5)=300
- ❌ C. 301 —— 多算1个单元，可能误算为A[5][6]
- ❌ D. 306 —— 计算错误

**易错点分析**

### 易错点1：偏移量计算漏掉列偏移j
❌ 错误理解：从A[0][0]到A[3][3]偏移量只算3整行3n，忽略列偏移
✅ 正确理解：完整偏移=i×n+j。A[3][3]偏移=3n+3，包含3整行+第4行前3列

### 易错点2：下标起点混淆
❌ 错误理解：不确定下标从0还是从1开始
✅ 正确理解：C语言数组下标从0开始，A[0][0]是第1个元素，A[i][j]偏移量恰好是i×n+j

**知识点对比**

| 存储方式 | 地址公式 | 本题应用 |
|---------|---------|---------|
| 行优先 | Loc(A[i][j])=Loc(A[0][0])+(i×n+j)×size | 220=100+3n+3→n=39 |
