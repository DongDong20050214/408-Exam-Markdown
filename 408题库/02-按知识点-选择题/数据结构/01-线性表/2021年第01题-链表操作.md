# 2021年408真题第01题

## 题目

已知头指针 h 指向一个带头结点的非空单循环链表，结点结构为 data next，其中 next 是指向直接后继结点的指针，p 是尾指针，q 是临时指针。现要删除该链表的第一个元素，正确的语句序列是（）。

A. h -> next = h -> next -> next; q = h -> next; free(q);

B. q = h -> next; h -> next = h -> next -> next; free(q);

C. q = h -> next; h -> next = q -> next; if(p != q) p = h; free(q);

D. q = h -> next; h -> next = q -> next; if(p == q) p = h; free(q);

## 答案

D

## 考点分析

### 考点：单循环链表的删除操作

本题考查单循环链表中删除第一个元素的正确操作步骤，特别是需要处理尾指针的维护问题。

### 解题思路：

1. **单循环链表的特点**：
   - 带头结点，头指针 h 指向头结点
   - 第一个元素是 h->next（头结点的下一个）
   - 尾结点的 next 指针指向头结点（而非 NULL）
   - p 是尾指针，指向最后一个数据结点

2. **删除第一个元素的步骤**：
   - 用临时指针 q 保存要删除的结点：`q = h->next`
   - 修改头结点的 next 指针，跳过第一个元素：`h->next = q->next`
   - 释放被删除结点：`free(q)`

3. **特殊情况处理**：
   - 如果被删除的是唯一的数据结点（即 p == q），删除后尾指针 p 应指向头结点
   - 条件判断：`if(p == q) p = h;`

4. **选项分析**：
   - 选项 A：先修改指针再保存，导致无法正确释放
   - 选项 B：没有处理尾指针 p 的维护
   - 选项 C：判断条件错误，应该是 `p == q` 而非 `p != q`
   - 选项 D：正确，先保存、再修改、再处理特殊情况、最后释放

### 关键点：

- 删除结点前必须先保存指针，否则无法正确释放内存
- 单循环链表删除唯一数据结点时，尾指针需要指向头结点
- 判断条件：当 p == q 时说明只有一个数据结点

### 易错点：

- 忘记处理尾指针的维护
- 判断条件写反（p != q vs p == q）
- 指针修改顺序错误导致内存泄漏或野指针
