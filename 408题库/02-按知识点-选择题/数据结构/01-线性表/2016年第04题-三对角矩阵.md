# 三对角矩阵真题汇总

## 2016年第04题

**元数据**
- 年份：[../../01-真题原卷/2016年408真题#04|2016年第4题](../../01-真题原卷/2016年408真题#04|2016年第4题.md)
- 科目：#数据结构
- 知识点：#数据结构/数组/三对角矩阵
- 难度：⭐⭐⭐
- 关联知识：[知识库/数据结构/特殊矩阵](知识库/数据结构/特殊矩阵.md)

**题干**

有一个 100 阶的三对角矩阵 M，其元素 m_{i,j}（1≤i≤100, 1≤j≤100）按行优先依次压缩存入下标从 0 开始的一维数组 N 中。元素 m_{30,30} 在数组 N 中的下标是（）。

A. 86  B. 87  C. 88  D. 89

**答案**：B

**解题思路**

**三对角矩阵定义**：$|i-j| \leq 1$ 时有值，否则为0
```
主对角线、上对角线、下对角线有值
例：[* * 0 0]
    [* * * 0]
    [0 * * *]
    [0 0 * *]
```

**每行元素数**：
- 第1行：2个（主对角线+上对角线）
- 第2~n-1行：3个（下+主+上）
- 第n行：2个（下对角线+主对角线）

**100阶三对角矩阵**：
- 第1行：m_{1,1}, m_{1,2}（2个）
- 第2~99行：每行3个
- 第100行：m_{100,99}, m_{100,100}（2个）

**计算m_{30,30}位置**：

前29行元素总数：
- 第1行：2个
- 第2~29行：28×3 = 84个
- 总计：2 + 84 = 86个

第30行中，m_{30,30}是主对角线元素：
- m_{30,29}（第1个）
- m_{30,30}（第2个）

下标（从0开始）：86 + 1 = 87

**选项分析**：
- ❌ A. 86
- ✅ B. 87
- ❌ C. 88
- ❌ D. 89

**易错点分析**

❌ **错误1：每行都算3个元素**
- 忽略第1行只有2个
- ✅ 正确：第1行特殊，只有2个

❌ **错误2：m_{30,30}位置算错**
- 认为是第30行第3个
- ✅ 正确：是第2个（第1个是m_{30,29}）

**知识点对比**

| 矩阵类型 | 非零元素数 | 压缩存储 |
|---------|-----------|---------|
| **对称矩阵** | $\frac{n(n+1)}{2}$ | 存上/下三角 |
| **三对角矩阵** | $3n-2$ | 按行存储 |
| **上三角** | $\frac{n(n+1)}{2}$ | 按行/列存储 |

**三对角矩阵元素数公式**

n阶三对角矩阵：
- 第1行：2个
- 第2到n-1行：3(n-2)个
- 第n行：2个
- 总计：2 + 3(n-2) + 2 = 3n-2

100阶：3×100-2 = 298个元素

**前k行元素数**

| 行数k | 元素总数 |
|------|---------|
| 1 | 2 |
| 2~k | 2 + 3(k-1) = 3k-1 |

前29行：3×29-1 = 86

**关键理解**：
- 第1行只有2个元素（特殊）
- 第2~n-1行每行3个元素
- 按行存储时需考虑第1行的特殊性

## 解题思路

### 第一步：理解三对角矩阵的结构
三对角矩阵定义：只有主对角线、上对角线、下对角线有值，即当|i-j|≤1时m_{i,j}有值
```
对100阶矩阵：
第1行：m_{1,1}, m_{1,2} (2个元素)
第i行(2≤i≤99)：m_{i,i-1}, m_{i,i}, m_{i,i+1} (3个元素)
第100行：m_{100,99}, m_{100,100} (2个元素)
```

### 第二步：计算m_{30,30}的存储位置
按行优先存储，需要计算前29行的元素总数，再加上第30行中m_{30,30}的位置

**前29行元素总数**：
- 第1行：2个
- 第2~29行：28行×3个/行 = 84个
- 合计：2 + 84 = 86个

**第30行内位置**：
- m_{30,29}是第1个元素（下标86）
- m_{30,30}是第2个元素（下标87）

### 第三步：逐项分析

**A. 86 ❌**
```
详细分析：
  86是前29行的元素总数
  这是m_{30,29}的下标，不是m_{30,30}
错误原因：
  把前29行的总数误认为是m_{30,30}的位置
  忽略了m_{30,30}在第30行不是第一个元素
结论：错误
```

**B. 87 ✅**
```
详细分析：
  前29行共86个元素（占用下标0~85）
  第30行有3个元素：m_{30,29}, m_{30,30}, m_{30,31}
  m_{30,30}是第30行第2个元素
  下标计算：86 + 1 = 87
判断依据：
  下标从0开始，前面有86个元素
  m_{30,30}是第87个元素，下标为87
结论：正确 ✅
```

**C. 88 ❌**
```
详细分析：
  88是m_{30,31}的下标
错误原因：
  把m_{30,30}的位置多算了一个
  可能是把第30行第1个算成了m_{30,30}
结论：错误
```

**D. 89 ❌**
```
详细分析：
  89远大于正确答案87
错误原因：
  可能把第1行算成了3个元素
  错误计算：前29行=29×3=87，加上位置=89
  正确计算：前29行=2+28×3=86，加上位置=87
结论：错误
```

### 第四步：确认答案
前29行共86个元素，m_{30,30}是第30行第2个，下标为87 → 答案：**B**

## 易错点分析

### 易错点1：忽略第1行只有2个元素
❌ 错误理解：每行都有3个元素，前29行共29×3=87个
✅ 正确理解：第1行只有2个，第2~29行每行3个，共2+28×3=86个

**常见错误计算**：
```
错误：前29行 = 29×3 = 87个
正确：第1行2个 + 第2~29行28×3 = 2+84 = 86个
```

### 易错点2：混淆元素总数和下标
❌ 错误理解：前29行有86个元素，所以m_{30,30}的下标是86
✅ 正确理解：前29行86个元素占用下标0~85，m_{30,30}从86开始算起

| 元素 | 元素总数 | 下标范围 | 下一个元素下标 |
|------|---------|---------|---------------|
| 前29行 | 86个 | 0~85 | 86 |
| m_{30,29} | 第87个 | 86 | 87 |
| m_{30,30} | 第88个 | 87 | 88 |

### 易错点3：不清楚第i行元素在该行内的顺序
| 行号 | 该行元素 | m_{i,i}的行内位置 |
|------|---------|------------------|
| 1 | m_{1,1}, m_{1,2} | 第1个 |
| 2~n-1 | m_{i,i-1}, m_{i,i}, m_{i,i+1} | 第2个 |
| n | m_{n,n-1}, m_{n,n} | 第2个 |

**第30行具体分析**：
```
第30行(2≤30≤99)有3个元素：
位置1：m_{30,29} (下对角线)
位置2：m_{30,30} (主对角线) ← 求的就是这个
位置3：m_{30,31} (上对角线)
```
