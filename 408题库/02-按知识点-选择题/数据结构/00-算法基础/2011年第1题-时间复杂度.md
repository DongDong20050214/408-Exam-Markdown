# 2011年第1题

**元数据**
- 年份：2011年第1题
- 科目：#数据结构
- 难度：⭐⭐

**题干**

设 $n$ 是描述问题规模的非负整数，下面程序片段的时间复杂度是（ ）。

$$
\begin{array}{r l} & \mathbf {x} = 2; \\ & \text { while } (\mathbf {x} <   n / 2) \\ & \quad \mathbf {x} = 2 ^ {*} \mathbf {x}; \end{array}
$$

A. $O(\log_2 n)$ 

B. $O(n)$ 

C. $O(n\log_2n)$ 

D. $O(n^{2})$ 

**答案**：A

**考点分析**

本题考查算法时间复杂度分析。循环中 x 每次乘以 2（即 $x = 2x$），因此 x 的变化序列为 2, 4, 8, 16, ..., 直到 $x \geq n/2$ 时停止。假设循环执行 k 次，则有 $2^{k+1} \geq n/2$，即 $k+1 \geq \log_2(n/2) = \log_2 n - 1$，所以 $k = O(\log_2 n)$。这是典型的指数增长导致的对数时间复杂度。

## 解题思路（逐项分析法）

### 第一步：明确题目条件
```
x = 2
while (x < n/2)
    x = 2*x
```

### 第二步：分析循环次数
x的变化序列：
```
第1次：x = 2 = 2^1
第2次：x = 4 = 2^2
第3次：x = 8 = 2^3
第k次：x = 2^k
```
循环终止条件：$2^k \geq n/2$
即：$k \geq \log_2(n/2) = \log_2 n - 1$

### 第三步：逐项判断

**A. $O(\log_2 n)$ ✅**
```
循环k次满足：2^k ≥ n/2
k ≥ log₂n - 1
k = O(log n)
结论：正确 ✅
```

**B. $O(n)$ ❌**
```
线性复杂度：循环n次
本题：循环log n次
log n << n
结论：错误
```

**C. $O(n\log_2n)$ ❌**
```
常见于归并排序等
本题只有一层循环
结论：错误
```

**D. $O(n^{2})$ ❌**
```
二次复杂度需嵌套循环
本题单层循环
结论：错误
```

### 第四步：确认答案
答案：**A**

## 易错点分析

### 陷阱1：看到乘法就认为是O(n)
❌ 错误理解：x乘以2，循环n次
✅ 正确理解：x指数增长，循环log n次

### 陷阱2：混淆变量值和循环次数
❌ 错误理解：x增长到n，复杂度O(n)
✅ 正确理解：x增长速度快，次数少

## 知识点对比

| 循环方式 | 循环次数 | 复杂度 |
|---------|---------|--------|
| i++ | n | O(n) |
| i*=2 | log n | O(log n) |
| i+=k | n/k | O(n) |
