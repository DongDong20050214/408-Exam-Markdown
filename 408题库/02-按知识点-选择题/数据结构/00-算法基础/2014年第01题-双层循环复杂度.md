# 双层循环时间复杂度真题汇总

## 2014年第01题

**元数据**
- 年份：[../../01-真题原卷/2014年408真题#01|2014年第1题](../../01-真题原卷/2014年408真题#01|2014年第1题.md)
- 科目：#数据结构
- 知识点：#数据结构/算法/时间复杂度
- 难度：⭐⭐
- 关联知识：[知识库/数据结构/算法分析](知识库/数据结构/算法分析.md)

**题干**

下列程序段的时间复杂度是（）。

```
count = 0;
for (k=1; k<=n; k*=2)
    for (j=1; j<=n; j++)
        count++;
```

A. O(log₂n)  
B. O(n)  
C. O(nlog₂n)  
D. O(n²)

**答案**：C

**考点分析**
- 外层循环：k从1到n，每次×2，执行log₂n次
- 内层循环：每次执行n次
- 总次数：n×log₂n = O(nlog₂n)

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 外层循环：k=1, k<=n, k*=2（指数增长）
- 内层循环：j=1, j<=n, j++（线性增长）
- 操作：count++（常数时间）

### 第二步：回顾嵌套循环复杂度分析
嵌套循环时间复杂度 = 外层次数 × 内层次数
- 独立循环：分别计算后相乘
- 依赖循环：需要求和

### 第三步：分别计算循环次数
**外层循环**：
```
k的变化：1 → 2 → 4 → 8 → ... → n
每次翻倍：k = 2^i
循环条件：2^i ≤ n
循环次数：i ≤ log₂n，执行⌊log₂n⌋+1 ≈ log₂n次
```

**内层循环**：
```
j从1到n，每次+1
循环次数：n次
```

**总操作次数**：
外层每执行1次，内层执行n次
总次数 = log₂n × n = O(nlog₂n)

### 第四步：确认答案
外层O(log n)×内层O(n) → 答案：**C**

## 易错点分析

### 陷阱1：只看到两层循环就认为是O(n²)
❌ **错误理解**：双层循环都是O(n²)
✅ **正确理解**：要看每层循环的实际执行次数
- 外层log₂n × 内层n = O(nlog₂n)
- 不是n × n = O(n²)

### 陷阱2：误认为外层是O(n)
❌ **错误理解**：k从1到n，所以外层是O(n)
✅ **正确理解**：k每次翻倍，不是每次+1
- k*=2（指数增长）→ O(log n)
- k++（线性增长）→ O(n)

### 陷阱3：混淆独立循环和嵌套循环
❌ **错误理解**：外层O(log n)，内层O(n)，所以是O(log n + n) = O(n)
✅ **正确理解**：
- 串行执行（两个独立循环）：相加，O(log n + n) = O(n)
- 嵌套执行（内层在外层内部）：相乘，O(log n × n) = O(nlog n)

## 知识点对比

| 循环结构 | 代码示例 | 时间复杂度 | 说明 |
|---------|---------|-----------|------|
| 单层线性 | `for(i=1;i<=n;i++)` | O(n) | 基础循环 |
| 单层对数 | `for(i=1;i<=n;i*=2)` | O(log n) | 倍增循环 |
| 双层线性 | `for(i)for(j)` (都到n) | O(n²) | 常见嵌套 |
| 线性+对数 | `for(i到n)for(j倍增到n)` | O(nlog n) | **本题类型** |
| 对数+线性 | `for(i倍增到n)for(j到n)` | O(nlog n) | 与上等价 |
| 依赖嵌套 | `for(i到n)for(j到i)` | O(n²) | j上限依赖i |

**记忆口诀**：嵌套循环看相乘，独立循环看相加
