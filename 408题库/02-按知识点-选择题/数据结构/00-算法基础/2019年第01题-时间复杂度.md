# 2019年第1题

**元数据**
- 年份：2019年第1题
- 难度：⭐⭐

**题干**：设 $n$ 是描述问题规模的非负整数，下列程序段的时间复杂度是（ ）。

$$
\begin{array}{l} \mathrm{x=0;} \\ \text { while } (\mathrm{n>= (x+1)*(x+1)}) \\ \mathrm {x = x + 1;} \end{array}
$$

A. $O(\log n)$ 
B. $O(n^{1 / 2})$ 
C. $O(n)$ 
D. $O(n^{2})$ 

**答案**：B

**考点分析**：本题考查算法时间复杂度分析。循环条件是n>=(x+1)²，即(x+1)²≤n，解得x+1≤√n，即x<√n。循环次数约为√n次，因此时间复杂度为O(√n)=O(n^(1/2))。

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 初始值：x = 0
- 循环条件：n >= (x+1) × (x+1)
- 每次操作：x = x + 1（线性增长）

### 第二步：回顾平方根复杂度特征
当循环变量的平方与n比较时，复杂度通常是O(√n)
- n >= x² → x最大约为√n
- 循环√n次

### 第三步：推导循环次数
```
循环条件：n >= (x+1)²
即：(x+1)² <= n
    x+1 <= √n
    x <= √n - 1

x从0增长到√n-1：
第1次：x=0，判断n>=1
第2次：x=1，判断n>=4
第3次：x=2，判断n>=9
...
第k次：x=k-1，判断n>=k²

终止时：x ≈ √n
循环次数：√n次
```

### 第四步：确认答案
x线性增长到√n → 答案：**B**

## 易错点分析

### 陷阱1：看到x+1就认为是O(n)
❌ **错误理解**：x每次加1，所以是O(n)
✅ **正确理解**：x只增长到√n就停止了
- x的范围：0 → √n
- 不是：0 → n

### 陷阱2：混淆循环条件和复杂度
❌ **错误理解**：条件里有(x+1)²，所以是O(n²)
✅ **正确理解**：条件中的平方限制了x的范围
- (x+1)² ≤ n 推出 x ≤ √n
- 复杂度看循环次数，不是看条件表达式

### 陷阱3：误算成O(log n)
❌ **错误理解**：有比较大小，应该是对数
✅ **正确理解**：
- x是线性增长（每次+1），不是指数增长（每次×2）
- 线性增长到√n：O(√n)
- 指数增长到n：O(log n)

## 知识点对比

| 循环终止条件 | 循环变量范围 | 增长方式 | 时间复杂度 |
|-------------|------------|---------|-----------|
| x < n | 0 → n | 线性+1 | O(n) |
| x < √n | 0 → √n | 线性+1 | O(√n) |
| x² < n | 0 → √n | 线性+1 | O(√n) **本题** |
| x < log n | 0 → log n | 线性+1 | O(log n) |
| 2^x < n | 0 → log n | 线性+1 | O(log n) |

**记忆技巧**：看变量最终能到多大，线性增长的次数就是多少
