# 2019年408真题第01题

## 题目

设 $n$ 是描述问题规模的非负整数，下列程序段的时间复杂度是（ ）。

```c
x = 0;
while (n >= (x+1)*(x+1))
    x = x + 1;
```

A. $O(\log n)$  
B. $O(n^{1/2})$  
C. $O(n)$  
D. $O(n^{2})$

## 答案

B

## 考点分析

### 考点：算法时间复杂度分析

本题考查循环结构的时间复杂度分析，需要通过循环条件推导循环次数。

### 解题思路：

1. **分析循环条件**：
   - 循环条件是 `n >= (x+1)*(x+1)`
   - 即 $n \geq (x+1)^2$
   - 等价于 $x+1 \leq \sqrt{n}$
   - 即 $x < \sqrt{n}$

2. **确定循环次数**：
   - x 从 0 开始，每次加 1
   - 当 $x = \lfloor\sqrt{n}\rfloor$ 时，循环终止
   - 循环执行约 $\sqrt{n}$ 次

3. **时间复杂度**：
   - 循环体内只有常数时间操作
   - 总时间复杂度为 $O(\sqrt{n}) = O(n^{1/2})$

### 关键点：

- 从循环条件反推循环变量的取值范围
- 循环次数与 $\sqrt{n}$ 成正比
- 类似问题：求一个数的平方根的整数部分

### 易错点：

- 误认为是对数复杂度（需注意循环变量是线性增长，不是指数增长）
- 混淆 $O(n^{1/2})$ 和 $O(n)$ 的区别
