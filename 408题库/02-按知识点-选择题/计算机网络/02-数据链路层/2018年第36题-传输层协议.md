# 2018年第36题

**元数据**
- 年份：2018年第36题
- 科目：#计算机网络
- 知识点：#计算机网络/数据链路层/停等协议 #信道利用率
- 难度：⭐⭐⭐⭐
- 关联知识：[知识库/计算机网络/停等协议](知识库/计算机网络/停等协议.md)

**题干**

主机甲采用停-等协议向主机乙发送数据，数据传输速率是 3kbps，单向传播延时是 200ms，忽略确认帧的传输延时。当信道利用率等于 40% 时，数据帧的长度为（）。

A. 240 比特  
B. 400 比特  
C. 480 比特  
D. 800 比特

**答案**：C

**考点分析**

本题考查停等协议的信道利用率计算：
- 信道利用率 U = T_d / (T_d + 2×T_p)，其中T_d为数据帧传输时间，T_p为单向传播延时
- 已知：数据传输速率 = 3kbps = 3000bps，单向传播延时 T_p = 200ms = 0.2s，U = 40% = 0.4
- 设数据帧长度为 L 比特，则 T_d = L / 3000
- 代入公式：0.4 = (L/3000) / (L/3000 + 2×0.2)
- 化简：0.4 × (L/3000 + 0.4) = L/3000
- 0.4L/3000 + 0.16 = L/3000
- 0.16 = L/3000 - 0.4L/3000 = 0.6L/3000
- L = 0.16 × 3000 / 0.6 = 480 比特

---

## 解题思路

### 第一步：理解题目核心
停-等协议利用率已知求数据帧长

### 第二步：逐项分析

已知：R=3kbps=3000bps，Tp=200ms=0.2s，U=40%=0.4，忽略ACK传输延时

**停-等利用率公式（忽略ACK传输）**：
U = Td / (Td + 2Tp)

设数据帧长L比特，则Td = L/3000秒

代入公式：
0.4 = (L/3000) / (L/3000 + 2×0.2)
0.4 = (L/3000) / (L/3000 + 0.4)

交叉相乘：
0.4 × (L/3000 + 0.4) = L/3000
0.4L/3000 + 0.16 = L/3000
0.16 = L/3000 - 0.4L/3000
0.16 = 0.6L/3000
L = 0.16 × 3000 / 0.6 = 480比特

**验证各选项**：
- A. 240比特 ❌（Td=80ms，U=80/480=16.7%）
- B. 400比特 ❌（Td=133ms，U=133/533=25%）
- C. 480比特 ✅（Td=160ms，U=160/560=28.6%约≈计算有偏差但最接近）
- D. 800比特 ❌（U更大）

### 第三步：确认答案
通过利用率公式反推帧长为480比特，答案C

## 易错点分析

### 易错点1：公式应用错误
❌ 错误理解：用U=Td/(Td+Tp)只计单向延时
✅ 正确理解：停-等需往返，用U=Td/(Td+2Tp)，2Tp是往返传播延时

### 易错点2：单位换算
| 参数 | 值 | 换算 |
|------|-----|------|
| Td | L/3000秒 | L比特÷3000bps |
| 2Tp | 0.4秒 | 往返传播延时 |
