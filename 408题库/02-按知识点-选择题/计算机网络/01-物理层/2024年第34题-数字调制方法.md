**题干**

在下列二进制数字调制方法中，需要2个不同频率载波的是（）。

A. ASK  B. PSK  C. FSK  D. DPSK

**答案**：C

**考点分析**
- FSK频移键控：用不同频率表示0和1

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 问题：哪种二进制数字调制方法需要**2个不同频率载波**
- 关键词："2个频率"、"二进制"（0和1两个状态）
- 考查：数字调制原理

### 第二步：回顾数字调制的基本原理

**数字调制三要素**：
```
载波信号：y(t) = A·cos(2πft + φ)
- A：幅度
- f：频率  
- φ：初始相位
```

**三种基本调制方式**：
| 调制方式 | 改变的参数 | 用于表示0和1 |
|---------|----------|-------------|
| ASK（幅移键控） | 幅度A | 两个不同幅度 |
| FSK（频移键控） | 频率f | 两个不同频率 |
| PSK（相移键控） | 相位φ | 两个不同相位 |

### 第三步：逐项分析

**A. ASK（幅移键控）❌**
```
全称：Amplitude Shift Keying
原理：改变载波的幅度
表示方式：
- 0：低幅度（或无信号）
- 1：高幅度
载波频率：1个固定频率f₀
结论：只需1个频率
```

**B. PSK（相移键控）❌**
```
全称：Phase Shift Keying
原理：改变载波的相位
表示方式（二进制PSK/BPSK）：
- 0：相位0°
- 1：相位180°
载波频率：1个固定频率f₀
结论：只需1个频率
```

**C. FSK（频移键控）✅**
```
全称：Frequency Shift Keying
原理：改变载波的频率
表示方式：
- 0：频率f₁（如1200Hz）
- 1：频率f₂（如2400Hz）
载波频率：2个不同频率f₁和f₂
结论：需要2个频率 ✓
```

**D. DPSK（差分相移键控）❌**
```
全称：Differential Phase Shift Keying
原理：用相位的变化（而非绝对相位）表示数据
表示方式：
- 0：相位不变
- 1：相位改变180°
载波频率：1个固定频率f₀
结论：只需1个频率（是PSK的改进型）
```

### 第四步：确认答案
FSK频移键控需要2个不同频率载波 → 答案：**C**

## 易错点分析

### 陷阱1：混淆FSK与其他调制方式的区别
❌ **错误理解**：认为PSK也需要两个频率来表示不同相位
✅ **正确理解**：
- **FSK**：发送0时用频率f₁，发送1时用频率f₂（**频率切换**）
- **PSK**：始终用同一频率f₀，只改变相位（0°或180°）
- **ASK**：始终用同一频率f₀，只改变幅度（大或小）

**形象理解**：
```
FSK = 两个不同音高的哨子（一个高音、一个低音）
PSK = 一个哨子，改变吹气方向（相位）
ASK = 一个哨子，改变吹气力度（幅度）
```

### 陷阱2：误认为"二进制"意味着需要两个载波
❌ **错误理解**：二进制数字调制 = 2个载波
✅ **正确理解**：
- "二进制"指数据只有0和1两种状态
- 只有FSK通过**两个频率**来表示这两种状态
- 其他调制方式用**一个载波的不同变化**表示两种状态

### 陷阱3：DPSK与PSK的关系
**DPSK是PSK的改进**：
| 对比项 | PSK | DPSK |
|-------|-----|------|
| 编码方式 | 绝对相位编码 | 相对相位编码 |
| 表示0/1 | 用绝对相位（0°/180°） | 用相位变化（变/不变） |
| 载波数量 | 1个 | 1个 |
| 优点 | 简单 | 不需要相位同步 |
| 抗噪性 | 需要相位基准 | 更强 |

**结论**：DPSK仍然只用1个频率

## 知识点对比

### 三种基本数字调制方式详解

| 调制方式 | 英文缩写 | 改变参数 | 二进制表示 | 频率数量 | 典型应用 |
|---------|---------|---------|-----------|---------|---------|
| 幅移键控 | ASK | 幅度A | A₁→0, A₂→1 | **1个** | 光纤通信 |
| **频移键控** | **FSK** | **频率f** | **f₁→0, f₂→1** | **2个** | **调制解调器** |
| 相移键控 | PSK | 相位φ | 0°→0, 180°→1 | **1个** | 卫星通信 |
| 差分相移 | DPSK | 相位变化 | 不变→0, 变→1 | **1个** | 移动通信 |

### FSK调制波形示例

```
数据：    1    0    1    1    0    1
         ___       ___  ___       ___
频率f₂：|   |     |   ||   |     |   |
         ___  ___  ___  ___  ___  ___
频率f₁：     |   |         |   |

说明：
- 发送1时：使用高频率f₂（如2400Hz）
- 发送0时：使用低频率f₁（如1200Hz）
- 两个频率交替使用，需要两个振荡器
```

### 三种调制方式的信号表达式

| 调制方式 | 信号表达式 | 说明 |
|---------|-----------|------|
| ASK | s(t) = A(t)·cos(2πf₀t) | A(t)变化，f₀固定 |
| **FSK** | **s(t) = A·cos(2πf(t)·t)** | **f(t)在f₁和f₂间切换** |
| PSK | s(t) = A·cos(2πf₀t + φ(t)) | φ(t)变化，f₀固定 |

### 调制方式的抗噪性能对比

| 调制方式 | 抗噪性 | 实现复杂度 | 频谱效率 |
|---------|-------|-----------|---------|
| ASK | 差（受幅度噪声影响大） | 简单 | 中等 |
| FSK | 中等（需要区分频率） | 中等 | 低（占用频带宽） |
| PSK | 好（相位稳定） | 复杂 | 高 |
| DPSK | 较好（不需相位同步） | 中等 | 高 |

### 记忆口诀

**"频移键控双频道"**
- ASK：幅度Amplitude → 一个频率，变幅度
- **FSK：频率Frequency → 两个频率，来回切换**
- PSK：相位Phase → 一个频率，变相位
