# 信道容量与实际传输速率真题汇总

## 2016年第34题

**元数据**
- 年份：[2016年第34题](../../../01-真题原卷/2016年408真题.md#34)
- 科目：#计算机网络
- 知识点：#计算机网络/物理层/信道容量 #香农公式
- 难度：⭐⭐⭐
- 关联知识：[2023年第34题-信道容量](2023年第34题-信道容量.md)

**题干**

若连接R2和R3链路的频率带宽为8kHz，信噪比为30dB，该链路实际数据传输速率约为理论最大数据传输速率的50%，则该链路的实际数据传输速率约是（）。

A. 8kbps  
B. 20kbps  
C. 40kbps  
D. 80kbps

**答案**：C

**考点分析**
- 香农公式：C = W × log₂(1 + S/N)
- 信噪比30dB：S/N = 10^(30/10) = 1000
- 理论最大速率：C = 8000 × log₂(1001) ≈ 8000 × 10 = 80000 bps = 80kbps
- 实际速率 = 80kbps × 50% = 40kbps

**计算过程**
- W = 8kHz = 8000Hz
- S/N(dB) = 10lg(S/N) = 30 → S/N = 1000
- log₂(1001) ≈ 10 (因为2^10=1024)
- C = 8000 × 10 = 80000bps
- 实际 = 80kbps × 0.5 = 40kbps

---

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 频率带宽：W = 8kHz
- 信噪比：30dB
- 实际速率 = 理论最大速率 × 50%
- 求：实际数据传输速率

### 第二步：回顾香农公式

**香农公式（有噪声信道容量）**：
```
C = W × log₂(1 + S/N)
```
- C：信道容量（理论最大速率）
- W：带宽（Hz）
- S/N：信噪比（功率比）

**信噪比dB转换**：
```
信噪比(dB) = 10 × lg(S/N)
已知：30dB = 10 × lg(S/N)
解得：lg(S/N) = 3
     S/N = 10³ = 1000
```

### 第三步：计算理论最大速率

**步骤1：转换信噪比**
```
30dB = 10lg(S/N)
S/N = 10^(30/10) = 10³ = 1000
```

**步骤2：计算log₂(1+S/N)**
```
log₂(1 + S/N) = log₂(1 + 1000)
              = log₂(1001)
              ≈ log₂(1024)    (近似：1001≈1024=2¹⁰)
              = 10
```

**步骤3：应用香农公式**
```
C = W × log₂(1 + S/N)
  = 8000Hz × 10
  = 80000 bps
  = 80 kbps
```

### 第四步：计算实际速率

```
实际速率 = 理论最大速率 × 50%
        = 80 kbps × 0.5
        = 40 kbps
```

### 第五步：逐项验证

**A. 8kbps ❌**
```
如果实际是8kbps，则理论最大 = 8/0.5 = 16kbps
但计算得理论是80kbps
结论：错误
```

**B. 20kbps ❌**
```
如果实际是20kbps，则理论最大 = 20/0.5 = 40kbps
但计算得理论是80kbps
结论：错误
```

**C. 40kbps ✅**
```
实际 = 80kbps × 50% = 40kbps
计算结果匹配
结论：正确 ✓
```

**D. 80kbps ❌**
```
80kbps是理论最大速率
题目要求实际速率（50%）
结论：错误
```

### 第六步：确认答案
实际速率 = 80kbps × 50% = 40kbps → 答案：**C**

## 易错点分析

### 陷阱1：信噪比dB与信噪比比值混淆
❌ **错误计算**：直接用S/N = 30代入公式
✅ **正确计算**：
```
信噪比dB = 10lg(S/N)
30 = 10lg(S/N)
lg(S/N) = 3
S/N = 10³ = 1000
```

**常见错误**：
- 误用20lg：那是电压比，功率比用10lg
- 误用自然对数ln：应该用常用对数lg（以10为底）

### 陷阱2：log₂计算技巧
❌ **错误方法**：用计算器精确计算log₂(1001)
✅ **正确方法**：利用2的幂次近似
```
2¹⁰ = 1024 ≈ 1001
所以 log₂(1001) ≈ 10
```

**记忆常用值**：
| 2的幂 | 数值 | 应用 |
|------|------|------|
| 2¹⁰ | 1024 | 近似1000 |
| 2⁷ | 128 | 近似100 |
| 2³ | 8 | 8QAM |
| 2² | 4 | QPSK |

### 陷阱3：忘记乘以实际速率系数
❌ **错误答案**：选D（80kbps），忘记×50%
✅ **正确理解**：
- 80kbps是**理论极限**（香农公式给出）
- 实际系统达不到理论极限（编码开销、保护间隔等）
- 题目明确说"实际为理论的50%"

**实际vs理论**：
```
理论最大速率：香农公式的结果（上限）
实际速率：考虑工程实现的限制（通常60%-80%）
本题：实际 = 理论 × 50%
```

### 陷阱4：单位换算错误
注意单位统一：
- 带宽：8**k**Hz = 8000Hz
- 速率：80000bps = 80**k**bps
- 避免：8kHz × 10 = 80Hz（忘记k）

## 知识点对比

### 香农公式 vs 奈奎斯特定理

| 对比项 | 香农公式 | 奈奎斯特定理 |
|-------|---------|-------------|
| 公式 | C = W×log₂(1+S/N) | C = 2W×log₂V |
| 适用条件 | **有噪声信道** | **无噪声信道** |
| 影响因素 | 带宽W、信噪比S/N | 带宽W、信号状态数V |
| 物理意义 | **实际信道极限** | **理想信道极限** |
| 是否可超越 | 不可超越（热噪声限制） | 可接近但受噪声限制 |
| 本题应用 | ✅ 有噪声（给出信噪比） | ❌ 无噪声 |

### 信噪比dB转换公式

| 信噪比类型 | 公式 | 适用 |
|----------|------|------|
| **功率信噪比** | **10lg(S/N)** | **本题（一般情况）** |
| 电压信噪比 | 20lg(Vs/Vn) | 电压测量 |

**推导关系**：
```
功率 ∝ 电压²
功率比 = (电压比)²
10lg(P₁/P₂) = 10lg(V₁²/V₂²) = 20lg(V₁/V₂)
```

### 常见信噪比对应的信道容量

**假设带宽W固定，信噪比变化**：
| 信噪比(dB) | S/N比值 | log₂(1+S/N) | 容量C |
|-----------|---------|-------------|-------|
| 0dB | 1 | log₂(2)=1 | W×1 |
| 10dB | 10 | log₂(11)≈3.5 | W×3.5 |
| 20dB | 100 | log₂(101)≈6.7 | W×6.7 |
| **30dB** | **1000** | **log₂(1001)≈10** | **W×10** |
| 40dB | 10000 | log₂(10001)≈13.3 | W×13.3 |

**规律**：信噪比每增加10dB，容量约增加3.3倍

### 实际系统效率

| 系统类型 | 实际/理论 | 原因 |
|---------|----------|------|
| 理想系统 | 100% | 不存在 |
| 良好设计 | 70%-80% | 编码开销小 |
| 一般系统 | 50%-60% | 有保护间隔、纠错码 |
| **本题** | **50%** | **题目给定** |
