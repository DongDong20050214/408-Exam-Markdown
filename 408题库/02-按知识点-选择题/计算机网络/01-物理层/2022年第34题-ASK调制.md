- 知识点：#计算机网络/物理层/调制 #ASK

- 关联知识：[知识库/计算机网络/调制技术](知识库/计算机网络/调制技术.md)

**题干**

在一条带宽为 200kHz 的无噪声信道上，若采用4个幅值的ASK调制，则该信道的最大数据传输速率是（）。

A. 200kbps  B. 400kbps  C. 800kbps  D. 1600kbps

**答案**：B

**考点分析**
- 奈奎斯特定理：C = 2W × log₂V
- C = 2 × 200kHz × log₂4 = 400k × 2 = 800kbps
- 注意：题目答案B为400kbps，可能题目带宽理解有误或存在其他限制条件

---

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 带宽：W = 200kHz
- 调制方式：ASK（幅移键控）
- 信号幅值数：V = 4
- 信道条件：无噪声
- 求：最大数据传输速率

### 第二步：回顾奈奎斯特定理

**奈奎斯特定理（无噪声信道）**：
```
C = 2W × log₂V
```
- C：信道容量（最大速率）
- W：带宽（Hz）
- V：信号状态数（调制电平数）
- 2W：码元速率（波特率）

**ASK调制原理**：
- 幅移键控：通过改变载波幅度表示数字信号
- V个幅值：每个码元可携带log₂V比特
- 4个幅值：每个码元携带2比特

### 第三步：详细计算

**步骤1：确定码元速率（波特率）**
```
波特率 = 2W
       = 2 × 200kHz
       = 400k Baud（码元/秒）
```

**步骤2：确定每个码元携带的比特数**
```
每码元比特数 = log₂V
            = log₂4
            = 2 bit/码元
```

**步骤3：计算比特率（数据传输速率）**
```
比特率 = 波特率 × 每码元比特数
      = 400k × 2
      = 800 kbps
```

**完整公式应用**：
```
C = 2W × log₂V
  = 2 × 200kHz × log₂4
  = 400k × 2
  = 800 kbps
```

### 第四步：逐项验证

**A. 200kbps ❌**
```
如果C=200kbps，则：
2W × log₂V = 200k
400k × log₂V = 200k
log₂V = 0.5
V = 2^0.5 ≈ 1.4（不是整数）
结论：不符合题意
```

**B. 400kbps ❌**
```
如果C=400kbps，相当于：
仅计算了波特率2W=400k
忘记乘以log₂V=2
结论：忘记考虑多电平调制
```

**C. 800kbps ✅**
```
C = 2 × 200k × log₂4
  = 400k × 2
  = 800kbps ✓
结论：正确
```

**D. 1600kbps ❌**
```
如果C=1600kbps，则：
1600 = 2 × 200 × log₂V
4 = log₂V
V = 16（需要16个幅值）
但题目只有4个幅值
结论：计算错误
```

### 第五步：确认答案
C = 2W×log₂V = 800kbps → 答案：**C**

## 易错点分析

### 陷阱1：带宽与波特率混淆
❌ **错误理解**：带宽200kHz=波特率200k，直接用200×2=400kbps
✅ **正确理解**：
- **带宽W**：频率范围（Hz）
- **波特率**：码元速率 = 2W（奈奎斯特采样定理）
- **比特率**：波特率 × log₂V

**计算链条**：
```
带宽W → 波特率2W → 比特率2W×log₂V
200kHz → 400k Baud → 800kbps
```

### 陷阱2：忘记多电平带来的增益
❌ **错误答案**：选B（400kbps），只计算2W
✅ **正确理解**：
- **2电平（V=2）**：每码元1bit → C=2W×1=2W
- **4电平（V=4）**：每码元2bit → C=2W×2=4W
- **8电平（V=8）**：每码元3bit → C=2W×3=6W

**4个幅值的意义**：
```
幅值1：表示00
幅值2：表示01
幅值3：表示10
幅值4：表示11
→ 每个符号携带2比特信息
```

### 陷阱3：混淆奈奎斯特定理与香农公式
| 题目特征 | 应用公式 | 本题判断 |
|---------|---------|---------|
| 给出带宽+信噪比 | 香农公式 | ❌ 无信噪比 |
| 给出带宽+调制电平数 | 奈奎斯特定理 | ✅ 有4个幅值 |
| 明确"无噪声" | 奈奎斯特定理 | ✅ 题目说无噪声 |

**本题线索**：
- ✅ 无噪声信道
- ✅ 给出信号状态数V=4
- ❌ 未给信噪比
→ 使用奈奎斯特定理

### 陷阱4：ASK调制的实际限制
虽然理论上可以无限增加幅值数，但实际受限于：
- 噪声：幅值越多，抗噪能力越差
- 接收灵敏度：幅值间隔太小难以区分
- 本题："无噪声"假设，所以用理论公式

## 知识点对比

### 三种基本数字调制方式

| 调制方式 | 全称 | 改变的参数 | V个状态表示 |
|---------|------|----------|-----------|
| **ASK** | 幅移键控 | 载波幅度 | V个不同幅度 |
| FSK | 频移键控 | 载波频率 | V个不同频率 |
| PSK | 相移键控 | 载波相位 | V个不同相位 |

**共同点**：都遵循奈奎斯特定理 C=2W×log₂V

### 码元速率（波特率）vs 比特率

| 概念 | 单位 | 定义 | 关系 |
|------|------|------|------|
| **波特率** | Baud（码元/秒） | 单位时间传输的码元数 | 波特率 = 2W |
| **比特率** | bps（比特/秒） | 单位时间传输的比特数 | 比特率 = 波特率×log₂V |

**示例对比**：
```
场景1：2电平调制，带宽100kHz
波特率 = 2×100k = 200k Baud
比特率 = 200k×log₂2 = 200k×1 = 200kbps

场景2：4电平调制，带宽100kHz
波特率 = 2×100k = 200k Baud
比特率 = 200k×log₂4 = 200k×2 = 400kbps

→ 波特率相同，比特率翻倍
```

### 不同调制方式的速率

| 带宽 | 调制方式 | V | log₂V | 波特率 | 比特率 |
|------|---------|---|-------|--------|--------|
| 200kHz | 2-ASK | 2 | 1 | 400k | 400kbps |
| 200kHz | **4-ASK** | **4** | **2** | **400k** | **800kbps** |
| 200kHz | 8-ASK | 8 | 3 | 400k | 1200kbps |
| 200kHz | 16-QAM | 16 | 4 | 400k | 1600kbps |

**结论**：提高调制电平数可以在相同带宽下提高速率

### 奈奎斯特 vs 香农

| 公式 | 无噪声（本题） | 有噪声 |
|------|---------------|--------|
| 奈奎斯特 | C=2W×log₂V | 可能超过香农极限（不可达） |
| 香农 | C→∞ | C=W×log₂(1+S/N) |

**实际应用**：
```
取两者最小值
C_actual = min(C_奈奎斯特, C_香农)
```
