# 信道容量真题汇总

## 2023年第34题

**元数据**
- 年份：[2023年第34题](../../../01-真题原卷/2023年408真题.md#34)
- 科目：#计算机网络
- 知识点：#计算机网络/物理层/信道容量 #QAM调制
- 难度：⭐⭐
- 关联知识：[2023年第34题-信道容量](2023年第34题-信道容量.md)

**题干**

某无噪声理想信道带宽为 4MHz，采用QAM调制，若该信道的最大数据传输速率是48Mbps，则该信道采用的QAM调制方案是（）。

A. QAM-16  B. QAM-32  C. QAM-64  D. QAM-128

**答案**：C

**考点分析**
- 奈奎斯特定理：C = 2W × log₂V
- 48 = 2×4 × log₂V → log₂V = 6 → V = 64

---

## 解题思路（逐项分析法）

### 第一步：明确题目条件
- 信道类型：无噪声理想信道
- 带宽：W = 4MHz
- 调制方式：QAM（正交幅度调制）
- 最大数据传输速率：C = 48Mbps
- 求：QAM调制方案（信号状态数V）

### 第二步：回顾奈奎斯特定理

**奈奎斯特定理（无噪声信道）**：
```
C = 2W × log₂V
```
- C：信道容量（bps）
- W：带宽（Hz）
- V：信号状态数（调制电平数）

**QAM调制命名**：
- QAM-16：16个信号状态（V=16）
- QAM-64：64个信号状态（V=64）
- 数字表示信号星座图上的点数

### 第三步：计算信号状态数

**已知条件**：
```
C = 48Mbps = 48×10⁶ bps
W = 4MHz = 4×10⁶ Hz
```

**代入奈奎斯特公式**：
```
C = 2W × log₂V
48×10⁶ = 2 × 4×10⁶ × log₂V
48 = 8 × log₂V
log₂V = 48/8 = 6
V = 2⁶ = 64
```

**确定QAM方案**：
```
V = 64 → QAM-64
```

### 第四步：逐项验证

**A. QAM-16 ❌**
```
V = 16
log₂16 = 4
C = 2 × 4 × 4 = 32 Mbps ≠ 48Mbps
结论：速率不够
```

**B. QAM-32 ❌**
```
V = 32
log₂32 = 5
C = 2 × 4 × 5 = 40 Mbps ≠ 48Mbps
结论：速率不够
```

**C. QAM-64 ✅**
```
V = 64
log₂64 = 6
C = 2 × 4 × 6 = 48 Mbps ✓
结论：正确
```

**D. QAM-128 ❌**
```
V = 128
log₂128 = 7
C = 2 × 4 × 7 = 56 Mbps > 48Mbps
结论：速率过高（虽然可行但不是题目要求的）
```

### 第五步：确认答案
V = 64，对应QAM-64 → 答案：**C**

## 易错点分析

### 陷阱1：奈奎斯特公式系数2被遗漏
❌ **错误计算**：C = W × log₂V = 4 × log₂V
✅ **正确计算**：C = **2**W × log₂V = 8 × log₂V

**原因分析**：
```
奈奎斯特采样定理：
采样频率 ≥ 2 × 信号最高频率
→ 码元速率 = 2W

比特率 = 码元速率 × 每码元比特数
      = 2W × log₂V
```

### 陷阱2：混淆信号状态数V与比特数
❌ **错误理解**：QAM-64表示每个码元6比特
✅ **正确理解**：
- **QAM-64**：64个信号状态（V=64）
- **每码元比特数**：log₂64 = 6 bit/码元
- 命名规则：QAM-V（V是信号状态数）

**对应关系**：
| QAM方案 | V（状态数） | log₂V（比特/码元） |
|---------|-----------|------------------|
| QAM-4 | 4 | 2 |
| QAM-16 | 16 | 4 |
| QAM-64 | 64 | 6 |
| QAM-256 | 256 | 8 |

### 陷阱3：单位换算错误
注意单位统一：
- 带宽：4**M**Hz = 4×10⁶ Hz
- 速率：48**M**bps = 48×10⁶ bps
- 避免忘记"M"（百万）

**计算检查**：
```
48M = 2 × 4M × log₂V
简化：48 = 8 × log₂V
log₂V = 6
```

### 陷阱4：选择过高的调制方案
虽然QAM-128也能满足48Mbps（甚至更高），但：
- 题目问"最大速率**是**48Mbps"
- 应选择**恰好达到**48Mbps的方案
- QAM-64：48Mbps ✓
- QAM-128：56Mbps（超过要求，实际不会选择更复杂的调制）

## 知识点对比

### QAM调制原理

**QAM（正交幅度调制）**：
- 全称：Quadrature Amplitude Modulation
- 原理：**同时调制幅度和相位**
- 优势：在相同带宽下传输更多比特

**信号星座图**：
```
QAM-16 (4×4矩阵)：16个点
  • • • •
  • • • •
  • • • •
  • • • •

QAM-64 (8×8矩阵)：64个点
  • • • • • • • •
  • • • • • • • •
  ......(省略)
  • • • • • • • •
```

### 不同调制方案的速率对比

**假设带宽W=4MHz**：
| 调制方案 | V | log₂V | 速率C=2W×log₂V |
|---------|---|-------|---------------|
| QAM-4 | 4 | 2 | 16 Mbps |
| QAM-16 | 16 | 4 | 32 Mbps |
| QAM-32 | 32 | 5 | 40 Mbps |
| **QAM-64** | **64** | **6** | **48 Mbps** |
| QAM-128 | 128 | 7 | 56 Mbps |
| QAM-256 | 256 | 8 | 64 Mbps |

**规律**：
- V每翻倍，速率增加2W
- V从64→128，速率从48M→56M（+8M）

### 调制复杂度与抗噪性的权衡

| 对比项 | V小（如QAM-16） | V大（如QAM-256） |
|-------|----------------|-----------------|
| 传输速率 | 低 | 高 |
| 实现复杂度 | 简单 | 复杂 |
| 抗噪声能力 | 强（点距大） | 弱（点距小） |
| 功耗 | 低 | 高 |
| 应用场景 | 长距离、高噪声 | 短距离、低噪声 |

**实际选择原则**：
- 无噪声/低噪声环境：选择高阶QAM（如QAM-256）
- 高噪声环境：选择低阶QAM（如QAM-16）
- 本题"无噪声理想信道"，理论上可选任意方案，但题目指定48Mbps

### 常见调制方式速率对比

**假设带宽W=4MHz，二进制调制**：
| 调制方式 | V | log₂V | 速率 |
|---------|---|-------|------|
| 2-ASK | 2 | 1 | 8 Mbps |
| 2-FSK | 2 | 1 | 8 Mbps |
| 2-PSK (BPSK) | 2 | 1 | 8 Mbps |
| 4-PSK (QPSK) | 4 | 2 | 16 Mbps |
| 8-PSK | 8 | 3 | 24 Mbps |
| **QAM-64** | **64** | **6** | **48 Mbps** |

**结论**：QAM通过同时调制幅度和相位，实现更高的频谱效率
