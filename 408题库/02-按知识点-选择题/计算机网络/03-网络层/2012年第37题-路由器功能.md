# IP路由器功能真题汇总

## 2012年第37题

**元数据**
- 年份：[2012年第37题](../../../01-真题原卷/2012年408真题.md#37)
- 科目：#计算机网络
- 知识点：#计算机网络/网络层/路由器功能
- 难度：⭐⭐
- 关联知识：[路由器](../../../../AI-知识点原子库/路由器.md)

**题干**

下列关于IP路由器功能的描述中，正确的是（）。

I. 运行路由协议，设置路由表  
II. 监测到拥塞时，合理丢弃IP分组  
III. 对收到的IP分组头进行差错校验，确保传输的IP分组不丢失  
IV. 根据收到的IP分组的目的IP地址，将其转发到合适的输出线路上

A. 仅III、IV  
B. 仅I、II、III  
C. 仅I、II、IV  
D. I、II、III、IV

**答案**：C

**考点分析**
- I正确：路由器运行路由协议维护路由表
- II正确：拥塞时可丢弃分组
- III错误：IP层只检测差错，不保证不丢失
- IV正确：根据目的地址转发

---

## 解题思路

### 第一步：理解题目核心
路由器核心功能是转发，但IP层不保证可靠传输

### 第二步：逐项分析

**I. 运行路由协议，设置路由表** ✅
路由器运行RIP、OSPF、BGP等路由协议，动态维护路由表

**II. 监测到拥塞时，合理丢弃IP分组** ✅
拥塞时主动丢弃分组是正常的拥塞管理策略(如RED、尾部丢弃等)

**III. 对收到的IP分组头进行差错校验，确保传输的IP分组不丢失** ❌
路由器对IP首部做校验和检测，发现错误就丢弃，但不重传也不确认。IP是不可靠服务，不保证分组不丢失

**IV. 根据收到的IP分组的目的IP地址，将其转发到合适的输出线路上** ✅
这是路由器的核心功能，查路由表转发分组

**A. 仅III、IV** ❌  III错误
**B. 仅I、II、III** ❌  III错误
**C. 仅I、II、IV** ✅  I、II、IV都正确
**D. I、II、III、IV** ❌  III错误

### 第三步：确认答案
C。路由器不保证分组不丢失

## 易错点分析

### 易错点1：误认为路由器保证分组不丢失
❌ 错误理解：认为路由器做差错校验就能确保IP分组不丢失，选III正确
✅ 正确理解：路由器对IP首部做校验和检测，发现错误就丢弃分组，并不重传或确认。IP网络层提供的是不可靠服务，不保证分组不丢失。拥塞时主动丢弃分组是正常行为

### 易错点2：路由器核心功能
| 功能 | 是否支持 | 说明 |
|-------|-------|------|
| 运行路由协议维护路由表 | ✓ | RIP/OSPF/BGP等 |
| 拥塞时丢弃分组 | ✓ | 主动拥塞管理 |
| 根据目的IP转发 | ✓ | 查路由表转发 |
| 保证分组不丢失 | ✗ | IP不可靠,不确认重传 |
