# 2012年第43题

**真题出处**：[2012年第43题](../../../01-真题原卷/2012年408真题.md#43)

**考点**：Cache缺失 | 主存带宽 | DMA | 交叉存储

**题干**：
某计算机的CPU主频为80MHz，CPI为4，并且平均每条指令访存1.5次，主存与Cache之间交换的块大小为16B，Cache的命中率为 99%，存储器总线宽度为32位。请回答下列问题。

1）该计算机的 MIPS 数是多少？平均每秒 Cache 缺失的次数是多少？在不考虑 DMA 传送的情况下，主存带宽至少达到多少才能满足 CPU 的访存要求？

2) 假定在 Cache 缺失的情况下访问主存时，存在 0.0005% 的缺页率，则 CPU 平均每秒产生多少次缺页异常？若页面大小为 4KB，每次缺页都需要访问磁盘，访问磁盘时 DMA 传送采用周期挪用方式，磁盘 I/O 接口的数据缓冲寄存器为 32 位，则磁盘 I/O 接口平均每秒发出的 DMA 请求次数至少是多少?

3）CPU 和 DMA 控制器同时要求使用存储器总线时，哪个优先级更高？为什么？

4) 为了提高性能，主存采用 4 体交叉存储模式，工作时每 1/4 个存储周期启动一个体。若每个体的存储周期为 50ns，则该主存能提供的最大带宽是多少?

**答案**：
## 43.【王道书7.3节】

## 1)【标准答案】

平均每秒 CPU 执行的指令数为 80M/4 = 20M，故 MIPS 数为 20；（1 分）

平均每秒 Cache 缺失的次数为 $20M \times 1.5 \times (1 - 99\%) = 300000 = 300k$ ; (1 分)

当 Cache 缺失时，CPU 访问主存，主存与 Cache 之间以块为单位传送数据，此时，主存带宽为 $16 \mathrm{~B} \times 300 \mathrm{k} / \mathrm{s} = 4.8 \mathrm{~MB} / \mathrm{s}$ 。在不考虑 DMA 传输的情况下，主存带宽至少达到 $4.8 \mathrm{~MB} / \mathrm{s}$ 才能满足 CPU 的访存要求。（2 分）

## 2)【标准答案】

平均每秒钟“缺页”异常次数为 $300000 \times 0.0005\% = 1.5$ 次。

因为存储器总线宽度为32位，每传送32位数据，磁盘控制器发出一次DMA请求，故平均

每秒磁盘 DMA 请求的次数至少为 $1.5 \times 4KB \div 4B = 1.5K = 1536$ 。（2 分）

## 3)【标准答案】

CPU 和 DMA 控制器同时要求使用存储器总线时，DMA 请求优先级更高；（1 分）因为，若 DMA 请求得不到及时响应，I/O 传输数据可能会丢失。（1 分）

## 4)【标准答案】

4体交叉存储模式能提供的最大带宽为 $4\times 4\mathrm{B}\div 50\mathrm{ns} = 320\mathrm{MB / s}$ 。（2分）

【总结】本题涉及多个考点：计算机的性能指标、存储器的性能指标、DMA 的性能分析、DMA 方式的特点及多体交叉存储器的性能分析。

## 解题思路

**第1问**：
- MIPS=80M/4=20M
- Cache缺失次数=20M×1.5×(1-99%)=300k
- 主存带宽=16B×300k=4.8MB/s

**第2问**：缺页次数=300k×0.0005%=1.5次，DMA请求=1.5×4KB÷4B=1536

**第3问**：DMA优先级高，防止I/O数据丢失

**第4问**：4体交叉带宽=4×4B÷50ns=320MB/s

## 易错点分析

### 易错点1：主存带宽计算
❌ 错误理解：带宽=访存次数×4B
✅ 正确理解：带宽=缺失次数×块大小=300k×16B（块大小16B）

### 易错点2：DMA请求次数
| 错误计算 | 正确计算 | 说明 |
|---------|---------|------|
| 请求次数=缺页次数 | 请求次数=缺页次数×页面大小÷总线宽度 | 每次传输总线宽度数据发出一次DMA请求 |
