请回答下列问题。

1）该机的指令系统最多可定义多少条指令？

2) 若 inc、shl 和 sub 指令的操作码分别为 01H、02H 和 03H，则以下指令对应的机器代码各是什么？

```asm
inc R1 ; (R1) + 1→R1
shl R2, R1 ; (R1) << 1→R2
sub R3, (R1), R2 ; ((R1)) - (R2) → R3 
```

3) 假设寄存器 X 的输入和输出控制信号分别为 Xin 和 Xout，其值为 1 表示有效，为 0 表示无效（如 PCout = 1 表示 PC 内容送总线）；存储器控制信号为 MEMop，用于控制存储器的读（read）和写（write）操作。写出题图 a 中标号①～⑧处的控制信号或控制信号的取值。

4）指令"sub R1, R3, (R2)"和"inc R1"的执行阶段至少各需要多少个时钟周期？

**答案**：
44.【王道书4.2节】

1)【标准答案】

指令操作码有7位，因此最多可定义 $2^{7} = 128$ 条指令。（1分）

2)【标准答案】

各条指令的机器代码分别如下：（3分）

① “inc R1” 的机器码为 0000001 0 01 0 00 0 00，即 0240H。

② “shl R2, R1” 的机器码为 0000010 0 10 0 01 0 00，即 0488H。

③ “sub R3, (R1), R2” 的机器码为 0000011 0 11 1 01 0 10，即 06EAH。

3)【标准答案】

各标号处的控制信号或控制信号取值如下：（4分）

① 0; ② mov; ③ mova; ④ left; ⑤ read; ⑥ sub; ⑦ mov; ⑧ SRout 

【评分说明】答对两个给1分，共4分。

4)【标准答案】

指令 “sub R1, R3, (R2)” 的执行阶段至少包含 4 个时钟周期；（1 分）指令 “inc R1” 的执行阶段至少包含 2 个时钟周期。（1 分）
---

## 解题思路
操作码7位→2^7=128条指令；寄存器编号2位→2^2=4个寄存器(R0~R3)。指令机器码按格式拼接：操作码+目的寻址位+目的寄存器+源寻址位+源寄存器。控制信号从图中数据流推断：取指阶段PCout+MARin，执行阶段根据指令类型选择ALU操作和寄存器读写。执行周期数=访存次数+运算次数。

## 易错点分析

### 易错点1：指令机器码编码字段顺序错误
❌ 错误编码：sub R3,(R1),R2认为源操作数在前，编码为0000011 1 01 0 11 0 10，顺序混乱。
✅ 正确理解：指令格式固定为OP(7位)+Ma(1位)+Ra(2位)+Ms(1位)+Rs(2位)，目的操作数字段在前。sub R3,(R1),R2编码：OP=03H=0000011, Ma=0(R3直接), Ra=11(R3), Ms=1((R1)间接), Rs=01(R1)，拼接为06EAH。

### 易错点2：执行周期计算遗漏访存次数
| 指令类型 | 访存次数 | 执行周期 |
|-------|-------|------|
| inc R1(寄存器直接) | 0次 | 2周期(读R1+ALU+写R1) |
| sub R1,R3,(R2)(含间接寻址) | 1次(取(R2)内容) | 4周期(读R3+访存+ALU+写R1) |
