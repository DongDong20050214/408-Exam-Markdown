# 2017年第44题

**真题出处**：[2017年第44题](../../../01-真题原卷/2017年408真题.md#44)

**考点**：机器指令 | 指令编码 | 相对寻址 | 大小端

**题干**：
在按字节编址的计算机M上，题43中f1的部分源程序（阴影部分）与对应的机器级代码（包括指令的虚拟地址）如下：

```txt
int fl(unsigned n)
1 00401020 55 push ebp
... ... ...
for(unsigned i=0; i<=n -1; i++)
... ... ...
20 0040105E 39 4D F4 cmp dword ptr [ebp-0Ch], ecx
... ... ...
{ power * = 2;
... ... ...
23 00401066 D1 E2 sh1 edx, 1
... ... ...
return sum;
... ... ...
35 0040107F C3 ret 
```

其中，机器级代码行包括行号、虚拟地址、机器指令和汇编指令。请回答下列问题。

1）计算机 M 是 RISC 还是 CISC？为什么？

2）f1 的机器指令代码共占多少字节？要求给出计算过程。

3）第20条指令cmp通过i减n-1实现对i和n-1的比较。执行f1(0)的过程中，当 $i = 0$ 时，cmp指令执行后，进/借位标志CF的内容是什么？要求给出计算过程。

4）第23条指令shl通过左移操作实现了power*2运算，在f2中能否也用shl指令来实现power*2？为什么？

**答案**：
44.【王道书4.3节】

1)【标准答案】

M 为 CISC。(1 分)

M 的指令长短不一，不符合 RISC 指令系统特点。(1 分)

## 2)【标准答案】

f1 的机器代码占 96B。(1 分)

因为 f1 的第一条指令 “push ebp” 所在的虚拟地址为 0040 1020H，最后一条指令 “ret” 所在的虚拟地址为 0040 107FH，所以，f1 的机器指令代码长度为 0040 107FH - 0040 1020H + 1 = 60H = 96 字节。（1 分）

## 3)【标准答案】

CF = 1。(1 分)

cmp 指令实现 $i$ 与 $n - 1$ 的比较功能, 进行的是减法运算。在执行 f1(0)过程中, $n = 0$ , 当 $i = 0$ 时, $i = 000000000\mathrm{H}$ , 并且 $n - 1 = \mathrm{FFFF}\mathrm{FFFFH}$ 。因此, 当执行第 20 条指令时, 在补码加/减运算器中执行“0 减 FFFF FFFFH”的操作, 即 $00000000\mathrm{H} + 00000000\mathrm{H} + 1 = 00000001\mathrm{H}$ , 此时, 进位输出 $\mathbf{C} = \mathbf{0}$ , 减法运算时的借位标志 $\mathrm{CF} = \mathrm{C} \oplus 1 = 1$ 。（2 分）

## 4)【标准答案】

f2 中不能用 shl 指令实现 power*2。(1 分)

因为 shl 指令用来将一个整数的所有有效数位作为一个整体左移; 而 f2 中的变量 power 是 float 型, 其机器数中不包含最高有效数位, 但包含了阶码部分, 将其作为一个整体左移时并不能实现 “乘 2” 的功能, 因此 f2 中不能用 shl 指令实现 power*2。(2 分)

【注意】浮点数运算比整型运算要复杂，耗时也较长。

## 解题思路
指令长度不一(1~5字节)，CISC特征。机器码占用字节数=末条指令地址-首条指令地址+末条指令长度=0040107FH-00401020H+1=96字节。无符号数减法i-(n-1)，i=0,n=0时0-FFFFFFFFH，补码运算借位CF=1。shl左移整数所有位实现×2；float型包含阶码和尾数，左移破坏格式无法正确乘2。

## 易错点分析

### 易错点1：机器码长度计算遗漏最后一条指令本身长度
❌ 错误计算：0040107FH-00401020H=5FH=95字节，认为地址差即代码长度。
✅ 正确理解：地址差只计算到最后一条指令首地址，需加上该指令长度。ret指令长1字节，总长度=95+1=96字节。地址范围包含末尾指令的所有字节。

### 易错点2：浮点数左移等价于×2
| 整数shl | 浮点数左移 | 区别 |
|-------|-------|------|
| 所有位作为整体左移，低位补0 | 阶码和尾数独立存储，格式特殊 | 整数×2=逻辑左移1位 |
| 00000010→00000100(2→4) | 阶码部分被左移破坏表示范围 | 浮点数×2需阶码+1，不能用shl |
