**答案**：
## 44.【王道书5.6节】

## 1)【标准答案】

x 的机器码为 $[x]_{补} = 1111\ 1101\ 1111\ 1111B$ ，即指令执行前 (R1) = FDFFH，右移 1 位后为 1111 1110 1111 1111B，即指令执行后 (R1) = FEFFH。（2 分）

【评分说明】仅正确写出指令执行前的(R1)可给1分。

## 2)【标准答案】

至少需要 $4 + (5 - 1) = 8$ 个时钟周期。(2 分)

## 【分析】

每个时钟周期只能有一条指令进入流水线，从第5个时钟周期开始，每个时钟周期都会有一条指令执行完毕。

## 3)【标准答案】

$I_{3}$ 的 ID 段被阻塞的原因：因为 $I_{3}$ 与 $I_{1}$ 和 $I_{2}$ 都存在数据相关，需等到 $I_{1}$ 和 $I_{2}$ 将结果写回寄存器后， $I_{3}$ 才能读寄存器内容，所以 $I_{3}$ 的 ID 段被阻塞（1 分）。 $I_{4}$ 的 IF 段被阻塞的原因：因为 $I_{4}$ 的前一条指令 $I_{3}$ 在 ID 段被阻塞，所以 $I_{4}$ 的 IF 段被阻塞（1 分）。

【分析】若 $\mathrm{I}_4$ 的IF段不被阻塞，则会覆盖指令寄存器的内容，导致 $\mathrm{I}_3$ 段的译码结果出错。

【注意】要求“按序发射，按序完成”，故2）中下一条指令的IF可以和上一条指令的ID并行，以免因上一条指令发生冲突而导致下一条指令先执行完。

## 解题思路

### 第一步：理解算术右移的补码特性
-513补码=1111110111111111B(FDFFH)。算术右移保持符号位，右移1位：最高位保持1，其余位右移。结果1111111011111111B=FEFFH，对应-257。

### 第二步：计算流水线无阻塞执行时间
5段流水线，第1条指令需5周期完成全部阶段，后续每条指令每周期进入。4条指令总时间=5+(4-1)=8周期。首条铺满流水线+其余依次进入。

### 第三步：分析数据相关引发的阻塞
I3需I1和I2写回R1、R2后才能读取，I3的ID阻塞到I2的WB后(第6周期)。I4的IF在I3阻塞期间也阻塞(按序发射)，否则会覆盖IR内容导致I3丢失。数据相关向后传播阻塞。

### 第四步：绘制指令执行时序图
I1: LOAD R1,[x]需访存；I2: LOAD R2,[a]需访存；I3: SHL R1等I1写R1；I4: ADD R1,R2等I2写R2和I3写R1；I5: STORE R2,[x]等I4写R2。每条等待前驱写回，阻塞3周期。

## 易错点分析

### 易错点1：算术右移当作逻辑右移处理
❌ 错误计算：-513的FDFFH逻辑右移，高位补0，得01111110111111111B=7EFFH(正数)，符号错误。
✅ 正确理解：算术右移保持符号位不变，负数右移高位补1。FDFFH→FEFFH，最高位始终为1保持负数属性。逻辑右移高位补0用于无符号数，算术右移保持符号用于有符号数。

### 易错点2：流水线阻塞时IF段继续取指
❌ 错误理解：I3的ID阻塞时，I4的IF正常进行取下一条指令，I4完成IF后等待I3完成ID。
✅ 正确理解：按序发射要求前一条未完成时后一条不能进入流水线。I3的ID阻塞导致I4的IF也阻塞，否则I4的指令会覆盖IR中I3的译码信息。流水线阻塞是级联的，上游阻塞下游也阻塞。

### 易错点3：x=2*x+a指令序列时钟周期计算
| 指令序列 | 阻塞点 | 总周期 |
|-------|-------|------|
| I1: LOAD R1,[x] | 无 | 1~5 |
| I2: LOAD R2,[a] | 无 | 2~6 |
| I3: SHL R1 | 等I1写R1，阻塞3周期 | 3→6~9 |
| I4: ADD R1,R2 | 等I2写R2和I3写R1 | 6→10~13 |
| I5: STORE R2,[x] | 等I4写R2 | 10→14~17 |
