# 2017年第43题

**真题出处**：[2017年第43题](../../../01-真题原卷/2017年408真题.md#43)

**考点**：浮点数表示 | IEEE754 | 整数溢出 | 类型转换

**题干**：
已知 $f(n) = \sum_{i=0}^{n} 2^i = 2^{n+1} - 1 = \overbrace{11\cdots1}^{n+1 \text{位}} \mathrm{B}$，计算 $f(n)$ 的 C 语言函数 f1 如下：

```c
int f1(unsigned n) {
    int sum = 1, power = 1;
    for (unsigned i=0;i<=n-1;i++) {
    power *= 2; 
    sum += power;
}
return sum;
} 
```

将 f1 中的 int 都改为 float，可得到计算 $f(n)$ 的另一个函数 f2。假设 unsigned 和 int 型数据都占 32 位，float 采用 IEEE 754 单精度标准。请回答下列问题。

1）当 $\mathbf{n} = 0$ 时，f1会出现死循环，为什么？若将f1中的变量i和n都定义为int型，则f1是否还会出现死循环？为什么？

2）f1(23)和f2(23)的返回值是否相等？机器数各是什么（用十六进制数表示）？

3）f1(24)和f2(24)的返回值分别为33554431和33554432.0，为什么不相等？

4） $f(31)=2^{32}-1$，而 $f1(31)$ 的返回值却为 -1，为什么？若使 $f1(n)$ 的返回值与 $f(n)$ 相等，则最大的 n 是多少？

5) f2(127)的机器数为 7F80 0000H，对应的值是什么？若使 f2(n) 的结果不溢出，则最大的 n 是多少？若使 f2(n) 的结果精确（无舍入），则最大的 n 是多少？

**答案**：
43.【王道书2.3节】

1)【标准答案】

i 和 n 是 unsigned 型，故 “i <= n - 1” 是无符号数比较，n = 0 时，n - 1 的机器数为全 1，值是 $2^{32} - 1$ ，为 unsigned 型可表示的最大数，条件 “i <= n - 1” 永真，因此出现死循环。（2 分）若 i 和 n 改为 int 类型，则不会出现死循环。（1 分）

因为 “i <= n - 1” 是带符号整数比较，n = 0 时，n - 1 的值是 -1，当 i = 0 时条件 “i <= n - 1” 不成立，此时退出 for 循环。（1 分）

2)【标准答案】

f1(23)与 f2(23)的返回值相等。(1 分)

f1(23)的机器数是 00FF FFFFH。(1 分)

f2(23)的机器数是 4B7F FFFFH。(1 分)

【分析】

$f(23)=2^{23+1}-1=2^{24}-1$ ，它的二进制形式是24个1。int占32位，没有溢出。float有1个符号位，8个指数位，23个底数位，23个底数位可以表示24位的底数。所以两者返回值相等。显而易见前者是24个1，即0000 0000 1111 1111 1111 1111 1111 1111 $_{(2)}$ ，后者符号位是0，指数位为 $23+127_{(10)}=1001\ 0110_{(2)}$ ，底数位是111 1111 1111 1111 1111 1111 $_{(2)}$ 。

3)【标准答案】

当 n=24 时， $f(24)=1\ 1111\ 1111\ 1111\ 1111\ 1111\ 1111B$ ，而 float 型数只有 24 位有效位，舍入后数值增大，所以 f2(24) 比 f1(24) 大 1。（1 分）

【评分说明】只要说明 f2(24)需舍入处理即可给分。

4)【标准答案】

显然 f(31)已超出了 int 型数据的表示范围，用 f1(31)实现时得到的机器数为 32 个 1，作为 int 型数解释时其值为 -1，即 f1(31)的返回值为 -1。（1 分）

因为 int 型最大可表示数是 0 后面加 31 个 1, 故使 $f1(n)$ 的返回值与 $f(n)$ 相等的最大 n 值是 30。(1 分)

【评分说明】对于第二问，只要给出 $\mathbf{n} = 30$ 即可给分。

5)【标准答案】

IEEE 754 标准用 “阶码全 1、尾数全 0” 表示无穷大。f2 返回值为 float 型，机器数 7F80 0000H 对应的值是 +∞。（1 分）

当 n = 126 时， $f(126) = 2^{127} - 1 = 1.1 \cdots 1 \times 2^{126}$ ，对应阶码为 $127 + 126 = 253$ ，尾数部分舍入后阶码加 1，最终阶码为 254，是 IEEE 754 单精度格式表示的最大阶码。故使 f2 结果不溢出的最大 n 值为 126。（1 分）

当 n = 23 时，f(23)为 24 位 1，float 型数有 24 位有效位，所以不需舍入，结果精确。故使 f2 获得精确结果的最大 n 值为 23。（1 分）

【评分说明】对于第二问，只要给出 $n = 23$ ，即可给分。对于第三问，只要给出 $n = 126$ ，即可给分。

## 解题思路

**问题1：unsigned类型死循环分析**
n=0时unsigned类型n-1机器数全1（$2^{32}-1$），条件i<=n-1永真。改为int型n-1=-1，i=0时条件不成立退出循环。

**问题2：f1(23)和f2(23)机器数**
$f(23)=2^{24}-1$=24个1，int型00FF FFFFH无溢出。float有效位24位够表示，指数127+23=150=10010110B，机器数4B7F FFFFH，两者数值相等。

**问题3：f2(24)舍入误差**
$f(24)=2^{25}-1$需25位有效位，float只有24位，舍入后变为$2^{25}$=33554432.0，比f1(24)=33554431大1。

**问题4：int溢出和最大n**
$f(31)=2^{32}-1$=32个1，int最高位1表示负数，值为-1。int最大正数=0后跟31个1=$2^{31}-1$=$f(30)$，最大n=30。

**问题5：float溢出和精度上限**
7F80 0000H阶码全1尾数全0表示+∞。不溢出最大n=126（阶码254）。精确结果最大n=23（24位有效位无需舍入）。

## 易错点分析

### 易错点1：混淆float有效位数和尾数位数
❌ 错误理解：float尾数23位，f(22)需23位能精确表示。
✅ 正确理解：IEEE 754隐藏首位1，尾数23位实际表示24位有效位。$f(23)=2^{24}-1$有24个1，float刚好表示无需舍入，最大n=23。

### 易错点2：判断int溢出时忽略符号位
❌ 错误理解：int 32位能表示$2^{32}$个数，f(31)=$2^{32}-1$不溢出。
✅ 正确理解：int有符号，范围$[-2^{31}, 2^{31}-1]$，正数最高位必须为0。f(31)机器数32个1最高位1表示负数-1。f(30)=0+31个1=$2^{31}-1$是int最大正数。
