# 2009年第43题

**真题出处**：[2009年第43题](../../../01-真题原卷/2009年408真题.md#43)

**考点**：中断方式 | DMA方式 | CPU占用率计算 | IO性能

**题干**：
某计算机的 CPU 主频为 500 MHz，CPI 为 5 (即执行每条指令平均需 5 个时钟周期)。假定某外设的数据传输率为 0.5 MB/s，采用中断方式与主机进行数据传送，以 32 位为传输单位，对应的中断服务程序包含 18 条指令，中断服务的其他开销相当于 2 条指令的执行时间。请回答下列问题，要求给出计算过程。

1）在中断方式下，CPU 用于该外设 I/O 的时间占整个 CPU 时间的百分比是多少？

2) 当该外设的数据传输率达到 5 MB/s 时，改用 DMA 方式传送数据。假定每次 DMA 传送块大小为 5000 B，且 DMA 预处理和后处理的总开销为 500 个时钟周期，则 CPU 用于该外设 I/O 的时间占整个 CPU 时间的百分比是多少 (假设 DMA 与 CPU 之间没有访存冲突)?

**答案**：

## 1）【标准答案】

外设每秒传送0.5MB，中断时每次传送4B。中断方式下，CPU每次用于数据传送的时钟周期为 $5 \times 18 + 5 \times 2 = 100$ 。

(2分)

为达到外设 0.5MB/s 的数据传输率，外设每秒申请的中断次数为 $0.5MB \div 4B = 125000$ 。（1 分）

1s 内用于中断的开销为 $100 \times 125000 = 12500000 = 12.5M$ 个时钟周期。（1 分）

CPU 用于外设 I/O 的时间占整个 CPU 时间的百分比为 12.5M/500M = 2.5%。（1 分）

## 2）【标准答案】

当外设数据传输率提高到 5MB/s 时，改用 DMA 方式传送，每次 DMA 传送 5000B，1s 内需产生的 DMA 次数为 $5MB \div 5000B = 1000$ 。（1 分）

CPU 用于 DMA 处理的总开销为 $1000 \times 500 = 500000 = 0.5\mathrm{M}$ 个时钟周期。（1 分）

CPU 用于外设 I/O 的时间占整个 CPU 时间的百分比为 0.5M/500M = 0.1%。（1 分）

【评分说明】如果考生只给出正确的计算结果，未给出计算过程，每个给2分。

## 解题思路

### 第一步：计算中断频率和单次开销
传输率0.5MB/s，每次传输4B(32位)，每秒中断次数=0.5MB/4B=125000次。单次中断开销=指令数×CPI=(18+2)×5=100时钟周期。总开销=125000×100=12.5M周期。

### 第二步：计算中断方式CPU占用百分比
CPU主频500MHz，1秒有500M个时钟周期。中断占用=12.5M/500M=2.5%。中断方式下CPU需频繁响应每次数据传输，开销与传输次数成正比。

### 第三步：计算DMA方式下的块传输次数
传输率提升到5MB/s，每次DMA传输5000B，每秒DMA次数=5MB/5000B=1000次。单次DMA开销=500周期(预处理+后处理)。总开销=1000×500=0.5M周期，占用0.5M/500M=0.1%。

## 易错点分析

### 易错点1：中断次数按字节计算而非按传输单位
❌ 错误计算：0.5MB/s数据传输，每秒中断0.5M次，开销=0.5M×100=50M周期，占用10%。
✅ 正确理解：中断以传输单位触发，每次传输32位(4字节)产生1次中断。中断次数=0.5MB/4B=125000次/秒，不是按每字节中断。传输单位决定中断粒度，是IO效率的关键参数。

### 易错点2：DMA预处理开销按每字节计算
❌ 错误计算：DMA传输5MB数据，每字节500周期开销，总开销=5M×500，远超CPU总周期数，计算错误。
✅ 正确理解：DMA的500周期开销是每次块传输的预处理+后处理总和，与块大小无关。5000B作为一个块一次性传输，仅产生1次500周期开销。DMA优势在于按块传输减少CPU介入次数。

### 易错点3：中断方式与DMA方式对比
| 中断方式 | DMA方式 | 性能差异 |
|-------|-------|------|
| 每次传输4B触发1次中断 | 每次传输5000B触发1次DMA | DMA传输单位更大 |
| 单次开销100周期，频率高 | 单次开销500周期，频率低 | DMA单次开销高但总次数少 |
| CPU占用2.5%(0.5MB/s) | CPU占用0.1%(5MB/s) | DMA适合高速外设 |
