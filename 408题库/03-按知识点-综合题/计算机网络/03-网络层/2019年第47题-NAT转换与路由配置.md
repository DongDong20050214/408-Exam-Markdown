# 2019年第47题

**真题出处**：[2019年第47题](../../../01-真题原卷/2019年408真题.md#47)

**考点**：NAT转换 | 路由器配置 | 网络互联 | HTTP协议

**题干**：
某校园网有两个局域网，通过路由器R1、R2和R3互联后接入Internet，S1和S2为以太网交换机。局域网采用静态IP地址配置，路由器部分接口以及各主机的IP地址如下图所示。

![image](https://cdn-mineru.openxlab.org.cn/result/2026-06-09/61a7441a-e21c-435b-9476-47f9ff01a858/aa183189b890fda563c84a81ea49d1729df36e6da34649248df01079c66caba1.jpg)

假设NAT转换表结构为

|外网||内网||
|IP 地址|端口号|IP 地址|端口号|
||||

请回答下列问题:

1）为使H2和H3能够访问Web服务器（使用默认端口号），需要进行什么配置？

2) 若 H2 主动访问 Web 服务器时，将 HTTP 请求报文封装到 IP 数据报 P 中发送，则 H2 发送 P 的源 IP 地址和目的 IP 地址分别是什么？经过 R3 转发后，P 的源 IP 地址和目的 IP 地址分别是什么？经过 R2 转发后，P 的源 IP 地址和目的 IP 地址分别是什么?

**答案**：
47.【王道书4.7节】

## 1)【标准答案】

设备 1: 路由器, 设备 2: 以太网交换机, 设备 3: 以太网交换机。

## 【分析】

以太网交换机（无 VLAN 功能）连接的若干 LAN 仍然是一个网络（同一个广播域），路由器可以连接不同的 LAN、不同的 WAN 或把 WAN 和 LAN 互联起来，隔离了广播域。IP 地址 192.168.1.2/26 与 192.168.1.3/26 的网络前缀均为 192.168.1.0，视为 LAN1。IP 地址 192.168.1.66/26 与 192.168.1.67/26 的网络前缀均为 192.168.1.64，视为 LAN2。

## 2)【标准答案】

设备 1 的接口需要配置 IP 地址; 设备 1 的 IF1、IF2 和 IF3 接口的 IP 地址分别是: 192.168.1.254、192.168.1.1 和 192.168.1.65。

## 【分析】

设备 1（路由器）的 IF1 接口与路由器 R 相连，R 端接口的 IP 地址为 192.168.1.253/30，253 的二进制表示形式为 11111101，故 IF1 接口的网络前缀也应为 192.168.1.111111，已分配 192.168.1.253，去除全 0 全 1，IF1 接口的 IP 地址应为 192.168.1.254。LAN1 的默认网关为 192.168.1.1，LAN2 的默认网关为 192.168.1.65，默认网关就是连接本机或子网的路由器接口的 IP 地址，即路由器的 LAN 端的 IP 地址。

## 3)【标准答案】

R 需要提供 NAT 服务。

## 【分析】

H1～H4 均为 C 类私有地址（192.168.0.0～192.168.255.255），R 连接设备 1 的点对点链路也是 C 类私有地址，若要确保 H1～H4 能够访问 Internet，R 需要提供 NAT 服务。

## 4)【标准答案】

主机 H4 会接收该数据报。

## 【分析】

IP 数据报的目的地址为 192.168.1.127，主机号为全 1，为本网络的广播地址，由于路由器可以隔离广播域，因此只有主机 H4 会接收数据报。
