# 2014年第43题

**真题出处**：[2014年第43题](../../../01-真题原卷/2014年408真题.md#43)

**考点**：路由表 | IP转发 | TTL | 路由聚合

**题干**：
请根据题42描述的网络，继续回答下列问题。

1）假设路由表结构如下表所示，请给出题42图中R1的路由表，要求包括到达题42图中子网192.1.x.x的路由，且路由表中的路由项尽可能少。

| 目的网络 | 下一跳 | 接口 |

2）当主机192.1.1.130向主机192.1.7.211发送一个 TTL = 64 的IP分组时，R1通过哪个接口转发该IP分组？主机192.1.7.211收到的IP分组TTL是多少？

3）若 R1 增加一条 Metric 为 10 的链路连接 Internet，则题 42 表中 R1 的 LSI 需要增加哪些信息？

**答案**：
## 43.【王道书4.4节】

## 1)【标准答案】

子网 192.1.6.0/24 和 192.1.7.0/24 在 R1 的路由表中可聚合为一个子网 192.1.6.0/23。于是得到 R1 的路由表如下：（6 分）

<table><tr><td>目的网络</td><td>下一条</td><td>接口</td></tr><tr><td>192.1.1.0/24</td><td>-</td><td>E0</td></tr><tr><td>192.1.6.0/23</td><td>10.1.1.2</td><td>L0</td></tr><tr><td>192.1.5.0/24</td><td>10.1.1.10</td><td>L1</td></tr></table>

## 【分析】

题目要求路由表中的路由项尽可能少，因此可以把从同一接口转发的目的网络进行路由聚合。

## 【评分说明】

① 每正确解答一个路由项，给2分，共6分。

② 路由项解答不完全正确，或路由项多于3条，可酌情给分。

## 2)【标准答案】

R1 通过 L0 接口转发该 IP 分组。(1 分) 主机 192.1.7.211 收到的 IP 分组的 TTL 是 61。(1 分)

## 【分析】

每经过一个路由器 TTL 减 1，该分组要经过 3 个路由器（R1, R2, R4）。

## 3)【标准答案】

R1 的 LSI 需要增加一条特殊的直连网络，网络前缀 Prefix 为 “0.0.0.0/0”，Metric 为 10。（1 分）此外，路由器连接到互联网的路由

## 【分析】

互联网（Internet）包括了无数的网络集合，不可能在路由表项中一一列出，因此 R1 到互联网的路由只能采用默认路由的方式，默认路由的网络前缀为 0.0.0.0/0。

【评分说明】考生只要回答：增加前缀 Prefix 为 “0.0.0.0/0”，Metric 为 10，同样给分。
