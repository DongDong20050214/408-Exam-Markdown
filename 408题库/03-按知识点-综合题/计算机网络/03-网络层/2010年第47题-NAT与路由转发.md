# 2010年第47题

**真题出处**：[2010年第47题](../../../01-真题原卷/2010年408真题.md#47)

**考点**：路由表 | NAT | Web访问 | IP地址转换

**题干**：
某校园网有两个局域网，通过路由器R1、R2和R3互联后接入Internet，S1和S2为以太网交换机。局域网采用静态IP地址配置，路由器部分接口以及各主机的IP地址如下图所示。

![image](https://cdn-mineru.openxlab.org.cn/result/2026-06-09/61a7441a-e21c-435b-9476-47f9ff01a858/aa183189b890fda563c84a81ea49d1729df36e6da34649248df01079c66caba1.jpg)

假设NAT转换表结构为

<table><tr><td colspan="2">外网</td><td colspan="2">内网</td></tr><tr><td>IP 地址</td><td>端口号</td><td>IP 地址</td><td>端口号</td></tr><tr><td></td><td></td><td></td><td></td></tr></table>

请回答下列问题:

1）为使H2和H3能够访问Web服务器（使用默认端口号），需要进行什么配置？

2) 若 H2 主动访问 Web 服务器时, 将 HTTP 请求报文封装到 IP 数据报 P 中发送, 则 H2 发送 P 的源 IP 地址和目的 IP 地址分别是什么? 经过 R3 转发后, P 的源 IP 地址和目的 IP 地址分别是什么? 经过 R2 转发后, P 的源 IP 地址和目的 IP 地址分别是什么?

**答案**：

## 1)【标准答案】

需要配置H2和H3的默认网关为R3的接口地址。

## 2)【标准答案】

H2发送P时：源IP=10.2.128.100，目的IP=Web服务器IP地址。

经R3转发后：源IP经NAT转换，目的IP不变。

经R2转发后：根据路由表转发，IP地址按NAT规则处理。

---

## 解题思路

本题考查NAT地址转换和路由转发过程。关键点：
1. 私有地址需通过NAT转换访问Internet
2. 路由器转发时修改MAC地址，NAT转换修改IP地址
3. Web服务器响应报文需原路返回

## 易错点分析

### 易错点1：NAT转换时机
❌ 错误理解：每个路由器都进行NAT转换
✅ 正确理解：只有配置了NAT的路由器才进行地址转换

### 易错点2：源目IP变化
| 位置 | 源IP | 目的IP | 说明 |
|------|------|--------|------|
| H2发出 | 10.2.128.100 | Web服务器 | 私有地址 |
| 经NAT | 公网地址 | Web服务器 | 地址转换 |
