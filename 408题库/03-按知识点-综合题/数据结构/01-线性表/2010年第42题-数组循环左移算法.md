# 2010年第42题

**考点**：数组 | 循环移位 | 算法设计

**题干**：
设将 $n (n > 1)$ 个整数存放到一维数组 $\mathbb{R}$ 中。试设计一个在时间和空间两方面都尽可能高效的算法, 将 $\mathbb{R}$ 中保存的序列循环左移 $p (0 < p < n)$ 个位置, 即将 $\mathbb{R}$ 中的数据由 $(x_0, x_1, \cdots, x_{n-1})$ 变换为 $(x_p, x_{p+1}, \cdots, x_{n-1}, x_0, x_1, \cdots, x_{p-1})$ 。要求:

1）给出算法的基本设计思想。

2) 根据设计思想，采用 C、C++ 或 Java 语言描述算法，关键之处给出注释。

3）说明你所设计算法的时间复杂度和空间复杂度。

**答案**：
## 42.【王道书2.2节】

## 1)【标准答案】

算法的基本设计思想：

先将 n 个数据 $X_{0}, X_{1}, \cdots, X_{p}, \cdots, X_{n-1}$ 原地逆置，得到 $X_{n-1}, \cdots, X_{p}, X_{p-1}, \cdots, X_{0}$ ，然后将前 n-p 个和后 p 个元素分别原地逆置，得到最终结果： $X_{p}, X_{p+1}, \cdots, X_{n-1}, X_{0}, X_{1}, \cdots, X_{p-1}$ 。

## 【分析】

将问题视为把数组 $ab$ 转换成数组 $ba$ （ $a$ 代表数组前 $p$ 个元素， $b$ 代表数组余下的 $n-p$ 个元素），先将整个 $ab$ 逆置得到 $(ab)^{-1} = b^{-1}a^{-1}$ ，再将 $b^{-1}$ 逆置得到 $ba^{-1}$ ，最后将 $a^{-1}$ 逆置得到 $ba$ 。设 Reverse() 执行数组逆置的操作，对 abcdefgh 循环左移 3 位（ $p=3$ ）的过程如下：

Reverse(0,n-1)得到 hgfedcba;

Reverse(0,n-p-1)得到 defghcba;

Reverse(n-p,n-1)得到 defghabc。

注：在 Reverse() 中，两个参数分别表示数组中待转换元素的始末位置。

## 2)【标准答案】

算法实现：

```txt
void Reverse(int R[], int left, int right) {
    int k = left, j = right, temp; // k 等于左边界 left, j 等于右边界 right
    while (k < j) { // 交换 r[k] 与 r[j]
    temp = r[k];
    r[k] = r[j];
    r[j] = temp;
    k++; // k 右移一个位置
    j--; // j 左移一个位置
    }
}
void leftShift(int R[], int n, int p) {
    if (p > 0 && p < n) {
    Reverse(r, 0, n - 1); // 将全部数据逆置
    Reverse(r, 0, n - p - 1); // 将前 n - p 个元素逆置
```

Reverse(r, n-p, n-1); // 将后 p 个元素逆置
}

## 3)【标准答案】

上述算法的时间复杂度为 $O(n)$ ，空间复杂度为 $O(1)$ 。

【另解】借助辅助数组来实现。

算法思想：创建大小为 $p$ 的辅助数组 $S$ ，将 $R$ 中前 $p$ 个整数依次暂存在 $S$ 中，同时将 $R$ 中后 $n - p$ 个整数左移，然后将 $S$ 中暂存的 $p$ 个数依次放回到 $R$ 中的后续单元。

该算法的时间复杂度为 $O(n)$ ，空间复杂度为 $O(p)$ 。
---

## 解题思路

**核心思想**：三次逆置法，(ab)^-1 = b^-1 a^-1 → ba

**解题步骤**：
1. 全部逆置Reverse(0,n-1)：abcdefgh → hgfedcba
2. 前n-p个逆置Reverse(0,n-p-1)：hgfedcba → defghcba
3. 后p个逆置Reverse(n-p,n-1)：defghcba → defghabc

**关键操作**：Reverse用双指针交换，左右逼近

## 易错点分析

### 易错点1：空间复杂度不达标
❌ 错误理解：创建辅助数组S[p]暂存前p个元素，空间O(p)
✅ 正确理解：三次逆置原地操作，空间O(1)更优

### 易错点2：逆置边界混淆
| 错误边界 | 正确边界 | 说明 |
|---------|---------|------|
| Reverse(0,p-1)和Reverse(p,n-1) | Reverse(0,n-1)→Reverse(0,n-p-1)→Reverse(n-p,n-1) | 先整体逆置是关键 |
