# 电子邮件

## 定义
电子邮件（Email）是互联网应用中重要的服务，用于异步的文字、图片、文件传输。

## 核心概念
- SMTP（Simple Mail Transfer Protocol）：邮件发送协议
- POP3（Post Office Protocol v3）：邮件接收协议
- IMAP（Internet Message Access Protocol）：邮件访问协议
- 邮件服务器：存储和转发邮件的服务器
- 邮件用户代理（MUA）：邮件客户端软件
- 邮件传输代理（MTA）：邮件服务器软件

## 算法流程/关键公式
1. 发送邮件：用户 → SMTP → 服务器
2. 接收邮件：服务器 → POP3/IMAP → 用户
3. SMTP 端口：25，用于邮件发送
4. POP3 端口：110，用于邮件接收（删除）
5. IMAP 端口：143，用于邮件同步

## 易混淆点
| SMTP | POP3 | IMAP |
|-----|------|------|
| 发送 | 接收并删除 | 接收并同步 |
| push | pull | pull |
