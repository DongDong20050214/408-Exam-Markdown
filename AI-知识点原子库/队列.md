# 队列

## 定义
队列是只允许在一端进行插入、另一端进行删除的线性表，遵循先进先出（FIFO）原则。允许插入的一端称为队尾，允许删除的一端称为队头。

## 核心概念
- 队头指针front：指向队头元素
- 队尾指针rear：指向队尾元素的下一个位置
- 入队（enqueue）：在队尾插入元素
- 出队（dequeue）：删除队头元素

## 算法流程/关键公式
1. 循环队列判空：front == rear
2. 循环队列判满：(rear + 1) % MaxSize == front
3. 队列长度：(rear - front + MaxSize) % MaxSize
4. 入队：Q[rear] = x; rear = (rear + 1) % MaxSize;

## 易混淆点
| 顺序队列 | 循环队列 | 区别 |
|-------|-------|------|
| 存在假溢出 | 解决假溢出 | 循环利用空间 |
| rear达到MaxSize即满 | 通过取模运算循环 | 空间利用率不同 |
