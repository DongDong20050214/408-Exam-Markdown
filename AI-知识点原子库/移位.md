# 移位

## 定义
移位是把数据的各位向左或向右移动的操作。

## 核心概念
- 左移：向左移动，右端补 0
- 右移：向右移动，左端补符号或 0
- 逻辑移位：补 0
- 算术移位：补符号位
- 循环移位：移出的位从另一端进入

## 算法流程/关键公式
1. 左移 n 位等于乘以 2^n
2. 右移 n 位等于除以 2^n（向下取整）
3. 循环移位不丢失数据

## 易混淆点
| 逻辑 | 算术 | 方式 |
|-----|------|------|
| 补 0 | 补符号 | 填充方式 |
| 无符号 | 有符号 | 数据类型 |
