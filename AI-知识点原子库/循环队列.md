# 循环队列

## 定义
循环队列是队列的一种变体，将队列用数组实现时，通过将数组尾部和头部连接成循环，解决普通队列的"假溢出"问题。

## 核心概念
- 头指针（front）：指向队首元素
- 尾指针（rear）：指向队尾元素的下一位置
- 循环：数组下标到末尾后回绕到开始
- 假溢出：普通队列由于不回绕导致的溢出
- 空/满判断：需要特殊处理，通常空时 front==rear，满时 (rear+1)%size==front

## 算法流程/关键公式
1. 初始化：front = rear = 0
2. 入队：rear = (rear + 1) % size
3. 出队：front = (front + 1) % size
4. 长度：(rear - front + size) % size
5. 队列满：(rear + 1) % size == front
6. 队列空：front == rear

## 易混淆点
| 普通队列 | 循环队列 | 优点 |
|----------|---------|------|
| 产生假溢出 | 充分利用空间 | 存储高效 |
| 需要搬移元素 | 无需搬移 | 时间高效 |
