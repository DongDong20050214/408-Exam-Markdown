# 设备管理

## 定义
设备管理是操作系统对 IO 设备的管理，包括设备驱动、中断处理、缓冲、调度等，向应用程序提供统一的设备访问接口。

## 核心概念
- IO 设备：输入/输出设备（硬盘、键盘、显示器等）
- 设备驱动：控制设备的程序代码
- 中断：设备完成 IO 后通知 CPU
- DMA：直接内存访问，设备不经 CPU 直接访问内存
- 设备控制器：管理设备的硬件部件
- 设备队列：等待 IO 的请求队列

## 算法流程/关键公式
1. IO 操作：应用程序 → 系统调用 → 驱动程序 → 设备控制器
2. 中断处理：CPU 收到中断信号 → 保存现场 → 执行中断程序 → 恢复现场
3. DMA 传输：设备与内存直接通信，仅在开始和结束时中断 CPU
4. 设备队列排序：改善 IO 响应时间

## 易混淆点
| 中断 | DMA | 区别 |
|-----|------|------|
| 设备操作完通知 | 设备与内存直接传输 | CPU 介入程度 |
| 需要 CPU 参与数据传输 | CPU 不参与传输 | 效率不同 |
