# 数组

## 定义
数组是由相同数据类型的元素组成、采用连续存储方式的线性数据结构。通过下标快速访问任意元素。

## 核心概念
- 下标：元素的位置标识，从 0 或 1 开始
- 连续存储：内存地址连续
- 随机访问：O(1) 时间访问任意元素
- 内存地址计算：Addr(A[i]) = BaseAddr + i × ElemSize
- 多维数组：行主序、列主序存储

## 算法流程/关键公式
1. 一维数组元素访问：O(1)
2. 插入/删除：需要移动元素，O(n)
3. 二维数组 m×n 存储：行主序 Addr = Base + (i×n + j)×ElemSize
4. 可动态扩展数组：初始容量，超出时扩容（通常翻倍）

## 易混淆点
| 数组 | 链表 | 区别 |
|-----|------|------|
| 连续存储 | 离散存储 | 访问时间不同 |
| 访问 O(1) | 访问 O(n) | 查询快 |
| 插入 O(n) | 插入 O(1) | 修改慢 |
