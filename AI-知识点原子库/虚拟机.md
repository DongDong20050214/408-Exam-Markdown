# 虚拟机

## 定义
虚拟机是通过软件模拟硬件环境，在一台物理机上运行多个独立操作系统的技术。分为系统虚拟机和进程虚拟机。

## 核心概念
- 宿主机：运行虚拟机的物理机器
- 客户机：虚拟机中运行的系统
- Hypervisor：虚拟机监控器
- Type 1：直接运行在硬件上
- Type 2：运行在宿主操作系统上
- 全虚拟化：客户机无需修改
- 半虚拟化：客户机需要修改

## 算法流程/关键公式
1. Type 1：Hypervisor→硬件，性能好
2. Type 2：宿主OS→Hypervisor→硬件
3. 特权指令：陷入Hypervisor处理
4. I/O虚拟化：虚拟设备映射到物理设备
5. 内存虚拟化：影子页表

## 易混淆点
| Type 1 | Type 2 | 区别 |
|-------|-------|------|
| 直接在硬件上 | 在OS上 | Type 1性能更好 |
| 如VMware ESXi | 如VirtualBox | 应用场景不同 |
