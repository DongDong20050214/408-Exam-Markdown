# 二分查找

## 定义
二分查找是一种在有序顺序表中查找特定元素的搜索算法。每次将查找区间缩小一半，直到找到目标或区间为空。

## 核心概念
- 前提条件：顺序存储且有序
- 时间复杂度：O(log n)
- 空间复杂度：O(1)
- 判定树高度：⌈log₂(n+1)⌉

## 算法流程/关键公式
1. 取中间位置：mid = (low + high) / 2
2. 比较key与mid位置元素
3. key < a[mid]：在左半区查找，high = mid - 1
4. key > a[mid]：在右半区查找，low = mid + 1
5. key == a[mid]：查找成功

## 易混淆点
| 二分查找 | 顺序查找 | 区别 |
|-------|-------|------|
| 要求有序 | 无要求 | 适用条件不同 |
| O(log n) | O(n) | 时间复杂度不同 |
