# 进程状态

## 定义
进程状态是进程在其生命周期内所处的各种状态，反映进程的执行情况和资源占用情况。

## 核心概念
- 创建态：进程刚被创建，等待分配资源
- 就绪态：已获得除 CPU 外的所有资源，等待 CPU
- 运行态：占用 CPU，正在执行
- 阻塞态（等待态）：因 IO 或其他原因释放 CPU，等待事件
- 终止态：进程执行完毕或被强制终止，等待回收资源

## 算法流程/关键公式
1. 创建 → 就绪：资源分配完毕
2. 就绪 → 运行：获得 CPU
3. 运行 → 就绪：时间片用完或被高优先级进程抢占
4. 运行 → 阻塞：请求 IO 或等待资源
5. 阻塞 → 就绪：事件发生，资源可用
6. 运行/就绪 → 终止：进程结束

## 易混淆点
| 就绪 | 阻塞 | 区别 |
|-----|------|------|
| 等待 CPU | 等待事件 | 等待对象不同 |
| 随时可运行 | 不能立即运行 | 可运行性不同 |
