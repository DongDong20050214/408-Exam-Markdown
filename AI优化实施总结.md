## 📋 优化标准确定

### Metadata标准（题目文件）

新增8个字段：
```yaml
year: 2024                    # 年份
question_id: "2024-DS-01"     # 唯一ID
difficulty: 2                 # 难度（1-5星）
exam_frequency: "高频"         # 考频
exam_count: 8                 # 历年考察次数
has_image: false              # 是否含配图
related_questions: []         # 相关题目
related_knowledge: []         # 关联知识点
```

### Metadata标准（知识点文件）

新增4个字段：
```yaml
ai_summary: "一句话总结"
difficulty_level: 2           # 知识点难度
exam_frequency: "高频"
typical_questions: []         # 代表题目
```

### 内容增强标准

**题目文件追加**：
- 解题思路（分步骤详解）
- 易错点分析（至少2个）
- 知识点对比/总结
- 记忆口诀/技巧

**知识点文件追加**：
- 易混淆点（至少2个）
- 常见提问QA（至少5个）
- 表格对比
- 实战应用场景
